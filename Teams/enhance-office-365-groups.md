<a name="enhance-existing-microsoft-365-groups-with-microsoft-teams"></a>Microsoft Teams를 통해 기존 Microsoft 365 그룹 개선
=======================================================

Microsoft Teams 사용자는 Teams 기능을 사용하여 기존 Microsoft 365 그룹을 향상시킬 수 있습니다. 공용 Microsoft 365 그룹 개선을 볼 때 사용자는 구성원 수가 5,000명 미만인 경우 이를 할 수 있습니다.

이를 위해 사용자는 Microsoft Teams 클라이언트에서 새 팀을 만드는 흐름을 진행해야 합니다.   >  **Microsoft 365** 그룹에서 만들기를 선택한 다음 Teams로 강화하려는 기존 그룹을 선택합니다. 기존 그룹 구성원은 자동으로 팀에 구성원으로 추가됩니다.

> [!IMPORTANT]
> Microsoft 365 그룹 소유자만 Teams를 통해 기존 그룹을 향상시킬 수 있는 권한이 있으며 해당 그룹에는 5,000명 미만의 사용자가 포함되어야 합니다. 이미 개선된 팀은 목록에 표시되지 않습니다.
>
>이러한 조건이 충족되지 않으면 Microsoft 365 그룹에서 팀을 만드는 옵션을 사용할 수 없습니다.

사용자는 메일 목록을 팀에 초대할 수도 있으며 해당 메일 목록의 구성원이 팀에 추가됩니다. 일회성 추가 기능으로, 나중에 메일 그룹의 그룹 멤버 자격 변경 내용이 Teams에 복제되거나 동기화되지 않습니다.

![Microsoft 365 그룹에서 팀을 만드는 옵션 스크린샷](media/Enhance_Existing_Office_365_groups_with_Microsoft_Teams_image2.png)

메일 사용이 가능한 보안 그룹을 팀의 구성원으로 추가할 수도 있습니다. 하지만 나중에 보안 그룹에 구성원을 더 추가하면 해당 구성원이 팀에 자동으로 추가되지 않습니다. 새 구성원을 별도로 추가하거나 보안 그룹을 팀에 다시 추가해야 합니다. (보안 그룹을 다시 추가하는 경우 중복 제거를 통해 구성원이 한 번만 추가됩니다.)

Microsoft 365 그룹의 개인 정보 설정에는 공개 및 비공개의 두 **가지 유형이 있습니다.** Teams에 대해 두 그룹 유형을 모두 사용할 수 있는 반면 셀프 서비스에서는 약간의 차이가 있습니다.

-   사용자는 공개 그룹을 검색할 수 있으며 팀 소유자의 승인 없이도 직접 참가할 수 있습니다.

-   비공개 그룹은 검색할 수 없습니다. 팀 소유자가 구성원으로 추가하지 않으면 사용자는 참가할 수 없습니다.

Teams에서 새 팀을 만들 때 기존 비공개 Microsoft 365 그룹의 소유자는 Microsoft 365 그룹의 멤버 자격을 사용하여 팀을 만들 수 있는 옵션이 있습니다. 사용자는 SharePoint용 탭을 추가하고 OneNote 파일을 통합하여 기존 SharePoint 및 OneNote 파일을 추가할 수 있습니다.
