# <a name="configure-the-skype-meetings-app-to-work-with-teams"></a>Teams에서 작업할 수 있도록 Skype 모임 앱 구성

사용자가 Microsoft Teams로 업그레이드된 후 관리자는 Microsoft Teams 관리 센터를 사용하여 사용자가 비즈니스용 Skype 모임에 참가하는 데 사용할 기본 설정 앱을 지정할 수 있습니다.

모임용 Skype 앱을 기본 설정 앱으로 지정하려면 다음을 수행합니다.

1. Microsoft Teams 관리 센터에 로그인합니다.
2. 왼쪽 창의 **Teams** 에서 **Teams 업그레이드 설정을** 선택합니다.
3. Teams 업그레이드 페이지의 **앱 기본 설정** 에서 사용자가 **비즈니스용 Skype 모임** 드롭다운 목록에 **참가할 수 있도록 기본 설정 앱에서 Skype 모임 앱을** 선택합니다.

    ![사용자가 비즈니스용 Skype 모임에 참가할 수 있도록 기본 설정 앱을 선택합니다.](media/teams-configure-skype-meetings-app-to-work-with-teams-image1.png)

## <a name="known-limitations"></a>알려진 제한 사항

Teams에서 Skype 모임 앱을 사용하는 사용자에게는 다음과 같은 제한 사항이 적용됩니다.

- 사용자는 비디오 디바이스를 변경할 수 있는 옵션이 없습니다.
- 사용자가 Teams로 업그레이드된 후 사용자가 Skype 모임 앱을 사용하여 모임에 참가한 다음 Teams에서 전화를 받으면 Skype 모임 앱의 모임이 보류되지 않습니다. 대신 사용자는 두 호출 모두에 연결됩니다.
