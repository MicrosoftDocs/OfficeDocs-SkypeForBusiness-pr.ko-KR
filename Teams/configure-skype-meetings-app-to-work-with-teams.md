# <a name="configure-the-skype-meetings-app-to-work-with-teams"></a>Teams에서 작업할 수 있도록 Skype 모임 앱 구성

사용자가 Microsoft Teams 업그레이드한 후 관리자는 Microsoft Teams 관리 센터를 사용하여 사용자가 모임에 참가하는 데 사용할 기본 비즈니스용 Skype 수 있습니다.

모임 앱의 Skype 기본 설정 앱으로 지정하는 방법은 다음을 실행합니다.

1. Microsoft Teams 관리 센터에 로그인합니다.
2. 왼쪽 창에서 **Teams 업그레이드 설정을** Teams **선택합니다**.
3. 업그레이드 Teams 앱 기본 설정에서 사용자가 기본 설정 Skype 모임 드롭다운 목록에 참가  할 비즈니스용 Skype 선택합니다.

    ![사용자가 모임에 참가할 수 있도록 기본 비즈니스용 Skype 선택하세요.](media/teams-configure-skype-meetings-app-to-work-with-teams-image1.png)

## <a name="known-limitations"></a>알려진 제한 사항

Skype 모임 앱을 사용하는 Teams 제한 사항이 적용됩니다.

- 사용자는 비디오 디바이스를 변경할 수 있는 옵션이 없습니다.
- 사용자가 Teams 업그레이드한 후 사용자가 Skype 모임 앱을 사용하여 모임에 있는 경우 Teams 모임 앱의 모임이 Skype 보류되지 않습니다. 대신 사용자가 두 호출에 연결됩니다.
