# <a name="usb-audio-and-video-devices-certified-for-microsoft-teams"></a>Microsoft 팀에 인증 된 USB 오디오 및 비디오 장치

주변 기기 인증 프로그램은 헤드셋, 스피커폰, 웹캠, 모니터 등의 USB 주변 장치에서 다양 한 오디오 또는 비디오 품질을 제공 하도록 보장 합니다. 인증 된 장치는 추가 구성이 필요 하지 않은 (플러그 & 재생), Microsoft 팀과 비즈니스용 Skype를 사용 하 여 통화 제어권을 제공 합니다. 이 제품은 최종 사용자와 관리자에 게 훌륭한 경험을 제공 합니다.

테스트 되 고 Microsoft 팀에 대해 인증 된 디바이스를 살펴보기 위해 [팀 스토어](https://products.office.com/microsoft-teams/across-devices/devices)에 걸쳐 있습니다.

팀과의 통화 제어 지원에 대 한 자세한 내용은 [팀에서 헤드셋을 사용 하 여 제어 통화](https://support.office.com/article/Control-calls-using-a-headset-in-Teams-65d6e104-444d-4013-b8c2-f11317dd69a8)를 참조 하세요.

> [!TIP]
> 비즈니스용 Skype에 대해 인증 된 디바이스를 찾고 계신가요? [비즈니스용 Skype에 대해 인증 된 USB 오디오 및 비디오 장치로](https://docs.microsoft.com/skypeforbusiness/certification/devices-usb-devices)이동 합니다.

## <a name="understanding-certification-categories"></a>인증 범주 이해

비즈니스용 Skype에서 팀으로 업그레이드할 때 사용자는 이전에 인증 된 장치를 교체할 필요가 없습니다.  모든 인증 된 장치는 Microsoft 팀 사용자와 비즈니스용 Skype 사용자 모두에 게 지원 되며 몇 가지 기능에 차이가 있습니다.  또한 인증서는 특정 디바이스 범주에 대 한 추가 지정을 제공 합니다.

- **팀 업그레이드를 통해 비즈니스용 Skype 인증** 이 수준에서 인증 된 디바이스는 처음에 비즈니스용 Skype에 대해 인증 되었습니다. 가능한 한 많은 수명 값을 사용자에 게 제공 하기 위해 Microsoft는 장치 제조업체와 협력 하 여 원래 하드웨어를 변경 하지 않고 이러한 기존 장치를 통해 가능한 많은 팀 통합을 제공 합니다. *Hookswitch* 버튼은 통화 중이 아닐 때 팀 버튼으로 작동 하며 시간에 따른 추가 기능을 포함할 수 있습니다.  이 범주의 디바이스는 마케팅 자료에 대 한 팀 인증 배지를 전달할 수 있습니다.
- **Microsoft 팀 인증** 이 수준에서 인증 된 디바이스는 손상 되지 않은 팀 경험을 위해 인증 되는 최신 장치를 나타냅니다. 이러한 디바이스는 이전 인증의 모든 이점을 제공 하며, LED 표시기가 있는 전용 팀 단추도 포함 합니다. 대부분의 폼 팩터를 사용 하는 팀 단추는 팀 로고가 있는 쉽게 찾을 수 있는 촉각 단추입니다. Microsoft는 장치 제조업체와 협력 하 여 시간이 지남에 따라 펌웨어 업데이트를 통해 새 값을 전달 합니다.

### <a name="special-usage-designations"></a>특별 한 용도 명칭

- **열린 Office에 대 한 프리미엄 마이크** 모든 헤드셋은 깨끗 한 오디오를 제공 하도록 테스트 되지만, 열려 있는 office 및 기타 잡음이 있는 환경에서는 헤드셋 사용자의 음성을 명확 하 게 선택 하지만 인접 한 소리는 전송 하지 않습니다.
- **회의실 크기** 스피커폰 및 카메라와 같은 공유 장치는 지원 되는 사용 거리와 권장 거리를 가지 며, 제조업체에 따라 다르게 보고 될 수 있습니다. 고객에 게 명확한 지침을 제공 하기 위해 모든 공유 디바이스는 잘 정의 된 회의실 크기와 소음 환경에 대해 인증을 통해 테스트 됩니다.

[Microsoft 팀에 대해 인증 된 디바이스를](https://products.office.com/microsoft-teams/across-devices/devices)찾습니다.

## <a name="end-user-experience"></a>최종 사용자 환경

### <a name="new-teams-certification-benefits"></a>신규 팀의 인증 혜택

- 전용 팀 단추 및 LED 표시등 표시등
- 팀 클라이언트 (예: 모임 시작, 음성 메일 또는 부재 중 전화)의 이벤트 및 알림에 대 한 사용자 알림
- 팀 클라이언트를 활성화 하거나 단추 (예: 모임 참가)를 사용 하 여 알림에 응답할 수 있습니다.
- 확장성을 위해 디자인 된 인증 (펌웨어 업데이트를 통해 새 장치/클라이언트 기능 제공)

### <a name="core-certification-benefits"></a>핵심 인증 이점

- 오디오 주변 기기가 여러 개 있는 경우 기본 오디오 장치 및 우선 순위 자동 선택
- 디바이스는 자동으로 팀 클라이언트에 등록 되어 사용할 준비가 되어 있습니다.
- 오디오 장치 (예: 헤드셋 및 스피커폰)는 다음과 같이 제공 됩니다.
  - PC 또는 Mac 장치에서 사용 하는 경우의 기본 통화 제어. 기본 컨트롤에는 answer/끊기, 음소거/음소거 해제, 볼륨 조절 등이 포함 됩니다.
  - 에코, 왜곡 또는 과도 한 결함 없이 광대역 오디오를 사용 합니다. 장치 에서도 반향 취소가 지원 됩니다.
- 비디오 장치는 노이즈, 색, 이미지 세부 정보, 지터, 대기 시간, 프레임 속도 및 보기 필드에 대 한 요구 사항을 충족 합니다.

## <a name="administrator-experience"></a>관리자 환경

- 구축이 필요 하지 않음
- 모든 장치에서 새로운 기능과 성능 개선을 허용 하는 펌웨어 업데이트를 지원 합니다.

제조업체이 고 인증 프로그램에 참가 하려면 요구 사항 및 사용 가능한 프로그램에 [참가 하는 방법을](https://docs.microsoft.com/skypeforbusiness/certification/how-to-join) 참조 하세요.
