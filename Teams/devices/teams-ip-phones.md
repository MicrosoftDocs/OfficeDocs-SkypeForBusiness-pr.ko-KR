# <a name="ip-phones-certified-for-microsoft-teams"></a>Microsoft Teams에 대해 인증된 IP 전화기

IP 전화 인증 프로그램은 인증된 표준 전화기 및 전화 회의 전화를 사용하는 고객이 고품질 음성 통화 및 모임 환경을 경험할 수 있도록 합니다.

## <a name="announcing-certified-for-microsoft-teams"></a>Microsoft Teams용 인증 발표

고객에게 다양한 비즈니스용 Skype 인증 IP 전화기 제공 외에도 Teams IP 전화 인증 프로그램의 일부로 Microsoft Teams에 대해 인증될 새 IP 전화 하드웨어의 가용성을 발표하고 있습니다. 이 인증 프로그램은 더 높은 성능 목표 및 오디오 품질 메트릭을 사용하여 더 높은 품질 표시줄을 보장합니다. 전화 공급업체는 이 높은 기준을 충족하기 위해 적극적으로 작업하고 있으며 올해 새 요구 사항에 맞출 수 있습니다. 또한 일부 파트너는 Teams IP 전화와 동일한 하드웨어에서 비즈니스용 Skype 서비스에 대한 지원을 제공하기 위해 작업 중입니다. Teams 인증 IP 휴대폰의 이점은 다음과 같습니다.

- 하드웨어 버튼 통합 및 LED 알림이 있는 네이티브 Teams 환경
- 일정 통합 및 원터치 모임 참가
- Intune 통합을 사용하여 Microsoft 디바이스 관리 솔루션을 통해 배포 및 관리
