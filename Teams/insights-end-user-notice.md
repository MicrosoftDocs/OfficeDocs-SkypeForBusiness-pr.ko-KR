# <a name="education-insights-end-user-notice"></a>교육 Insights 사용자 알림

이 애플리케이션은 Microsoft에서 제공하지만 이 애플리케이션의 일부가 Microsoft Teams.
이 애플리케이션을 사용하려면 이 애플리케이션("Application Services")을 Microsoft 365 또는 기타 Microsoft 서비스 구독이 필요할 수 있습니다.
이 애플리케이션 및 해당 Application Services에 대한 액세스는 (a) Microsoft 서비스 계약 또는 (b) Microsoft와의 조직의 구독 계약을 통해 적용되는 이러한 Application Services에 대한 기존 약관에 따라 적용됩니다.
i) 조직의 구독이 만료되거나 ii) 조직이 해당 구독, 애플리케이션 또는 Application Services에 대한 액세스를 종료하기로 결정한 경우 이 애플리케이션에 대한 액세스 권한이 손실될 수 있습니다. 애플리케이션에는 별도의 조건에 따라 라이선스가 부여된 타사 구성 요소를 포함할 수 있습니다.
이러한 타사 구성 요소에 대한 타사 알림은 타사 알림 에서 온라인으로 액세스할 [수 있습니다.](https://eduinsights.microsoft.com/thirdpartynotice.txt)
