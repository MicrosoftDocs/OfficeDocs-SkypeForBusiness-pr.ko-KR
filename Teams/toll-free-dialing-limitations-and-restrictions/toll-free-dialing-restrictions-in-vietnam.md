# <a name="toll-free-dialing-restrictions-in-vietnam"></a>베트남의 무료 전화 걸기 제한

[다른 국가 또는 지역](../toll-free-dialing-limitations-and-restrictions.md)또는 무료 전화 걸기 제한 사항에 대 한 자세한 정보를 확인 하세요.


| **세부 정보**                                      | **설명**                                                                                                                                                                                                                                                                                                                                                                                           |
|:-------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 무료 전화를 받을 수 있나요?  <br/>           | 예  <br/>                                                                                                                                                                                                                                                                                                                                                                                                |
| 전화 걸기 형식  <br/>                         | 120-11-3XX <br/>  122-XXX-XX <br/>  1800 XXXX: 형식이 더 이상 제공 되지 않음 <br/>                                                                                                                                                                                                                                                                                                                            |
| 국가/지역 ISO 코드  <br/>                   | VN  <br/>                                                                                                                                                                                                                                                                                                                                                                                                 |
| 섬/지역 포함  <br/>                  | 해당 사항 없음  <br/>                                                                                                                                                                                                                                                                                                                                                                                     |
| 섬/지역 포함 안 함  <br/>              | 해당 사항 없음  <br/>                                                                                                                                                                                                                                                                                                                                                                                     |
| 유선/고정 회선 가용성  <br/>             | Partial. 메모를 참조 하세요.  <br/>                                                                                                                                                                                                                                                                                                                                                                             |
| 모바일 네트워크 가용성  <br/>               | Partial. 메모를 참조 하세요.  <br/>                                                                                                                                                                                                                                                                                                                                                                             |
| 메모, 제한 사항 및 액세스 문제  <br/> | 120 XX XXX 번호 형식은 베트남 통신 국제 (VTI) 및 VNPT (베트남 Post 및 통신) 고정 및 모바일 네트워크에서 작동 합니다. <br/> <br/> 2018 년 6 월부터 120 XX XXX 번호 형식은 Mobifone의 모바일 네트워크에서 호출할 수 없습니다. <br/> <br/>**참고:** Mobifone &amp; Vinaphone는 VNPT의 일부입니다. 숫자 형식 122 XXX XX 네트워크 적용 범위: Viettel (고정 및 모바일). 통신의 베트남 Ministry에 따라 허용 되는 최대 통화 기간은 2 시간입니다. |
