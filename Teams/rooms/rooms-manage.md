# <a name="manage-microsoft-teams-rooms"></a>Microsoft Teams 룸 관리

조직에서 Microsoft 팀의 회의실 인증 디바이스를 보유 하 고 있는 경우 Microsoft 팀 관리 센터를 사용 하 여 중앙 위치에서 관리할 수 있습니다. 당신은 할 수 있어요:

- 장치를 다시 시작 하거나 차단 하 고 장치 로그를 다운로드 하는 등의 디바이스 관리 수행
- 팀 특정 설정 적용
- Microsoft 팀 실내 장치 및 카메라, 디스플레이, 마이크 등의 주변 기기 상태 확인
- 현재 및 과거 모임 활동 검토 (예: 통화 품질, 네트워크 상태 및 연결 및 참가자 수에 대 한 세부 정보)
- Microsoft 팀 대화방 장치에 연결 된 주변 기기 (예: 카메라 및 프로젝터) 보기

팀 회의실 장치를 관리 하려면 [Microsoft 팀 관리 센터](https://admin.teams.microsoft.com) 를 열고 **장치**  >  **팀 대화방**으로 이동 하세요.

:::image type="content" source="../media/teams-rooms-summary.png" alt-text="팀 관리 센터의 팀 대화방 요약 페이지":::

> [!IMPORTANT]
> 팀 관리 센터를 사용 하 여 디바이스를 관리 하려면 팀 서비스 관리자 또는 전역 관리자 역할을 할당 해야 합니다.

## <a name="make-changes-to-teams-rooms-devices"></a>팀 회의실 장치 변경 하기

팀 대화방 장치가 두 개 이상인 경우 여러 장치에서 동시에 대부분의 작업을 수행할 수 있습니다. 예를 들어 모든 장치에서 팀 앱 설정을 동시에 설정할 수 있습니다.

### <a name="device-settings"></a>장치 설정

조직에서 하나 이상의 장치에 대 한 설정을 변경할 수 있습니다. 설정을 변경 하려면 관리 하려는 장치를 선택 하 고 **설정 편집**을 선택 합니다. 장치에서 변경할 수 있는 모든 설정을 포함 하는 새 창이 열립니다. 다음 표에는 팀 관리 센터를 사용 하 여 변경할 수 있는 설정이 나와 있습니다. 일부 설정은 단일 장치를 선택한 경우에만 사용할 수 있습니다.

둘 이상의 장치를 선택 하는 경우 일괄 편집을 지 원하는 설정은 다음 두 가지 옵션을 표시 합니다.

- **기존 값 유지** 이 옵션을 선택 하면 선택한 장치에 대 한 설정이 변경 되지 않습니다.
- **기존 값을 다음으로 바꾸기** 이 옵션을 선택 하면 사용자가 선택한 장치의 설정을 사용자가 제공한 값으로 업데이트할 수 있습니다.
    > [!CAUTION]
    > 업데이트 하도록 선택한 설정의 기존 값이 사용자가 입력 한 값으로 대체 됩니다. 기존 값 목록에 추가 하려는 경우에는 추가 하려는 값으로 기존 값을 포함 해야 합니다. 예를 들어 설정에이의 기존 도메인 목록이 있는 경우 추가 하려는 경우 제공 해야 하는 값이 됩니다 `contoso.com, fabrikam.com` `northwindtraders.com` `contoso.com, fabrikam.com, northwindtraders.com` .

### <a name="device-restart-options"></a>장치 다시 시작 옵션

장치 설정에 대 한 변경 사항은 장치를 다시 시작한 후에만 적용 됩니다. 다시 시작 해야 하는 변경 작업을 수행할 때 장치를 즉시 다시 시작할지 아니면 다시 시작할지를 예약할 지 선택할 수 있습니다. 사용할 수 있는 다시 시작 옵션은 다음과 같습니다.

- **즉시 다시 시작** 이 옵션을 선택 하면이 옵션을 선택 하는 즉시 변경 하는 모든 장치가 다시 시작 됩니다.
- **예약 된 다시 시작** 이 옵션을 선택 하면 사용자가 변경 하는 장치를 조직에 대해 덜 방해가 되는 시간에 다시 시작할 수 있습니다.
  - **날짜 및 시간을 선택** 합니다. 장치를 다시 시작할 특정 날짜 및 시간을 선택 합니다. 선택 하는 날짜와 시간은 다시 시작할 장치에 대 한 로컬입니다. 
  - **야간 재부팅을 위해 업데이트 남기기** 유지 관리를 수행 하기 위해 모든 장치가 야 간에 다시 시작 됩니다. 장치에 대해 변경한 내용은이 컴퓨터를 다시 시작 하는 동안 적용 됩니다.

> [!CAUTION]
> 다시 시작 될 때 사용 중인 장치를 다시 시작 프로세스 동안 사용할 수 없게 됩니다. 진행 중인 모임에서 연결이 끊어져 새 모임에 참가할 수 없습니다.

### <a name="remove-or-block-a-device"></a>장치 제거 또는 차단

장치를 **제거** 하면 디바이스는 조직에서 제거 되 고 팀 관리 센터의 팀 대화방 장치 목록에 더 이상 표시 되지 않습니다. 

장치를 **차단** 하면 팀이 디바이스와 더 이상 통신 하지 않습니다. 차단 된 장치는 대량 편집 중인 장치 그룹에 포함 된 경우에도 명령을 보낼 수 없습니다. 상태가 **차단 됨**상태의 팀 대화방 디바이스 목록에 계속 표시 됩니다.

장치가 차단 되거나 제거 되는지 여부와 상관 없이 유효한 사용자 이름 및 암호를 사용 하 여 구성 된 경우에는 Microsoft 365에 연결 되는 경우 팀 대화방 디바이스 목록에 자동으로 다시 추가 됩니다.

하나 이상의 장치를 제거 하려면 다음을 수행 합니다.

1. **장치**  >  **팀 대화방** 으로 이동 하 여 제거 하려는 장치를 선택 합니다.
1. **제거**를 선택 합니다.

장치를 차단 하려면 다음을 수행 합니다.

1. **장치**  >  **팀 대화방** 으로 이동 하 여 차단할 디바이스의 이름을 선택 합니다.
1. 장치 세부 정보 페이지에서 페이지의 오른쪽 위 모서리에 있는 **작업** 을 선택 합니다.
1. **블록**을 선택 합니다.

장치를 차단 해제 하려면 다음을 수행 합니다.

1. **장치**  >  **팀 대화방** 으로 이동 하 여 차단할 디바이스의 이름을 선택 합니다.
1. 장치 세부 정보 페이지에서 페이지의 오른쪽 위 모서리에 있는 **작업** 을 선택 합니다.
1. **차단 해제**를 선택 합니다.

## <a name="download-device-logs"></a>장치 로그 다운로드

Microsoft support에서 요청 하는 경우 장치의 진단 로그 파일 복사본을 다운로드할 수 있습니다. 로그 파일은 팀 관리 센터에서 다운로드할 수 있는 zip 파일로 압축 됩니다.

팀 대화방 디바이스의 로그를 컴퓨터에 다운로드 하려면 다음을 수행 합니다.

1. **장치**  >  **팀 대화방** 으로 이동 하 여 로그를 다운로드 하려는 디바이스의 이름을 선택 합니다.
1. **장치 로그 다운로드**를 선택 합니다. 장치 로그를 사용할 수 있게 되는 데 몇 분 정도 걸릴 수 있습니다.
1. **기록** 탭을 선택한 다음 **진단 파일**에서 로그 파일 링크를 선택 합니다. 장치의 진단 로그 파일이 포함 된 zip 파일이 브라우저의 기본 다운로드 폴더에 다운로드 됩니다.

## <a name="view-device-information"></a>장치 정보 보기

팀 관리 센터에서 조직의 모든 장치에 대 한 전반적인 상태를 보고 각 장치의 세부 정보를 개별적으로 볼 수 있습니다.

### <a name="teams-rooms-system-dashboard"></a>팀 대화방 시스템 대시보드

팀 대화방 시스템 대시보드에는 모든 디바이스의 상태와 상태가 한눈에 표시 됩니다.

### <a name="device-details-view"></a>장치 정보 보기

장치에 대 한 자세한 정보를 보려면 장치 목록에서 해당 이름을 선택 합니다. 자세히 보기에서 장치에 대 한 다음 정보를 볼 수 있습니다.

- **상태** 팀 실내 장치의 전반적인 상태를 표시 합니다. 상태는 **정상** 또는 **비정상**중 하나일 수 있습니다.
- **이후 오프 라인** Microsoft 365에서 장치와 통신할 수 있었던 마지막 시간을 보여 줍니다.
- **장치 상태** 장치의 현재 상태 ( **유휴**, **팀 모임**, **Skype 모임**또는 **수집**)를 표시 합니다.
- **주변 장치** 팀 실내 장치에 연결 된 주변 기기와 해당 상태를 표시 합니다. 상태는 **연결** 되거나 **연결이 끊어질**수 있습니다.
- **상태** 팀 대화방 장치, 네트워크 연결, 필수 서비스에 로그인 상태 및 소프트웨어 버전 정보에 연결 된 주변 기기에 대 한 자세한 정보를 표시 합니다.
- **세부 정보** 제조업체 정보, 네트워크 IP 주소, 팀 실 장치 직렬/MAC 주소를 표시 합니다.
- **활동** 모임 날짜 및 시간, 참가자 수, 기간 및 오디오 품질을 비롯 한 지난 모임 세부 정보를 표시 합니다. 모임 세부 정보에 대 한 자세한 내용은이 문서의 뒷부분에 나오는 [모임 활동 세부 정보](#meeting-activity-details) 섹션을 참조 하세요.
- **기록** 구성 업데이트, 장치 다시 시작 및 장치 로그 다운로드 링크를 포함 하 여 팀 대화방 장치에서 관리 활동 기록을 표시 합니다.

#### <a name="meeting-activity-details"></a>모임 활동 세부 정보

팀 대화방 디바이스 세부 정보에 있는 **활동** 탭에는 시간이 지남에 따라 장치가 참가 한 모든 모임에 대 한 상위 수준 및 자세한 정보가 표시 됩니다. **활동** 탭에서 모임이 보류 된 때, 모임에 참가 하는 참가자의 수, 모임 중에 오디오 품질을 확인할 수 있습니다.

:::image type="content" source="../media/teams-rooms-meeting-activity-summary.png" alt-text="팀 대화방 디바이스 활동 요약 목록":::

특정 모임에 대 한 자세한 정보를 보려면 모임의 날짜와 시간을 선택 합니다. 모임에 참가자가 두 명만 있으면 참가자 세부 정보 페이지가 표시 되 고, 그렇지 않으면 참가자 요약 페이지가 표시 됩니다.

##### <a name="participant-summary"></a>참가자 요약

참가자 요약 페이지에는 모임에 참석 하는 모든 참가자가 표시 됩니다. 각 참가자가 모임에 참가 한 시간, 해당 이름, 오디오 품질, 세션 중에 사용 된 기능을 확인할 수 있습니다. 참가자 세션의 세부 정보를 보려면 해당 참가자의 세션 시작 시간을 선택 합니다.

##### <a name="participant-details"></a>참가자 세부 정보

참가자 세부 정보 페이지에 해당 참가자의 세션에 대 한 종단 간 진단 정보가 표시 됩니다. 다음 그래픽 에서처럼 참가자와 팀 대화방 장치에 대 한 **장치**, **시스템**및 **연결** 정보를 제공 합니다. 참가자와 팀 대화방 장치 간의 **네트워크** 진단 정보도 함께 제공 됩니다. 자세한 정보를 보려는 컨텍스트에 대 한 아이콘을 선택 합니다. 추가 진단 정보를 보려면 **고급** 탭을 선택 합니다.
