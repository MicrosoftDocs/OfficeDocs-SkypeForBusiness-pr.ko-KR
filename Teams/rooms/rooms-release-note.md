## <a name="microsoft-teams-rooms-feature-introduction-and-issue-resolution"></a>Microsoft Teams Rooms 기능 소개 및 문제 해결

### <a name="48190-04062021"></a>4.8.19.0 (04/06/2021)

이 업데이트에서 소개된:
- Teams Rooms에 대한 정부 커뮤니티 클라우드 높음(GCCH) 지원 기존 Teams Rooms 디바이스가 있는 GCCH 고객은 Microsoft Teams Rooms 디바이스를 수동으로 업데이트하여 버전 4.8.18.0을 [다운로드할 수 있습니다.](manual-update.md)
- 더 나은 비디오 품질(720p 지원)으로 확대/축소 모임에 참가하고 참가자의 비디오 갤러리를 수신합니다.
- Teams 기본 모드에 대해 비즈니스용 Skype 로그인 실패 배너가 제거되었습니다. 이 변경은 비즈니스용 Skype 인프라를 제거하는 조직을 지원합니다.
- 이제 팀 모임 조인 링크 구문 분석은 외부 Teams에 원활하게 참가할 수 있도록 Microsoft Defender 고급 스레드 보호 안전 링크를 처리합니다.
- 공유자 PC에 Windows에 사용자 지정 DPI 집합이 있는 경우 비즈니스용 Skype 모임에서 공유 콘텐츠 크기 조정 문제 해결
- 품질 및 안정성 수정

- 품질 및 안정성 수정

### <a name="47150-12112020"></a>4.7.15.0 (12/11/2020)

이 업데이트에서 소개된:

- Teams 모임에서 모임 참가자에게 HDMI 오디오 공유

- Teams Room이 참석자로 참가할 때 오디오 사용 권한을 기반으로 음역을 금지합니다. 자세한 내용은 Teams 모임에서 참석자 [오디오 권한 관리를 참조하세요.](https://support.microsoft.com/office/manage-attendee-audio-permissions-in-teams-meetings-f9db15e1-f46f-46da-95c6-34f9f39e671a)
- Teams Room 콘솔에서 다른 사람의 비디오에 집중 조명을 표시하고 방 디스플레이에서 스포트라이트 비디오 사용

> [!NOTE]
> Cortana 음성 기술은 미국에 있는 테넌트에 대한 일부 오디오 주변 장치를 사용할 수 있습니다. 향후 추가 국가 또는 지역이 추가될 예정입니다. 자세한 내용은 [Teams의 Cortana 음성 지원을 참조하세요.](../cortana-in-teams.md)

<sup>2</sup> IT 관리자가 Microsoft Whiteboard를 켜야 합니다. 또한 회의실 전면에 터치가 설정된 경우 디바이스 관리자 로그인을 사용하여 Windows 설정을 사용하여 여러 터치 디스플레이를 보정하여 Teams 모임에 공유된 회의실 디스플레이에서 공동 작업을 위해 Microsoft Whiteboard를 사용하여 공동 작업을 시작해야 합니다.

### <a name="release-to-market-1272016"></a>시장에 릴리스(2016년 12월 7일)

**기능:**

 **비즈니스용 Skype용 기본 제공**

- Skype 모임의 원터치 조인
- 화면 채우기 HD 비디오 및 HD 와이드 밴드 오디오가 있는 회의실에 최적화된 Skype 모임 환경
- 모든 참가자는 어디에 있든 원하는 디바이스를 사용하여 Skype 모임에 연결할 수 있습니다.
- 디렉터리에서 해당 가용성을 즉시 확인할 수 있는 사용자 초대 또는 전화 통화를 통해
- 비즈니스용 Skype PSTN 회의 및 PSTN 통화를 지원하여 회의실의 독립 실행형 전화 교체

 **회의실 변환**

- 테이블 터치 컨트롤러 중심 및 회의실 디스플레이 전면에 최적화된 전용 Skype 모임 앱
- 방 디스플레이 또는 프로젝터 앞에서 기존 투자를 다시 사용
- 허들 공간에서 대규모 회의실까지 모든 유형의 모임 공간에서 작동합니다.
- 비즈니스용 Skype 오디오 및 비디오 디바이스는 다양한 방 크기에 사용할 수 있습니다.
- 회의실 및 Skype 모임에 데스크톱 공유를 프로젝트할 수 있는 기본 제공 유선 인제스트

 **배포하기 쉽고 관리가 간편합니다.**

- 룸에서 사용자 감지 시 자동으로 디스플레이가 깨어나는 Always-On 어플라이언스
- UWP(유니버설 Windows 플랫폼) Skype 모임 앱의 간단한 배포 및 업데이트
- Windows AppLocker가 디바이스를 Skype 모임 앱으로 잠그기
- Intune 및 MDM(구성 관리자)을 통해 Windows 10 Enterprise 디바이스로 모니터링 및 관리
- 엔터프라이즈급 안정성
- 익숙한 Skype 사용자 인터페이스로 인해 최종 사용자의 낮은 교육 노력
- Surface Pro 4 태블릿에서 실행
