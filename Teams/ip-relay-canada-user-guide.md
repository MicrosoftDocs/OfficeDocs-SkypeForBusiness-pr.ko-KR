# <a name="ip-relay-in-canada---user-guide"></a>캐나다의 IP 릴레이 - 사용자 가이드

[!INCLUDE [sfbo-retirement](../Skype/Hub/includes/sfbo-retirement.md)]

## <a name="scenario-a"></a>시나리오 A
  
청각 또는 음성 장애가 있는 사람이 비장애인에게 연락하고자 하는 경우 특수 웹 기반 클라이언트를 통해 통화 응시자라는 에이전트와 "채팅"합니다.

그런 다음 통화 담당자는 채팅 세션을 통해 청각 또는 음성 장애인과 통신하고 필요에 따라 캐나다 또는 미국 전화 네트워크를 통해 비장애인에게 연락합니다.

## <a name="scenario-b"></a>시나리오 B

비장애인이 등록된 청각 또는 음성 장애가 있는 사용자와 대화하려는 경우 Microsoft에서 제공하는 무료 전화 번호를 호출합니다.

그런 다음 통화 응시자는 비장애인과 통신하고 채팅을 통해 청각 또는 음성 장애인에게 연락하고 앞뒤로 의사 소통합니다.

청각 또는 음성 장애가 있는 사람이 채팅 프로그램에 온라인 상태가 아닌 경우, 전자 메일 주소를 사용할 수 있는 경우 발신자가 요청한 경우 발신자가 청각 또는 음성 장애가 있는 사람에게 이메일을 보냅니다.

## <a name="scenario-c"></a>시나리오 C

청각 또는 음성 장애가 있는 사람이 응급 지원이 필요한 경우 적절한 단추(911 – 경찰 / 911 – 화재 / 911 – 구급차)를 선택할 수 있습니다.

그러면 북부 IP 릴레이 운영자에게 긴급 알림이 전송되며, 이 운영자는 이 목적을 위해 설계된 911 계정에 액세스합니다. 호출 응시자의 주소가 확인되면 해당 PSAP에 연락하여 필요한 기간 동안 줄을 유지합니다.

## <a name="how-to-place-a-text-chat-to-voice-call"></a>음성 통화에 텍스트 채팅을 배치하는 방법

https://aka.ms/IPRelay

음성 통화에 대한 텍스트를 시작하려면 위의 URL로 이동하여 IP Message Relay 사용자 이름 및 암호를 사용하여 로그인합니다.

로그인하면 화면 왼쪽에 지침이 표시됩니다.

### <a name="how-to-make-a-text-to-voice-call"></a>음성으로 텍스트 통화를 하는 방법

1. 화면의 오른쪽 아래 모서리에서 채팅을 클릭합니다.
2. IPRelay 사용자를 클릭합니다.
3. 팝업되는 새 상자의 아래쪽에 메시지를 입력합니다.

### <a name="how-to-receive-a-voice-to-text-call"></a>음성 텍스트 변환 통화를 받는 방법

- 음성 사용자는 음성을 문자 통화에 배치하기 위해 IP 메시지 릴레이 사용자 이름을 알고 있어야 합니다.

- 음성 사용자는 (866) 660-8613으로 전화하여 IP 메시지 릴레이 운영자와 연결할 수 있습니다.

- 통화 또는 채팅을 받으려면 IP Message Relay 포털에 로그인해야 합니다.

### <a name="how-to-place-a-911-call"></a>911 통화를 하는 방법

응급 상황이 발생하면 화면 아래쪽에 있는 적절한 비상 버튼을 선택할 수 있습니다. 911 – 경찰 / 911 – 화재 / 911 – 구급차

IP 메시지 릴레이 운영자는 911에 연락하고, 주소를 확인하고, 더 이상 필요하지 않을 때까지 다양한 응급 센터로 번역하고, 해당 부서가 사용자의 위치로 파견됩니다.

> [!Warning]
> 해당 기관이 단순히 테스트라고 표시하더라도 적절한 기관에 연락하여 발송될 수 있으므로 이 서비스를 테스트하지 마십시오. 응급 기관에서 거짓 디스패치 요금이 부과될 수 있습니다.
