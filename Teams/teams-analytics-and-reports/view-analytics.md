# <a name="view-analytics-in-teams"></a>팀에서 분석 보기

Microsoft 팀에서 사용자는 자신이 속한 팀과 채널에 대 한 분석을 볼 수 있습니다. 이 정보는 사용자가 팀의 사용 패턴 및 활동에 대 한 통찰력을 제공 합니다. 사용자는 세 가지 수준에서 활성 사용자 수, 게시물, 회신 등의 데이터를 볼 수 있습니다.

- **팀 간 분석** 을 통해 사용자는 단일 목록 보기의 구성원 또는 소유자 인 모든 팀의 사용 현황 데이터에 대 한 폭넓은 개요를 제공할 수 있습니다.
- **팀 단위 분석** 을 통해 사용자에 게 특정 팀의 사용량 데이터를 보여 주는 보다 세부적인 보기를 제공할 수 있습니다.
- **채널당 분석** 을 통해 사용자는 특정 채널에 대 한 사용량 데이터를 표시 하는 더욱 세밀 한 보기를 제공할 수 있습니다.

사용자는 이러한 보기 중 하나를 필터링 하 여 지정 된 기간의 데이터를 볼 수 있습니다.

## <a name="view-cross-team-analytics"></a>팀 간 분석 보기

1. 팀의 팀 목록 아래에서 **팀 참가 또는 만들기**옆에 있는 팀 **관리**를 클릭 합니다.
2. **분석** 탭을 클릭 합니다.
3. 구성원 또는 소유자 인 모든 팀의 사용량 데이터를 표시할 날짜 범위를 선택 합니다.

    ![팀 간 분석 보기의 스크린샷](../media/view-analytics-cross-team.png)

    |항목 |설명  |
    |--------|-------------|
    |**이름**   |팀의 이름입니다. |
    |**활성 사용자**   |지정 된 기간 동안 팀 활동 및 추세 행의 활성 사용자 수입니다.
    |**사용자**   |지정 된 기간에 팀의 총 사용자 수입니다. 여기에는 팀 소유자, 팀 구성원 및 게스트가 포함 됩니다.|
    |**게스트로**   |지정 된 기간에 팀의 게스트 수입니다. |
    |**게시물**   |지정한 기간 동안 팀 채팅에 게시 된 새 메시지 수입니다. |
    |**회신**   |지정 된 기간에 팀 채팅의 답 글 수입니다. |
    |**유형**   |팀이 비공개 팀 인지 또는 공개 팀입니다.|

## <a name="view-per-team-analytics"></a>팀 단위 분석 보기

1. 팀에서 원하는 팀으로 이동 하 고 **기타 옵션 (...)** 을 클릭 한 다음 **팀 관리**를 클릭 합니다.
2. **분석** 탭을 클릭 합니다.
4. 팀에 대 한 사용 현황 데이터를 표시할 날짜 범위를 선택 합니다.  

    ![팀 단위 분석 보기의 스크린샷](../media/view-analytics-per-team.png)

    |항목 |설명  |
    |--------|-------------|
    |**요약**   |다음을 포함 한 팀 활동 요약:<ul><li>**사용자**: 지정 된 기간의 총 사용자 수입니다. 여기에는 팀 소유자, 팀 구성원 및 게스트가 포함 됩니다.</li> <li>**게시물**: 지정한 기간 동안 팀 채팅에 게시 된 새 메시지 수입니다.</li><li>**회신**: 지정 된 기간에 팀 채팅의 답 글 수입니다.</li> <li>**앱**: 팀에 추가 된 앱 수입니다.</li><li>**모임**: 팀 수준에서 구성 된 팀 모임 수입니다.</li> </ul> |
    |**활성 사용자**   |활성 및 비활성 사용자의 수입니다.|
    |**역할인**   |팀 소유자, 팀 구성원, 게스트를 비롯 한 역할별 사용자 수|
    |**활성 사용자** 차트  |일별 활성의 수입니다. 지정 된 날짜의 점을 마우스로 가리켜 해당 날짜의 활성 사용자 수를 확인 합니다.|
    |**메시지** 차트  |날짜별로 팀 채팅에 게시 된 총 메시지 수입니다. 지정 된 날짜의 점을 가리키면 해당 날짜에 게시 된 새 게시물 및 회신 수가 표시 됩니다.|

> [!TIP]
> 팀 [분석 보기](#view-cross-team-analytics)에서 팀 단위 목록을 클릭 하 여 팀 별로 분석을 볼 수도 있습니다.

## <a name="view-per-channel-analytics"></a>채널별 분석 보기

1. 팀에서 원하는 채널로 이동 하 고 **기타 옵션 (...)** 을 클릭 한 다음 **채널 관리**를 클릭 합니다.
2. **분석** 탭을 클릭 합니다.
3. 채널에 대 한 사용 현황 데이터를 표시할 날짜 범위를 선택 합니다.  

    ![채널당 분석 보기의 스크린샷](../media/view-analytics-per-channel.png)

    |항목 |설명  |
    |--------|-------------|
    |**요약**   |다음을 포함 한 채널 활동 요약:<ul><li>**사용자**: 지정 된 기간의 총 사용자 수입니다. 여기에는 팀 소유자, 팀 구성원 및 게스트가 포함 됩니다.</li> <li>**게시물**: 지정한 기간 동안 채널에 게시 된 새 메시지 수입니다.</li><li>**회신**: 지정 된 기간 동안 채널의 응답 수입니다.</li> <li>**앱**: 채널에 추가 된 앱 수입니다.</li> </ul> |
    |**메시지** 차트  |날짜별로 채널 채팅에 게시 된 총 메시지 수입니다. 지정 된 날짜의 점을 가리키면 해당 날짜에 게시 된 새 게시물 및 회신 수가 표시 됩니다.|

> [!TIP]
> 또한 [각 팀 분석 보기](#view-per-team-analytics)의 드롭다운 목록 상자에서 채널을 선택 하 여 채널별 분석을 볼 수 있습니다.
    
> [!NOTE]
> 데스크톱 클라이언트, 모바일 클라이언트, 웹 클라이언트에서 의도적인 작업을 수행 하는 사용자로 활성 사용자를 정의 합니다. 의도적인 작업의 예로는 채팅 시작, 전화 걸기, 파일 공유, 팀 내에서 문서 편집, 모임 참여 등이 포함 됩니다. 자동 부팅, 화면 최소화 또는 앱 닫기와 같은 수동 작업을 제거 합니다. 또한 단일 사용자 ID로 모든 작업을 해제 합니다.
