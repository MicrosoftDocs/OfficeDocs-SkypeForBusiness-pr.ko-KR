# <a name="assign-change-or-remove-a-phone-number-for-a-user"></a>사용자의 전화 번호 할당, 변경 또는 제거

통화 플랜을 설정할 때 사용자에게 전화 번호를 할당합니다. Microsoft Teams에서 사용자가 통화를 클릭하면 지정한 전화 번호가 **나열됩니다.**

![Teams에 표시되는 사용자의 전화 번호입니다.](media/teams-phone-number.png)

사용자가 전화를 걸고 받을 수 있도록 사용자를 설정할 때 먼저 Microsoft Teams 관리 센터를 사용하여 전화 번호를 할당해야 합니다. 필요한 경우 전화 번호를 변경하거나 제거할 수 있습니다.
  
Teams에서 통화 요금제와 비용에 대한 자세한 내용은 Teams 추가 기능 [라이선스를 참조합니다.](https://docs.microsoft.com/microsoftteams/teams-add-on-licensing/microsoft-teams-add-on-licensing)
  
> [!NOTE]
> 사용자에게 라이선스가 할당되어 있는지 여부를 확인하려면 Microsoft Teams 관리 센터로 > **있습니다.** 라이선스가 할당된 경우 페이지에 표시됩니다.  Microsoft 365 관리 센터를 사용할 수도 있습니다.
  
## <a name="assign-a-phone-number-to-a-user"></a>사용자에게 전화 번호 할당
 
![Microsoft Teams 로고를 나타내는 아이콘](media/teams-logo-30x30.png) **Microsoft Teams 관리 센터 사용**
    
1. 왼쪽 탐색 모음에서 **음성 전화 번호를**  >  **클릭합니다.**
2. 전화 번호 **페이지에서** 목록에서 부호 없는 번호를 선택한 다음 편집을 **클릭합니다.**  
3. 편집 **창의** **[할당]** 아래에서 표시 이름 또는 사용자 이름으로 사용자를 검색한 다음 할당을 **클릭합니다.**
4. 관련된 긴급 위치를 할당하거나 변경하려면 긴급 위치 아래에서 위치를 검색한 다음 선택합니다.
5. 전화 번호 정보를 통해 사용자에게 전자 메일을 보낼지 여부에 따라 전화 번호 정보가 있는 전자 메일 사용자를 끄거나 **끄거나 끄면 됩니다.** 기본적으로 이 설정은 설정되어 있습니다. 
6. **저장** 을 클릭합니다.
    > [!NOTE]
    > Microsoft 365 또는 Office 365와 Teams 간의 대기 시간 때문에 사용자가 사용하도록 설정하는 데 최대 24시간이 걸릴 수 있습니다. 24시간 후에 전화 번호가 올바르게 할당되지 않으면 비즈니스 제품 지원에 [문의하세요. 관리자 도움말](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products) 도움을 드리세요!

  
## <a name="change-a-phone-number-for-a-user"></a>사용자의 전화 번호 변경
 
![Microsoft Teams 로고를 나타내는 아이콘](media/teams-logo-30x30.png) **Microsoft Teams 관리 센터 사용**
    
1. 왼쪽 탐색 모음에서 **사용자를** 클릭하고 원하는 사용자를 찾아 두 번 클릭하고 계정을 클릭한 다음 일반 정보에서 사용자에게 할당된 전화 번호를 메모합니다.
2. 왼쪽 탐색 모음에서 **음성 전화 번호를**  >  **클릭합니다.**
3. 전화 번호 **페이지에서** 1단계에서 식별한 번호를 선택한 다음 편집을 **클릭합니다.**  
4. 편집 **창의** 할당된 **창에서** **X를** 클릭하여 사용자를 제거합니다.
5. **저장** 을 클릭합니다.
6. 전화 번호 **페이지에서** 목록에서 부호 없는 번호를 선택한 다음 편집을 **클릭합니다.**  
7. 편집 **창의** **[할당]** 아래에서 표시 이름 또는 사용자 이름으로 사용자를 검색한 다음 할당을 **클릭합니다.**
8. 관련된 긴급 위치를 할당하거나 변경하려면 긴급 위치 아래에서 위치를 검색한 다음 선택합니다.
9. **저장** 을 클릭합니다.

## <a name="remove-a-phone-number-from-a-user"></a>사용자에서 전화 번호 제거
 
![Microsoft Teams 로고를 나타내는 아이콘](media/teams-logo-30x30.png) **Microsoft Teams 관리 센터 사용**

1. 왼쪽 탐색 모음에서 **사용자를** 클릭하고 원하는 사용자를 찾아 두 번 클릭하고 계정을 클릭한 다음 일반 정보에서 사용자에게 할당된 전화 번호를 메모합니다.
2. 왼쪽 탐색 모음에서 **음성 전화 번호를**  >  **클릭합니다.**
3. 전화 번호 **페이지에서** 2단계에서 식별한 번호를 선택한 다음 편집을 **클릭합니다.**  
4. 편집 **창의** 할당된 **창에서** **X를** 클릭하여 사용자를 제거합니다.
5. **저장** 을 클릭합니다.
