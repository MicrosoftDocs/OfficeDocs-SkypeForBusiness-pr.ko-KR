# <a name="release-notes-for-microsoft-teams-admin-features"></a>Microsoft Teams 관리자 기능에 대한 릴리스 정보

## <a name="january-27-2021"></a>2021년 1월 27일

### <a name="virtual-visits-with-teams---integration-into-ehr"></a>Teams를 통해 가상 방문 - EHR에 통합

Microsoft Teams EHR(Electronic Health Record) 커넥터용 인터페이스와, EHR 시스템에서 직접 Teams의 다른 공급자와 가상 환자 방문 또는 상담을 쉽게 시작하게 하는 커넥터를 업데이트했습니다. [자세한 정보](https://docs.microsoft.com/microsoftteams/expand-teams-across-your-org/healthcare/ehr-admin)를 확인합니다.

## <a name="january-21-2021"></a>2021년 1월 21일

### <a name="offline-documentation"></a>오프라인 설명서

오프라인 상태이거나 대역폭이 낮은 조건에서 실행되는 경우에도 Teams가 계속 실행됩니다. Teams는 기존 채팅을 위해 보내지 않은 모든 메시지를 저장하고(최대 24시간 동안) 다시 온라인이 되자마자 해당 메시지를 전송합니다. [자세한 정보](https://docs.microsoft.com/MicrosoftTeams/connectivity-issues)를 확인합니다.

## <a name="january-19-2021"></a>2021년 1월 19일

### <a name="teams-approvals-app-availability"></a>Teams 승인 앱 가용성

승인 앱은 모든 Microsoft Teams 사용자의 개인 앱으로 사용할 수 있습니다. 승인 앱은 Teams에서 구조적 및 비구조적 승인에 감사, 규정 준수, 책임 및 워크플로를 가져오는 간단한 방법을 제공합니다. [자세한 정보](https://docs.microsoft.com/MicrosoftTeams/approval-admin)를 확인합니다.

## <a name="december-18-2020"></a>2020년 12월 18일

### <a name="updated-meeting-settings-policies-for-breakout-rooms"></a>회의실에 대한 모임 설정 정책 업데이트

 Teams의 소규모 회의실은 모임을 별도의 더 작은 회의실로 분할하는 데 사용됩니다. 이 설정은 모임 이끌이가 예약하거나 시작하는 모임에서 회의실을 만들고 관리할 수 있는 옵션이 있는지 여부를 제어합니다. 이 정책 설정은 모임 시작을 포함하여 모든 모임에 영향을 미치고 있습니다. [자세한 정보](https://docs.microsoft.com/microsoftteams/meeting-policies-in-teams#meeting-policy-settings---allow-breakout-rooms)를 확인합니다.

## <a name="december-9-2020"></a>2020년 12월 9일

### <a name="advisor-for-teams-with-microsoft-365-learning-pathways"></a>Microsoft 365 학습 경로를 통해 Teams용 Advisor

Microsoft 365 학습 경로는 사용자를 교육하도록 사용자 지정할 수 있는 주문형 학습 솔루션으로, 조직 내 Teams 사용 및 채택률을 높일 수 있습니다. Teams용 어드바이저와 학습 경로를 사용하여 사용자를 신속하게 시작하고 채택을 유도할 수 있습니다.

학습 경로를 통해 SharePoint Online 사이트 서식 파일과 사용자를 위한 학습 사이트를 쉽게 구축할 수 있습니다. 학습 경로 교육 포털을 사용자 정의하여 사용자의 필요에 맞는 교육 및 지원 콘텐츠를 포함할 수 있습니다. Microsoft 온라인 카탈로그의 Teams 재생 목록을 사용하고 재생 목록을 추가할 수 있습니다. [자세한 정보](https://docs.microsoft.com/microsoftteams/use-advisor-teams-roll-out#advisor-for-teams-and-microsoft-365-learning-pathways)를 확인합니다.

[자세한 정보](https://docs.microsoft.com/microsoftteams/expand-teams-across-your-org/shifts/manage-shift-based-access-flw)를 확인합니다.

### <a name="download-a-participant-report-in-a-teams-meeting"></a>Teams 모임에서 참가자 보고서 다운로드

모임 이끌이, 특히 교사는 Teams 모임에 누가 참가한지 알고 있는 경우가 종종 있습니다. 곧 참가자의 참가 및 휴가 시간을 포함하는 참석자 보기에 있는 참가자 보고서를 다운로드할 수 있습니다. 모임 이끌이는 참석자 보기에서 사용할 수 있습니다. 이끌이가 있는 동안 참가한 사용자가 포함된 보고서를 다운로드할 수 있습니다. 이 기능은 모임이 활성화된 동안에만 모임 내에서 사용할 수 있습니다. 데스크톱(Windows 및 Mac) 및 웹에서 사용할 수 있습니다. [자세한 정보](../teams-analytics-and-reports/meeting-attendance-report.md)를 확인합니다.
