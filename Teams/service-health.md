<a name="verify-service-health-for-microsoft-teams"></a>Microsoft 팀의 서비스 상태 확인
===========================================

Microsoft 팀의 서비스 상태는 Office 365 관리 포털 기본 페이지에 표시 됩니다. 문제를 해결 하기 전에 팀 서비스가 정상 인지 확인 하는 것이 좋습니다.

또한 Microsoft 팀은 추가 Office 365 서비스를 기반으로 구축 되므로 서비스 상태를 확인할 때 Exchange, SharePoint, 비즈니스용 OneDrive의 상태도 확인 해야 한다는 점에 유의 하세요. 이러한 다른 서비스에 대 한 서비스 상태 문제는 팀에 영향을 주는 것 (예: Exchange에서 주소록 다운로드를 사용할 수 없음)을 자동으로 의미 하지 않지만 영향을 받는 서비스에 대 한 권고를 검토 하 여 영향을 받는지 확인 해야 합니다. Microsoft 팀.
