# <a name="microsoft-teams-add-on-licensing"></a>Microsoft 팀 추가 기능 라이선스

## <a name="what-are-add-on-licenses"></a>추가 기능 라이선스는 무엇 인가요?

추가 기능 라이선스는 특정 Microsoft 팀 기능에 대 한 라이선스입니다. 일부 기업에서는 경쟁력 있는 가격으로 특정 기능만 구입 하는 유연성이 필요 합니다. 기능을 추가 하려면 사용할 각 사용자에 대해 하나의 추가 기능 라이선스를 구입 합니다.

Microsoft 팀 라이선스는 최대한 많은 유연성을 제공 하도록 디자인 되었습니다. 예를 들어, while에 기본 팀 기능을 사용한 후에는 오디오 회의 또는 Office 365의 전화 시스템 등 더 많은 기능을 사용할 준비가 되 면 추가 기능 라이선스를 구입할 수 있습니다.

## <a name="how-does-add-on-licensing-affect-cost"></a>추가 기능 라이선스의 비용에 미치는 영향

대부분의 비즈니스에서 엔터프라이즈 계획에 소프트웨어 번들을 구입 하면 전체적인 비용을 줄일 수 있습니다. 일반적으로, 계획의 일부가 아닌 여러 추가 기능 라이선스를 개별적으로 구입 하는 경우 비용을 합산 하는 것이 더 높습니다.

자세한 비용 정보는 다음 중 하나를 참조 하세요.

- [오디오 회의에 대 한 가격](https://products.office.com/microsoft-teams/online-meeting-solutions)

- [전화 시스템 및 통화 요금제의 가격 책정](https://products.office.com/microsoft-teams/voice-calling)

## <a name="how-to-see-prices-and-buy-add-on-licenses"></a>가격을 확인 하 고 추가 기능 라이선스를 구입 하는 방법
<a name="bkmk_how"> </a>

 **Office 365 계획을 구입한 후에는 관리 센터를 통해 가격을 표시 하 고 추가**기능을 구입 합니다.

원하는 추가 기능에 따라 이러한 기능을 포함 하는 계획으로 전환 하는 [것이 더 저렴 한 비용을 비교](https://go.microsoft.com/fwlink/?linkid=844053) 하는 것이 좋습니다.

구매할 라이선스와 가격을 확인 하 고 추가 기능 라이선스를 구입 하는 방법에 대 한 자세한 내용은 "요금제에 따른 라이선스 옵션"에서 요금제를 확인 하세요.
