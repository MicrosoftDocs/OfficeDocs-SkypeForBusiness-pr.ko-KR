# <a name="office-365-enterprise-e5-without-audio-conferencing"></a>Office 365 Enterprise E5(오디오 회의 제외)

Office 365 Enterprise E5 (오디오 회의 없음) 계획을 이미 구매한 경우, 추가 기능을 얻으려면 구입 해야 하는 Microsoft 팀 추가 기능 라이선스가 있습니다.

|이 기능을 추가 하려면|수행 해야 하는 작업은 다음과 같습니다. |
|:------------------|:--------------------------|
|오디오 회의 <br/> | 이 추가 기능을 얻으려면 다음을 수행 합니다. <ol><li>E5 계획을 이미 구매한 경우 해당 국가나 지역에서 오디오 회의 기능을 사용할 수 없는 경우에는 타사 오디오 회의 공급자를 사용 해야 합니다. [Microsoft](https://go.microsoft.com/fwlink/?LinkId=797530)에서 타사 오디오 회의 공급자를 찾습니다.  <br/>국가 또는 지역에서 오디오 회의를 사용할 수 있는지 확인 하려면이 문서를 참조 하세요. [오디오 회의 및 통화 요금제의 국가 및 지역 사용 가능 여부](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) </li><li/>사용할 수 있는 경우 [요금제 전환 단추](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a) 를 사용 하 여 오디오 회의를 포함 하는 Office 365 Enterprise E5 요금제로 이동 합니다. 더 많은 기능을 이용할 수 있습니다. </li></ol> |
|전화 시스템이 포함 되어 있습니다! <br/> | Office 365에 대 한 [통화 요금제](../calling-plans-for-office-365.md) 를 구입 합니다. <br/>  또는 [기존 공급자의 전화 서비스](microsoft-teams-add-on-licensing.md#bkmk_existing)를 사용 합니다.  <br/> |
|클라우드 보이스 메일이 포함 되었습니다! <br/> |다른 작업을 수행 하지 않아도 됩니다.  <br/> |
