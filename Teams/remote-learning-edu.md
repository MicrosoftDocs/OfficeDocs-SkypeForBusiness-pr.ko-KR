# <a name="get-started-with-microsoft-teams-for-remote-learning"></a>Microsoft 팀의 원격 학습 시작 하기

Microsoft 팀은 대화, 콘텐츠, 과제, 앱을 한 곳에 모아 두는 플랫폼입니다. 단일 환경에서 공동 작업 강의실, 전문 학습 커뮤니티에 연결, 동료와의 연결을 구성 합니다.

이 문서의 모범 사례를 사용 하 여 교육에 대 한 Microsoft 팀의 사용을 시작 하 여 원격 학습 기능을 사용 하도록 설정 해야 합니다. Microsoft 팀은 대화에서 수업 공동 작업을 가능 하 게 하 고, 가상 회의 플랫폼을 제공 하 고, 과제를 배포 하는 데 사용 될 수 있습니다. 학교 관리자와 직원은 팀을 사용 하 여 최신 상태를 유지 하 고 알림 및 topical 대화를 할 수 있습니다. 강사는 전문 학습 커뮤니티를 사용 하 여 교육용 자료를 공유할 수 있습니다.

Microsoft 팀은 데스크톱 (Windows, Mac, Linux), 웹, 모바일 (Android 및 iOS)에 사용할 수 있는 [클라이언트](get-clients.md) 를 보유 하 여 모든 직원과 학생이 연결을 유지할 수 있도록 합니다.

[교육 웹 세미나 시리즈 팀](https://aka.ms/TeamsEDUWebinars)의 Microsoft 팀 사용 시나리오에 대해 자세히 알아보세요.

## <a name="user-accounts-licenses-and-identity-security"></a>사용자 계정, 라이선스 및 id 보안

Microsoft 팀은 Microsoft 365 기능을 활용 하 여 사용자를 인증 하 고 서비스를 제공 합니다. 공동 작업을 쉽게 할 수 있도록 직원, 강사, 학생 들의 id가 설정 되어 있어야 합니다. Id가 아직 없는 경우이 프로세스를 따라 설정 합니다.

팀 접근 권한 값을 사용 하기 전에 [사용자가 팀 라이선스를 사용할 수 있도록 설정 해야](user-access.md) 합니다. 팀은 [Office 365 그룹](Office-365-groups.md), [Exchange](Exchange-Teams-interact.md), [SharePoint, OneDrive](SharePoint-OneDrive-interact.md) 와 같은 추가 Microsoft 365 기능을 사용 하 여 공동 작업 시나리오를 지원 합니다. 이러한 서비스를 모두 사용 하는 경우 사용자에 게 최상의 팀 경험을 받으실 수 있습니다. [Google에서 호스트 하는 전자 메일을 사용 하는 사용자는 팀이 지원 됩니다](https://docs.microsoft.com/microsoft-365/education/deploy/enabling-teams-for-education-for-google-users).

## <a name="easily-set-up-microsoft-teams"></a>간편 하 게 Microsoft 팀 설정

다음은 팀을 시작 하 고 실행 하기 위해 수행 해야 하는 두 가지 사항입니다.

### <a name="1-allow-users-to-create-teams"></a>1. 사용자가 팀을 만들 수 있도록 허용

학생 및 강사는 최소 장벽으로 팀을 사용할 수 있고, 요구 사항에 맞게 사용자를 조정 하는 유연성을 제공 합니다. 사용자가 팀 환경을 조정할 수 있는 한 가지 방법은 해당 요구에 맞는 팀을 만드는 기능을 제공 하는 것입니다. **기본적으로 모든 사용자가 Office 365 그룹 및 팀을 만들 수 있습니다**. 이 접근 권한 값이 적절 하지 않은 경우가 있습니다. 예를 들어 일부 고객은 기본 보조 학생이 팀을 만들지 못하도록 제한 하려고 할 수 있습니다. 필요한 경우 Office 365 그룹 및 팀 만들기를 해당 환경 내의 [특정 보안 그룹으로 제한할](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups) 수 있습니다.

높은 교육 고객은 학생을 포함 하 여 모든 사용자가 수업, 리서치, 그룹 프로젝트 및 연구 그룹을 위한 팀을 만들 때 혜택을 누릴 수 있습니다. 기본-보조 학교는 학생 들에 게 모든 학생이 모든 학생 들이 성인으로 포함 되어 있는지 확인 하기 위해 팀을 만드는 것을 제한 해야 할 수 있습니다. 이 경우 Office 365 그룹 및 팀 만들기를 모든 교육자 및 교직원에 게 제한할 수 있습니다.

### <a name="2-configure-user-experiences-using-policies"></a>2. 정책을 사용 하 여 사용자 환경 구성

[팀 정책은](teams-policies.md) 특정 사용자 또는 사용자 그룹에 대해 제공 되는 옵션을 제어 하는 기능을 제공 합니다. 개인 채팅, 비공개 통화, 모임 예약, 공유할 수 있는 콘텐츠 형식 등을 사용할 수 있는 사용자를 정의 하기 위해 정책을 적용할 수 있습니다.

**높은 교육 직원, 교육자 및 학생 들은** 기본 (전역) 정책에 포함 된 기능을 통해 혜택을 누릴 수가 있습니다. 일부 추가 정책 설정을 사용 하면 [메시징 정책에서 번역 기능을 사용](https://docs.microsoft.com/microsoftteams/messaging-policies-in-teams#messaging-policy-settings) 하도록 설정 하 고 모임 정책에서 자동 모임 기록을 허용 하는 등 Microsoft 팀에 더 많은 기능을 추가할 수 있습니다.

**기본-보조 학교 학생이** 학생에 게 제공 되는 제한 된 접근 권한 값이 필요할 수 있습니다. 정책은 학생이 수행할 수 있는 작업에 대 한 경계를 설정 합니다. 학생 집단은 일반적으로 가장 큰 사용자 집합 이므로 가장 제한적인 설정을 받을 수 있으므로 학생 정책이 ' 전역 (조직 차원의 기본) ' 정책을 변경 하는 것이 좋습니다.

다음은 학생 간의 unmoderated 통신을 제한 하기 위해 기본 보조 학생에 게 할당 되는 일반적인 비 기본 정책 구성 집합입니다.

#### <a name="messaging-policy"></a>메시지 정책

- ' 끄기 '로 설정 변경
- Giphy 콘텐츠 등급이 ' strict '으로 설정 되어 있습니다.
- ' 설정 '으로 설정한 메시지 번역
- 우선 순위 알림을 사용 하 여 긴급 메시지 보내기 ' 해제 '로 설정
- 그룹 채팅에서 사용자 제거를 ' 끄기 '로 설정

#### <a name="meeting-policy"></a>모임 정책

- Outlook 추가 기능을 ' 해제 '으로 설정할 수 있도록 허용
- 채널 모임 예약 허용을 ' 해제 '로 설정
- 비공개 모임의 일정을 ' 해제 '으로 설정할 수 있습니다.
- 비공개 모임에서 모임 시작 허용을 ' 해제 '로 설정

#### <a name="live-events-policy"></a>라이브 이벤트 정책

- 일정 설정 허용 ' 해제 '

#### <a name="calling-policy"></a>통화 정책

- 비공개 통화를 ' 꺼짐 '으로 설정

#### <a name="teams-policy"></a>팀 정책

- 개인 채널 만들기가 ' 해제 '로 설정

**주 학교 직원과** 강사는 학생을 제한할 수 있는 핵심 기능을 부여 하는 정책을 할당 받아야 합니다. 개인 채팅 및 모임 예약을 허용 하는 새 정책을 만듭니다 (새 정책의 기본 설정). [보안 그룹 구성원 자격을 통해 교직원 및 교육자에 이러한 정책을 할당](https://docs.microsoft.com/microsoftteams/assign-policies#assign-a-policy-to-a-group)합니다.

## <a name="start-using-teams"></a>팀 사용 시작

### <a name="create-class-teams-for-secure-classroom-use"></a>보안 강의실 사용을 위한 수업 팀 만들기

교육을 위한 Microsoft 팀은 교육용으로 [특정 팀 유형을](https://support.office.com/article/choose-a-team-type-to-collaborate-in-microsoft-teams-0a971053-d640-4555-9fd7-f785c2b99e67) 제공 합니다. [수업 팀 유형은](https://support.office.com/article/create-a-class-team-in-microsoft-teams-fae422eb-58b7-4431-9ff2-a4b9b6ae7c5b) 과제, OneNote 교실 수첩, 학생을 위한 읽기 전용 콘텐츠를 보호 하는 [수업 자료 폴더](https://support.office.com/article/Use-folders-to-create-read-only-files-for-students-or-other-team-members-0e7791d7-8c9c-4749-9bca-984289477988) , 그리고 학생 들의 음소거 기능을 비롯 한 특정 기능을 사용 하 여 강의실으로 설계 되었습니다. 수업 팀을 배포할 수 있는 몇 가지 방법이 있습니다.

1. 이를 사용 하 여 school [데이터 동기화](https://sds.microsoft.com/) (SDS)를 **설정**하 여 학교 정보 시스템의 정보에 따라 모든 클래스에 대해 수업 팀을 만들 수 있습니다. 이 프로세스는 각 섹션에 대 한 팀을 프로 비전 하 고 강사와 학생 rosters 동기화 상태를 유지 합니다. 교육자는 학생을 admitting 하기 전에 [팀을 준비할 수 있습니다](https://support.office.com/article/activate-early-access-class-teams-created-with-school-data-sync-0d154696-66ab-4fcf-b22f-c3d9a82aaf78) . 또는 강사에서 팀을 사용 하지 않는 경우 강사는 ' 활성화 '를 클릭 하지 않기 때문에 학생 들이 팀에 게 참석할 수 없습니다. SDS는 데이터 가져오기에 대 한 80 가지 이상의 SIS (School Information Systems)를 지원 하 고, [sds 지원 팀](https://aka.ms/SDSSupport) 은 계획 및 구성에 대 한 지원을 제공할 준비가 되었습니다.
1. **교육자** 는 고유한 수업 유형 팀을 설정 하 고 학생을 초대 합니다. 교사 [는 팀에 학생을 추가](https://support.office.com/article/add-a-student-to-a-class-team-b88263bb-ace1-4702-8a48-f8a2cf4af954)하거나, [참가 코드를 공유](https://support.office.com/article/Create-a-link-or-a-code-for-joining-a-team-11b0de3b-9288-4cb4-bc49-795e7028296f)하거나, [팀에 대 한 링크를 공유](https://support.office.com/article/Create-a-link-or-a-code-for-joining-a-team-11b0de3b-9288-4cb4-bc49-795e7028296f)하 여이 작업을 수행할 수 있습니다. 가능 하다 면 교사 들이 팀에 학생을 추가 하 여 학생 들에 게 액세스 하 고 팀에 추가 되었다는 알림을 받는 것이 가장 좋습니다.

팀을 설정한 후에 팀 소유자 [는 팀](https://support.office.com/article/find-your-class-team-s-settings-in-microsoft-teams-2592d4de-581d-4952-9028-02317880c158) [사진](https://support.office.com/article/change-your-team-picture-02ea2af6-b49d-4de8-9551-1a5e472993c0)추가, 수업 주제 또는 그룹 공동 작업 영역에 대 한 [채널 만들기](https://support.office.com/article/create-student-project-groups-channels-in-microsoft-teams-f85b3c07-fb87-4b94-883b-9be55f4b1e45?ui=en-US&rs=en-US&ad=US) , Quizlet/Flipgrid/하세요 등의 [앱을 추가](https://support.office.com/article/add-an-app-to-teams-b2217706-f7ed-4e64-8e96-c413afd02f77) 하 여 기존 교육 콘텐츠를 진행 하 고, 모든 사용자에 게 알리고 대화를 시작 하는 [첫 번째 게시물에 대 한 팀을 멘 션](https://support.office.com/article/using-the-conversation-tab-in-microsoft-teams-53d1c530-3797-4a6f-9892-6760f8763df2) 합니다.

### <a name="districtuniversity-events-or-updates"></a>학구/대학 이벤트 또는 업데이트

일부 명령에는 더 많은 대상 그룹과 추가 생산 능력이 필요 합니다. 이러한 모임에는 주로 발표자, 제작자, 중재 하는 Q&A가 정의 되어 있습니다. Microsoft 팀은 [Microsoft 팀의 라이브 이벤트](teams-live-events/what-are-teams-live-events.md)를 사용 하 여 이러한 세션을 지원 합니다. 라이브 이벤트는 학구 또는 대학 전체 업데이트, 리더십 주소, 대형 수업 또는 학생 그룹에 대 한 지침 또는 커뮤니티에 확장 하는 것과 같은 시나리오에 사용할 수 있습니다. 라이브 세션 수행에 대 한 자세한 내용은 [라이브 이벤트 계획 및 예약](https://support.office.com/article/video-plan-and-schedule-a-live-event-f92363a0-6d98-46d2-bdd9-f2248075e502), 라이브 이벤트 [생성](https://support.office.com/article/video-produce-a-live-event-34c89e79-ffd4-4a6a-baf6-77055e0709cb), [라이브 이벤트에 참가](https://support.office.com/article/video-attend-a-live-event-d837ad8d-ce34-44d0-9744-9beb50e943ac)중재 하기, [질문&a를](https://support.office.com/article/video-moderating-a-q-a-4984e582-8c66-4ea3-aaaf-d93cf62e1b76)참고 하세요.

## <a name="recommended-tips--tricks"></a>추천 팁 & 요령

Microsoft 팀을 교육에서 사용 하는 방법에 대 한 자세한 내용은 [Microsoft 팀의 교육](https://support.office.com/article/Teams-5aa4431a-8a3c-4aa5-87a6-b6401abea114)기관용을 참고 하세요.
