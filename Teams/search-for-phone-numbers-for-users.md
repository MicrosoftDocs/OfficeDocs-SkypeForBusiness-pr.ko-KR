# <a name="search-for-phone-numbers-for-users"></a>사용자의 전화 번호 검색

전화를 걸고 받을 조직의 사용자를 설정 하는 경우 **Microsoft 팀 관리 센터** 를 사용 하 고 먼저 사용자에 게 할당할 수 있는 전화 번호를 받아야 합니다. 사용자에 게 할당 하는 전화 번호는 조직에 대해 선택한 전화 번호 이며, 사용자의 속성을 편집 하 고 **할당**을 클릭할 때 드롭다운 목록에 나열 됩니다.
  
사용자에 게 전화 번호를 지정 하려면 먼저 **새 번호 가져오기** 페이지를 사용 하 여 영역 내에서 사용할 수 있는 번호를 검색 해야 합니다. **국가 또는 지역**, **번호 형식**, **위치로**검색할 수 있으며, 사용자에 게 필요한 전화 번호 수를 입력 합니다. 
  
전화 번호를 확인 하는 데 도움이 필요한 경우 [조직의 전화 번호 관리](/microsoftteams/manage-phone-numbers-for-your-organization) 또는 [비즈니스 제품에 대 한 고객 지원 문의-관리자 도움말](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b) 을 참조 하세요.
  
## <a name="search-for-phone-numbers"></a>전화 번호 검색

**사용자의 전화 번호를 검색 하려면**

![](media/teams-logo-30x30.png) **Microsoft 팀 관리 센터를 사용 하 여** 팀 로고를 표시 하는 아이콘
  
1. **Microsoft 팀 관리 센터로**이동 합니다.

2. 왼쪽 탐색 창에서 **음성** > **전화 번호** > 를 선택 하 여**새 번호를 가져옵니다**.
  
    > [!IMPORTANT]
    > Microsoft 팀 관리 센터의 왼쪽 탐색에서 **음성** 옵션을 보려면 먼저 하나 이상의 **Enterprise E5 라이선스**, 하나의 **전화 시스템** 추가 기능 라이선스 또는 하나의 **오디오 회의** 추가 기능 라이선스를 구입 해야 합니다.  

3. **위치 및 수량 선택** 페이지의 **국가 또는 지역** 드롭다운 목록에서 위치를 선택 합니다.

4. **숫자 형식** 드롭다운 목록에서 숫자 형식을 선택 합니다.

5. **위치** 상자에 사용자가 있는 도시의 이름을 입력 한 다음 목록에서 위치를 선택 합니다. 원하는 위치가 목록에 표시 되지 않는 경우 **위치 추가를** 클릭 합니다.

6. 위치에 해당 하는 지역 번호를 선택 합니다.

7. **수량**에서 조직에 사용할 전화 번호 수를 입력 하 고 **다음**을 클릭 합니다. 전화 번호를 선택 하는 데 10 분이 걸립니다. 10 분 이상 소요 되는 경우 숫자가 전화 번호 풀로 반환 됩니다.

    > [!NOTE]
    > 사용할 수 있는 전화 번호의 수 (라이선스 수에 따라)는 **수량**옆에 표시 되어 있습니다. 
  
8. **번호 가져오기** 페이지에서 원하는 전화 번호를 선택 하 고 **번호 받기를**클릭 한 후 **다음**을 클릭 합니다.

    > [!IMPORTANT]
    > 라이선스를 보유 한 것 보다 더 많은 전화 번호를 얻을 수 있습니다. 가져올 수 있는 전화 번호 수를 확인 하려면 라이선스 수를 취하고 라이선스 수의 10%를 추가한 다음 10을 더 합니다. 예를 들어 100 **국내 통화 요금제** 및/또는 **국제 통화 요금제** 라이선스가 있는 경우 해당 100 사용자에 대 한 일부 전화 번호를 아직 취득 하지 않은 것으로 가정 하 여 120 전화 번호를 예약할 수 있습니다. 자세한 내용은 [받을 수 있는 전화 번호 수](/microsoftteams/how-many-phone-numbers-can-you-get) 를 확인 하세요.

9. **확인** 페이지에서 선택 사항을 확인 한 다음 **주문을**클릭 합니다.

10. **전화 번호** 페이지로 돌아가면 할당할 전화 번호 또는 번호를 선택한 다음 **편집** 을 클릭 하 여 사용자에 게 할당 합니다.
