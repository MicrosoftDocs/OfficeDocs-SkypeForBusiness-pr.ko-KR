## <a name="definitions"></a>용어

팀 보고서에는 활성 사용자 및 활성 채널에 대 한 데이터가 표시 됩니다. 예를 들어 조직의 사용자가 보고서에 대해 지정한 날짜 범위 중 팀에서 활성화 되어 있지 않은 경우 해당 사용자의 데이터는 해당 보고서에 포함 되지 않습니다.

|항목  |고해상도  |
|---------|---------|
|활성 사용자     |지정 된 날짜 범위 동안 팀에서 작업을 수행 하는 고유 사용자의 수를 측정 합니다.    |
|활성 채널    |지정 된 날짜 범위 동안 사용자가 작업을 수행 하는 팀의 채널 수를 측정 합니다.           |
