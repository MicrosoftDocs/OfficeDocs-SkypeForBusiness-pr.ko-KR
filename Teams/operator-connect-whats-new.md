# <a name="whats-new-for-operator-connect-operators"></a>연산자 연결 연산자의 새로운 기능

이 문서에서는 Operator Connect의 새로운 기능에 대해 설명합니다. 업데이트를 자주 확인합니다.

## <a name="dr-to-oc-migration-api"></a>DR에서 OC로 마이그레이션 API

2022년 7월 1일에 Microsoft는 직접 라우팅에서 운영자 연결로 숫자를 마이그레이션할 수 있는 운영자용 API를 게시했습니다. 

자세한 내용은 [Operator Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#tag/TnMigration-(New))를 참조하세요.

## <a name="number-status"></a>숫자 상태

2022년 7월 1일에 Microsoft는 동의한 테넌트에 대해 지정된 숫자에 대한 운영자 상태 정보를 제공하는 데 사용되는 연산자용 API를 게시했습니다.

자세한 내용은 [Operator Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#operation/NumberStatus_GetNumberStatus)를 참조하세요.
