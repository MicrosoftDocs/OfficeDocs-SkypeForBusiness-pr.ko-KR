# <a name="edit-guest-information"></a>게스트 정보 편집

웹 사이트 또는 Microsoft 365 관리 센터 포털에서 게스트 정보를 Azure Active Directory 있습니다.

## <a name="guest-users-in-the-microsoft-365-admin"></a>관리자 관리자의 Microsoft 365 사용자

1. 에서 게스트 정보를 [편집하려면](https://admin.microsoft.com)Microsoft 365 관리 센터 **게스트 사용자를**  >  **선택합니다.**

   ![편집할 게스트 사용자 정보입니다.](media/access-guest-user.png)

2. 게스트 사용자에서 **사용자를 선택합니다.**

3. 게스트 사용자 정보에서 연락처 **정보 관리를 선택합니다.**

   ![연락처 정보를 관리합니다.](media/guest-user-data1.png)

4. 연락처 정보 관리에서 표시  이름을 제외한 선택한 필드를 편집한 다음 **변경** 내용 **저장을 선택합니다.**

   ![게스트 사용자 연락처 정보를 편집합니다.](media/manage-guest-contact.png)

또한 포털 에서 [게스트를 편집할 Azure Active Directory 있습니다.](https://aad.portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/MsGraphUsers)
