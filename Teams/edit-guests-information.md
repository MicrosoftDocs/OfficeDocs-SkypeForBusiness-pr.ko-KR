<a name="edit-guest-user-information"></a>게스트 사용자 정보 편집
===========================

게스트 계정 (예: 표시 이름 또는 프로필 사진)을 편집 하려면 Azure Active Directory 포털로 이동 합니다. 자세한 내용은 [Office 365 id 및 Azure Active Directory 이해](https://support.office.com/article/Understanding-Office-365-Identity-and-Azure-Active-Directory-06a189e7-5ec6-4af2-94bf-a22ea225a7a9)를 참조 하세요.

현재 Microsoft 팀 관리 센터, Microsoft 365 관리 센터 또는 Exchange 관리 센터에서 게스트 정보를 편집할 수 없습니다.
