# <a name="dimensions-and-measures-available-in-call-quality-dashboard"></a>통화 품질 대시보드에서 사용할 수 있는 크기 및 측정값

Microsoft 팀 및 비즈니스용 Skype Online에 대 한 CQD (통화 품질 대시보드)는 Microsoft 팀과 비즈니스용 Skype 서비스를 사용 하 여 만든 통화 품질에 대 한 통찰력을 얻을 수 있도록 합니다. 이 항목에서는 CQD에 의해 노출 되는 차원과 측정값에 대 한 자세한 정보를 제공 합니다. CQD 및이를 사용 하는 방법에 대 한 자세한 내용은 [Microsoft 팀과 비즈니스용 Skype Online에 대 한 통화 품질 대시보드 켜기 및 사용](turning-on-and-using-call-quality-dashboard.md)을 참조 하세요.



## <a name="first-and-second-endpoint-classification"></a>첫 번째 및 두 번째 끝점 분류

CQD (통화 품질 대시보드)에 있는 대부분의 차원과 측정값에는 첫 번째 또는 두 번째 레이블이 지정 되어 있습니다. 다음 논리는 스트림이나 통화와 관련 된 끝점에 첫 번째 레이블을 지정 합니다.

- 서버가 스트림/통화와 관련 된 경우 첫 번째는 항상 서버 끝점 (AV MCU, 중재 서버 등)입니다.

- 두 서버 끝점 사이에 스트림이 없으면 항상 클라이언트 끝점이 됩니다.

- 두 끝점의 유형이 같은 경우 첫 번째와 두 번째의 순서는 사용자 에이전트 범주의 내부 순서 지정을 기준으로 합니다. 이렇게 하면 순서가 일치 합니다.
