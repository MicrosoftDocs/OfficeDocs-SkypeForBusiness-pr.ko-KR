# <a name="manage-messaging-policies-in-teams"></a>Teams에서 메시지 정책 관리

<!--- Add zone marker here--->

메시징 정책은 Microsoft Teams에서 [사용자(소유자 및 구성원)](assign-roles-permissions.md)가 사용할 수 있는 채팅 및 채널 메시징 기능을 제어하는 데 사용됩니다. 자동으로 생성되는 전역(조직 전체 기본값) 정책을 사용하거나 사용자 지정 메시징 정책을 만들고 할당할 수 있습니다.

사용자 지정 정책을 만들고 할당하지 않는 한 조직의 사용자는 자동으로 전역 정책을 받습니다. 전역 정책에서 설정을 편집하거나 하나 이상의 사용자 지정 정책을 만들고 할당하여 원하는 기능을 켜거나 끕니다.

> [!NOTE]
> 정책 변경 후 동기화를 보장하려면 특정 인스턴스에 대해 다시 부팅이 필요할 수 있습니다. 

## <a name="create-a-custom-messaging-policy"></a>사용자 지정 메시징 정책 만들기

1. Microsoft Teams 관리 센터의 왼쪽 탐색 영역에서 **메시징 정책** 으로 이동합니다.
2. **추가** 를 선택합니다.
3. 정책의 이름과 설명을 입력합니다.
4. 원하는 설정을 선택합니다.
5. **저장** 을 선택합니다.

예를 들어 보낸 메시지가 삭제되거나 변경되지 않았는지 확인하려고 합니다. "보낸 메시지 보존"이라는 새 사용자 지정 정책을 만들고 다음 설정을 끕니다.

- 소유자는 보낸 메시지를 삭제할 수 있습니다.
- 사용자는 보낸 메시지를 삭제할 수 있습니다.
- 사용자는 보낸 메시지를 편집할 수 있습니다.

그런 다음, 사용자에게 정책을 할당합니다.

## <a name="edit-a-messaging-policy"></a>메시징 정책 편집

전역 정책 및 직접 만든 사용자 지정 정책을 편집할 수 있습니다.

1. Microsoft Teams 관리 센터의 왼쪽 탐색 영역에서 **메시징 정책** 으로 이동합니다.
2. 정책 이름 왼쪽을 클릭하여 정책을 선택한 다음 **편집** 을 선택합니다.
3. 여기서 원하는 대로 변경합니다.
4. **저장** 을 선택합니다.

## <a name="assign-a-custom-messaging-policy-to-users"></a>사용자에게 사용자 지정 메시징 정책 할당

[!INCLUDE [assign-policy](includes/assign-policy.md)]

사용자는 한 번에 하나의 메시징 정책만 할당할 수 있습니다.

> [!NOTE]
> 사용자에게 할당된 정책을 삭제할 수 없습니다. 먼저 영향을 받는 모든 사용자에게 다른 정책을 할당한 후에 처음의 정책을 삭제할 수 있습니다.

## <a name="messaging-policy-settings"></a>메시징 정책 설정

구성할 수 있는 메시징 정책 설정은 다음과 같습니다.

- **소유자는 보낸 메시지를 삭제할 수 있습니다**  .  이 설정을 사용하여 소유자가 사용자가 보낸 채널 메시지 또는 게시물을 삭제할 수 있습니다.
- **보낸 메시지 삭제** 이 설정을 사용하여 사용자가 채팅에서 보낸 개별 메시지를 삭제할 수 있습니다.
- **채팅 삭제** 사용자가 채팅 목록에서 전체 대화를 삭제할 수 있도록 하려면 이 설정을 사용합니다. 이렇게 하면 작업을 수행하는 사용자의 채팅 콘텐츠만 삭제됩니다. 채팅의 다른 사용자는 여전히 모든 메시지를 볼 수 있습니다. 채팅을 삭제한 사용자는 활동 피드에 기존 메시지가 표시되지 않거나 삭제된 채팅에 대한 검색 결과가 표시되지 않습니다. 이 사용자는 삭제된 채팅에서도 제거됩니다. 사용자에게 채팅 삭제 결정을 확인하라는 메시지가 표시됩니다. 이 설정은 기본적으로 설정됩니다. 이 설정이 꺼져 있으면 채팅에 대한 삭제 옵션이 더 이상 표시되지 않습니다.
- **보낸 메시지 편집** 사용자가 채팅에서 보낸 메시지를 편집할 수 있도록 하려면 이 설정을 사용합니다.
- **읽음** 읽은 영수증을 사용하면 받는 사람이 1:1로 메시지를 읽었을 때 채팅 메시지의 보낸 사람에게 알림을 보내고 20명 이하의 그룹 채팅을 할 수 있습니다. 메시지 읽기 영수증은 메시지를 읽었는지 여부에 대한 불확실성을 제거하고 팀 커뮤니케이션을 개선합니다. 읽기 영수증은 eDiscovery 보고에서 캡처되지 않습니다.  
  - **사용자 제어** 즉, 사용자가 읽은 영수증을 켜거나 끌지 여부를 결정하게 됩니다. 앱 내의 기본 설정은 ON입니다. 그러면 사용자가 끄기를 할 수 있습니다.
  - **모든 사용자에 대해 켜짐** 즉, 테넌트 내의 모든 사용자는 기능을 해제할 수 있는 옵션 없이 ON 기능을 갖습니다. **모든 사용자에 대해 켜** 기 설정을 사용하는 경우 전체 테넌트에서 영수증을 설정하는 유일한 방법은 전체 테넌트("전역(조직 전체 기본값)"라는 기본 정책)에 대해 하나의 메시징 정책만 포함하거나 테넌트에서 모든 메시징 정책이 영수증에 동일한 설정을 사용하도록 하는 것입니다. 읽음 확인 기능은 **모든 사용자에 대해 사용** 으로 설정된 경우 가장 효과적입니다.
  - **모든 사용자에 대해 꺼짐** 즉, 기능이 비활성화되어 있고 테넌트에서 읽기 영수증을 가지고 있지 않으며 기능을 켤 수 없습니다.
<a name="bkchat"> </a>

- **채팅**  조직의 사용자가 Teams 앱을 사용하여 다른 사용자와 채팅할 수 있도록 하려면 이 설정을 켭니다.
- **그룹과 채팅** 사용자는 메일 그룹, 메일 사용 가능 보안 그룹 및 Microsoft 365 그룹으로 채팅을 시작할 수 있습니다.
- **대화에서 Giphy 사용**  Giphys를 켜면 사용자는 다른 사용자와의 채팅 대화에 Giphys를 포함할 수 있습니다. Giphy는 사용자가 애니메이션 GIF 파일을 검색하고 공유할 수 있는 온라인 데이터베이스 및 검색 엔진입니다. 각 Giphy에는 콘텐츠 등급이 할당됩니다. 이 설정을 켜는 것 외에도 대화에서 Giphys를 허용하도록 [선택적 연결된 환경을](/deployoffice/privacy/manage-privacy-controls#policy-setting-for-optional-connected-experiences) 사용하도록 설정해야 합니다.
- **Giphy 콘텐츠 등급**
  - **제한 없음** 즉, 사용자는 콘텐츠 등급에 관계없이 채팅에 Giphy를 삽입할 수 있습니다.
  - **온건한**  즉, 사용자가 채팅에 Giphys를 삽입할 수 있지만 성인 콘텐츠에서 적당히 제한됩니다.
  - **엄격한**  즉, 사용자가 채팅에 Giphys를 삽입할 수 있지만 성인 콘텐츠에서 엄격하게 제한됩니다.
- **대화의 밈** 밈을 켜면 사용자는 다른 사용자와의 채팅 대화에 밈을 포함할 수 있습니다.
- **대화의 스티커** 이 기능을 켜면 사용자가 다른 사용자와의 채팅 대화에 스티커를 포함할 수 있습니다.
- **URL 미리 보기** 이 설정을 사용하여 메시지에서 자동 URL 미리 보기를 켜거나 끕니다.
- **메시지 번역** 이 설정을 켜면 사용자가 Teams 메시지를 Microsoft 365 또는 Office 365 대한 개인 언어 설정으로 지정된 언어로 자동으로 번역할 수 있습니다.
- **메시지에 대한 몰입형 판독기** 사용자가 Microsoft 몰입형 리더 메시지를 볼 수 있도록 하려면 이 설정을 켭니다. 몰입형 리더 텍스트의 가독성을 높이기 위해 전체 화면 읽기 환경을 제공하는 학습 도구입니다.
- **우선 순위 알림을 사용하여 긴급 메시지 보내기** 이 기능을 켜면 사용자는 [우선 순위 알림을](https://support.microsoft.com/article/mark-a-message-as-important-or-urgent-in-teams-ea99d5b6-1317-4550-8d75-86ff14cd4462) 사용하여 메시지를 보낼 수 있습니다. 우선 순위 알림은 20분 동안 또는 *긴급* 으로 표시된 메시지가 수신자가 선택하고 읽을 때까지 2분마다 사용자에게 알립니다. 이 기능을 사용하면 메시지가 적시에 처리될 가능성이 높아집니다. 긴급 메시지를 보낸 후에는 편집할 수 없습니다.
- **음성 메시지 만들기**
  > [!Important]
  > 오디오 메시지는 eDiscovery 보고에서 캡처되지 않습니다.
  - **채팅 및 채널에서 허용됨** 즉, 사용자는 채팅과 채널 모두에서 오디오 메시지를 남길 수 있습니다.
  - **채팅에서만 허용됨** 즉, 사용자는 채팅에 오디오 메시지를 남길 수 있지만 채널에는 그대로 둘 수 없습니다.
  - **사용하도록 설정되지 않음** 즉, 사용자는 채팅 또는 채널에서 오디오 메시지를 만들 수 없습니다.  
- **모바일 디바이스에서 최근 채팅 위의 즐겨찾는 채널을 표시합니다.** 사용자가 스크롤하여 찾을 필요가 없도록 즐겨찾는 채널을 모바일 디바이스 화면의 맨 위로 이동하려면 이 설정을 사용하도록 설정합니다.
- **그룹 채팅에서 사용자 제거** 사용자가 그룹 채팅에서 다른 사용자를 제거할 수 있도록 하려면 이 설정을 켭니다. 이 기능을 사용하면 채팅 기록을 잃지 않고 소규모 그룹과 채팅을 계속할 수 있습니다.
- **텍스트 예측** 사용자가 채팅 메시지에 대한 텍스트 예측을 받을 수 있도록 하려면 이 설정을 켭니다.
- **추천 회신**  채팅 메시지에 대해 제안된 회신을 사용하도록 설정하려면 이 설정을 켭니다.
- **채팅 권한 역할** 이 설정을 사용하여 사용자의 감독된 채팅 역할을 정의합니다. [감독되는 채팅](supervise-chats-edu.md)에 대해 자세히 알아보세요.
- **전체 채팅 권한이 있는 사용자는 모든 메시지를 삭제할 수 있습니다.** 이 설정을 사용하여 모든 권한이 있는 사용자가 그룹 또는 모임 채팅 메시지를 삭제할 수 있습니다.
- **비디오 메시지** 조직의 사용자가 Teams 앱을 사용하여 채팅의 다른 사용자에게 비디오 메시지를 보낼 수 있도록 하려면 이 설정을 켭니다.

> [!NOTE]
> Giphys 사용과 같은 이러한 설정 중 일부는 팀 소유자가 팀 수준에서 구성하고 채널 소유자가 비공개 또는 공유 채널 수준에서 구성할 수도 있습니다.
