| 1  | 조직 Microsoft 365 Office 365 만들기| 조직에서 조직의 이점을 Teams 전에 조직 또는 Microsoft 365 Office 365 설정해야 합니다. 조직에 Microsoft 365 Office 365 없는 경우 추가 정보 열의 지침을 **참조** 하세요. | | [비즈니스용 Microsoft 365 Office 365 설정](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) <br/><br/>[추가 지원이 필요한 경우 Microsoft FastTrack Microsoft 365 Office 365 지원할 수 있습니다.](https://fasttrack.microsoft.com/office) |

| 4  | 선택 사항: 통신 크레딧 사용 | Communications Credits는 조직의 전화 회의 브리지 전화 번호에 대한 무료 액세스를 제공하거나 회의 이끌이에게 국제 모임 참가자에게 전화를 걸 수 있는 기능을 제공하는 데 사용하는 선택적 기능입니다. <br/><br/>사용자당 표준 오디오 회의 라이선스 외에도 볼륨 및 라이선스 조직은 오디오 회의 기능을 사용하도록 분당 유료 제안을 선택할 수 있습니다. <br/><br/>조직에 통신 크레딧이 필요한지 여부를 결정하고, 필요한 경우 테넌트에 Communications 크레딧 추가 기능 라이선스를 추가하여 사용하도록 설정합니다. | | [통신 크레딧이란?](what-are-communications-credits.md) <br/><br/>[조직에 사용할 통신 크레딧 설정](set-up-communications-credits-for-your-organization.md) <br/><br/>[오디오 회의 분당 요금](audio-conferencing-pay-per-minute.md) |

| 6  | ID에 대한 환경을 구성합니다. | 사용자를 직접 Microsoft 365 또는 Office 365(온라인 배포 모델에서)에서 만들거나, 프레미스 Active Directory에서 사용자 또는 조직으로 Microsoft 365 Office 365 있습니다. <br/><br/>클라우드 ID, 동기화된 ID 또는 페더리드 ID를 사용할지 여부를 결정합니다. 적절한 ID 형식을 결정하는 것은 이 검사 목록의 범위를 벗어날 수 있습니다. 그러나 추가 정보 열에서 이러한 옵션에 대한 정보에 대한 링크를 **찾을 수** 있습니다. <br/><br/>**참고:** 동기화된 ID 또는 페더링된 ID를 사용하는 경우 ONNS(프레미스 사용자 주체 이름)가 MICROSOFT 365 또는 Office 365 UPNS와 일치하고 모든 필수 특성이 Azure AD 커넥트. 온라인에 필요한 특성의 경우 Teams 온라인에 대한 특성 비즈니스용 Skype 사용하세요. | | [ID Microsoft 365 Office 365 및 Azure Active Directory](https://support.office.com/article/Understanding-Office-365-identity-and-Azure-Active-Directory-06a189e7-5ec6-4af2-94bf-a22ea225a7a9) <br/><br/>[디렉터리 동기화를 통해 사용자를 Microsoft 365 또는 Office 365](https://support.office.com/article/Prepare-to-provision-users-through-directory-synchronization-to-Office-365-01920974-9e6f-4331-a370-13aea4e82b3e) <br/><br/>[Azure AD 커넥트 동기화: 동기화된 특성 Azure Active Directory](/azure/active-directory/connect/active-directory-aadconnectsync-attributes-synchronized) |
| 7  | 테넌트 관리자 확인 | 보안 팀과 함께 관리 모델을 Microsoft 365 Office 365 수 있습니다. <br/><br/>모든 테넌트 및 서비스 관리자를 식별하고 문서화해야 합니다. | | [관리자 Microsoft 365 또는 Office 365 역할에 대해](https://support.office.com/article/About-Office-365-admin-roles-da585eea-f576-4f55-a1e0-87090b6aaa9d) |
