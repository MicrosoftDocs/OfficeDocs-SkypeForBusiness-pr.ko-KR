# <a name="support-remote-government-workers-using-microsoft-teams"></a>Microsoft Teams를 사용하여 정부 기관 원격 작업자 지원하기

정부 기관용 Office 365의 일부로 Microsoft Teams을 실행할 경우 이 지침을 사용하여 원격 직원을 지원하세요.


## <a name="it-guidance"></a>IT 지침

조직에서 Teams를 준비하는 방법에 대해 알아보세요.

  - 모든 사용자가 [Teams를 포함한 유료 구독](https://docs.microsoft.com/office365/servicedescriptions/teams-service-description)을 통해 Teams에 액세스할 수 있는지 확인합니다. 

  - 구독이 이미 있는 경우에는 모든 사용자가 **[Teams를 켜도록](../user-access.md)** 합니다.

  - 신속한 배포를 위해 [**Teams FastPath PowerPoint**](https://aka.ms/TeamsGovFastPath)를 방문하여 조직에 배포되는 Teams를 가져옵니다.

  - 조직에서 **[Teams에서 모임](https://docs.microsoft.com/MicrosoftTeams/tutorial-meetings-in-teams)** 을 시작하는 방법을 이해하세요.

## <a name="end-user-communication"></a>최종 사용자 커뮤니케이션

사용자에게 Teams를 배포할 준비가 되었나요? **[정부 기관용 고객 성공 키트](https://aka.ms/TeamsCSKGov)** 에서 이러한 멋진 전자 메일 템플릿을 확인할 수 있습니다.

  - **공****문** 및 **카운트다운** **템플릿**을 사용하여 재미를 느껴보세요.

  - 관련 전자 메일의 주간 접점 만들기:
    
      - **카운트다운 이메일**로 Teams 실행을 재촉하세요.
    
      - **공지 전자 메일**로 사용자에게 사용 가능함을 알리세요.
    
      - **팁과 트릭**으로 협업하는 새로운 방법을 사용자에게 보여 주세요.

## <a name="end-user-guidance"></a>최종 사용자 가이드

이러한 자산과 비디오를 공유하여 최종 사용자가 빠르게 시작할 수 있도록 돕습니다.

  - **[정부 기관용 빠른 시작 가이드](https://aka.ms/quickstartgov)** 를 다운로드하여 Teams의 기본 사항을 알아보세요.

  - Teams를 안내하는 **[짧은 빠른 시작 동영상](https://support.office.com/article/video-what-is-microsoft-teams-422bf3aa-9ae8-46f1-83a2-e65720e1a34d?wt.mc_id=otc_microsoft_teams)** 과 **[모임 동영상](https://support.office.com/article/join-a-teams-meeting-078e9868-f1aa-4414-8bb9-ee88e9236ee4)** 을 시청하고 Teams에서 모임을 활용하는 방법을 알아보세요.

## <a name="learn-more"></a>자세히 알아보기

  - 조직에서 Teams를 설치하고 실행하기 위해 마련된 라이브 시리즈인 **[대화형 교육](https://aka.ms/TeamsLiveTraining)** 을 통해 Teams의 여정을 시작해보세요.

  - **[온라인 수업](../instructor-led-training-teams-landing-page.yml)** 중 하나에 참여하여 Teams가 실제로 작동하는 모습을 확인하고 질문에 대한 답변을 받으세요.

  - 자세한 정보를 알고 싶어하는 사용자와 **[자세한 교육용 비디오](https://www.youtube.com/playlist?list=PLXPr7gfUMmKzR7_jXN5s886apYoHNC3Xk)** 를 공유합니다.

  - **[Teams 관리자 설명서](https://docs.microsoft.com/MicrosoftTeams/)** 를 읽고 Teams를 배포하고 관리하는 방법을 알아보세요.
