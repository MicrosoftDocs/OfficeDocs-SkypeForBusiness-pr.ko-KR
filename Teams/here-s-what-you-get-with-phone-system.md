# <a name="heres-what-you-get-with-phone-system-in-office-365"></a>Office 365에서 전화 시스템을 사용 하 여 얻을 수 있는 기능

PBX는 비즈니스 내의 전화 시스템입니다. Office 365 추가 기능에서 휴대폰 시스템을 사용 하면 PBX 기능을 사용할 수 있지만 복잡 하 고 저렴 한 장비를 제공 하지 않습니다. 
  
휴대폰 시스템 추가 기능을 사용 하면 인터넷에 접속 하 여 거의 모든 곳에서 전화를 걸고 받고, 통화를 전송 하거나, 음소거 또는 해제 통화와 같은 기본적인 작업을 수행할 수 있습니다. 전세계에서 사용할 수 있습니다! 
  
자세한 내용은 [전화 시스템의 가격 산정](https://products.office.com/skype-for-business/cloud-pbx#requirements)을 참조 하세요.

## <a name="plan-and-set-up-the-phone-system-in-office-365"></a>Office 365에서 전화 시스템 계획 및 설정
<a name="bkmk_setup"> </a>

비즈니스에 가장 적합 한 방법으로 전화 시스템을 설정 합니다.
  
- 전화 시스템 추가 기능을 회사의 기존 전화선 또는 통화 [계획 서비스와 함께](calling-plan-landing-page.md) 사용 하 여 인바운드와 아웃 바운드 통화를 할 수 있습니다.
    
- Office 365에서 새 전화 번호를 구입 하거나 기존 전화 번호를 비즈니스용 Skype로 전송할 수 있습니다. 
    
비즈니스에서 전화 시스템을 계획 하 고 설정 하려면 다음 문서를 참조 하세요.

- [통화 요금제가 포함 되어 있는 전화 시스템](calling-plan-landing-page.md)

- [통화 요금제 설정](set-up-calling-plans.md):이 문서에서는 전화 번호를 사용 하 여 설정 하거나 기존 전화 번호를 비즈니스용 Skype로 이전 하는 단계별 지침을 제공 합니다.
    
- [Microsoft 전화 통신 솔루션](https://docs.microsoft.com/en-us/SkypeForBusiness/hybrid/msft-telephony-solutions):이 문서에서는 모든 배포 옵션의 레이아웃을 설명 합니다. 또한 전체 클라우드 전용 솔루션 계획에 대 한 정보도 있습니다. 
    
- [Microsoft 팀 추가 기능 라이선스](teams-add-on-licensing/microsoft-teams-add-on-licensing.md): Office 365에서 전화 시스템을 받기 위해 구입 해야 하는 라이선스에 대해 알아보세요.
    
- [클라우드 보이스 메일 설정-관리자 도움말](set-up-phone-system-voicemail.md)
    
- [비즈니스용 Skype 살펴보기](https://support.office.com/article/8a3491a3-c095-4718-80cf-cbbe4afe4eba): 회사 사용자가 비즈니스용 skype 기능을 사용 하는 방법에 대해 알아보세요.
