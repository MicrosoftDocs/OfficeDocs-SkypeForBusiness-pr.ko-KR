# <a name="view-guest-users-in-a-team"></a>팀에서 게스트 사용자 보기

팀의 모든 구성원은 게스트를 포함하여 팀의  모든 구성원을 볼 수 Microsoft Teams. 자세한 [내용은](https://support.office.com/article/see-who-s-on-a-team-or-in-a-channel-5c6be9be-9c45-4a0f-a1a0-f332b23cb6b7) 팀 또는 채널에 있는 사람 보기를 참조하세요.

게스트 사용자를 식별하는 경우:

1.  에 [Microsoft 365 관리 센터.](https://admin.microsoft.com)
2.  관리 **센터를**  >  **Teams.**
3.  관리 Teams 에서 **사용자를 선택합니다.** 게스트가  모든 게스트 사용자의 표시 이름에 추가됩니다.
  
소유한 팀 내에서 게스트 사용자를 식별할 수도 있습니다.
