데이터베이스 및 관리 서버 cmdlet은 Microsoft Lync Server 2013 백 엔드 데이터베이스와 프런트 엔드 관리 서비스를 모두 관리 하는 데 사용 됩니다. 이러한 cmdlet을 사용 하 여 Lync Server 2013에서 사용 되는 모든 데이터베이스를 설치 하거나 제거할 수 있을 뿐 아니라 중앙 관리 저장소에 대 한 Active Directory 서비스 제어 지점을 구성할 수도 있습니다.
