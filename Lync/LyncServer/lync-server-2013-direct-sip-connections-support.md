# <a name="direct-sip-connections-support-in-lync-server-2013"></a>Lync Server 2013의 직접 SIP 연결 지원

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-06-29_

Lync Server 2013는 direct SIP 연결을 사용 하 여 Lync Server 2013을 다음 중 하나에 연결할 수 있습니다.

  - IP-PBX

  - PSTN 게이트웨이

Lync Server 2013 풀의 중재 서버는 전화 통신 서비스 공급자가 제공 하는 여러 게이트웨이, 세션 경계 컨트롤러 (SBCs) 또는 그에 대 한 일부 조합을 제어할 수 있습니다. 또한 풀의 여러 중재 서버는 단일 게이트웨이와 상호 작용할 수 있습니다.

엔터프라이즈 전화 통신 인프라에 대 한 Microsoft 통합 커뮤니케이션 오픈 상호 운용성 프로그램을 사용 하 여 검증 된 PSTN 게이트웨이, IP-Pbx 및 SIP 트렁크 서비스를 찾을 수 있습니다. 자세한 내용은에서 [http://go.microsoft.com/fwlink/p/?linkId=203309](http://go.microsoft.com/fwlink/p/?linkid=203309)Microsoft 통합 커뮤니케이션 오픈 상호 운용성 프로그램 웹 사이트를 참조 하세요.

직접 SIP 연결에 대 한 토폴로지 및 배포 옵션에 대 한 자세한 내용은 계획 설명서의 [Lync Server 2013에서 직접 sip 연결](lync-server-2013-direct-sip-connections.md) 을 참조 하세요.
