# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>Lync Server 2013에서 권한이 부여 된 기존 호스트 제거 (선택 사항)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-05_

원격 통화 제어를 Lync Server 배포로 마이그레이션할 때 레거시 배포에서 SIP/CSTA 게이트웨이를 위해 권한이 부여 된 레거시 호스트 항목 (Lync Server에서는 *신뢰할 수 있는 응용 프로그램 항목* 이라고 함)을 제거 해야 합니다. 관리 도구를 사용하여 해당 배포에서 권한이 부여된 호스트 항목을 각각 제거해야 합니다.
