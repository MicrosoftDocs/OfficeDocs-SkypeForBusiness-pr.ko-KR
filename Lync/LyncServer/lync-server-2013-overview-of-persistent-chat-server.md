# <a name="overview-of-persistent-chat-server-in-lync-server-2013"></a>Lync Server 2013의 영구 채팅 서버 개요

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-29_

Lync Server 2013, 영구 채팅 서버를 통해 사용자는 시간에 따라 유지 되는 단체의 토픽 기반 대화에 참여할 수 있습니다. 영구 채팅 서버는 조직에서 다음을 수행할 수 있도록 지원 합니다.

  - 지리적으로 분산 된 팀과 부서간 업무 흐름도 간의 의사 소통을 개선 합니다. 영구 채팅을 사용 하면 팀에서 정보, 아이디어 및 의사 결정을 효율적으로 공유할 수 있습니다. 채팅방에 게시 된 메시지 (토론 포럼)는 계속 해 서 온라인 상태가 아닌 경우에도 다른 위치와 부서의 사용자가 참여할 수 있도록 유지할 수 있습니다 (즉, 시간에 따라 사용 가능). 사용자가 채팅방에 연결 되 면 사용자에 게 대화에 대 한 컨텍스트를 제공 하기 위해 채팅 실 (구성 가능한 채팅 기록 번호)이 채팅방에 자동으로 로드 됩니다.

  - 정보 인식 개선. 사용자는 클라이언트측 필터를 사용 하 여 메시지 내용의 키워드 또는 메시지의 "보낸 사람" 필드 값을 정의할 수 있으며, 이러한 조건이 충족 되는 경우에는 영구 채팅 인스턴트 메시지 또는 채팅방 메시지에서 알림을 받습니다. 이렇게 하면 사용자가 가장 관심 있는 콘텐츠를 사용 하 여 최신 상태를 유지할 수 있습니다.

  - 확장 된 조직과의 통신을 개선 합니다. 조직의 다른 사용자와의 장기 실행 항목을 쉽게 공동으로 작업 하 고 정보를 공유할 수 있는 지속적인 장소를 제공 함으로써 지속적인 채팅을 통해 통신을 향상 시킬 수 있습니다.

  - 정보 과부하를 줄입니다. 사용자는 클라이언트측 필터를 사용 하 여 가장 관심 있는 채팅방과 메시지를 팔 로우 할 수 있으며 팔 로우 하는 채팅방을 자신의 연락처 목록에 추가할 수 있습니다.

  - 중요 한 지식과 정보의 dispersion을 늘립니다. 문서 및 링크는 모든 팀의 액세스를 위해 대화 내에 포함 될 수 있습니다. 광범위 한 팀에 질문을 게시 하면 사용자가 주제별 전문가의 응답을 통해 혜택을 누릴 수 있습니다. 다른 정보 시스템과 통합 하면 중요 한 조직 데이터가 대규모 그룹에 쉽게 전달 될 수 있습니다.

Lync Server 2013에서 채팅방을 사용 하도록 설정 하려면 Lync Server 2013 영구 채팅을 배포 합니다. 채팅방을 사용 하는 방법에 대 한 자세한 내용은의 영구 <http://go.microsoft.com/fwlink/p/?linkid=270945>채팅 도움말을 참조 하세요. Lync Server에 사용자를 설정 하 고 Lync Server 지원이 배포 되는 경우 사용자는 Lync 2013 영구 채팅을 설치 하 고 사용 하 여 채팅방 지원을 제공할 수 있습니다.

조직이 준수 규정을 따라야 하는 경우에는 선택적으로 영구 채팅 준수 서비스를 배포할 수 있습니다.
