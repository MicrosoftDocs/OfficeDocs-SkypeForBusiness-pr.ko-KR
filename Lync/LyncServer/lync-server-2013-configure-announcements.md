# <a name="configure-announcements-in-lync-server-2013"></a>Lync Server 2013에서 알림 구성

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-12_

알림을 구성할 때는 할당 되지 않은 번호로 전화를 처리 하는 방법을 실제로 구성 합니다. 오디오 파일 또는 TTS (텍스트 음성 변환) 파일 일 수 있는 음성 안내를 재생 하거나, 확인 메시지를 표시 하지 않고 지정 된 대상으로 통화를 전송할 수 있습니다.

할당 되지 않은 번호 테이블을 정의 하기 전에 알림을 만들어야 합니다. 오디오 음성 안내, TTS 프롬프트 또는 음성 안내를 사용 하는 모든 알림에 대해이 단계를 수행 해야 합니다.
