공유 풀에서 Lync Server 2013에 호스트 되는 최대 전화 회의 크기 (인스턴트 메시징 (IM), 현재 상태, 회의 및 Enterprise Voice를 포함 하 여 모든 Lync Server 2013 워크 로드를 호스트 하는 풀)는 250 사용자입니다.
