# <a name="integration-of-persistent-chat-management-tools-in-lync-server-2013"></a>Lync Server 2013의 영구 채팅 관리 도구 통합

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-08-16_

Lync Server 2013는 관리 도구를 Lync 서버의 나머지 부분에 사용 되는 도구와 통합 하 여 영구 채팅 서버의 관리를 간소화 합니다.

영구 채팅 서버에는 Lync Server 제어판과 통합 된 관리 사용자 인터페이스 환경이 포함 되어 있습니다. 또한 영구 채팅 서버에는 영구 채팅 서버 범주를 관리 하 고 관리 하는 Windows PowerShell cmdlet의 컬렉션이 포함 되며, 채팅방을 삭제 하 고 오래 된 콘텐츠를 제거 하는 등의 다양 한 기능을 활용할 수 있습니다.
