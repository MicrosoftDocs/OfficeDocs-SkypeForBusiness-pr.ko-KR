# <a name="integration-of-persistent-chat-management-tools-in-lync-server-2013"></a>Lync Server 2013의 영구 채팅 관리 도구 통합

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-08-16_

Lync Server 2013에서는 관리 도구를 나머지 Lync Server에 사용 된 도구와 통합 하 여 영구 채팅 서버를 간편 하 게 관리할 수가 있습니다.

영구 채팅 서버에는 Lync Server 제어판에 통합 된 관리 사용자 인터페이스 환경이 포함 되어 있습니다. 또한 영구 채팅 서버에는 영구 채팅 서버 범주를 관리 하 고 관리 하는 Windows PowerShell cmdlet 모음 (채팅방 삭제 및 사용 되지 않는 콘텐츠 제거) 및 추가 기능이 포함 되어 있습니다.
