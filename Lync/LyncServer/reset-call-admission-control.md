# <a name="reset-call-admission-control"></a>통화 허용 콘트롤 재설정

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-11_

Lync Server 2010 프런트 엔드 풀이 CAC (통화 허용 제어)를 호스팅하고 있는 경우 lync server 2010 프런트 엔드 풀을 제거 하려면 먼저 CAC 호스팅을 Lync Server 2013 풀로 이동 해야 합니다.

<div>

## <a name="to-reset-cac"></a>CAC를 다시 설정하려면

1.  토폴로지 작성기를 엽니다.

2.  사이트 노드를 마우스 오른쪽 단추로 클릭한 다음 **속성 편집**을 클릭합니다.

3.  **통화 허용 제어 설정** 아래에서 **통화 허용 제어 사용**이 선택되었는지 확인합니다.

4.  **Cac (통화 허용 제어)를 실행 하려면 프런트 엔드 풀**에서 cac를 호스팅할 Lync Server 2013 풀을 선택 하 고 **확인**을 클릭 합니다.

5.  토폴로지를 게시합니다.
