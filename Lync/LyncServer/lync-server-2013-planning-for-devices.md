_**마지막으로 수정 된 항목:** 2013-02-28_

Lync Server 2013에는 Lync Phone Edition, 적격 장치에서 실행 되는 소프트웨어 및 일반적인 및 고급 전화 통신 기능, 통합 보안, 관리 효율성 등이 제공 됩니다. Lync Phone Edition은 lync Server 2010에서와 마찬가지로 Lync Server 2013와 동일한 방식으로 작동 합니다. 장치 계획에 대 한 자세한 내용은 Lync Server 2010 TechNet 라이브러리의 [장치에 대 한 계획](https://go.microsoft.com/fwlink/p/?linkid=285880) 을 참조 하세요.
