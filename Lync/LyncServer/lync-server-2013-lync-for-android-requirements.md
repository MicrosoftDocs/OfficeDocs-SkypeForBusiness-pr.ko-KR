# <a name="lync-for-android-requirements-in-lync-server-2013"></a>Lync Server 2013의 Android 용 lync 요구 사항

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2014-04-24_

Microsoft Lync 2013 Android 용 Microsoft Lync 2013는 Android 장치에서 연결 하는 조직의 사용자에 게 IM (인스턴트 메시징), 향상 된 현재 상태 및 Lync 모임 참가 기능을 제공 합니다. 이 항목에서는 구성 요소, 기술 요구 사항, 필수 구성 요소를 포함 하 여 Android 용 Lync 2013의 고려 사항에 대해 설명 합니다.

<div>

## <a name="lync-for-android-prerequisite"></a>Android 용 Lync 필수 조건

Android 용 Lync 2013을 지원 하려면 Android 장치가 다음 요구 사항을 충족 해야 합니다.

  - Android 장치는 Tegra2 칩이 아닌 태블릿을 포함 하 여 Android 4.0 또는 이후 휴대폰 또는 태블릿 기반 운영 체제를 실행 중 이어야 합니다.

  - 장치는 1.2 GHz 듀얼 코어 이상의 CPU를 사용 해야 합니다.

  - 장치 카메라 (전면/후면) 해상도는 VGA 이상 이어야 합니다.

  - 다른 하드웨어 요구 사항은 Android 4.0 호환성 정의 문서에 맞춰야 합니다.

</div>

<div>

## <a name="other-technical-considerations"></a>기타 기술 고려 사항

Android 장치 플랫폼에서는 Lync 응용 프로그램이 백그라운드에서 실행 될 수 있습니다. 따라서 다른 모바일 장치 플랫폼과 달리 푸시 알림은 Android 장치에 필요 하지 않습니다. Android 장치에서 Lync 응용 프로그램을 종료 하는 유일한 방법은 Lync에서 명시적으로 로그 아웃 하는 것입니다. 이 Lync 응용 프로그램 버전은 Tegra 2 칩셋을 사용 하는 장치에서 지원 되지 않습니다.
