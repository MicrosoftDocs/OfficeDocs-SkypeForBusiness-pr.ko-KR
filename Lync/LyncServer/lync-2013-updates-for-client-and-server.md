# <a name="updates-for-lync-2013-client-and-lync-server-2013"></a>Lync 2013 클라이언트 및 Lync Server 2013에 대 한 업데이트

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2014-11-12_

최신 버전의 Lync 클라이언트를 사용할 수 있도록 하려면 Microsoft Lync 2013에 대 한 모든 클라이언트 릴리스 버전의 업데이트 된이 자주 되는 목록을 확인 하세요.

<div>

## <a name="determining-your-lync-desktop-client-version"></a>Lync 데스크톱 클라이언트 버전 확인

사용 중인 Lync 데스크톱 클라이언트의 버전을 확인 하려면 옵션 아이콘을 클릭 하 여 **lync** 대화 상자를 엽니다. **옵션** 메뉴에서 **도움말** 을 선택한 다음 **Microsoft Lync 정보**를 선택 합니다. 클라이언트 버전 번호는 일반적으로 **Lync** 대화 상자의 첫 번째 줄에 표시 됩니다.

</div>

<div>

## <a name="lync-client-updates"></a>Lync 클라이언트 업데이트

Lync 클라이언트 문제를 해결 하는 데 가장 일반적으로 수행 되는 검사 중 하나는 최신 버전의 Lync 클라이언트를 실행 중인지, 모든 패치가 설치 되어 있는지 여부를 확인 하는 것입니다. 보고 되는 대부분의 Lync 클라이언트 문제는 제품의 최신 릴리스 또는 업데이트로 업그레이드 하 여 해결 됩니다. 따라서 Microsoft 기술 지원 서비스에 문의 하기 전에 최신 버전의 Lync 클라이언트와 최근 업데이트를 함께 설치 하는 것이 좋습니다. 이 [문서](https://docs.microsoft.com/SkypeForBusiness/sfb-client-updates#lync-2013-client-updates) 는 올바른 릴리스 버전 번호를 식별 하는 데 도움이 됩니다.

Lync 클라이언트에 대 한 각 업데이트에는 자체 버전 번호가 있습니다. 다음 표에 연결 된 관련 기술 자료 문서를 살펴보면 업데이트와 관련 된 버전 번호를 확인할 수 있습니다.

</div>

<div>

## <a name="lync-server-updates"></a>Lync Server 업데이트

Lync Server 2013의 업데이트에 대 한 자세한 내용은 Microsoft 지원 웹에서 [Lync server 2013에 대 한 업데이트](https://support.microsoft.com/help/2809243/updates-for-lync-server-2013) 를 참조 하십시오.

</div>
