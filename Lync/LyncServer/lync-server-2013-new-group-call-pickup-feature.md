# <a name="new-group-call-pickup-feature-in-lync-server-2013"></a>Lync Server 2013의 새 그룹 통화 픽업 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-12_

Lync Server에 대 한 누적 업데이트 2013:2 월 2013 그룹 통화 픽업을 새 Enterprise Voice 기능으로 도입 합니다. 그룹 호출 픽업를 사용 하 여 다른 사용자가 자신의 전화에서 그룹에 속하는 사용자에 게 들어오는 호출에 응답할 수 있도록 그룹에 사용자를 할당할 수 있습니다.

그룹 통화 픽업는 통화 대기 응용 프로그램을 기반으로 합니다. 통화 대기와 마찬가지로 그룹 통화 픽업의 경우 그룹 번호로 사용할 가상 번호 범위를 설정 합니다. 사용자가 그룹 번호를 사용 하 여 해당 그룹에 있는 다른 사용자에 게 신호음으로 연결 되는 통화를 선택 합니다.
