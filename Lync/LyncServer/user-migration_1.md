# <a name="user-migration"></a>사용자 마이그레이션

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-19_

일반적으로 마이그레이션에 허용 되는 모범 사례는 여러 테스트 사용자를 만들고이를 사용 하 여 시스템 테스트를 수행 하는 것입니다. 이러한 계정을 성공적으로 이동 하 고 테스트 한 후에는 파일럿 프로덕션 사용자 그룹을 식별 하 고 해당 계정을 이동 하 고 유효성 검사 테스트를 수행 해야 합니다. 만족 스러운 결과를 얻을 경우 나머지 사용자를 새 배포로 이동할 수 있습니다.
