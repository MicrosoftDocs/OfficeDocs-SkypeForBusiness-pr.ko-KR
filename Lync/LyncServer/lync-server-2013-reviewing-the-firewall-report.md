# <a name="reviewing-the-firewall-report-in-lync-server-2013"></a>Lync Server 2013에서 방화벽 보고서 검토

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-02-21_

Lync Server 2013에는 잠재적으로 복잡 한 방화벽 규칙 집합이 있습니다. 계획 도구는 디자이너의 입력 기준에 따라 모든 방화벽 요구 사항에 대 한 세부 정보를 정의 하는 보고서를 생성 하 여 이러한 복잡성을 줄입니다. IT 방화벽 관리자가이 보고서를 사용 하 여 필요한 규칙을 구성 하 고 정의할 수 있습니다.

방화벽 관리의 관점에서 보고서를 검토 하 여 종료 되는 방화벽 규칙과 충돌이 없는지, 위반 될 수 있는 정책이 나 절차가 없는지 확인 해야 합니다.
