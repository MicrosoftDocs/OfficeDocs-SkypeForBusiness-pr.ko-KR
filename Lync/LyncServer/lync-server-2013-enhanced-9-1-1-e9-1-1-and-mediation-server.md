# <a name="enhanced-9-1-1-e9-1-1-and-mediation-server-in-lync-server-2013"></a>Lync Server 2013의 향상 된 9-1-1 (E9-1-1) 및 중재 서버

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-29_

중재 서버는 향상 된 9-1-1 (E9-1-1) 서비스 공급자와 올바르게 상호 작용할 수 있도록 확장 된 기능을 제공 합니다. 중재 서버에서는 특별 한 구성이 필요 하지 않습니다. E9-1-1 상호 작용에 필요한 SIP 확장은 기본적으로 중재 서버의 SIP 프로토콜에 포함 되어 있으며, 게이트웨이 피어와의 상호 작용 (PSTN 게이트웨이, IP-PBX 또는 인터넷 전화 통신 서비스 공급자의 SBC, 즉 E9-1 서비스를 포함 합니다. 자가

E9-1-1 서비스 공급자에 대 한 SIP 트렁크를 기존 중재 서버 풀에서 종료 하거나 독립 실행형 중재 서버가 필요한 지는 E9-1-1 SBC가 중재 서버 풀과 상호 작용할 수 있는지 여부에 따라 달라 집니다. 자세한 내용은 [Lync Server 2013의 M:n 트렁크](lync-server-2013-m-n-trunk.md)를 참조 하십시오.
