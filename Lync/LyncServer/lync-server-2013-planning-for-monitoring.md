# <a name="planning-for-monitoring-in-lync-server-2013"></a>Lync Server 2013의 모니터링 계획

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-05_

Microsoft Lync Server 2013의 모니터링 서비스는 관리자가 조직에서 발생 하는 통신 세션에 대 한 사용 현황, 추세 및 서비스 품질 데이터를 수집할 수 있는 방법을 제공 합니다. Lync Server 2013의 모니터링에는 더 이상 별도의 서버 역할이 필요 하지 않습니다. 대신 각 프런트 엔드 서버에 모니터링 서비스가 기본 제공 됩니다. 그러나 Lync Server 2013에서는 기본적으로 모니터링을 사용할 수 없습니다. 이 문서는 조직에서 모니터링을 사용 해야 하는지 여부를 결정 하는 데 도움이 됩니다.
