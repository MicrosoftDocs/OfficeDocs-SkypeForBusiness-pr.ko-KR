# <a name="setting-up-permissions-for-archiving-in-lync-server-2013"></a>Lync Server 2013에서 보관에 대 한 사용 권한 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-01_

Lync Server 2013에서 특정 작업은 이러한 작업을 수행 하는 사용자가 하나 이상의 특정 그룹의 구성원 이어야 한다는 것을 여전히 필요로 합니다. 그러나 RBAC (역할 기반 액세스 제어)를 사용 하 여 미리 정의 된 Lync Server 관리 역할에 사용자를 할당 하 여 권한을 부여할 수도 있습니다. 보관을 배포 하기 전에 적절 한 사용자 권한 및 사용 권한이 현재 위치에 있어야 하며 특정 RBAC 역할에 할당 하려는 사용자가 해당 역할에 할당 되어 있는지 확인 합니다. 보관에 대 한 지원을 배포 하는 사용자 권한, 사용 권한 및 역할에 대 한 자세한 내용은 계획 문서 및 배포 설명서에서 제공 되는 [Lync Server 2013에서 보관에 대 한 배포 검사 목록을](lync-server-2013-deployment-checklist-for-archiving.md)참조 하세요. RBAC에 대 한 자세한 내용은 계획 문서에서 [Lync Server 2013의 역할 기반 액세스 제어 계획](lync-server-2013-planning-for-role-based-access-control.md) 을 참조 하세요.
