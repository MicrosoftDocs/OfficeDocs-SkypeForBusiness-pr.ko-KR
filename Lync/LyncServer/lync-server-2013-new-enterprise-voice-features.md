# <a name="new-enterprise-voice-features-in-lync-server-2013"></a>Lync Server 2013의 새로운 Enterprise Voice 기능

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<span> </span>

_**마지막으로 수정 된 항목:** 2013-05-01_

Lync Server 2013에는 Enterprise Voice를 향상 시키는 몇 가지 새로운 라우팅 및 통화 관리 기능이 도입 되었습니다.

Lync Server 2013는 중재 서버와 게이트웨이 간에 여러 트렁크를 지원 합니다. *트렁크* 는 포트 번호와 중재 서버 간의 논리적 연결로, 포트 번호와 게이트웨이를 사용 합니다. 즉 중재 서버가 서로 다른 게이트웨이에 여러 트렁크 있을 수 있으며, 게이트웨이는 중재 서버 마다 여러 트렁크 있을 수 있습니다. Intertrunk 라우팅은 Lync Server 2013에서 ip-pbx를 PSTN (공중 전화망) 게이트웨이와 상호 연결 하거나 여러 IP PBX 시스템을 상호 연결할 수 있도록 합니다. Lync Server 2013는 서로 다른 전화 통신 시스템 간의 붙이기 (즉, 상호 연결) 역할을 합니다.

Microsoft Lync Server 2013에서는 착신 전환, 동시 전화 신호, 음성 메일 처리 및 발신자 번호 표시 영역에서 개선 된 기능을 지원 합니다. 이러한 기능은 기업 음성 통화 환경을 보강 합니다.

Lync Server 2013에는 다음과 같은 새로운 향상 된 Enterprise Voice 기능이 도입 되었습니다.

  - [Lync Server 2013의 새로운 통화 기능](lync-server-2013-new-call-features.md)

  - [Lync Server 2013의 새 발신자 번호 기능](lync-server-2013-new-caller-id-feature.md)

  - [Lync Server 2013의 새로운 음성 메일 기능](lync-server-2013-new-voice-mail-feature.md)

  - [Lync Server 2013의 새로운 트렁크 기능](lync-server-2013-new-trunk-feature.md)

  - [Lync Server 2013의 새로운 인터 트렁크 기능](lync-server-2013-new-intertrunk-feature.md)

  - [Lync Server 2013의 새 통화 관리 기능](lync-server-2013-new-call-management-features.md)
