# <a name="remove-legacy-archiving-and-monitoring-servers"></a>레거시 보관 및 모니터링 서버 제거

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-19_

레거시 배포에 보관 서버 또는 모니터링 서버가 포함 된 경우, Lync Server 2013으로 마이그레이션한 후에는 모든 사용자가 나머지 레거시 풀에서 제거 되 면 레거시 환경에서 해당 서버를 제거할 수 있습니다. 모든 순서로 보관 서버 또는 모니터링 서버를 제거할 수 있습니다. 주요 요구 사항은 모든 사용자가 나머지 레거시 풀에서 제거 되었다는 것입니다.

[4 단계: 테스트 사용자를 파일럿 풀로 이동](phase-4-move-test-users-to-the-pilot-pool.md)에 설명 된 절차에 따라 lync server 2010에서 lync server 2013로 사용자를 이동할 수 있습니다.

모든 사용자가 남아 있는 풀에서 제거 되었는지 확인 한 후에 [http://go.microsoft.com/fwlink/p/?linkId=246227](http://go.microsoft.com/fwlink/p/?linkid=246227)다운로드할 수 있는 "Microsoft Lync Server 2010 제거 및 서버 역할 제거"의 절차를 따릅니다.
