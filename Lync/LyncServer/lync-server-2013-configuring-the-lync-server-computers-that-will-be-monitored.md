# <a name="configuring-the-lync-server-computers-that-will-be-monitored-in-lync-server-2013"></a>Lync Server 2013에서 모니터링 되는 Lync Server 컴퓨터 구성

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-20_

Lync Server 2013에서 Microsoft Lync Server 2010에 사용 되는 중앙 검색 프로세스를 사용 하지 않기 때문에 모니터링할 각 Lync Server 2013 컴퓨터는 관리 서버에 대 한 존재 여부를 자체 보고할 수 있어야 합니다. 이를 가능 하 게 하려면 모니터링할 각 컴퓨터에 Operations Manager 에이전트 파일을 설치 해야 합니다. 에이전트 파일을 설치한 후에는 컴퓨터를 System Center 프록시로 작동 하도록 구성 해야 합니다. 이러한 절차는 이러한 컴퓨터에 Lync Server를 설치 하 고 구성한 후에 수행 해야 합니다.
