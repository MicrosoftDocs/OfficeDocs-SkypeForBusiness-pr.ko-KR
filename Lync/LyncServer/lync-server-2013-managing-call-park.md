# <a name="managing-call-park-in-lync-server-2013"></a>Lync Server 2013의 통화 대기 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-10_

통화 대기 응용 프로그램을 사용 하면 Enterprise Voice 사용자가 전화를 통해 통화를 한 번에 전화를 걸어 나중에 모든 전화기에서 전화를 걸 수 있습니다. 사용자가 전화를 걸 때 Lync Server는 통화가 전화를 걸거나 시간이 초과 될 때까지 통화를 대기 하는 *궤도*라는 임시 번호로 통화를 전송 합니다.

이 섹션의 항목에서는 배포에서 통화 대기 응용 프로그램을 사용자 지정 하 고 유지 관리 하기 위해 수행할 수 있는 작업에 대 한 단계별 절차를 제공 합니다.
