# <a name="managing-call-park-in-lync-server-2013"></a>Lync Server 2013에서 통화 공원 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-10_

전화 공원 응용 프로그램을 사용 하면 엔터프라이즈 음성 사용자가 한 전화기에서 통화를 대기 상태로 전환 하 고 나중에 모든 전화기에서 통화를 검색할 수 있습니다. 사용자가 통화를 전환 하면 Lync Server가 통화를 다른 사람이 검색 하거나 시간 초과 될 때까지 통화가 대기 하는 *회전*이라고 하는 임시 번호로 착신 전환 합니다.

이 섹션의 항목에서는 배포에서 통화 공원 응용 프로그램을 사용자 지정 하 고 유지 관리 하기 위해 수행할 수 있는 작업에 대 한 단계별 절차를 제공 합니다.
