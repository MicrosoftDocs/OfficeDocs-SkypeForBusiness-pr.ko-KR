# <a name="setting-up-the-infrastructure-for-archiving-in-lync-server-2013"></a>Lync Server 2013에서 보관용 인프라 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-01_

보관에 대 한 인프라 요구 사항은 저장소를 제외 하 고 Lync Server 배포의 경우와 동일 합니다. Exchange 2013 저장소, 보관 데이터베이스 또는 둘 다를 사용 하 여 저장소를 설정 하는 것을 제외 하면 추가 인프라 설정이 필요 하지 않습니다. Lync Server 2013의 인프라 요구 사항에 대 한 자세한 내용은 배포 설명서에서 계획 설명서 및 lync server [에 대 한 인프라 및 시스템 준비 2013](lync-server-2013-preparing-the-infrastructure-and-systems.md) 에서 [lync server 2013에 대 한 인프라 요구 사항 결정](lync-server-2013-determining-your-infrastructure-requirements.md) 을 참조 하십시오. 보관에 대 한 저장소 요구 사항에 대 한 자세한 내용은 계획 설명서의 Lync server 2013에서 보관을 위한 [기술 요구 사항](lync-server-2013-technical-requirements-for-archiving.md) , 배포 설명서의 [lync Server 2013에서 보관용 시스템 플랫폼 설정](lync-server-2013-setting-up-system-platforms-for-archiving.md) 및 배포 설명서의 [lync Server 2013에서 보관용 저장소 설정을](lync-server-2013-setting-up-storage-for-archiving.md) 참조 하십시오.
