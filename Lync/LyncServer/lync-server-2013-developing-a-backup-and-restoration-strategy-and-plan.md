# <a name="developing-a-backup-and-restoration-strategy-and-plan-for-lync-server-2013"></a>Lync Server 2013에 대 한 백업 및 복원 전략 및 계획 개발

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-17_

Lync Server 백업 및 복원 작업의 효과는 백업 및 복원 전략 및 계획에 따라 달라 집니다. 조직의 전반적인 전략에 맞는 Lync Server 백업 및 복원 전략을 설정 하 고, 데이터 및 설정을 백업 하기 위한 포괄적이 고 간결한 계획을 수립 해야 하며, 중단 시에 서비스를 복원 하는 계획을 세우는 것이 좋습니다.

프런트 엔드 풀의 견고한 재해 복구를 위해 Lync Server 2013에 도입 된 쌍 풀 재해 복구 토폴로지를 사용 합니다. 자세한 내용은 [Lync Server 2013에서 고가용성 및 재해 복구 계획](lync-server-2013-planning-for-high-availability-and-disaster-recovery.md)을 참조 하세요.
