# <a name="new-im-and-presence-features-in-lync-server-2013"></a>Lync Server 2013의 새로운 메신저 및 현재 상태 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-19_

Microsoft Lync Server 2013는 사용자의 Lync 환경을 보강 하기 위해 다음과 같은 새로운 IM (인스턴트 메시징) 및 현재 상태 기능을 추가 합니다.

  - 조직에서 Exchange 2013도 실행 하는 경우 사용자는 통합 된 연락처 저장소를 활용할 수 있습니다. 사용자는 Outlook 2013, Outlook Web App 및 Lync 2013에서 연락처를 관리할 수 있습니다.

  - 사용자는 Lync Server 2013의 **Xmpp 통합** 기능으로 인해 Google 대화 등의 확장 가능한 메시징 및 현재 상태 프로토콜을 사용 하는 공용 IM 공급자의 사용자와 인스턴트 메시지 및 현재 상태 정보를 교환할 수 있습니다. XMPP 통합은 프런트 엔드 서버 및 Edge 서버에 기본 제공 되며이 기능을 허용 하도록 설정 하 고 구성할 수 있습니다.
