_**마지막으로 수정 된 항목:** 2012-03-21_

Lync Server 2010 모범 사례 분석기 도구는 Lync Server 2010 배포에서 구성 정보를 수집 하 고 구성이 Microsoft 모범 사례에 따라 설정 되는지 여부를 확인 합니다. Microsoft .NET Framework 3.5 서비스 팩 1을 실행 하는 클라이언트 컴퓨터에 또는 Lync Server 2010를 실행 하는 서버에서 직접 도구를 설치할 수 있습니다. 이 도구는 클라이언트 컴퓨터에 설치하고 실행하는 것이 좋습니다. 모범 사례 분석기가 전체 데이터 집합을 수집할 수 있도록 Lync Server 2010 관리 도구는 클라이언트 컴퓨터에 로컬로도 설치 해야 합니다.

Microsoft 다운로드 센터에서 Lync Server 2010 모범 사례 분석기를 다운로드할 수 있습니다 <https://go.microsoft.com/fwlink/p/?linkid=246173> .
