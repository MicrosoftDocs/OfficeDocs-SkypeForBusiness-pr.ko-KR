# <a name="run-best-practices-analyzer"></a>모범 사례 분석기 실행

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-03-21_

Lync Server 2010 모범 사례 분석기 도구는 Lync Server 2010 배포에서 구성 정보를 수집 하 고 구성이 Microsoft 모범 사례에 따라 설정 되었는지 여부를 확인 합니다. Microsoft .NET Framework 3.5 서비스 팩 1 또는 Lync Server 2010를 실행 하는 서버에서 직접 실행 하는 클라이언트 컴퓨터에이 도구를 설치할 수 있습니다. 클라이언트 컴퓨터에서이 도구를 설치 하 고 실행 하는 것이 좋습니다. 또한 모범 사례 분석기에서 전체 데이터 집합을 수집할 수 있도록 Lync Server 2010 관리 도구를 클라이언트 컴퓨터에 로컬로 설치 해야 합니다.

Microsoft 다운로드 센터에서 Lync Server 2010 모범 사례 분석기를 다운로드할 수 있습니다 <http://go.microsoft.com/fwlink/p/?linkid=246173> .
