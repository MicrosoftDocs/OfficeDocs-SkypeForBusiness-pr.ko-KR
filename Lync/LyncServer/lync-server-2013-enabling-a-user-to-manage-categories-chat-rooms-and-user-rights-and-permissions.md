# <a name="enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions-in-lync-server-2013"></a>Lync Server 2013에서 사용자가 범주, 채팅방, 사용자 권한 및 사용 권한을 관리할 수 있도록 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-11-01_

**CsPersistentChatAdministrator** 역할 (영구 채팅 관리자)의 구성원은 다른 사용자와 자신에 게 채팅방 관리자 권한 및 사용 권한을 부여할 수 있습니다.

영구 채팅 관리자는 다음을 수행할 수 있습니다.

  - 방 범주와 채팅방을 만듭니다.

  - 모든 범주와 채팅방의 구성원 자격을 설정 합니다.

  - 모든 범주와 채팅방의 모든 설정을 관리 합니다.

  - 영구 채팅 서버에 대 한 정책을 사용 하거나 사용 하지 않도록 설정 합니다.

  - 영구 채팅 서버 풀에서 구성 설정을 설정 하 고 관리 합니다.

자세한 내용은 배포 설명서의 [Lync Server 2013에서 영구 채팅 관리자 추가](lync-server-2013-adding-a-persistent-chat-administrator.md) 를 참조 하세요.
