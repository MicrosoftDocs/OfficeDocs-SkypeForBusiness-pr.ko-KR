# <a name="enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions-in-lync-server-2013"></a>Lync Server 2013에서 사용자가 범주, 대화방, 사용자 권한 및 사용 권한을 관리할 수 있도록 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-11-01_

**CsPersistentChatAdministrator** 역할 (영구 채팅 관리자)의 구성원은 다른 사용자와 자신에 게 대화방 관리자 권한을 부여할 수 있습니다.

영구 채팅 관리자는 다음을 수행할 수 있습니다.

  - 방 범주 및 채팅방 만들기

  - 모든 범주 및 채팅방의 구성원 자격 설정

  - 모든 범주와 채팅방의 모든 설정 관리

  - 영구 채팅 서버에 대 한 정책을 사용 하거나 사용 하지 않도록 설정

  - 영구 채팅 서버 풀에서 구성 설정을 설정 및 관리 합니다.

자세한 내용은 배포 설명서에서 [Lync Server 2013의 영구 채팅 관리자 추가](lync-server-2013-adding-a-persistent-chat-administrator.md) 를 참조 하십시오.
