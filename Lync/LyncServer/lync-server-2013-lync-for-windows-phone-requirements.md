# <a name="lync-for-windows-phone-requirements-in-lync-server-2013"></a>Lync Server 2013의 Windows Phone 요구 사항에 대 한 lync

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-02-20_

Windows Phone 용 Microsoft Lync 2013는 참가자 또는 Windows Professional 모바일 장치에서 연결 하는 조직의 사용자에 게 IM (인스턴트 메시징), 향상 된 현재 상태 및 전화 접속을 제공 합니다. 모바일 장치를 통해 사용자는 Lync 2013의 범위를 연장할 수 있습니다. 이 항목에서는 필수 구성 요소와 기술 요구 사항, 필수 구성 요소, 배포 지침을 포함 하 여 Windows Phone 용 Lync 2013에 대 한 계획 고려 사항을 설명 합니다.

<div>

## <a name="lync-for-windows-phone-prerequisites"></a>Windows Phone 용 Lync 필수 조건

다음은 Windows Phone 필수 구성 요소에 대 한 Lync 2013입니다.

  - Windows Phone 8, codenamed "아폴로" 또는 최신 버전

  - Windows Phone 장치에는 Microsoft에서 제공 하는 최신 업데이트가 있어야 합니다. 자세한 내용은에서 [http://go.microsoft.com/fwlink/p/?LinkID=281961](http://go.microsoft.com/fwlink/p/?linkid=281961)Windows Phone 8 업데이트 내역을 참조 하세요.

  - 장치에 사용 가능한 디스크 공간이 12MB 여야 합니다.

  - 사용자는 통신 회사의 음성 및 데이터 요금제를 보유 하 고 있어야 합니다.
