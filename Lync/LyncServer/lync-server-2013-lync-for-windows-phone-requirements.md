# <a name="lync-for-windows-phone-requirements-in-lync-server-2013"></a>Lync Server 2013의 Windows Phone 요구 사항에 대 한 lync

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-20_

Microsoft Lync 2013 for Windows Phone은 smartphone 또는 Windows Professional 모바일 장치에서 연결 하는 조직의 사용자에 게 IM (인스턴트 메시징), 향상 된 현재 상태 및 전화 통신을 제공 합니다. 모바일 장치를 통해 Lync 2013의 범위를 확장할 수 있습니다. 이 항목에서는 필수 구성 요소 및 기술 요구 사항 식별, 필수 컴포넌트 및 배포 지침을 포함 하는 Windows Phone 용 Lync 2013에 대 한 고려 사항에 대해 설명 합니다.

<div>

## <a name="lync-for-windows-phone-prerequisites"></a>Lync for Windows Phone 필수 구성 요소

다음은 Windows Phone 필수 구성 요소에 대 한 Lync 2013입니다.

  - Windows Phone 8, 코드 이름 "아폴로," 또는 최신 버전

  - Windows Phone 장치에는 Microsoft에서 제공 하는 최신 업데이트가 있어야 합니다. 자세한 내용은 Windows Phone 8 업데이트 내역을 참조 하십시오 [https://go.microsoft.com/fwlink/p/?LinkID=281961](https://go.microsoft.com/fwlink/p/?linkid=281961).

  - 장치에 12MB의 사용 가능한 디스크 공간이 있어야 합니다.

  - 사용자가 통신사의 음성 및 데이터 요금제에 가입한 상태여야 합니다.
