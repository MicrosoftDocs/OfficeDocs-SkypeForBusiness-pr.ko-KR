# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>반드시 Lync Server 2013에서 알림 배포 확인

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-25_

알림을 설치 및 구성한 후에는 지정되지 않은 번호에 대한 통화가 예상대로 작동하는지 구성을 확인해야 합니다. 최소한 다음 사항을 확인하십시오.

  - 조직에 유효한 번호로 전화를 걸어 할당 되지 않은 번호입니다.

  - 지정되지 않은 번호에 전화를 걸어 올바른 알림이 재생되는지 확인합니다.
