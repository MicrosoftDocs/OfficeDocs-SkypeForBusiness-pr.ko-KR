# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>) Lync Server 2013에서 알림 배포 확인

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-02-25_

알림을 설치 하 고 구성한 후에는 지정 하지 않은 숫자에 대 한 호출이 예상 대로 작동 하는지 확인 하도록 구성 해야 합니다. 최소한 다음 사항을 확인 하세요.

  - 조직에 유효한 번호로 전화를 걸고 지정 되지 않은 번호입니다.

  - 지정 하지 않은 번호를 호출 하 고 올바른 알림이 재생 되는지 확인 합니다.
