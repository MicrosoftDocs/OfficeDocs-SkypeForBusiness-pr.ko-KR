# <a name="choosing-an-e9-1-1-service-provider-for-lync-server-2013"></a>Lync Server 2013 용 E9-1-1 서비스 공급자 선택

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-06-06_

E9-1-1 서비스 공급자는 통화 내에 포함 된 위치 정보를 기반으로 Lync Server의 긴급 통화를 올바른 PSAP (공개 보안 응답 지점)로 라우팅합니다.

Lync Server 배포의 일부로 E9-1-1을 지원 하려면 Lync Open a E9-1-1 서비스 공급자 로부터 E9-1-1 라우팅 서비스를 가져와야 합니다. 조직의 요구 사항에 가장 적합한 공급자를 선택하십시오.
