# <a name="choosing-an-e9-1-1-service-provider-for-lync-server-2013"></a>Lync Server 2013의 E9-1-1 서비스 공급자 선택

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-06-06_

E9-1 서비스 공급자는 Lync Server에서 시작 된 비상 통화를 통화 내에 포함 된 위치 정보에 따라 올바른 PSAP (공개 안전 응답 시점)로 라우팅합니다.

Lync Server 배포의 일부로 E9-1을 지원 하려면 Lync Open Interop 프로그램 정식 E9-1-1 서비스 공급자에서 E9-1 라우팅 서비스를 얻어야 합니다. 조직의 요구 사항에 가장 잘 맞는 공급자를 선택 합니다.
