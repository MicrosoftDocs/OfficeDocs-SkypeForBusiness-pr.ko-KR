# <a name="components-used-by-group-call-pickup-in-lync-server-2013"></a>Lync Server 2013의 그룹 통화 픽업에서 사용 되는 구성 요소

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-01-30_

그룹 통화 픽업 기능은 Enterprise Voice 및 통화 대기 응용 프로그램을 배포할 때 자동으로 배포 됩니다. 통화 대기 번호 표는 call pickup 그룹 번호로 지정 된 별도의 숫자 범위로 구성한 다음, 사용자를 지정 하 여 pickup 그룹에 전화를 걸어 사용자를 그룹 통화 픽업으로 설정 하 여 그룹 통화 픽업을 사용 하도록 설정 합니다. 다음 Lync Server 구성 요소는 그룹 통화 픽업를 지원 합니다.

  - **응용 프로그램 서비스**   응용 프로그램 서비스는 통화 대기 응용 프로그램 등의 통합 통신 응용 프로그램을 배포, 호스팅 및 관리 하기 위한 플랫폼을 제공 합니다. 응용 프로그램 서비스는 프런트 엔드 풀의 모든 프런트 엔드 서버와 모든 Standard Edition 서버에 자동으로 설치 됩니다.

  - **통화 대기 응용**   프로그램 통화 대기 응용 프로그램은 응용 프로그램 서비스에서 호스팅하는 통합 통신 응용 프로그램 중 하나입니다. 그룹 통화 픽업는 통화 대기 응용 프로그램을 기반으로 합니다.

  - **Lync server 관리 셸**   lync server 관리 셸을 사용 하 여 그룹 통화 픽업 그룹을 관리 합니다.

  - **SEFAUtil resource kit tool**   SEFAUtil (보조 확장 기능 활성화 유틸리티)를 사용 하 여 사용자를 통화 픽업 그룹에 할당 하 고 사용자를 위해 call pickup를 사용 하거나 사용 하지 않도록 설정할 수 있습니다.
