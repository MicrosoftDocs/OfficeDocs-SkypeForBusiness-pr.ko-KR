Microsoft는 모든 사용자가 제품 및 서비스를 쉽게 사용할 수 있도록 하기 위해 노력 하 고 있습니다. 다음 섹션에서는 장애가 있는 사용자가 Lync Server 2013에 더욱 쉽게 액세스할 수 있도록 지 원하는 기능, 제품 및 서비스에 대 한 정보를 제공 합니다.

## <a name="accessibility-features-of-lync-server-2013"></a>Lync Server 2013의 접근성 기능

Lync Server 2013의 다음 기능은 장애가 있는 사용자에 게 더욱 쉽게 액세스할 수 있도록 지원 합니다.

  - 바로 가기 키

  - 그림에 대 한 대체 텍스트

또한 Windows의 접근성 기능 및 도구에 따라 장애가 있는 Lync Server 사용자에 게 도움이 될 수 있습니다. Windows PowerShell 크기 및 색 변경으로 Lync Server Management Shell을 사용할 때 접근성 옵션이 제공 됩니다. Windows PowerShell 접근성 옵션에 대 한 자세한 내용은 TechNet 라이브러리에서 "Windows PowerShell의 접근성 2.0을 참조 하세요 [http://go.microsoft.com/fwlink/p/?linkId=98964](http://go.microsoft.com/fwlink/p/?linkid=98964).

## <a name="alternate-text-for-figures"></a>그림에 대 한 대체 텍스트

스크린샷, 다이어그램, 순서도, 기타 그림을 포함 하 여 Lync Server 2013 도움말의 모든 그림에는 대체 텍스트가 연결 되어 있습니다. 그림이 표시 되지 않는 사용자는 그림에서 커서를 일시 중지 하 여 대체 텍스트를 읽을 수 있습니다. 대체 텍스트는 그림에 표시 된 항목을 설명 합니다.

## <a name="accessibility-features-of-windows"></a>Windows의 접근성 기능

Windows 운영 체제에는 입력 하거나 마우스를 사용 하는 데 어려움이 있거나 시각 장애인 또는 청각 장애가 있거나 청각 장애가 있는 사용자에 게 유용 하 게 사용할 수 있는 다양 한 접근성 기능이 기본적으로 제공 됩니다. 이 기능은 설치 중에 설치 됩니다. 이러한 기능에 대 한 자세한 내용은 Windows 도움말 또는 Microsoft 접근성을 [http://go.microsoft.com/fwlink/p/?linkId=18139](http://go.microsoft.com/fwlink/p/?linkid=18139)참조 하세요.

  - **무료 단계별 자습서**   Microsoft는 컴퓨터에서 접근성 옵션 및 설정을 조정 하는 자세한 절차를 제공 하는 일련의 단계별 자습서를 제공 합니다. 이 정보는 마우스, 키보드 또는 두 가지 모두의 조합을 사용 하는 방법을 배울 수 있도록 나란히 형식으로 제공 됩니다.
    
    Microsoft 제품에 대 한 단계별 자습서를 찾으려면의 [http://go.microsoft.com/fwlink/p/?linkId=18139](http://go.microsoft.com/fwlink/p/?linkid=18139)microsoft 접근성을 참조 하세요.

  - ****   장애가 있는 사용자를 위해 컴퓨터를 더욱 쉽게 사용할 수 있도록 Windows 용 보조 기술 제품에는 다양 한 보조 기술 제품이 있습니다. 에서 [http://go.microsoft.com/fwlink/p/?linkId=18139](http://go.microsoft.com/fwlink/p/?linkid=18139)Microsoft 접근성 웹 사이트의 Windows에서 실행 되는 보조 기술 제품 카탈로그를 검색할 수 있습니다.
    
    보조 기술을 사용 하는 경우에는 소프트웨어 또는 하드웨어를 업그레이드 하기 전에 호환성 문제를 확인 하기 전에 보조 기술 공급 업체에 문의 해야 합니다.
