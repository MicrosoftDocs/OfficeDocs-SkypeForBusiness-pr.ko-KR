# <a name="unassigned-number-cmdlets-in-lync-server-2013"></a>Lync Server 2013의 할당 되지 않은 번호 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-03-21_

배정 되지 않은 번호는 조직에 속하지만 사용자 또는 전화기에 게 배정 되지 않은 전화 번호입니다. 할당 되지 않은 번호는 해당 번호로 전화를 건 경우 다양 한 유형의 알림을 재생 하도록 구성할 수 있습니다.

<div>
