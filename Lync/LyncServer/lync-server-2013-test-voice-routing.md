# <a name="test-voice-routing-in-lync-server-2013"></a>Lync Server 2013에서 음성 라우팅 테스트

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-24_

Lync Server 제어판 **테스트 음성 라우팅** 탭을 사용 하 여 테스트 사례 시나리오를 구성할 수 있습니다. 테스트 사례를 정의 하려면 지정 된 전화 번호를 테스트할 다이얼 플랜, 음성 정책, PSTN 사용 및 음성 경로를 지정 합니다.

음성 라우팅 구성을 실제로 배포 하기 전에 다양 한 전화 번호에서이를 테스트 하 여 예상 되는 결과를 확인 하는 것이 좋습니다.

<div>


> [!TIP]  
> <STRONG>테스트 사례 내보내기</STRONG> 및 <STRONG>테스트 사례 가져오기</STRONG> 명령을 사용 하 여 음성 라우팅 테스트 사례를 저장 하 고 다른 컴퓨터에서 사용 하도록 가져올 수 있습니다.



</div>

<div>


> [!WARNING]  
> 다이얼 플랜, 음성 정책, 음성 경로 또는 전화와 같은 음성 라우팅 구성 부분을 삭제 하는 경우에는 음성 라우팅 테스트 사례를 검토 하 고 업데이트 해야 합니다. Lync Server 제어판은 변경 된 구성으로 인해 더 이상 유효 하지 않은 테스트 사례를 알려 주지 않습니다.



</div>
