_**마지막으로 수정 된 항목:** 2014-05-05_

다음 단계에 따라 디렉터에 Lync Server 2013 구성 요소를 설치 합니다.

<div>

## <a name="to-install-lync-server-components-on-a-director"></a>디렉터에 Lync Server 구성 요소를 설치하려면

1.  Lync server 배포 마법사의 Lync Server 2013 페이지에서 **2 단계: Lync Server 구성 요소 설치 또는 제거**옆의 **실행**을 클릭 합니다.

2.  **Lync Server 구성 요소 설치** 페이지에서 **다음** 을 클릭 하 여 게시 된 토폴로지에 정의 되어 있는 대로 구성 요소를 설정 합니다.

3.  Lync Server 구성 요소 설치가 완료 되 면 **마침을**클릭 합니다.
