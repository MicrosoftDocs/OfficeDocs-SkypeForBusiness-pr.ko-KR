# <a name="disabling-uploading-and-downloading-files-in-chat-rooms-in-lync-server-2013"></a>Lync Server 2013의 채팅방에서 파일 업로드 및 다운로드 사용 안 함

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-12_

기본적으로 사용자는 자신이 게시 한 메시지에 파일을 업로드 하 고 다운로드할 수 있습니다. 이 기능을 사용 하지 않도록 설정 하면 사용자가 다음 두 가지 방법으로 파일을 업로드 하 고 다운로드 하지 못하도록 할 수 있습니다.

  - 모든 사용자가 특정 영구 채팅방 또는 채팅방 범주에 있는 파일을 업로드 하 고 다운로드 하지 못하도록 합니다.

  - 특정 사용자가 채팅방에서 파일을 업로드 하 고 다운로드 하지 못하도록 합니다. 사용자는 파일 업로드 및 다운로드를 허용 하는 채팅방을 포함 하 여 모든 채팅방에 파일을 업로드 하 고 다운로드할 수 없습니다.

파일 업로드 및 다운로드를 사용 하거나 사용 하지 않도록 설정 하는 방법에 대 한 자세한 내용은 배포 설명서의 [Lync Server 2013에서 범주 구성](lync-server-2013-configure-categories.md) 및 [범주 관리](manage-categories.md) 를 참조 하세요.
