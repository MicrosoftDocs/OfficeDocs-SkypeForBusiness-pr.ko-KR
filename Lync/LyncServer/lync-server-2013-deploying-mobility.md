# <a name="deploying-mobility-in-lync-server-2013"></a>Lync Server 2013에서 모바일 기능 배포

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-08_

Lync Server 2013 모바일 기능을 배포할 때 모바일 사용자는 IM (인스턴트 메시징), 현재 상태 및 연락처 등의 Lync 기능에 지원 되는 모바일 장치를 사용할 수 있습니다.

모바일 기능을 배포 하기 위한 요구 사항에 대 한 자세한 내용은 [Lync Server 2013의 Mobility 계획](lync-server-2013-planning-for-mobility.md)을 참조 하십시오.

이 섹션에서는 자동 검색 기능과 모바일 기능을 배포 및 확인하는 단계를 안내합니다.
