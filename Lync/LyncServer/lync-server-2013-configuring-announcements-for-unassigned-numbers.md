# <a name="configuring-announcements-for-unassigned-numbers-in-lync-server-2013"></a>Lync Server 2013에서 지정되지 않은 번호에 대한 알림 구성

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-11_

알림 신청은 할당 되지 않은 확장명 (조직에 대해 유효 하지만 사용자 또는 휴대폰에 할당 되지 않은 확장명)에 대 한 호출을 구성할 수 있는 Enterprise Voice 기능입니다. 예를 들어 지정 하지 않은 번호로 통화를 구성 하 여 메시지를 재생 하거나 다른 대상 또는 둘 다로 전송할 수 있습니다.

알림 응용 프로그램은 엔터프라이즈 음성을 배포할 때 프런트 엔드 서버 또는 Standard Edition 서버에 응답 그룹 응용 프로그램의 기능으로 설치 됩니다. 오디오 파일을 업로드 하거나 텍스트 읽어주기 (TTS)를 구성 하 고 할당 되지 않은 번호 표를 구성 하 여 알림을 구성 해야 합니다.

이 섹션에서는 Lync Server 알림의 구성 과정을 안내 합니다. 이 예제에서는 알림과 관련 된 계획 섹션을 이미 읽고 enterprise Edition server 또는 Enterprise Voice를 사용 하는 스탠더드 버전 서버를 배포 했다고 가정 합니다.
