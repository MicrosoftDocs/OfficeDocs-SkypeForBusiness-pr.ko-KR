# <a name="reviewing-the-summary-report-in-lync-server-2013"></a>Lync Server 2013에서 요약 보고서 검토

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-21_

Lync Server 관리자 보고서는 디자인을 자세히 문서화 하는 가장 중요 한 4 가지 보고서입니다. 이 보고서의 정보와 3 개의 관련 보고서에는 정보 기술 팀을 위한 훌륭한 문서가 있습니다.

![일반 요약 관리 보고서](images/Gg558626.9c529ef7-cb1b-4ce1-a8bc-3ec79aba2377(OCS.15).jpg "일반 요약 관리 보고서")

  - 인증서 보고서

  - 방화벽 보고서

  - DNS 보고서

요약 보고서에는에 지 네트워크와 연결 된 일반 구성 정보가 나열 됩니다. 위치, FQDN (정규화 된 도메인 이름) 및 IP 주소, 네트워크 유형 및 지정 된 역할과 관련 된 설명이 문서화 됩니다.

인프라를 배포, 관리 및 유지 관리 하는 각 팀과 디자이너는 요약 보고서의 정확성을 검토 하 고 오류가 최소로 나타나는지 확인 해야 합니다.
