# <a name="resetting-or-deleting-external-user-access-policies-in-lync-server-2013"></a>Lync Server 2013에서 외부 사용자 액세스 정책 다시 설정 또는 삭제

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-08_

더 이상 사용 하지 않을 외부 사용자 액세스 정책을 만들거나 구성한 경우 다음을 수행할 수 있습니다.

  - 만든 사이트 또는 사용자 정책을 삭제 합니다.

  - 전역 정책을 기본 설정으로 다시 설정 합니다. 기본 전역 정책 설정에서는 외부 사용자 액세스를 거부 합니다. 전역 정책은 삭제할 수 없습니다.
