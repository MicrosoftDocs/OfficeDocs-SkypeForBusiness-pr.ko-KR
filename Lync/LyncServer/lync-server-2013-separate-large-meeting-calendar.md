# <a name="separate-large-meeting-calendar-in-lync-server-2013"></a>Lync Server 2013에서 대용량 모임 일정 구분

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-01_

각 대규모 모임 풀에 대해 해당 풀에 예약 된 대규모 모임의 일정을 개별적으로 유지 관리 해야 합니다. 예를 들어 대규모 모임 풀의 단일 사용자 계정을 홈으로 설정 하 고 Lync 2013의 Exchange 및 온라인 모임 추가 기능을 통해 Outlook을 사용 하 여 별도의 일정을 유지할 수 있습니다. 여러 사용자 계정을 사용 하 여 지원 직원이 대규모 모임을 만들 수 있도록 설정 하는 경우 지원 직원의 구성원이 만든 모든 대규모 모임을 집계 하는 별도의 일정을 설정할 수 있습니다.

별도의 대규모 모임 일정을 유지 관리 하면 충돌을 방지 하 고 한 번에 하나의 큰 모임만 활성화 되도록 할 수 있습니다.
