# <a name="scanning-for-viruses-and-checking-virus-definitions-in-lync-server-2013"></a>Lync Server 2013에서 바이러스를 검사 하 고 바이러스 정의를 확인 합니다.

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2014-05-01_

IM 수준 바이러스 백신 제품을 설치 하는 것이 좋습니다. IM은 조직 전체에서 바이러스와 악성 소프트웨어를 신속 하 게 분배 하는 데 잘 알려진 원본입니다. Lync Server에 대 한 Microsoft Forefront® 보안은 바이러스, 악성 소프트웨어, 파일 및 키워드 필터 보호 및 Office Communications Server와의 원활한 통합으로 다중 엔진 검색을 제공 합니다.

Lync Server에 대 한 Forefront 보안 외에도, 서버 파일 시스템을 보호 하기 위해 파일 수준 바이러스 백신 솔루션을 설치 하는 것이 좋습니다.

스캐너 엔진과 바이러스 정의를 업데이트 된 상태로 유지 하는 것이 매우 중요 합니다. 업데이트 상태를 구성 하 고 모니터링 하면 Office 통신 서버와 파일 시스템을 보호 하기 위해 최신 스캐닝 정보를 사용 하 고 있는지 확인할 수 있습니다.

<div>


> [!IMPORTANT]  
> Lync server 2013 및 Lync Server에 대 한 Forefront Security를 실행 하는 서버에서 타사의 파일 수준 바이러스 백신 소프트웨어를 사용 하는 경우 Lync Server 및 Lync server 용 Forefront Security이 설치 되어 있는 폴더가 검사 되지 않도록 해야 합니다. 손상 되었습니다. 제외의 전체 목록은를 참조 <A class=uri href="http://support.microsoft.com/kb/943620">http://support.microsoft.com/kb/943620</A>하세요.
