# <a name="create-response-group-workflows-in-lync-server-2013"></a>Lync Server 2013에서 응답 그룹 워크플로 만들기

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-12_

워크플로는 다른 사용자가 전화를 건 시간으로 울릴 때 까지의 통화 동작을 정의 합니다. 워크플로는 통화를 보류 하는 데 사용할 큐를 지정 하 고, 헌트 그룹 또는 대화형 응답 그룹에 사용할 질문 및 답변에 사용할 라우팅 방법을 지정 합니다. 워크플로는 환영 메시지, 보류 중인 음악, 업무 시간, 휴일 등의 설정도 정의 합니다.

응답 그룹 구성 도구를 사용 하 여 워크플로를 만듭니다. Lync Server 제어판의 응답 그룹 페이지에서 응답 그룹 구성 도구에 액세스할 수 있습니다.

<div>


> [!NOTE]  
> 에이전트 그룹과 큐를 사용 하는 워크플로를 만들기 전에 만들어야 합니다.
