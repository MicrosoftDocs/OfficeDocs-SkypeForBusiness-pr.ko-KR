# <a name="manage-sip-federated-domains-for-your-organization-in-lync-server-2013"></a>Lync Server 2013에서 조직의 SIP 페더레이션된 도메인 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-19_

이 문서는 예비 문서로, 변경 될 수 있습니다. 빈 항목은 개체 틀로 포함 됩니다.

페더레이션 할 수 있는 SIP 도메인을 관리 하 고 구성 하려면 다음을 수행 하면 됩니다.

  - SIP 페더레이션 파트너 도메인의 허용 된 도메인 목록을 만들거나 편집 합니다.

  - SIP 페더레이션 도메인의 차단 된 도메인 목록을 만들거나 편집 합니다.

이러한 작업을 수행 하려면이 섹션의 절차를 사용 합니다.
