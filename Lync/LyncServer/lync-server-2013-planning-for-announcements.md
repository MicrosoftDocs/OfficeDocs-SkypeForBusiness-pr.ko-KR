# <a name="planning-for-announcements-in-lync-server-2013"></a>Lync Server 2013의 알림 계획

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-13_

Lync Server 알림 응용 프로그램을 사용 하면 전화 건 번호가 조직에 대해 유효 하지만 사용자 또는 전화에 할당 되지 않은 수신 전화 통화에 대 한 처리를 구성할 수 있습니다. 이러한 통화를 미리 정의 된 대상 (전화 번호, SIP URI 또는 음성 메일)으로 전송 하거나 오디오 알림을 재생 하거나 두 가지를 모두 수행할 수 있습니다. 알림 응용 프로그램을 사용 하면 발신자가 통화를 잘못 전화를 걸거나 SIP 클라이언트에서 오류 메시지를 수신 하는 상황을 방지할 수 있습니다. 이 섹션에는 알림 응용 프로그램과 관련 된 계획 정보에 대 한 정보가 포함 됩니다.
