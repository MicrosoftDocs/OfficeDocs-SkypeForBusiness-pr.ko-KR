# <a name="advanced-enterprise-voice-cmdlets-in-lync-server-2013"></a>Lync Server 2013의 고급 엔터프라이즈 음성 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2016-04-12_
