# <a name="media-quality-diagnostic-reports-in-lync-server-2013"></a>Lync Server 2013의 미디어 품질 진단 보고서

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-22_

미디어 품질 진단 보고서에서는 통화 품질에 대 한 정보 및 실패 한 통화에 대 한 진단 및 문제 해결 정보를 제공 합니다.

<div>

## <a name="in-this-section"></a>이 섹션의 내용

  - [Lync Server 2013](lync-server-2013-media-quality-summary-report.md)     의 미디어 품질 요약 보고서 Enterprise Voice 피어 투 피어 통화, Enterprise Voice 전화 회의 통화 및 최소한의 공중 전화망 (PSTN)을 사용 하는 통화를 포함 하 여 서로 다른 끝점 유형에 대 한 전체 품질 데이터를 제공 합니다.

  - [Lync Server 2013](lync-server-2013-media-quality-comparison-report.md)     의 미디어 품질 비교 보고서 서로 다른 종류의 오디오 통화 (예: 무선 네트워크를 통한 통화와 유선 연결을 통한 통화 간의 통화 품질 값 비교)를 제공 합니다.

  - [Lync server 2013](lync-server-2013-server-performance-report.md)     의 서버 성능 보고서 성능 저하, 패킷 손실, 지터 등 주요 품질 메트릭 측정값을 기준으로 대부분의 문제가 발생 한 서버를 나열 합니다.

  - [Lync Server 2013](lync-server-2013-location-report.md)     의 위치 보고서 네트워크 위치 목록과 각 위치에서 발생 하는 호출의 미디어 품질에 대 한 요약을 제공 합니다. 이 보고서의 목적을 위해 위치는 IP 서브넷을 기반으로 합니다.

  - [Lync Server 2013](lync-server-2013-device-report.md)     의 장치 보고서 Enterprise Voice 통화에 사용 되는 장치에 대 한 요약 정보를 제공 하며, 장치별 통화의 평균 미디어 품질을 포함 합니다.

  - [Lync Server 2013](lync-server-2013-call-list-report.md)     의 통화 목록 보고서 조직 내에서 수행 하거나 받은 전화 통화에 대 한 자세한 정보를 제공 합니다.

  - [Lync Server 2013](lync-server-2013-call-detail-report.md)     의 통화 정보 보고서 조직 내에서 보내거나 받는 전화 통화에 대 한 자세한 정보를 제공 합니다.

  - [Lync server 2013](lync-server-2013-server-media-quality-trend-report.md)     의 서버 미디어 품질 추세 보고서 통화 량, 불량 통화 율, 패킷 손실 및 지터와 같은 경험 현황 메트릭의 최대 5 대의 서버를 그래픽으로 비교할 수 있는 방법을 제공 합니다.

  - [Lync Server 2013](lync-server-2013-media-quality-metrics-distribution-report.md)     의 미디어 품질 메트릭 배포 보고서 지터 또는 패킷 손실과 같은 경력 수준의 품질 메트릭에 대 한 분포 값을 보여 주는 그래프를 제공 합니다.

  - [Lync Server 2013](lync-server-2013-location-trend-report.md)     의 위치 추세 보고서 네트워크 위치에 대 한 통화 품질 추세 정보를 제공 합니다.
