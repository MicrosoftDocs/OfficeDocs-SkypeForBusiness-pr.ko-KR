# <a name="overview-of-conferencing-in-lync-server-2013"></a>Lync Server 2013의 회의 개요

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-30_

회의를 배포 하는 경우 웹 회의 및 A/V 회의를 사용 하도록 설정 하 고 사용 하거나 웹 회의만을 선택할 수 있습니다. 전화 접속 회의는 오디오 회의의 하위 집합이 며 추가 구성이 필요 합니다.

모든 형태의 회의를 사용 하는 경우 사용자는 인스턴트 메시지 (IM), 오디오, 비디오, 데스크톱 공유, 슬라이드 프레젠테이션, 첨부 파일 공유, 응용 프로그램 공유 등을 조합 하 여 다양 가능한 회의 환경을 즐길 수 있습니다. 회의가 예약 되거나 예약 되지 않을 수 있으며 사용자는 발생 하는 동안 간편 하 게 통신 양식을 추가할 수 있습니다. 예를 들어 메신저에서 시작 하 여 문서 공동 작업을 추가한 다음 음성 또는 비디오를 추가 합니다. 새로운 참가자도 실시간으로 진행 중인 회의에 추가 될 수 있습니다.

다음 항목에서는 웹 회의, A/V 회의 및 전화 접속 회의에서 제공 하는 특정 기능에 대 한 개요를 제공 합니다.
