# <a name="running-voice-routing-tests-in-lync-server-2013"></a>Lync Server 2013에서 음성 라우팅 테스트 실행

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-02-21_

Lync Server 2013는 음성 경로를 테스트 하는 두 가지 방법, 즉 모든 전화 번호 및 음성 경로에 대해 비공식적인 특별 한 테스트를 수행할 수 있습니다. 또는 음성 경로 테스트 사례를 사용 하 여 더 많은 공식적인 테스트를 할 수 있습니다. 공식적인 테스트를 사용 하 여 전화를 걸고 사용할 다이얼 플랜 및 음성 정책 등의 항목을 정의한 다음 Lync Server는 해당 조건을 지정 하면 제공 된 번호를 PSTN 네트워크로 성공적으로 라우팅할 수 있는지 확인할 수 있습니다. 이 두 방법 모두이 설명서의 후속 섹션에서 설명 합니다.
