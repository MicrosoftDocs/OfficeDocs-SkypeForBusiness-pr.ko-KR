</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-08-03_

Lync Server 2013, 영구 채팅 서버를 장애 조치 및 장애 복구 하려면 Microsoft SQL Server 2008 R2 이상에 대 한 복제 및 장애 조치 프로세스를 잘 알고 있어야 합니다. 또한 영구 채팅 서버 서비스에 대해 잘 알고 있어야 합니다.
