# <a name="voice-policy-cmdlets-in-lync-server-2013"></a>Lync Server 2013의 음성 정책 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-02-18_

Enterprise Voice 관리에는 음성 정책 및 다이얼 플랜 등의 구성과 음성 경로를 음성으로 연결 하는 기능이 포함 됩니다. 음성 정책 관리와 관련 된 cmdlet을 사용 하 여 동시 연결 등의 기능을 설정 하 고, 다른 사용자가 사무실 전화를 걸 때마다 두 번째 전화기를 연결 하는 기능, 착신 전환 및 전화 걸기 요구 사항을 설정할 수 있습니다.
