# <a name="managing-enhanced-9-1-1-and-the-location-service-in-lync-server-2013"></a>Lync Server 2013에서 고급 9-1-1 및 위치 서비스 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-11-01_

Lync Server 2013는 Lync 클라이언트 및 Lync Phone Edition 장치에서 향상 된 9-1-1 (E9-1) 통화를 지원 합니다. E9-1-1에 대 한 Lync Server 2013를 구성 하는 경우 Lync 2013 또는 Lync Phone Edition에서 제공 하는 비상 전화에는 위치 정보 서비스 데이터베이스의 ERL (비상 응답 위치) 정보가 포함 됩니다. 이 섹션의 절차를 사용 하 여 위치 정책을 관리 합니다.

<div>


> [!NOTE]  
> E9-1-1 및 위치 정보 서비스와 같은 고급 엔터프라이즈 음성 기능을 배포 하는 방법에 대 한 자세한 내용은 <A href="lync-server-2013-deploying-advanced-enterprise-voice-features.md">Lync Server 2013의 고급 엔터프라이즈 음성 기능 배포</A>를 참조 하세요.
