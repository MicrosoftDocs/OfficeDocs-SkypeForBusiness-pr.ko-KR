# <a name="performing-an-abc-front-end-pool-failover-in-lync-server-2013"></a>Lync Server 2013에서 ABC 프런트 엔드 풀 장애 조치 (failover) 수행

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-21_

이 섹션의 두 항목에서는 Lync Server 2013에서 ABC 풀 장애 조치 (failover)를 수행 하는 절차에 대해 설명 하 고, Lync Server 프런트 엔드 풀 A 및 B가 있고, 풀 A를 복구할 수 없게 됩니다. 이 절차를 사용 하 여 새 FQDN (정규화 된 도메인 이름)을 사용 하 여 새 프런트 엔드 풀 C를 만듭니다. Pool C는 failed pool A의 정보를 통해 생성 됩니다. 이 절차에는 또한 풀 B와 C에 함께 페어링이 포함 됩니다.

  - [Lync Server 2013의 ABC 풀 장애 조치 (failover)를 위한 백업 필수 구성 요소](lync-server-2013-backup-prerequisites-for-abc-pool-failover.md)

  - [Lync Server 2013의 프런트 엔드 풀 ABC 장애 조치 (failover) 절차](lync-server-2013-front-end-pool-abc-failover-procedure.md)
