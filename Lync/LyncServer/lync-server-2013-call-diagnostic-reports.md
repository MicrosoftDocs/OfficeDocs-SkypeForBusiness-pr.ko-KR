# <a name="call-diagnostic-reports-in-lync-server-2013"></a>Lync Server 2013의 통화 진단 보고서

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-21_

통화 진단 보고서는 피어 투 피어 및 회의 세션에 대해 요약 정보 및 진단 데이터를 제공합니다.

<div>

## <a name="in-this-section"></a>이 섹션의 내용

  - [Lync Server 2013](lync-server-2013-call-diagnostic-summary-report.md)   의 통화 진단 요약 보고서는 실패 한 피어 투 피어 세션 및 회의 세션에 대 한 전체 요약을 제공 합니다. 피어 투 피어 세션은 두 명의 참가자만 포함 하는 세션입니다. 회의 세션에는 세 명 이상의 참가자가 포함 됩니다.

  - [Lync Server 2013](lync-server-2013-peer-to-peer-activity-diagnostic-report.md)   의 피어 투 피어 활동 진단 보고서는 실패 한 피어 투 피어 세션에 대 한 전체적인 추세 보기를 제공 합니다. 피어 투 피어 세션에는 두 명의 참가자만 포함 됩니다.

  - [Lync Server 2013](lync-server-2013-conference-diagnostic-report.md)   의 전화 회의 진단 보고서에서는 각 회의 모달에 대 한 실패 한 회의 세션 및 추세 보기에 대 한 전체적인 추세 보기를 제공 합니다. 회의 세션에는 참가자가 세 명 이상 포함 됩니다.

  - [Lync Server 2013](lync-server-2013-top-failures-report.md)   의 Top Failures 보고서에는 가장 자주 발생 하는 오류와 시간에 따른 추세에 대 한 목록이 제공 됩니다.

  - [Lync Server 2013](lync-server-2013-failure-distribution-report.md)   의 실패 분포 보고서는 실패 한 세션에 대 한 분석을 제공 합니다.

  - [Lync Server 2013](lync-server-2013-failure-list-report.md)   의 실패 목록 보고서는 실패 한 회의에 포함 된 개별 참가자에 대 한 자세한 정보를 제공 합니다.

  - [Lync Server 2013](lync-server-2013-diagnostic-report.md)   의 진단 보고서는 실패 한 세션에 대 한 진단 및 문제 해결 정보를 제공 합니다 (SIP 응답 코드 및 진단 헤더와 id 포함).
