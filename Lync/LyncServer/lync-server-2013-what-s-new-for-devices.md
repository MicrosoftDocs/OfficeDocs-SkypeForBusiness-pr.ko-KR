# <a name="whats-new-for-devices-in-lync-server-2013"></a>Lync Server 2013의 장치에 대한 새로운 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-06-22_

Lync Server 2013에는 Lync Phone Edition, 정식 장치에서 실행 되는 소프트웨어가 포함 되어 있으며, 기존 및 고급 전화 통신 기능, 통합 보안, 관리 효율성 등을 제공 합니다. Lync Phone Edition은 lync server 2010에서와 동일한 방식으로 Lync Server 2013와 동일 하 게 작동 합니다. 디바이스와 관련 된 최신 기능에 대 한 자세한 내용은 Lync Server 2010 TechNet 라이브러리의 [장치에 대 한 새로운](http://go.microsoft.com/fwlink/p/?linkid=256490) 기능을 참조 하세요.
