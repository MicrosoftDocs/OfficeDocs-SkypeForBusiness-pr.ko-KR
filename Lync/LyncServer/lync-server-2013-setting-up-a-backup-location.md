# <a name="setting-up-a-backup-location-for-lync-server-2013"></a>Lync Server 2013의 백업 위치 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-02-17_

Lync Server의 첫 번째 백업을 수행 하기 전에 백업을 저장 하 고 유지 관리 하기 위해 필요한 하드웨어 및 소프트웨어를 설정 합니다. 미디어 및 콘텐츠에 대 한 액세스 권한을 적절 하 게 확보 하 고 백업할 각 서버와 백업 미디어 간에 네트워크 연결을 제공 해야 합니다. 백업 및 복원 전략에 사용 하는 미디어 및 위치를 정의 해야 합니다. 일반 백업에 사용 하는 위치는 로컬 또는 원격 일 수 있지만, 보안이 유지 되어야 하며, 백업 및 복원에 대해 액세스 가능 해야 합니다. 원격 위치를 사용 하 여 기본 사이트의 치명적 이벤트를 보호 하는 것이 좋습니다.

개별 구성 요소를 설정 하 고 테스트 한 후 각 서버에서 백업에 대 한 접근성을 확인 합니다.
