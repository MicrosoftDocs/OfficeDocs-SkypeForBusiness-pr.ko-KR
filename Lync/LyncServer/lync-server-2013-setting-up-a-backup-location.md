# <a name="setting-up-a-backup-location-for-lync-server-2013"></a>Lync Server 2013의 백업 위치 설정

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-17_

Lync Server의 첫 번째 백업을 수행 하기 전에 백업을 저장 하 고 유지 관리 하기 위해 필요한 하드웨어 및 소프트웨어를 설정 해야 합니다. 필요에 따라 미디어 및 콘텐츠에 대한 액세스 권한을 얻고 백업할 각 서버와 백업 미디어 간의 네트워크 연결을 제공해야 합니다. 사용 하는 미디어와 위치는 백업 및 복원 전략에서 정의 해야 합니다. 일반 백업에 사용 하는 위치는 로컬 또는 원격 일 수 있지만 안전 하며 백업 및 복원에 대 한 액세스 가능 해야 합니다. 기본 사이트에서 치명적 이벤트 로부터 보호 하려면 원격 위치를 사용 하는 것이 좋습니다.

개별 구성 요소를 설정 및 테스트한 후에는 각 서버에서 백업에 대한 접근성을 확인합니다.
