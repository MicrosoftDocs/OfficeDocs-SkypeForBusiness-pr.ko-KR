# <a name="managing-the-lync-server-2013-network-infrastructure"></a>Lync Server 2013 네트워크 인프라 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2014-02-11_

Microsoft Lync Server 2013에는 CAC (통화 허용 제어) 및 미디어 바이패스에 대 한 지원이 포함 됩니다. 이러한 기능을 구현 하려면 지역, 사이트, 서브넷 등의 네트워크를 구성 하 여 오디오 및 비디오 전송을 제한 해야 하는 상황에서 대역폭을 관리할 수 있도록 해야 합니다. QoS (서비스 품질) 네트워킹 기술을 사용 하 여 오디오 및 비디오 통신을 위한 최적의 최종 사용자 환경을 제공 하는 데 도움이 될 수도 있습니다.

Lync Server 제어판을 사용 하 여 CAC, 미디어 바이패스 및 QoS를 설정 하 고 관리할 수 있습니다. 다음 항목에서는이 작업을 수행 하는 단계를 제공 합니다.
