응용 프로그램 끝점은 Active Directory 도메인 서비스 내의 연락처 개체입니다. Lync Server 관리 셸에서 cmdlet을 사용 하 여 Microsoft Lync Server 2013에서 이러한 연락처를 관리할 수 있습니다.

**응용 프로그램 끝점**

  - <span></span>  
    [Get-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398655(v=OCS.15))
