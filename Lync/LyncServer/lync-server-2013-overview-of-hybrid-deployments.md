# <a name="overview-of-lync-server-2013-hybrid-deployments"></a>Lync Server 2013 하이브리드 배포 개요

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2016-05-25_

Lync Server 하이브리드 배포는 contoso.com와 같은 도메인 사용자가 Lync Server 온-프레미스 및 Microsoft Lync Online을 사용 하는 방법으로 분할 되는 배포입니다. 일부 도메인 사용자는 온-프레미스 Lync Server에 있고 일부 사용자는 비즈니스용 Skype Online에 있습니다.

온-프레미스 Lync 배포를 비즈니스용 Skype Online과 하이브리드로 구성 하 고, Active Directory 동기화를 사용 하 여 온-프레미스 및 온라인 사용자의 동기화 상태를 유지할 수 있습니다. 온-프레미스 Exchange 및 SharePoint와의 통합 또는 Exchange Online 및 SharePoint Online을 비롯 한 Microsoft Office 365 응용 프로그램에 하이브리드 배포를 구성할 수도 있습니다.

이 섹션에서는 Lync Server 하이브리드 배포에 필요한 응용 프로그램을 배포한 다음 Lync Server 온-프레미스 및 비즈니스용 Skype Online 간에 사용자를 관리 하도록 배포를 구성 하는 과정을 안내 합니다.

비즈니스용 Skype Online과 하이브리드에 대 한 온-프레미스 Lync Server 배포를 구성 하는 방법에 대 한 자세한 내용은 다음 항목을 참조 하십시오.

  - [Lync Server 2013 하이브리드 배포 계획](lync-server-2013-planning-for-hybrid-deployments.md)

  - [Lync Server 2013 하이브리드 배포 구성](lync-server-2013-configuring-hybrid-deployments.md)

비즈니스용 Skype Online에 대 한 자세한 내용은 [Lync Online](http://go.microsoft.com/fwlink/p/?linkid=282396)를 참조 하세요.
