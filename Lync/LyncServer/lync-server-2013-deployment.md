# <a name="deployment-of-lync-server-2013"></a>Lync Server 2013 배포

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-18_

Lync Server 2013 통신 소프트웨어 배포에는 Active Directory 도메인 서비스 준비, 프런트 엔드 서버 및 다른 핵심 Lync Server 2013 내부 구성 요소 배포, 추가 서버 역할 및 기능 배포 등이 포함 되어 있습니다. 외부 사용자 액세스 및 Enterprise Voice 등의 조직에 필요할 수 있습니다.

이 설명서는 Lync Server 2013 배포를 위한 세 가지 시나리오에 대해 설명 합니다.

  - Lync Server 2013, Enterprise Edition의 새로운 배포

  - Lync Server 2013, Standard Edition의 새로운 배포

  - 기존 Lync Server 2010 Standard edition 또는 Enterprise Edition 배포에 Lync Server 2013 Standard Edition 또는 Enterprise Edition의 새로운 배포

기존 Microsoft Office Communications Server 2007 또는 Microsoft Office Communications Server 2007 R2 환경에서 Lync Server 2013을 배포 하는 방법에 대 한 자세한 내용은 [마이그레이션](migration.md) 설명서를 참조 하세요.
