# <a name="address-book-server-cmdlets-in-lync-server-2013"></a>Lync Server 2013의 주소록 서버 cmdlet

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-06-26_

Active Directory 도메인 서비스와 Microsoft Lync Server 2013 사이에 주소록 서버가 중개 됩니다. 주소록 서버는 Lync Server 2013에 저장 되어 있는 사용자 정보가 Active Directory에 저장 된 사용자 정보와 동기화 되도록 합니다. 이 작업은 주소록 파일을 사용자 데이터베이스에 저장된 정보와 주기적으로 동기화하는 방식으로 수행됩니다. Lync Server에는 주소록 서버를 관리 하기 위한 여러 개의 cmdlet이 포함 되어 있습니다.

<div>

## <a name="address-book-server-cmdlets"></a>주소록 서버 Cmdlet

Lync Server 제어판에서는 주소록 서버 설정을 구성할 수 없습니다. Windows PowerShell은 이러한 설정을 관리 하기 위한 기본 도구입니다. 다음은 주소록 서버 관리에 직접적으로 관련된 cmdlet 목록입니다.

**주소록 서버**

  - <span></span>
