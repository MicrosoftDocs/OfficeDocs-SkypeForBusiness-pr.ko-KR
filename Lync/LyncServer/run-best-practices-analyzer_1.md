# <a name="run-best-practices-analyzer"></a>모범 사례 분석기 실행

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-11_

Office Communications Server 2007 R2 모범 사례 분석기 도구는 Office Communications Server 2007 R2 배포에서 구성 정보를 수집 하 고 해당 구성이 Microsoft 모범 사례에 따라 설정 되는지 여부를 확인 합니다. Microsoft .NET Framework 2.0를 실행 하는 클라이언트 컴퓨터 또는 Office Communications Server 2007 r 2를 실행 하는 서버에서 직접 도구를 설치할 수 있습니다. 이 도구는 클라이언트 컴퓨터에 설치하고 실행하는 것이 좋습니다. 모범 사례 분석기가 전체 데이터 집합을 수집할 수 있도록 Office Communications Server 2007 R2 관리 도구를 클라이언트 컴퓨터에 로컬로 설치 해야 합니다.

Microsoft 다운로드 센터에서 Office Communications Server 2007 R2 모범 사례 분석기를 다운로드할 수 있습니다 [http://go.microsoft.com/fwlink/p/?LinkId=268702](http://go.microsoft.com/fwlink/p/?linkid=268702).
