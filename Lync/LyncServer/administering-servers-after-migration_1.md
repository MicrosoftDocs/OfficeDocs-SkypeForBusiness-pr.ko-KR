# <a name="administering-servers-after-migration"></a>마이그레이션 후 서버 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-29_

일반적으로 관리하려는 서버 버전에 해당하는 관리 도구를 사용해야 합니다. Lync Server 2013 및 Office Communications Server 2007 R2 관리 도구는 동일한 컴퓨터에 설치할 수 없습니다. 또한 Lync Server 2013 제어판은 각 서버에 자동으로 설치 되지 않습니다. Lync Server 2013 제어판을 설치 하려면 배포 설명서의 [Install Lync Server 2013 관리 도구](lync-server-2013-install-lync-server-administrative-tools.md) 항목에 포함 된 절차를 따르세요.
