# <a name="performing-physical-environmental-checks"></a>물리적 환경 검사 수행

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2014-04-30_

Lync Server 2013 배포의 성능, 가용성 및 기능을 확인 하기 전에 실제 환경을 확인 해야 합니다. 예를 들어 서버 대화방 온도를 낮추지 않거나 네트워크 케이블을 교체 해야 할 수도 있습니다. 최상의 결과를 위해 다음과 같은 물리적 환경 검사를 수행 합니다.

  - **실제 보안**   잠금, 문 및 제한 된 액세스 대화방과 같은 물리적 보안 보호를 보호 해야 합니다. 승인 되지 않은 항목과 강제 입력 및 장비 손상 징후를 확인 합니다.

  - **온도 및 습도**   높은 온도, 불량 공기 흐름 및 습도로 인해 하드웨어 구성 요소가 과열 될 수 있습니다. 온도와 습도를 확인 하 여 난방 및 공기 조절 같은 환경 시스템이 하드웨어 제조업체의 사양 내에서 적합 한 조건을 유지 하 고 작동 하는지 확인 합니다. 최근에 새 장비가 설치 된 경우에도 서버에서 주고 받은 air이 unimpeded 하 고 제조업체 사양을 충족 하는지 확인 합니다.

  - **장치 및 구성 요소**   Lync Server 2013 조직은 작동 하는 물리적 네트워크와 관련 하드웨어를 사용 합니다. 라우터, 스위치, 허브, 실제 케이블 및 커넥터가 작동 하는지 확인 합니다.

이러한 검사를 수행 하는 방법에 대 한 자세한 내용은 설치 사이트 및 선택한 서버 하드웨어에 따라 크게 달라 집니다. 이 검사를 처음 수행할 때 하드웨어 설명서를 참조 하 고 나중에 참조할 수 있도록 원하는 매개 변수를 기록해 둡니다.
