# <a name="backing-up-archiving-and-monitoring-databases-in-lync-server-2013"></a>Lync Server 2013에서 보관 및 모니터링 데이터베이스 백업

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-17_

보관 또는 모니터링을 배포한 경우 조직의 SQL Server 백업 정책에 따라 이러한 데이터베이스를 백업해야 합니다.

<div>


> [!NOTE]  
> 보관 및 모니터링 설정은 중앙 관리 저장소를 백업할 때 백업 됩니다. 자세한 내용은 <A href="lync-server-2013-backing-up-core-data-and-settings.md">Lync Server 2013의 핵심 데이터 및 설정 백업</A>를 참조 하세요.



</div>

보관 및 모니터링에 대해서는 SQL Server Management Studio와 같은 SQL Server 도구를 사용하여 수동 백업을 수행하거나 SQL Server 관리 도구를 사용하여 자동 정기 백업을 예약할 수 있습니다.

</div>
