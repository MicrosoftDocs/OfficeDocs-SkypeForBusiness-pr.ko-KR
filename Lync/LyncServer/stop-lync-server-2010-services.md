# <a name="stop-lync-server-2010-services"></a>Lync Server 2010 서비스 중지

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-04_

Lync Server 제어판을 사용 하 여 특정 컴퓨터에서 실행 되는 모든 Lync Server 2010 서비스를 시작 또는 중지 하거나 특정 Lync Server 2010 서비스를 시작 하거나 중지할 수 있습니다.

<div>

## <a name="to-start-or-stop-all-lync-server-services-on-a-computer"></a>컴퓨터의 모든 Lync Server 서비스를 시작 또는 중지하려면

1.  Lync Server 제어판을 엽니다.

2.  왼쪽 탐색 모음에서 **토폴로지**를 클릭하고 **상태**를 클릭합니다.

3.  **상태** 페이지에서 시작하거나 중지할 서비스를 실행 중인 컴퓨터를 찾기 위해 필요한 경우 목록을 정렬하거나 검색한 다음 해당 서비스를 클릭합니다.

4.  **동작**을 클릭합니다.

5.  **모든 서비스 시작** 또는 **모든 서비스 중지**를 클릭합니다.

</div>

<div>

## <a name="to-start-or-stop-a-specific-service"></a>특정 서비스를 시작 또는 중지하려면

1.  Lync Server 제어판을 엽니다.

2.  왼쪽 탐색 모음에서 **토폴로지**를 클릭하고 **상태**를 클릭합니다.

3.  **상태** 페이지에서 시작하거나 중지할 서비스를 실행 중인 컴퓨터를 찾기 위해 필요한 경우 목록을 정렬하거나 검색한 다음 해당 서비스를 클릭합니다.

4.  **속성**을 클릭합니다.

5.  필요한 경우 서비스 목록을 정렬하고 시작 또는 중지할 서비스를 클릭합니다.

6.  **동작**을 클릭합니다.

7.  **서비스 시작** 또는 **서비스 중지**를 클릭합니다.

8.  **닫기**를 클릭합니다.

</div>
