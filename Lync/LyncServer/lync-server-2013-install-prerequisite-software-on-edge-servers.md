# <a name="install-prerequisite-software-on-edge-servers-for-lync-server-2013"></a>Lync Server 2013에 대 한에 지 서버에 필수 구성 요소 소프트웨어 설치

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-09-08_

Lync Server 2013을 설치 하기 전에 배포 하는 각에 지 서버에 필수 구성 요소 소프트웨어를 배포 해야 합니다. 여기에는 시스템 요구 사항을 충족하는 서버에 운영 체제를 설치하는 작업이 포함됩니다. 지원 되는 운영 체제를 비롯 하 여 시스템 요구 사항에 대 한 자세한 내용은 [Lync Server 2013의 외부 사용자 액세스 구성 요소에 대 한 시스템 요구 사항을](lync-server-2013-system-requirements-for-external-user-access-components.md)참조 하세요.
