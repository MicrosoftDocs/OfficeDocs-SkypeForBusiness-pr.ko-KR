# <a name="setting-up-the-director-in-lync-server-2013"></a>Lync Server 2013에서 디렉터 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2014-05-05_

에 지 서버를 배포 하 여 외부 사용자에 대 한 액세스를 사용 하도록 설정 하는 경우에는 디렉터를 배포 하는 옵션이 하나입니다. 디렉터는 사용자 요청을 인증 하는 Microsoft Lync Server 2013를 실행 하는 서버 이며, 사용자 계정은 그렇지 않습니다. 이제는 이것이 요구 사항은 아니지만 성능에 대해 걱정 하 고 인증 요청을 합리화 하려는 경우에는 매우 유용 합니다. 조직에서이를 결정 하는 것이 좋습니다. 디렉터 또는 디렉터 풀을 설정 하는 단계는 Enterprise Edition 프런트 엔드 풀 또는 Standard Edition 서버를 설정 하는 것과 비슷합니다. 토폴로지 작성기에서 디렉터를 정의한 후에는이 섹션의 단계를 수행 해야 합니다.
