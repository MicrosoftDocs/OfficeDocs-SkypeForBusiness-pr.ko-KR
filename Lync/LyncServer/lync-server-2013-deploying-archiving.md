# <a name="deploying-archiving-in-lync-server-2013"></a>Lync Server 2013에서 보관 배포

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-28_

Lync Server 2013는 Lync Server의 IM (인스턴트 메시징) 콘텐츠 및 회의 통신을 보관 하는 솔루션을 제공 합니다. 보관 저장소를 Exchange 2013 저장소와 통합 하거나, Lync Server 2013 보관 데이터를 저장 하는 SQL Server 데이터베이스를 사용 하거나, Lync Server 2013 및 Exchange 2013 저장소를 모두 사용 하 여 아카이빙 지원을 구현할 수 있습니다. 정책 및 보관 구성을 사용 하 여 데이터를 보관 하는 방법을 제어 합니다. 자세한 내용은 계획 설명서의 [Lync server 2013에서 보관 계획](lync-server-2013-planning-for-archiving.md) 및 lync server 2013의 계획 문서, 배포 설명서 또는 운영 설명서에서 보관을 수행 [하는 방법](lync-server-2013-how-archiving-works.md) 을 참조 하세요.

이 섹션의 정보를 사용 하 여 초기에 보관을 설정 및 구성할 수 있습니다. 배포 후에는 보관 설정을 변경할 수 있습니다. 일일 관리에 대 한 보관 지원을 구현 하거나 조직의 새로운 요구 사항을 충족 하는 방법에 대 한 자세한 내용은 운영 설명서에서 [Lync Server 2013 보관 관리](lync-server-2013-managing-archiving.md) 를 참조 하세요.
