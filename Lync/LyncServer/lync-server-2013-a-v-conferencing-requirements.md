# <a name="av-conferencing-requirements-in-lync-server-2013"></a>Lync Server 2013의 A/V 회의 요구 사항

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-30_

Lync Server 2013 회의에 대 한 인프라 요구 사항은 Lync Server 2013 배포와 동일 합니다. 자세한 내용은 계획 설명서에서 [Lync Server 2013의 인프라 요구 사항 확인](lync-server-2013-determining-your-infrastructure-requirements.md) 을 참조 하세요.

회의에 의해 생성 된 미디어 트래픽을 처리 하는 데 필요한 미디어 대역폭을 보유 하 고 있는 것이 중요 합니다. 필요한 대역폭을 계산 하는 데 사용할 수 있는 세부 정보는 [Lync Server 2013의 미디어 트래픽에 대 한 네트워크 대역폭 요구 사항을](lync-server-2013-network-bandwidth-requirements-for-media-traffic.md)참조 하세요. 대역폭을 제한 하는 방법에 대 한 자세한 내용은 [Lync Server 2013에서 대역폭 정책 프로필 만들기 또는 수정을](lync-server-2013-creating-or-modifying-bandwidth-policy-profiles.md)참조 하세요. 인프라 요구 사항에 대 한 자세한 내용은 [Lync Server 2013의 네트워크 인프라 요구 사항을](lync-server-2013-network-infrastructure-requirements.md)참조 하세요.

회의 기능을 사용 하기 위해 Lync Server 2013에서 특정 포트가 열려 있어야 합니다. 포트 요구 사항에 대 한 자세한 내용은 [Lync Server 2013의 내부 서버에 대 한 포트 및 프로토콜](lync-server-2013-ports-and-protocols-for-internal-servers.md)을 참조 하세요. 포트 구성에 대 한 자세한 내용은 [회의, 응용 프로그램 및 중재 서버에 대 한 Lync Server 2013에서 포트 범위 구성을](lync-server-2013-configuring-port-ranges-for-your-conferencing-application-and-mediation-servers.md)참조 하세요.
