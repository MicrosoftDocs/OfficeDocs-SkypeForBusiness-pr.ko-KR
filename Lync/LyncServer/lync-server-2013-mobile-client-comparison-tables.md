# <a name="mobile-client-comparison-tables-for-lync-server-2013"></a>Lync Server 2013 의 모바일 클라이언트 비교 표

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2015-09-21_

다음 표에서는 Lync 2013 모바일 클라이언트와 Lync 2013 데스크톱 클라이언트의 기능 및 기능을 다음 범주로 비교 합니다.

  - 로그인, 푸시 알림 및 일반 기능

  - 향상 된 현재 상태

  - 연락처 및 메일 그룹

  - 인스턴트 메시지 (IM)

  - Lync에서 Lync 오디오 및 비디오

  - 회의

  - 통신

  - 외부 사용자

  - 보관 및 규정 준수

다음 표는 Lync Server 2013의 온-프레미스 배포에서 Lync 사용자가 사용할 수 있는 기능을 나타냅니다. 표 각주에 달리 표시 되지 않는 한 비즈니스용 Skype Online 및 Microsoft Office 365 사용자도 동일한 기능을 사용할 수 있습니다.

<div>


> [!NOTE]  
> <UL>
> <LI>
> <P>비즈니스용 Skype에 대 한 모바일 클라이언트 비교 테이블을 찾고 계신가요? <A href="https://technet.microsoft.com/en-us/library/dn951412.aspx">비즈니스용 Skype에 대 한 모바일 클라이언트 비교 표를</A>참조 하세요.</P>
> <LI>
> <P>최종 사용자를 위한 온라인 도움말 및 리소스는 <A href="http://go.microsoft.com/fwlink/?linkid=286237">모바일 클라이언트용 Microsoft Lync 2013</A>을 참조 하세요.</P>
> <LI>
> <P>다른 Lync 2013 클라이언트에서 사용할 수 있는 기능을 비교 하려면 <A href="lync-server-2013-desktop-client-comparison-tables.md">Lync Server 2013에 대 한 클라이언트 비교 표</A>를 참조 하세요.</P>
> <LI>
> <P>Lync Server 2013는 Lync 2010 모바일 앱도 지원 합니다. 자세한 내용은 Lync Server 2010 설명서의 <A href="http://go.microsoft.com/fwlink/p/?linkid=234777">모바일 클라이언트 비교 표</A> 를 참조 하세요.</P></LI></UL>
