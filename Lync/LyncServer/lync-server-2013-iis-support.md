# <a name="iis-support-in-lync-server-2013"></a>Lync Server 2013의 IIS 지원

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-06-29_

여러 Lync Server 2013 구성 요소에는 IIS (인터넷 정보 서비스)가 필요 합니다. 지원 되는 IIS 버전에는 다음이 포함 됩니다.

  - IIS 8.0

  - IIS 7.5

Lync Server 2013 구성 요소에 대 한 IIS 요구 사항에 대 한 자세한 내용은 계획 설명서의 [Lync server 2013에서 iis (인터넷 정보 서비스) 요구 사항을](lync-server-2013-internet-information-services-iis-requirements.md) 참조 하세요.
