# <a name="optional-verify-the-group-call-pickup-deployment-in-lync-server-2013"></a>반드시 Lync Server 2013에서 그룹 통화 픽업 배포 확인

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-01-30_

사용자에 대해 그룹 통화 픽업을 사용 하도록 설정한 후 구성이 정상적으로 작동 하는지 확인 합니다. 최소한 다음 사항을 확인 합니다.

  - 그룹 통화 픽업를 사용할 수 있는 사용자에 게 전화를 걸어 다른 사용자에 게 통화를 검색 하도록 합니다. 다른 사용자는 같은 그룹에 있거나, 다른 그룹에 있거나, 그룹 통화 픽업을 사용 하도록 설정 되어 있지 않을 수 있습니다.

  - 그룹 통화 픽업를 사용할 수 있는 사용자에 게 전화를 걸어 통화에 응답 하지 않습니다.
