# <a name="optional-verify-the-group-call-pickup-deployment-in-lync-server-2013"></a>) Lync Server 2013에서 그룹 통화 픽업 배포 확인

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-01-30_

사용자 용 그룹 통화 픽업을 사용 하도록 설정한 후에는 구성이 예상 대로 작동 하는지 확인 합니다. 최소한 다음 사항을 확인 하세요.

  - 그룹 통화 픽업 기능을 사용 하도록 설정 된 사용자에 게 전화를 걸고 다른 사용자가 통화를 검색 하도록 합니다. 다른 사용자는 같은 그룹에 있거나 다른 그룹에 있거나 그룹 통화 픽업 사용이 설정 되어 있지 않을 수 있습니다.

  - 그룹 통화 픽업으로 설정 된 사용자에 게 전화를 걸고 전화를 받지 않습니다.
