# <a name="managing-lync-server-2013-archiving"></a>Lync Server 2013 보관 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-10_

조직의 보관을 배포 하는 경우 배포 중에 초기 구성을 지정 합니다. 그러나 일일 관리에 대 한 보관 지원을 구현 하는 방법이 나 조직의 새로운 요구 사항을 충족 하는 방법을 변경 해야 하는 경우가 있을 수 있습니다. 예를 들어 조직 내 특정 사이트, 풀 또는 사용자에 대해 보관 지원을 다르게 설정 해야 할 수 있습니다. Lync Server 2013에 속한 사용자의 경우, 보관 정책 및 구성을 만들고 사용자 지정 하는 것이 가능 합니다. Microsoft Exchange 통합을 사용 하는 경우 Exchange 2013 설정도 구성 해야 합니다. 이 섹션에서는 보관 배포를 변경 하는 데 사용할 수 있는 정보와 절차에 대해 설명 합니다.
