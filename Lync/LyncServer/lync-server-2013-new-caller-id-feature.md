# <a name="new-caller-id-feature-in-lync-server-2013"></a>Lync Server 2013의 새 발신자 번호 기능

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-05_

Lync Server 2013에서는 관리자에 게 발신자의 전화 번호 형식을 쉽게 수정할 수 있는 유연성을 제공 합니다. 이러한 발신자 번호 표시 기능을 통해 관리자는 필요한 경우 발신자의 전화 번호를 트렁크 피어가 이해할 수 있는 전화 걸기 형식으로 수정할 수 있습니다. 예를 들어 전화 걸기 문자열의 시작 부분에서 +44를 제거하고 대신 0144를 넣는 변환 규칙을 작성할 수 있습니다.

이제 Lync Server 2013를 사용 하 여 필요한 경우 발신자의 전화 번호와 수신자 전화 번호를 서로 다른 형식으로 변환할 수 있습니다. 이러한 유연성을 통해 Lync Server 2013이 서로 다른 전화 통신 시스템 간에 트렁크 변환기로 사용 될 수 있습니다.
