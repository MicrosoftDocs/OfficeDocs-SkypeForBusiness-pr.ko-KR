# <a name="new-caller-id-feature-in-lync-server-2013"></a>Lync Server 2013의 새 발신자 번호 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-05_

Lync Server 2013는 관리자에 게 전화 상대의 전화 번호 형식을 수정할 수 있는 유연성을 제공 합니다. 이 발신자 ID 프레젠테이션 기능을 통해 관리자는 전화 상대의 전화 번호를 필요한 경우 트렁크 피어에서 인식 하는 전화 걸기 형식으로 수정할 수 있습니다. 예를 들어 다이얼 문자열의 시작 부분에서 + 44을 제거 하 고 0144으로 바꾸는 번역 규칙을 작성할 수 있습니다.

이제 Lync Server 2013를 사용 하면 필요에 따라 호출자의 전화 번호와 호출 수신자의 전화 번호를 서로 다른 형식으로 변환할 수 있습니다. 이러한 유연성을 통해 Lync Server 2013는 서로 다른 전화 통신 시스템 간에 트렁크 번역자 역할을 할 수 있습니다.
