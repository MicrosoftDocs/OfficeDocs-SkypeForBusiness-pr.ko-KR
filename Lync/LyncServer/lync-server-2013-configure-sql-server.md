# <a name="configure-sql-server-in-lync-server-2013"></a>Lync Server 2013에서 SQL Server 구성

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2015-01-22_

배포 하는 각 데이터베이스에 대해 데이터베이스 서버에 collocated 수 있는 Lync Server 2013 배포의 모든 데이터베이스에 대해 단일 SQL Server 인스턴스를 사용할 수 있습니다. 데이터베이스 collocation에 대 한 자세한 내용은 지원 가능성 설명서의 [Lync server 2013에서 지원 되는 서버 위치](lync-server-2013-supported-server-collocation.md) 를 참조 하세요.

또한 데이터베이스를 설정 하거나 Windows PowerShell cmdlet을 사용 하 여 데이터베이스를 수동으로 만드는 토폴로지 작성기의 단계를 완료 하기 전에 각 SQL Server 인스턴스를 설치 하 고 사용할 수 있어야 합니다. SQL Server 지원 가능성에 대 한 자세한 내용은 [Lync Server 2013의 하드웨어 설정을](lync-server-2013-hardware-setup.md)참조 하세요.
