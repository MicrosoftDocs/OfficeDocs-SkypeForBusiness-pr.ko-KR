# <a name="requirements-for-unified-contact-store-in-lync-server-2013"></a>Lync Server 2013의 통합 연락처 저장소 요구 사항

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainBody">

_**마지막으로 수정 된 항목:** 2012-10-01_

다음 목록에서는 Lync Server 2013에서 통합 연락처 저장소를 구현 하기 위한 요구 사항에 대해 설명 합니다.

  - Microsoft Lync Server 2013 및 Exchange 2013을 실행 해야 합니다.

  - 사용자는 Lync 2013를 사용 하 여 Lync Server 2013에서 Exchange 2013로의 연락처 마이그레이션을 시작 해야 합니다.

  - 사용자 사서함은 Exchange 2013로 마이그레이션해야 합니다.

  - Lync Server 2013과 Exchange 2013 간에 서버 간 인증이 구성 되어 있어야 합니다.
    
    <div>
    

    > [!NOTE]  
    > Lync Server 2013 및 Exchange 2013 간에 인증을 설정 하는 방법에 대 한 자세한 요구 사항은 작업 설명서의 <A href="lync-server-2013-managing-server-to-server-authentication-oauth-and-partner-applications.md">Lync server 2013에서 서버 간 인증 (OAuth) 및 파트너 응용 프로그램 관리</A> 를 참조 하세요.
