# <a name="configure-iis-for-lync-server-2013"></a>Lync Server 2013에 대 한 IIS 구성

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2011-12-16_

Lync Server 2013에 대해 IIS (인터넷 정보 서비스)를 구성 하려면 Lync Server 2013에서 필요한 웹 서비스를 지원 하기 위한 올바른 구성 요소를 설치 해야 합니다. IIS를 설치 하는 방법에 대 한 자세한 내용은 [Lync Server 2013의 iis 구성을](lync-server-2013-iis-configuration.md)참조 하십시오. 서버에서 보안 구성 마법사를 실행 하는 데 필요한 정책 (예를 들어, 서비스에 적용 하거나 일반적인 유지 관리 작업의 일환으로)을 사용 하는 경우에는 [보안 구성 마법사가 iis에서 포트를 닫은 후 서버를 다시 활성화](lync-server-2013-re-activate-server-after-security-configuration-wizard-closes-ports-in-iis.md) 하는 방법에 대 한 자세한 내용은 Lync SERVER 2013 iis 구성에서 포트를 닫기 위해 마법사가 실행 되는 부작용에 대 한 정보
