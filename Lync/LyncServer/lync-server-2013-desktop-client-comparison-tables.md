# <a name="desktop-client-comparison-tables-for-lync-server-2013"></a>Lync Server 2013 용 데스크톱 클라이언트 비교 표

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2018-01-23_

다음 표에서는 다음 범주에 해당 하는 Lync 2013 클라이언트의 특징과 기능을 비교 합니다.

  - 향상 된 현재 상태 지원

  - 연락처 및 대화 상대 그룹 지원

  - IM 지원

  - 회의 지원

  - 전화 통신 지원

  - 외부 사용자 지원

  - 보관, 규정 준수 및 로깅 지원

  - 최신 인증

이러한 테이블은 Lync Server 2013의 온-프레미스 배포에서 Lync 사용자가 사용할 수 있는 기능을 나타냅니다. 별도로 지정 되지 않은 경우 비즈니스용 Skype Online 및 Office 365 사용자도 동일한 기능을 사용할 수 있습니다.

<div>


> [!IMPORTANT]  
> Lync 2013 모바일 클라이언트의 기능 가용성에 대 한 자세한 내용은 <A href="lync-server-2013-mobile-client-comparison-tables.md">Lync Server 2013에 대 한 모바일 클라이언트 비교 표</A> 를 참조 하십시오.



</div>

<div>


> [!NOTE]  
> <UL>
> <LI>
> <P>비즈니스용 Skype 온라인 구독 계획에 대 한 자세한 내용은에서 <A href="https://go.microsoft.com/fwlink/p/?linkid=282430">https://go.microsoft.com/fwlink/p/?LinkId=282430</A>비즈니스용 Skype Online 서비스 설명을 참조 하세요.</P>
> <LI>
> <P>Lync Server 2013에서는 이전에 출시 된 클라이언트 (Lync 2010, Lync 2010 Mobile, Lync Phone Edition 및 Lync 2010 Attendant)를 지원 합니다. 이러한 클라이언트에 대 한 자세한 내용은에서 <A href="https://go.microsoft.com/fwlink/p/?linkid=213798">https://go.microsoft.com/fwlink/p/?LinkID=213798</A>Lync 2010 버전의 클라이언트 비교 표를 참조 하세요.</P>
> <LI>
> <P>Lync 2013 클라이언트는 VDI (가상 데스크톱 인프라) 환경에서 오디오 및 비디오를 지원 합니다. VDI 환경의 Lync 2013 기능에 대 한 자세한 내용은 <A href="lync-server-2013-supported-virtualization-technologies-and-known-limitations.md">Lync Server 2013의 지원 되는 가상화 기술 및 알려진 제한 사항을</A>참조 하세요.</P></LI></UL>



</div>

## <a name="basic-client-limitations"></a>기본 클라이언트 제한 사항

아래 기능은 전체 클라이언트를 사용 하 여 사용할 수 있으며 기본 클라이언트에서는 사용할 수 없습니다.

  - 팀 호출 설정 관리

  - 대리인 관리

  - 다른 대화 상대 대신 전화 걸기(관리자/대리인 시나리오)

  - 대리인으로 구성된 경우 다른 대화 상대의 통화 처리

  - 많은 양의 통화 관리

  - 응답 그룹과 통화 시작

  - 통화 대기

  - 그룹 통화 받기

</div>

<span id="Online_Hybrid"></span>

<div>

## <a name="online-or-hybrid-user-account-limitations"></a>온라인 또는 하이브리드 사용자 계정 제한 사항

사용자 계정은 온라인 또는 온-프레미스로 존재할 수 있으며 해당 사용자가 사용할 수 있는 기능에 영향을 미칩니다. 비즈니스용 Skype Online의 계정을 가진 사용자는 전체 클라이언트 에서도 다음과 같은 기능에 액세스할 수 없습니다.

  - 향상 된 현재 상태: 모든 공용 사이트의 사진을 사진에 사용

  - 연락처: 응답 그룹 검색

  - IM 지원: 영구 채팅 (그룹 채팅) 통합

  - IM 지원: 영구 채팅방을 한 번 클릭으로 모임에 에스컬레이션 합니다.

  - 외부 사용자: 외부 사용자와의 두 사용자 간 또는 단체 통화 수행
