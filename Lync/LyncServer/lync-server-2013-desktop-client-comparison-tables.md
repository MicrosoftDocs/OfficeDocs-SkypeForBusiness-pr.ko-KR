# <a name="desktop-client-comparison-tables-for-lync-server-2013"></a>Lync Server 2013의 데스크톱 클라이언트 비교 표

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2018-01-23_

다음 표에서는 Lync 2013 클라이언트의 기능 및 기능과 다음 범주를 비교 합니다.

  - 향상 된 현재 상태 지원

  - 연락처 및 연락처 그룹 지원

  - 메신저 대화 지원

  - 회의 지원

  - 전화 통신 지원

  - 외부 사용자 지원

  - 보관, 준수 및 로깅 지원

  - 최신 인증

다음 표는 Lync Server 2013의 온-프레미스 배포에서 Lync 사용자가 사용할 수 있는 기능을 나타냅니다. 다른 방식으로 명시 되지 않는 한, 비즈니스용 Skype Online 및 Office 365 사용자도 동일한 기능을 사용할 수 있습니다.

<div>


> [!IMPORTANT]  
> Lync 2013 모바일 클라이언트의 기능 사용 가능성에 대 한 자세한 내용은 <A href="lync-server-2013-mobile-client-comparison-tables.md">Lync Server 2013의 모바일 클라이언트 비교 표</A> 를 참조 하세요.



</div>

<div>


> [!NOTE]  
> <UL>
> <LI>
> <P>비즈니스용 Skype Online 구독 요금제에 대 한 자세한 내용은의 비즈니스용 Skype Online 서비스 설명을 참조 하세요 <A href="https://go.microsoft.com/fwlink/p/?linkid=282430">https://go.microsoft.com/fwlink/p/?LinkId=282430</A>.</P>
> <LI>
> <P>Lync Server 2013는 Lync 2010, Lync 2010 모바일, Lync Phone Edition, Lync 2010 Attendant 등 이전에 릴리스된 다음 클라이언트를 지원 합니다. 이러한 클라이언트에 대 한 자세한 내용은에서 <A href="https://go.microsoft.com/fwlink/p/?linkid=213798">https://go.microsoft.com/fwlink/p/?LinkID=213798</A>Lync 2010 버전의 클라이언트 비교 표를 참조 하세요.</P>
> <LI>
> <P>Lync 2013 클라이언트는 VDI (가상 데스크톱 인프라) 환경에서 오디오 및 비디오를 지원 합니다. VDI 환경의 Lync 2013 기능에 대 한 자세한 내용은 <A href="lync-server-2013-supported-virtualization-technologies-and-known-limitations.md">Lync Server 2013의 지원 되는 가상화 기술 및 알려진 제한 사항을</A>참조 하세요.</P></LI></UL>



</div>

## <a name="client-limitations"></a>클라이언트 제한 사항

<span id="Full_Basic"></span>

<div>

## <a name="basic-client-limitations"></a>기본 클라이언트 제한 사항

아래 기능은 전체 클라이언트를 사용 하 여 사용할 수 있으며 기본 클라이언트에서는 사용할 수 없습니다.

  - 팀 통화 설정 관리

  - 대리인 관리

  - 다른 대화 상대를 대신 하 여 전화 걸기 (관리자/대리인 시나리오)

  - 대리인으로 구성 된 경우 다른 사람의 통화 처리

  - 대량 통화 관리

  - 응답 그룹에 대 한 통화 시작

  - 통화 공원

  - 그룹 통화 픽업

</div>

<span id="Online_Hybrid"></span>

<div>

## <a name="online-or-hybrid-user-account-limitations"></a>온라인 또는 하이브리드 사용자 계정 제한 사항

사용자 계정은 온라인 또는 온-프레미스로 존재할 수 있으며 해당 사용자가 사용할 수 있는 기능에 영향을 줍니다. 비즈니스용 Skype Online에 계정이 있는 사용자는 전체 클라이언트를 사용 하는 경우에도 다음 기능에 액세스할 수 없습니다.

  - 향상 된 현재 상태: 모든 공용 사이트의 사진을 사진으로 사용

  - 연락처: 응답 그룹 검색

  - 메신저 대화 지원: 영구 채팅 (그룹 채팅) 통합

  - IM 지원: 한 번의 클릭으로 영구 채팅방을 모임으로 확대

  - 외부 사용자: 외부 사용자와 함께 2-파티 또는 단체 통화를 실시 합니다.
