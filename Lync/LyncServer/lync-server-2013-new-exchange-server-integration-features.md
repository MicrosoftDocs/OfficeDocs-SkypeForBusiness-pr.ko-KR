# <a name="new-exchange-server-integration-features-in-lync-server-2013"></a>Lync Server 2013의 새로운 Exchange Server 통합 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-24_

Lync Server 2013는 다음 목록에 설명 된 대로 Microsoft Exchange Server 2013와 함께 배포 되는 경우 새로운 기능을 지원 합니다. 이러한 기능 각각에 대해 작업을 수행 하려면 Lync Server 2013 및 Exchange 2013을 모두 배포 해야 합니다.

  - Lync 대화 상대 목록이 Exchange 2013에 저장 되어 있는 **통일 된 연락처 저장소**를 사용할 수 있으며 lync 2013, outlook 2013 및 Outlook Web App에서 연락처 저장소를 관리할 수 있습니다.

  - 고해상도 **사진을** 사용 하 여 연락처를 만들 수 있습니다. 648x648 x 까지의 사진이 Exchange 2013에 저장 되며 Lync 2013, Outlook 2013, Microsoft Lync Web App, Outlook Web App을 비롯 한 클라이언트에서 사용할 수 있게 됩니다.

  - Lync Server 2013 보관을 2013 Exchange 2013에 속한 사용자에 게 통합 하는 **Lync 보관 통합**기능을 사용 하도록 설정 하 여 규정 준수 및 eDiscovery에 대 한 일반적인 경험을 관리자에 게 사용할 수 있습니다.
