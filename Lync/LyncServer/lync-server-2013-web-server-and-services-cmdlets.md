# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Lync Server 2013의 웹 서버 및 서비스 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-02-18_

대부분의 Microsoft Lync Server 2013 구성 요소는 웹 서비스 또는 웹 페이지를 사용 하 여 작업을 수행 합니다. 웹 서버 및 웹 서비스 cmdlet을 사용 하면 웹 서버 설정을 구성 하 고 단순 Url을 관리 하는 등의 작업을 수행할 수 있습니다. 간단한 Url을 통해 사용자가 모임 및 전화 회의에 보다 쉽게 참가할 수 있으며 관리자가 Lync Server 2013 제어판에 쉽게 로그온 할 수 있습니다.
