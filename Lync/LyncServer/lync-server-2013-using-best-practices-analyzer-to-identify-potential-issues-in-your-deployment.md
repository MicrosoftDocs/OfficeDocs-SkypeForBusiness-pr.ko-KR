# <a name="using-best-practices-analyzer-to-identify-potential-issues-in-your-lync-server-2013-deployment"></a>모범 사례 분석기를 사용 하 여 Lync Server 2013 배포의 잠재적인 문제 식별

</div>

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-21_
