# <a name="using-best-practices-analyzer-to-identify-potential-issues-in-your-lync-server-2013-deployment"></a>모범 사례 분석기를 사용 하 여 Lync Server 2013 배포에서 발생할 수 있는 문제 식별

</div>

<div id="mainSection">

<div id="mainBody">

_**마지막으로 수정 된 항목:** 2012-09-21_
