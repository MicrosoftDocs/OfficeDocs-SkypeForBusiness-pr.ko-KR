# <a name="deploying-devices-in-lync-server-2013"></a>Lync Server 2013에서 장치 배포

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-09-10_

Lync Server 2013에는 Lync Phone Edition, 적격 장치에서 실행 되는 소프트웨어 및 일반적인 및 고급 전화 통신 기능, 통합 보안, 관리 효율성 등이 제공 됩니다. Lync Phone Edition은 lync Server 2010에서와 마찬가지로 Lync Server 2013와 동일한 방식으로 작동 합니다. 장치 배포에 대 한 자세한 내용은 Lync Server 2010 TechNet Library에서 [Lync Phone Edition 배포](http://go.microsoft.com/fwlink/p/?linkid=285880) 를 참조 하십시오.
