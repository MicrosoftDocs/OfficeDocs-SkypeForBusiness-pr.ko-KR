# <a name="deploying-devices-in-lync-server-2013"></a>Lync Server 2013에서 장치 배포

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-09-10_

Lync Server 2013에는 Lync Phone Edition, 정식 장치에서 실행 되는 소프트웨어가 포함 되어 있으며, 기존 및 고급 전화 통신 기능, 통합 보안, 관리 효율성 등을 제공 합니다. Lync Phone Edition은 lync server 2010에서와 동일한 방식으로 Lync Server 2013와 동일 하 게 작동 합니다. 장치 배포에 대 한 자세한 내용은 Lync Server 2010 TechNet 라이브러리에서 [Lync Phone Edition 배포](http://go.microsoft.com/fwlink/p/?linkid=285880) 를 참조 하세요.
