# <a name="supported-devices-in-lync-server-2013"></a>Lync Server 2013에서 지원 되는 장치

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-08_

Lync Server는 다음 표에 나열 된 모든 전화를 지원 합니다. 지원 되는 모든 전화기에서 Lync Phone 에디션을 실행할 수 있는 것은 아닙니다 (일부는 이전 버전 으로만 실행할 수 있음). 이 표를 사용 하 여 Lync Phone Edition을 실행할 수 있는 전화와 지원 되는 모든 전화에 대 한 자세한 정보를 얻을 수 있는 위치를 알아봅니다.

<div>


> [!NOTE]  
> 지원 되는 통화에는 일반 전화기 전화 (책상에서 직원이 사용할 수 있도록 설계 된 수화기 IP 또는 USB 장치), 회의 장치 (회의실에서 사용 하도록 설계 된 사용이 가능 하지 않은 IP 또는 USB 전화기) 및 일반 지역 전화 (IP 전화)가 있습니다. 로비, kitchens, 공장, 공유 작업 영역 등의 공유 영역에서 사용 하도록 디자인 되었으며, 다양 한 Lync Phone 에디션 기능 집합을 제공 하도록 사용자 지정할 수 있습니다.



</div>

Lync와 작동 하도록 최적화 된 모든 장치에 대 한 자세한 비교 테이블은에서 <http://go.microsoft.com/fwlink/p/?linkid=208938>"Microsoft Lync 용으로 적격 한 전화 및 장치"를 참조 하세요. 이 웹사이트의 전화 비교 표에는 가격 정보와 기술 사양이 포함 되어 있습니다.
