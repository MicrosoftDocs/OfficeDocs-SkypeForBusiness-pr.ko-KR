# <a name="quality-of-experience-qoe-in-lync-server-2013"></a>Lync Server 2013의 환경 품질 (체감 품질)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-11-01_

환경 품질 (체감 품질)은 통화 및 세션과 관련 된 참가자, 장치 이름, 드라이버, IP 주소, 끝점 형식에 대 한 미디어 품질과 정보를 표시 하는 숫자 데이터를 기록 합니다. Lync Server 2013을 설치 하는 경우 체감 품질에 대 한 미리 정의 된 전역 구성 설정 모음도 설치 합니다. 이 섹션의 항목을 사용 하 여 체감 품질 설정을 구성 합니다.
