# <a name="running-large-meetings-in-lync-server-2013"></a>Lync Server 2013에서 대규모 모임 실행

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2019-01-17_

회의에 포함 된 사용자를 수백 명까지 사용 하는 경우에는 대규모 모임의 온라인 세션을 전담 전용으로 유지 하는 것이 좋습니다. 이 전담 관리자는 모임 이끌이의 대리자거나 조직의 대규모 모임 지원 스태프의 일원일 수 있습니다. 전담 모임 중재자를 모임이 예약된 시간에 발표자로 추가해야 하지만, 모임이 진행되는 동안에는 온라인 모임 참가자를 발표자 역할로 승격시킬 수 있습니다.

모임 중재자는 Lync Server 2013 클라이언트의 다음과 같은 발표자 기능을 사용 하 여 대규모 모임을 관리할 수 있습니다.

- 모임 액세스 유형을 변경 합니다.

- Lync 끌어서 놓기 기능, 휴대폰으로 전화 걸기 또는 전자 메일을 사용 하 여 모임 중에 추가 참가자를 초대 합니다.

- 콘텐츠 업로드, 콘텐츠 삭제 및 활성 콘텐츠 전환과 같은 모임 콘텐츠를 관리 합니다.
