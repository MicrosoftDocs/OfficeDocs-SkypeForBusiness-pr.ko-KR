# <a name="start-edge-servers-in-lync-server-2013"></a>Lync Server 2013에서 에지 서버 시작

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-01-16_

Edge 서버 및 부하 분산 장치 설정을 완료 한 후에 각에 지 서버에서 서비스를 시작 해야 합니다.

<div>

## <a name="to-start-the-services"></a>서비스를 시작 하려면

1.  각 Edge 서버에서 배포 마법사의 **4 단계: 서비스 시작**옆에 있는 **실행**을 클릭 합니다.

2.  **Lync Server 15 서비스 시작** 페이지에서 서비스 목록을 검토 하 고 **다음** 을 클릭 하 여 서비스를 시작 합니다.

3.  서비스가 시작 되 면 **마침을** 클릭 하 여 마법사를 닫습니다.

4.  **4 단계: 서비스 시작**에서 **서비스 상태 (선택 사항)** 를 클릭 합니다.

5.  서버의 MMC ( **서비스** Microsoft Management Console)에서 모든 Lync server 2013 서비스가 실행 중인지 확인 합니다.
