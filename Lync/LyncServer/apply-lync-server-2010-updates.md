# <a name="apply-lync-server-2010-updates"></a>Lync Server 2010 업데이트 적용

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-19_

Lync Server 2013로 마이그레이션하기 전에 Lync Server 2010 환경에 업데이트를 적용 해야 합니다. Lync Server 2010에 대 한 최신 정보는 **업데이트 리소스 센터 For lync** at <https://go.microsoft.com/fwlink/p/?linkid=232630>를 참조 하십시오.

Lync Server 2010에 대 한 업데이트를 설치 하려면 Microsoft 기술 자료 문서 "Lync Server 2010에 대 한 업데이트"에 설명 된 **방법 1 누적 서버 업데이트 설치 관리자** 절차를 따르는 것 [https://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=2493736](https://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=2493736)이 좋습니다.
