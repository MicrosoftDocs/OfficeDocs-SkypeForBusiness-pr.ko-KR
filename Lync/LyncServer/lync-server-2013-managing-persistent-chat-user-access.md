# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>Lync Server 2013에서 영구 채팅 사용자 액세스 관리

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-06_

다음 항목에서는 Lync Server 2013 제어판 및 **영구 채팅** 페이지를 사용 하 여 영구 채팅의 사용자 액세스를 관리 하는 방법을 설명 합니다. 영구 채팅 서버를 관리 하려면 시스템에 영구 채팅 관리자 권한 및 사용 권한이 있어야 합니다. 영구 채팅 서버를 설치 하는 경우 필요한 영구 채팅 관리자 권한 및 사용 권한이 있는 사용자를 하나 이상 지정 합니다. 추가 사용자 계정에 대 한 관리자 권한 및 사용 권한을 부여 하려면 **사용자 및 사용자 그룹 관리**를 사용 합니다. 자세한 내용은 [Lync Server 2013에서 사용자가 범주 관리, 채팅방, 사용자 권한 및 사용 권한 설정을](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md)참조 하세요.
