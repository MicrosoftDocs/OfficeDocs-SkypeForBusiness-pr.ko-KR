# <a name="windows-update-for-lync-server-2013"></a>Lync Server 2013 용 Windows Update

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-12-05_

Windows Update Services를 통해 업데이트와 보안 업데이트를 자주 확인하고 적용합니다. 이렇게 하면 공격자가 관리자 권한으로 Microsoft Lync Server 2013을 실행 하는 서버에 액세스 하 고 Lync Server 2013를 손상 시킬 수 있는 다른 시스템 구성 요소의 취약성을 방지할 수도 있습니다.

Microsoft SQL Server 2008 Express (64 비트 버전)에 대 한 업데이트는 업그레이드 해야 하는 경우를 제외 하 고, 각 Lync Server 2013 Standard Edition Server (백 엔드 데이터베이스용) 및 다른 모든 Lync Server 2013 서버 역할 (로컬 구성 저장소)에서 실행 됩니다. SQL Server 2008 R2 Express 데이터베이스 프런트 엔드 풀의 백 엔드 데이터베이스에 있는 SQL Server, 모니터링 데이터베이스 및 보관 데이터베이스와 같이 이러한 데이터베이스는 일상적인 보안 업데이트 유지 관리의 일부로 생각해야 합니다.

<div>

## <a name="best-practice"></a>유용한 정보

  - Windows Update를 통해 최신 상태를 유지합니다.
