# <a name="windows-update-for-lync-server-2013"></a>Lync Server 2013용 Windows Update

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-12-05_

Windows Update Services를 사용 하 여 업데이트 및 보안 업데이트를 자주 확인 하 고 적용 합니다. 이렇게 하면 공격자가 관리자 권한으로 Microsoft Lync Server 2013를 실행 하는 서버에 액세스할 수 있고 Lync Server 2013을 손상 시킬 수 있는 다른 시스템 구성 요소의 취약점이 방지 됩니다.

Microsoft SQL Server 2008 Express (64 비트 버전)에 대 한 업데이트는 각 Lync Server 2013 Standard Edition server (백 엔드 데이터베이스용) 및 다른 모든 Lync Server 2013 서버 역할 (로컬 구성 저장소의 경우)에서 실행 되며,이를 업그레이드 하지 않은 경우 SQL Server 2008 R2 Express 데이터베이스 프런트 엔드 풀, 모니터링 데이터베이스 및 보관 데이터베이스의 백 엔드 데이터베이스에 있는 SQL Server와 같이 루틴 보안 업데이트 유지 관리의 일부로 이러한 데이터베이스를 고려해 야 합니다.

<div>

## <a name="best-practice"></a>모범 사례

  - Windows Update를 사용 하 여 최신 상태로 유지
