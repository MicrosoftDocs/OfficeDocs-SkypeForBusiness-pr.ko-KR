# <a name="associate-subnets-with-network-sites-for-e9-1-1-in-lync-server-2013"></a>E9의 네트워크 사이트와 서브넷 연결 Lync Server 2013에서-1-1

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-02_

E9-1-1에 대해 사용 하려는 네트워크의 모든 서브넷은 특정 네트워크 사이트에 연결 되어 있어야 합니다. 이는 끝점이 위치한 네트워크 사이트를 확인 하는 데 서브넷 정보를 사용 하기 때문입니다.

E9-1-1에는 서브넷을 네트워크 사이트에 연결 하기 위한 특별 한 요구 사항이 없습니다. 토폴로지의 서브넷과 네트워크 사이트 간에 연결을 만들려면 [Lync Server 2013에서 서브넷을 네트워크 사이트와 연결](lync-server-2013-associate-a-subnet-with-a-network-site.md)의 절차를 따릅니다.
