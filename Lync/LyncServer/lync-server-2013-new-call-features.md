# <a name="new-call-features-in-lync-server-2013"></a>Lync Server 2013의 새로운 통화 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2012-10-10_

Lync Server 2013에서는 착신 전환 및 동시 신호 울림에 대 한 보다 광범위 한 구성 옵션을 제공 합니다. 예를 들어 조직에서 PSTN (공중 전화망)에 외부로 들어오는 전화가 전달 되지 않도록 하려는 경우 관리자는 특수 음성 정책을 적용 하 여이 제한을 배포할 수 있습니다.

또한 대리인은 이제 해당 관리자에 대 한 수신 전화에 대해 모바일 장치로 동시 신호음 울림을 설정할 수 있습니다. 이를 통해 대리인이 보다 유연 하 게 제공 되므로 일반 전화기에 연결 하지 않고도 관리자를 대신 하 여 통화에 응답할 수 있습니다.
