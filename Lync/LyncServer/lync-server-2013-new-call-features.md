# <a name="new-call-features-in-lync-server-2013"></a>Lync Server 2013의 새로운 통화 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-10_

Lync Server 2013는 착신 전환 및 동시 연결을 위한 광범위 한 구성 옵션을 제공 합니다. 예를 들어 조직이 PSTN (공개 전환 전화 네트워크)에 외부로 착신 전환 되는 것을 원하지 않는 경우 관리자는 특수 음성 정책을 적용 하 여이 제한을 배포할 수 있습니다.

또한 대리인은 이제 해당 관리자에 게 걸려오는 전화를 받을 수 있도록 모바일 장치에 동시 신호음을 설정 합니다. 이렇게 하면 대리자가 더 융통성이 제공 되어 일반 전화기에 연결 되지 않고 관리자 대신 전화를 받을 수 있습니다.
