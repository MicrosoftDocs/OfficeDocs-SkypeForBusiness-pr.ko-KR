# <a name="client-and-device-hardware-support-in-lync-server-2013"></a>Lync Server 2013의 클라이언트 및 장치 하드웨어 지원

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-01_
