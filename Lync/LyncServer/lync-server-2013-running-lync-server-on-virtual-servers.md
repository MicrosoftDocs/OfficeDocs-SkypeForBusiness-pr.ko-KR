# <a name="running-lync-server-2013-on-virtual-servers"></a>가상 서버에서 Lync Server 2013 실행

_**마지막으로 수정 된 항목:** 2014-03-13_

Lync Server 2013에서는 IM (인스턴트 메시징) 및 현재 상태, 회의, Enterprise Voice, 모니터링, 보관 및 영구 채팅을 비롯 한 모든 Lync Server 작업을 지 원하는 가상화 토폴로지를 지원 합니다. 가상 토폴로지의 Lync Server 성능은 사용 중인 작업, 사용자 수 및 호스트 하드웨어에 따라 크게 다를 수 있습니다. 가상 서버에서 Lync Server 2013을 실행 하는 방법에 대 한 자세한 지침은 [가상 서버에서 Lync server 2013 배포 계획](https://www.microsoft.com/download/details.aspx?id=41936)을 참조 하십시오.

Lync Server 2013는 Hyper-v 플랫폼과 Windows Server 가상화 유효성 검사 프로그램에서 지원 되는 모든 가상화 플랫폼에서 지원 됩니다. 이 프로그램에 대 한 자세한 내용은를 참조 하세요 <http://www.windowsservercatalog.com/svvp.aspx> .
