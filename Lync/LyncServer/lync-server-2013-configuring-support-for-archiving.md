# <a name="configuring-support-for-archiving-in-lync-server-2013"></a>Lync Server 2013에서 보관에 대 한 지원 구성

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-01_

토폴로지에 보관을 추가 하 고 새 토폴로지를 게시 한 후에는 배포에서 보관을 처음으로 구현 하는 방법에 대 한 옵션을 구성한 다음 배포에 대 한 보관을 사용 하도록 하나 이상의 보관 정책을 구성 해야 합니다. 필요에 따라 특정 사이트 및 사용자의 경우 Lync Server 2013 제어판을 사용 하 여이 작업을 수행할 수 있습니다.

<div>


> [!NOTE]  
> 배포 후 보관 설정을 변경 하 여 보관을 사용 하거나 사용 하지 않도록 설정할 수 있습니다. 일일 관리에 대 한 보관 지원을 구현 하는 방법에 대 한 자세한 내용 또는 배포 후 조직의 새 요구 사항을 충족 하는 방법에 대 한 자세한 내용은 운영 설명서에서 <A href="lync-server-2013-managing-archiving.md">Lync Server 2013 보관 관리</A> 를 참조 하세요.
