# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Lync Server 2013의 새로운 통합 연락처 저장소 기능

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-13_

Lync Server 2013에는 통합 된 연락처 저장소가 도입 되었습니다. 통합 된 연락처 저장소를 통해 사용자는 Microsoft Exchange Server 2013에서 모든 연락처 정보를 유지할 수 있습니다. 사용자의 Lync 대화 상대를 Exchange 2013으로 마이그레이션한 후 사용자는 Lync 2013, Outlook 또는 Outlook Web App에서 연락처에 액세스 하 고 관리할 수 있으며, 즐겨찾기는 동기화 된 상태로 유지 됩니다. 예를 들어 사용자가 Outlook에서 즐겨찾기에 연락처를 추가 하는 경우 Lync 2013의 즐겨찾기 그룹에 해당 연락처가 나타납니다. 사용자가 Outlook 또는 Outlook Web App에서 연락처를 관리 하기 위해 Lync에 로그인 할 필요는 없습니다.

통합 된 대화 상대 저장소는 기본적으로 사용 하도록 설정 됩니다. 사이트별, 테 넌 트 또는 개인 또는 개인 그룹 별로 통합 된 대화 상대 저장소의 사용자를 전역적으로 사용 하거나 사용 하지 않도록 설정할 수 있습니다.
