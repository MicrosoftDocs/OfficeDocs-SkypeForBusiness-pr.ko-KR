# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>System Center Operations Manager를 사용 하 여 Lync Server 2013 모니터링

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2015-05-06_

Lync server 배포를 모니터링 하기 위한 선택 사항에 대 한 모니터링 솔루션으로 서,이 MP (서버 관리 팩)가 있습니다.

MP는 기존 이벤트 로그 및 성능 카운터 기반 계측을 구현 하 고, 여러 주요 상태 표시기에 대 한 쌍 이벤트 (실패/성공)와 같은 Lync 서버에서 새로 사용할 수 있는 계측 기능을 사용할 수 있도록 하 고 새 가상\* 트랜잭션 (테스트-Cs Windows PowerShell cmdlet)을 완전 하 게 구현 합니다.

Lync Server 2013 관리 팩과 관련 설명서는에서 [http://go.microsoft.com/fwlink/p/?LinkId=400468](http://go.microsoft.com/fwlink/p/?linkid=400468)확인할 수 있습니다. System Center Operations Manager 2012를 실행 하는 경우이 방법을 권장 합니다.
