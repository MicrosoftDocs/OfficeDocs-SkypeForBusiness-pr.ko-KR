# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>System Center Operations Manager를 사용 하 여 Lync Server 2013 모니터링

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2015-05-06_

Lync server 관리 팩 (MP)은 Lync 서버 배포를 모니터링 하기 위해 선택할 수 있는 모니터링 솔루션입니다.

MP는 일반적인 이벤트 로그 및 성능 카운터 기반 계측을 구현 하며, 몇 가지 주요 상태 표시기에 대 한 쌍 이벤트 (오류/성공)와 같이 Lync Server에서 새로 사용 가능한 계측을 사용 하도록 설정 하 고, 새 가상 트랜잭션\* (테스트-Cs Windows PowerShell cmdlet)을 완벽 하 게 구현 합니다.

Lync Server 2013 관리 팩과 관련 설명서는에서 [https://go.microsoft.com/fwlink/p/?LinkId=400468](https://go.microsoft.com/fwlink/p/?linkid=400468)찾을 수 있습니다. System Center Operations Manager 2012를 실행 하는 경우이 방법이 권장 됩니다.
