<span data-ttu-id="b39e7-104">Lync Server 2013는 두 가지 주요 방법으로 RBAC (역할 기반 액세스 제어) 기능을 향상 시킵니다.</span><span class="sxs-lookup"><span data-stu-id="b39e7-104">Lync Server 2013 enhances the role-based access control (RBAC) feature in two major ways.</span></span> <span data-ttu-id="b39e7-105">이제 지정한 cmdlet 집합에 대 한 권한이 있는 사용자 지정 역할을 만들 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b39e7-105">You can now create custom roles, which each have privileges for only a set of cmdlets you specify.</span></span> <span data-ttu-id="b39e7-106">이러한 사용자 지정 역할은 cmdlet 스크립트를 실행 하기 위한 권한을 부여할 수도 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b39e7-106">These custom roles can also be given privilege to run scripts of cmdlets.</span></span>

<span data-ttu-id="b39e7-107">또한 Lync Server 2013에는 두 가지 새로운 미리 정의 된 역할이 포함 되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b39e7-107">Additionally, Lync Server 2013 includes two new predefined roles.</span></span>
