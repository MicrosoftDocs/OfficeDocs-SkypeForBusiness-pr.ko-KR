# <a name="setting-who-can-post-messages-in-an-auditorium-chat-room-in-lync-server-2013"></a>Lync Server 2013에서 강당 채팅방에서 메시지를 게시할 수 있는 사용자 설정

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-10-06_

Auditorium 채팅방에서 발표자 역할을 부여 받은 사용자만 메시지를 게시할 수 있습니다. 다른 모든 구성원은 메시지를 읽을 수만 있습니다. Auditorium 채팅방의 발표자는 채팅방의 구성원 이어야 합니다.

Windows PowerShell 명령줄 인터페이스를 사용 하 여 auditorium 채팅방을 관리 하는 방법에 대 한 자세한 내용은 배포 설명서에서 [채팅방 관리](manage-rooms.md) 를 참조 하세요.

영구적인 채팅방 관리자와 채팅방 관리자는 채팅방 설정을 관리할 수 있지만, **발표자**가 아닌 경우에는 auditorium 채팅방에 게시할 수 없습니다.
