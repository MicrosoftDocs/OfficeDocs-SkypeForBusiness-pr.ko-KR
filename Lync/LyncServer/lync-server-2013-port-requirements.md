# <a name="port-requirements-for-lync-server-2013"></a>Lync Server 2013의 포트 요구 사항

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2013-03-27_

Lync Server를 사용 하려면 방화벽의 특정 포트를 열어야 합니다. 또한 사용자의 조직에 인터넷 프로토콜 보안 (IPsec)을 배포 하는 경우 오디오, 비디오, 파노라마 비디오를 제공 하는 데 사용 되는 포트 범위를 통해 IPsec을 사용 하지 않도록 설정 해야 합니다.
