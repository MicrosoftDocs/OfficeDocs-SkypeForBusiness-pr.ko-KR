# <a name="planning-for-branch-site-voice-resiliency-in-lync-server-2013"></a>Lync Server 2013의 분기 사이트 음성 복구 계획

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-21_

가용성 엔터프라이즈 음성 서비스를 제공 하는 경우에는 다음과 같은 세 가지 방법으로 지점 사이트의 복원을 할 수 있습니다.

  - Survivable 분기 기기

  - Survivable Branch 서버

  - 지점 사이트의 전체 Lync Server 배포

이 가이드는 조직에 가장 적합 한 탄력성 솔루션을 평가 하는 데 도움이 되며 PSTN 연결 솔루션에 해당 하는 복원 솔루션을 기반으로 합니다. 또한 필수 구성 요소 및 기타 계획 고려 사항을 설명 하 여 선택한 솔루션을 배포 하도록 준비 하는 데 도움이 됩니다.
