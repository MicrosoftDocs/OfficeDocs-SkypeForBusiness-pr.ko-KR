# <a name="client-management-cmdlets-in-lync-server-2013"></a>Lync Server 2013의 클라이언트 관리 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정한 주제:** 2012-09-27_

클라이언트 관리는 주로 Microsoft Lync 2013 등의 클라이언트 응용 프로그램에서 Lync Server 2013에 로그온 하 고 해당 클라이언트 응용 프로그램에서 로그온 한 후 사용할 수 있는 기능을 결정 하는 것으로 구성 됩니다.
