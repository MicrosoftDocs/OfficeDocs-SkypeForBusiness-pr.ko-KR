# <a name="lync-server-2013-stress-and-performance-tool"></a>Lync Server 2013 스트레스 및 성능 도구

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**마지막으로 수정 된 항목:** 2013-01-25_

Lync Server 2013 스트레스 및 성능 도구에는 Lync Server 2013에 대 한 용량 계획을 간소화 하는 도구가 포함 되어 있습니다. Lync Server 2013 스트레스 및 성능 도구는 다음 작업을 수행 하는 데 도움이 됩니다.

  - Lync Server 2013에 대 한 하드웨어 계획을 간소화 합니다.

  - 성능 조정에 대 한 향상 된 지식과 모범 사례를 제공 합니다.

  - 의도 한 Lync Server 2013 배포의 성능을 측정 합니다.
