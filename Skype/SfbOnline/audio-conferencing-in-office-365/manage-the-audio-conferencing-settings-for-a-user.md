# <a name="manage-the-audio-conferencing-settings-for-a-user-in-skype-for-business-online"></a>비즈니스용 Skype Online에서 사용자의 오디오 회의 설정 관리

> [!Note]
> Microsoft Teams에서 사용자 설정을 관리하려면 Microsoft Teams에서 사용자의 오디오 회의 설정 관리를 [참조하세요.](/MicrosoftTeams/manage-the-audio-conferencing-settings-for-a-user-in-teams)

Microsoft 365 또는 Office 365 관리자는 조직의 개별 사용자에 대해 공급자, 기본 무료 또는 무료 번호, 전화 회의 ID 또는 PIN과 같은 오디오 회의 설정을 편집할 수 있습니다. 조직의 설정을 편집하려면 조직의 오디오 회의 설정 관리를 [참조하세요.](manage-the-audio-conferencing-settings-for-my-organization.md)

 
1. 직장 또는 학교 계정으로 로그인합니다.
    
2. 관리 **센터**  >  **비즈니스용 Skype를 선택하세요.**
    
3. 비즈니스용 Skype 관리 센터에서 사용자를 **선택하세요.**
    
4. 설정을 관리할 사용자를 선택한 다음 작업 창에서 편집 표시 아이콘 편집을 ![ ](../images/4d8bea48-be68-4e0e-a54c-73decf7ea4ec.png) 클릭합니다.
    
5. 왼쪽 **탐색 모음에서** 오디오 회의를 선택한 다음  사용자에 대한 속성 페이지에서 다음 중 원하는 것을 수정합니다.
    
|**설정**|**설명**|
|:-----|:-----|
|**공급자 이름** <br/> |목록에서 공급자를 선택합니다.  <br/><br/> **참고:** 이 표의 나머지 설정은 Microsoft를 오디오 회의 공급자로 선택하는 경우만 적용됩니다.           |
|**기본 전화** 번호(필수) <br/> |타사 공급자의 경우 이러한 전화 번호는 오디오 회의 공급자로부터 받은 전화 번호입니다. 사용자가 Microsoft를 오디오 회의 공급자로 사용하는 경우 오디오 회의 브리지에 설정된 번호가 됩니다. 비즈니스용 Skype 및 Microsoft Teams 모임 요청에 표시하려는 번호 서식을 지정합니다.  <br/> |
|**기본 무료 번호** <br/> |타사 공급자의 경우 이러한 전화 번호는 오디오 회의 공급자로부터 받은 전화 번호입니다. 사용자가 Microsoft를 오디오 회의 공급자로 사용하는 경우 오디오 회의 브리지에 설정된 번호가 됩니다. 비즈니스용 Skype 및 Microsoft Teams 모임 요청에 표시하려는 번호 서식을 지정합니다.  <br/> |
|**조직의 Microsoft 브리지에서 무료 번호를 사용하여 이 사용자의 모임에 참가할 수 있도록 허용** <br/> |사용자가 무료 번호로 모임에 참가할 수 있도록 허용하려면 이 옵션을 선택합니다.  <br/> |
|**전자 메일을 통해 회의 정보 보내기** <br/> |전화 회의 ID와 전화 번호가 있는 전자 메일을 사용자에게 즉시 보내려면 이 링크를 클릭합니다. (이 전자 메일에는 PIN이 포함되어 있지 않습니다.) 오디오 회의 정보가 있는 사용자에게 전자 메일 [보내기를 참조하세요.](send-an-email-to-a-user-with-their-dial-in-information.md)  <br/> |
|**회의 ID** <br/> |사용자의 **회의** ID를 다시 설정해야 하는 경우 재설정을 선택합니다. 자세한 내용은 사용자의 회의 [ID 재설정을 참조하세요.](reset-a-conference-id-for-a-user.md)  <br/> |
|**PIN** <br/> |사용자에 **대한** PIN을 다시 설정해야 하는 경우 재설정을 선택합니다. 자세한 내용은 오디오 회의 PIN 재설정을 [참조하세요.](reset-the-audio-conferencing-pin.md)  <br/> |
|**비인식 발신자만 모임의 첫 번째 사람이 될 수 있도록 허용** <br/> |처음으로 모임에 참가할 수 있도록 허용하려면 이 옵션을 선택합니다.  <br/> |
|**이 사용자의 모임에서 전화 걸기 제한** <br/> |국내로만 전화 접속을 제한하거나 모든 전화 걸기에서 모임을 차단하려는 경우 이 목록에서 옵션을 선택합니다.  <br/> |
