# <a name="getting-phones-for-skype-for-business-online"></a>비즈니스용 Skype Online 휴대폰 받기

[!INCLUDE [sfbo-retirement](../../../Hub/includes/sfbo-retirement.md)]

비즈니스용 Skype 온라인에서는 데스크톱 앱을 사용하는 대신 기존 전화 환경을 원하는 사용자를 위해 데스크톱 비즈니스용 Skype 지원합니다. 이 항목에서는 조직에서 휴대폰을 설정할 때 도움이 될 수 있는 비즈니스용 Skype 온라인에서 사용할 수 있도록 지원되는 휴대폰 및 펌웨어 버전에 대해 다를 수 있습니다.

> [!NOTE]
> Skype 비즈니스의 경우 Microsoft Teams 기본 통신 방법으로 Microsoft 365 Office 365.  자세한 내용은 Office 365 인텔리전트 [통신에 대한 새로운 비전을](https://www.microsoft.com/microsoft-365/blog/2017/09/25/a-new-vision-for-intelligent-communications-in-office-365/) 참조하세요.
>
>지원되는 디바이스에 대한 최신 업데이트 및 최신 정보를 얻은 경우 지능형 통신을 위한 Microsoft Teams [디바이스를 참조하세요.](https://products.office.com/microsoft-teams/across-devices?ms.url=officecomteamsdevices&rtc=1)
  
## <a name="supported-phones"></a>지원되는 휴대폰
  
Microsoft는 PiP(파트너 IP ip 전화 프로그램)를 통해 다양한 디바이스를 개발하고 인증하기 위해 Polycom, Yealink 및 AudioCodes와 긴밀히 협력하고 전화 시스템.
  
새 휴대폰을 주문할 비즈니스용 Skype 올바른 제품 ID로 휴대폰을 *구입하는 것이 중요합니다.* 이러한 제품 아이디는 수신하는 휴대폰에 온라인 자격을 갖춘 비즈니스용 Skype 이미 설치되어 있습니다.
  
|||
|:-----|:-----|
|**전화 파트너** <br/> |**비즈니스용 Skype 특정 제품 ID** <br/> |
|Polycom  <br/> |제품 ID -019  <br/> |
|Yealink  <br/> |SIP-TXXG 비즈니스용 Skype 버전  <br/> |
|AudioCodes  <br/> |UCXXXHDEG(SfB)  <br/> |
   
Polycom 휴대폰에 대한 자세한 내용은 Poly 설명서 라이브러리 [를 참조하세요.](https://documents.polycom.com/category/voice)
  
Yealink 휴대폰에 대한 자세한 내용은 IP 휴대폰 [비즈니스용 Skype 참조](http://www.yealink.com/products_list_10.html#filter2)
  
AudioCodes 휴대폰에 대한 자세한 내용은 IP 휴대폰 비즈니스용 Skype [참조](https://www.audiocodes.com/solutions-products/products/products-for-microsoft-365/ip-phones-room-solutions)
  
> [!NOTE]
> Lync 전화 버전은 비즈니스용 Skype 온라인에서 지원되지만 Lync Microsoft Teams. LPE 플랫폼에 대한 주류 지원은 2014년 4월 10일까지 종료되어 Lync Server 2013의 제품 지원 수명 주기에 맞게 2023년 4월 11일까지 확장된 지원을 제공합니다. [LPE 수명 주기에](https://support.microsoft.com/lifecycle/search?qid=&amp;alpha=Lync%20Phone%20Edition&amp;Filter=FilterNO) 대한 자세한 내용은 Microsoft 제품 수명 주기를 참조합니다. LPE CAP 모델은 온라인에서 지원되지 비즈니스용 Skype 없습니다.
>
> 올해 하반기에는 Office 365 1.2보다 오래된 TLS 버전을 지원하지 않습니다. LPE의 주 운영 체제는 TLS 1.2를 지원하지 않습니다. LPE는 더 이상 LPE에 Office 365 지원되지 않습니다. 자세한 [내용은 TLS 1.2의](https://support.microsoft.com/en-gb/help/4057306/preparing-for-tls-1-2-in-office-365) 필수 사용 준비를 Office 365 참조하세요.

## <a name="required-licenses"></a>필수 라이선스

비즈니스용 Skype Online에서는 사용자 라이선스 이외에 추가 Microsoft 라이선스가 필요하지 않습니다. 필요한 사용자 라이선스에 대한 자세한 내용은 추가 기능 비즈니스용 Skype 및 Microsoft Teams [참조합니다.](../../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md)
  
제조업체 라이선스 모델은 열려 있는 SIP와 인증된 펌웨어 비즈니스용 Skype 다를 수 있습니다. Open SIP 펌웨어를 사용하여 인증된 모델을 다시 구매하는 경우 제조업체에서 펌웨어 라이선스 요구 사항을 확인해야 합니다.
