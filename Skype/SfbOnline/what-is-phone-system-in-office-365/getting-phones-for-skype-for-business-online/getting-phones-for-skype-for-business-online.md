# <a name="getting-phones-for-skype-for-business-online"></a>비즈니스용 Skype Online 휴대폰 받기

비즈니스용 Skype Online은 비즈니스용 Skype 앱을 사용하는 대신 기존 전화 환경을 사용하려는 사용자를 위해 데스크톱 전화를 한정하고 지원합니다. 이 항목에서는 비즈니스용 Skype Online에서 사용할 수 있도록 지원되는 전화 및 펌웨어 버전 및 조직에서 휴대폰을 설정할 때 도움이 되는 기타 정보에 대해 다 업무를 진행합니다.

> [!NOTE]
> 비즈니스용 Skype는 Microsoft 365 및 Office 365의 기본 통신 방법으로 Microsoft Teams로 서서히 대체됩니다.  자세한 [내용은 Office 365의](https://www.microsoft.com/microsoft-365/blog/2017/09/25/a-new-vision-for-intelligent-communications-in-office-365/) 지능형 통신에 대한 새로운 비전을 참조하세요.
>
>지원되는 장치에 대한 최신 업데이트 및 최신 정보를 얻은 경우 지능형 통신에 [대한 Microsoft Teams 디바이스를 참조하세요.](https://products.office.com/microsoft-teams/across-devices?ms.url=officecomteamsdevices&rtc=1)
  
## <a name="supported-phones"></a>지원되는 휴대폰
  
Microsoft는 Polycom, Yealink 및 AudioCodes와 긴밀하게 협력하여 전화 시스템에 대한 PIP(파트너 IP 전화 프로그램)를 통해 다양한 디바이스를 개발하고 인증합니다.
  
비즈니스용 Skype의 새 휴대폰을 주문할 때 올바른 제품 ID로 휴대폰을 *구입하는 것이 중요합니다.* 이러한 제품 신분을 통해 받은 휴대폰에 비즈니스용 Skype Online 적격 버전이 이미 설치되어 있습니다.
  
|||
|:-----|:-----|
|**전화 파트너** <br/> |**비즈니스용 Skype 특정 제품 ID** <br/> |
|Polycom  <br/> |제품 ID -019  <br/> |
|Yealink  <br/> |SIP-TXXG 비즈니스용 Skype Edition  <br/> |
|AudioCodes  <br/> |UCXXXHDEG(SfB)  <br/> |
   
Polycom 휴대폰에 대한 자세한 내용은 [Poly 설명서 라이브러리를 참조하세요.](https://documents.polycom.com/category/voice)
  
Yealink 전화에 대한 자세한 내용은 비즈니스용 Skype IP 전화 번호를 [참조하세요.](http://www.yealink.com/products_list_10.html#filter2)
  
AudioCodes 휴대폰에 대한 자세한 내용은 비즈니스용 Skype IP 전화 번호를 [참조하세요.](https://www.audiocodes.com/solutions-products/products/products-for-microsoft-365/ip-phones-room-solutions)
  
> [!NOTE]
> Lync Phone Edition은 비즈니스용 Skype Online에서 지원되지만 Microsoft Teams에서는 지원되지 않습니다. LPE 플랫폼에 대한 기본 지원은 2014년 4월 10일까지 종료되었습니다. Lync Server 2013의 제품 지원 수명 주기에 맞게 2023년 4월 11일까지 확장된 지원이 있습니다. LPE 수명 주기에 대한 자세한 내용은 [Microsoft](https://support.microsoft.com/lifecycle/search?qid=&amp;alpha=Lync%20Phone%20Edition&amp;Filter=FilterNO) 제품 수명 주기를 참조합니다. LPE CAP 모델은 비즈니스용 Skype Online에서 지원되지 않습니다.
>
> 올해 말, Office 365는 1.2 이전 버전의 TLS를 지원하지 않습니다. LPE의 운영 체제는 TLS 1.2를 지원하지 않습니다. LPE는 더 이상 Office 365에 연결하기 위해 지원되지 않습니다. 자세한 [내용은 Office 365에서 TLS 1.2의](https://support.microsoft.com/en-gb/help/4057306/preparing-for-tls-1-2-in-office-365) 필수 사용 준비를 참조하세요.

지원되는 휴대폰이 전화 시스템과 함께 작동하기 위해 필요한 최소 소프트웨어 릴리스입니다.

|인증된 Polycom VVX 시리즈  <br/> |5.4.0A  <br/> |2015년 12월  <br/> |

현재 인증된 펌웨어 버전에 대한 자세한 내용은 비즈니스용 Skype IP 전화 번호를 [참조하세요.](https://docs.microsoft.com/skypeforbusiness/certification/devices-ip-phones#conference-phones)

> [!NOTE]
> 온-프레미스 배포에 대해 설정한 LPE(Lync Phone Edition) 휴대폰은 해당 사용자를 비즈니스용 Skype Online으로 이동하기 전에 필요한 최소 또는 이후 펌웨어로 업데이트해야 합니다. 휴대폰의 펌웨어를 업데이트하기 전에 사용자를 비즈니스용 Skype Online으로 이동하는 경우 해당 휴대폰은 비즈니스용 Skype Online에 연결할 수 없습니다. 
  
## <a name="required-licenses"></a>필수 라이선스

비즈니스용 Skype Online에는 사용자 라이선스 이외에 추가 Microsoft 라이선스가 필요하지 않습니다. 필요한 사용자 라이선스에 대한 자세한 내용은 비즈니스용 Skype 및 Microsoft Teams 추가 기능 [라이선스를 참조하세요.](../../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md)
  
제조업체 라이선스 모델은 개방형 SIP와 비즈니스용 Skype Certified 펌웨어 간에 다를 수 있습니다. Open SIP 펌웨어를 사용하여 인증된 모델을 다시 구매하는 경우 제조업체에 펌웨어 라이선스 요구 사항을 확인해야 합니다.
