# <a name="office-365-enterprise-e5-without-audio-conferencing"></a>Office 365 Enterprise E5(오디오 회의 제외)

오디오 회의 없이 Office 365 Enterprise E5 요금제를 이미 구입한 경우 추가 기능을 비즈니스용 Skype 구매해야 하는 추가 기능 라이선스는 다음과 같습니다.

|||
|:-----|:-----|
|**이 기능을 추가하기 위해** <br/> |**구입해야 할 것이 있습니다.** <br/> |
|**오디오 회의** <br/> | 이 추가 기능을 얻은 경우 다음을 합니다. <br/><br/> 1. 이미 E5 요금제를 구입했지만 해당 국가 또는 지역에서 오디오 회의 기능을 사용할 수 없는 경우 타사 오디오 회의 공급자를 사용해야 합니다. Microsoft PinPoint에서 타사 오디오 회의 [공급자를 찾을 수 있습니다.](https://go.microsoft.com/fwlink/?LinkId=797530)  <br/><br/>  이 문서를 확인하여 국가 또는 지역에서 오디오 회의를 사용할 수 있는지 확인: 오디오 회의 및 통화 계획에 대한 국가 및 지역 [가용성](/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans) <br/><br/> 2. 사용할 수 있는 [](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a) 경우 계획 전환 단추를 사용하여 포함된 Office 365 Enterprise E5로 이동합니다. 더 많은 기능을 사용할 수 있습니다. <br/> |
|**전화 시스템 포함!** <br/> | 통화 계획: 전화 [](/MicrosoftTeams/calling-plans-for-office-365) 요금제에서 Office 365. <br/>  또는 기존 [공급자의](../../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_existing)전화 서비스를 사용합니다.  <br/> |
|**클라우드 음성 사서함 포함!** <br/> |다른 것은 필요하지 않습니다.  <br/> |
