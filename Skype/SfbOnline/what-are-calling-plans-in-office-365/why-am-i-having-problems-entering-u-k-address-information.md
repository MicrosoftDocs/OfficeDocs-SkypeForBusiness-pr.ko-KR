# <a name="why-am-i-having-problems-entering-uk-address-information"></a>영국을 입력 하는 데 문제가 발생 하는 이유 주소 정보

미국 (영국)에서 올바른 주소 또는 주소를 확인 하는 데 문제가 있을 수 있는 몇 가지 이유는 다음과 같습니다.
  
## <a name="whats-wrong-with-the-company-name"></a>회사 이름에 문제가 있나요?

회사 이름을 입력 하는 데 문제가 있는 경우 다음 사항에 유의 합니다.
  
- 여기에는 문자, 숫자, 다음과 같은 특수 문자 (/-+)가 포함 될 수 있습니다. ' &amp; " _ 
    
    > [!CAUTION]
    > 영국 (영국)에서는 회사 이름에 + 기호를 사용할 수 없습니다. 
  
- 80 자 이하의 문자 여야 합니다.
    
- 특수 문자로 시작할 수 없습니다.
    
## <a name="whats-wrong-with-the-house-number"></a>집 번호에 문제가 있나요?

집 번호를 입력 하는 데 문제가 있는 경우 다음 사항에 유의 하십시오.
  
- 문자, 숫자, 다음과 같은 특수 문자를 포함할 수 있습니다.
    
- 숫자로 시작 해야 합니다.
    
- 20 자 이하의 문자를 사용할 수 있습니다.
    
- 대문자 또는 소문자를 포함할 수 있습니다.
    
## <a name="whats-wrong-with-the-house-name"></a>집 이름에 문제가 있나요?

집 이름을 입력 하는 데 문제가 있는 경우 다음 사항을 유의 하세요.
  
- 여기에는 문자, 숫자, 다음과 같은 특수 문자 (/-+)가 포함 될 수 있습니다. ' &amp; " _
    
- 50 자 이하의 문자 여야 합니다.
    
- 특수 문자로 시작할 수 없습니다.
    
## <a name="whats-wrong-with-the-street-name"></a>거리 이름에 문제가 있나요?

거리 이름을 입력 하는 데 문제가 있는 경우 다음 사항에 유의 합니다.
  
- 여기에는 문자, 숫자, 다음과 같은 특수 문자 (/-+)가 포함 될 수 있습니다. ' &amp; " _ 
    
- 50 자 이하의 문자 여야 합니다.
    
- 특수 문자로 시작할 수 없습니다. 
    
## <a name="whats-wrong-with-the-city-or-town"></a>구/군/시에 문제가 있나요?

도시 또는 도심 이름을 입력 하는 데 문제가 있는 경우 다음을 수행 합니다.
  
- 여기에는 문자, 숫자, 다음과 같은 특수 문자 (/-+)가 포함 될 수 있습니다. ' &amp; " _
    
- 50 자 이하의 문자 여야 합니다.
    
- 특수 문자로 시작할 수 없습니다. 
    
## <a name="whats-wrong-with-the-postal-code"></a>우편 번호에 문제가 있나요?

우편 번호를 입력 하는 데 문제가 있는 경우 다음 사항을 유의 하세요.
  
- 문자, 숫자 또는 공백만 포함할 수 있습니다.
    
- 숫자와 공백을 포함 하 여 8 자 이하의 문자를 사용 해야 합니다.
    
- 영국 **(영국)에**거주 하는 경우에는 문자 a = 문자 a-z, # 기호 = 0-9의 숫자, _ (밑줄) = 빈 공간 중 한 형식으로 우편 번호를 입력 **해야 합니다** . 몇 가지 예는 다음과 같습니다.
    
|**포맷할**|**예**|
|:-----|:-----|
|AA # A_ # AA  <br/> |EC3B 2BD  <br/> |
|# A_ # AA  <br/> |E1W 1AP  <br/> |
|A # _ # AA  <br/> |F1 2AG  <br/> |
|# #_ # AA  <br/> |C44 8HT  <br/> |
|AA # _ # AA  <br/> |DR3 6HG  <br/> |
|AA # #_ # AA  <br/> |FN35 1TH  <br/> |

> [!NOTE]
> 이 보다 더 많은 전화 번호를 받으려면 [비즈니스 제품에 대 한 고객 지원에 문의-관리자 도움말](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b) 을 참조 하세요.
