# <a name="why-am-i-having-problems-entering-uk-address-information"></a>영국 입력에 문제가 있는 이유 주소 정보가 있나요?

다음은 영국(영국)에서 주소 또는 주소를 올바르게 유효성 검사하는 데 문제가 있는 몇 가지 이유입니다.
  
## <a name="whats-wrong-with-the-company-name"></a>회사 이름에 문제가 있나요?

회사 이름을 입력하는 데 문제가 있는 경우 다음을 기억해야 합니다.
  
- 문자, 숫자 및 특수 문자를 사용할 수 있습니다. , / - + . ' &amp; " _ 
    
    > [!CAUTION]
    > 영국(영국)에서는 회사 이름에 + 기호를 사용할 수 없습니다. 
  
- 80자 이하입니다.
    
- 특수 문자로 시작할 수 있습니다.
    
## <a name="whats-wrong-with-the-house-number"></a>집 번호에 문제가 있나요?

집 번호를 입력하는 데 문제가 있는 경우 다음을 기억해야 합니다.
  
- 문자, 숫자 및 특수 문자를 사용할 수 있습니다. - / [ ]
    
- 숫자로 시작해야 합니다.
    
- 20자 이하입니다.
    
- 대문자 또는 소문자를 사용할 수 있습니다.
    
## <a name="whats-wrong-with-the-house-name"></a>집 이름에 문제가 있나요?

집 이름을 입력하는 데 문제가 있는 경우 다음을 기억해야 합니다.
  
- 문자, 숫자 및 특수 문자( , / - + )를 사용할 수 있습니다. ' &amp; " _
    
- 50자 이하입니다.
    
- 특수 문자로 시작할 수 있습니다.
    
## <a name="whats-wrong-with-the-street-name"></a>도로명에 문제가 있나요?

도로명을 입력하는 데 문제가 있는 경우 다음을 기억해야 합니다.
  
- 문자, 숫자 및 특수 문자( , / - + )를 사용할 수 있습니다. ' &amp; " _ 
    
- 50자 이하입니다.
    
- 특수 문자로 시작할 수 있습니다. 
    
## <a name="whats-wrong-with-the-city-or-town"></a>도시 또는 구정에 어떤 문제가 있나요?

구/구/시의 이름을 입력하는 데 문제가 있는 경우 다음을 기억해야 합니다.
  
- 문자, 숫자 및 특수 문자( , / - + )를 사용할 수 있습니다. ' &amp; " _
    
- 50자 이하입니다.
    
- 특수 문자로 시작할 수 있습니다. 
    
## <a name="whats-wrong-with-the-postal-code"></a>우편 번호에 문제가 있나요?

우편 번호 입력에 문제가 있는 경우 다음을 기억해야 합니다.
  
- 문자, 숫자 또는 공백만 사용할 수 있습니다.
    
- 숫자와 공백을 포함하여 8자 이하의 문자를 포함해야 합니다.
    
- **영국(영국)에** 있는 경우 다음 형식 중  하나에 우편 번호를 입력해야 합니다. 여기서 문자 A = 문자 A-Z, # 기호 = 0-9의 숫자 및 _(밑 줄) = 빈 공백으로 입력해야 합니다. 다음은 몇 가지 예입니다.
    
|**서식**|**예**|
|:-----|:-----|
|AA#A_#AA  <br/> |EC3B 2BD  <br/> |
|A#A_#AA  <br/> |E1W 1AP  <br/> |
|A#_#AA  <br/> |F1 2AG  <br/> |
|A##_#AA  <br/> |C44 8HT  <br/> |
|AA#_#AA  <br/> |DR3 6HG  <br/> |
|AA##_#AA  <br/> |FN35 1TH  <br/> |

> [!NOTE]
> 이 번호보다 많은 전화 번호를 필요로 하는 경우 비즈니스 제품에 대한 지원에 [문의하세요. 관리자 도움말](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
