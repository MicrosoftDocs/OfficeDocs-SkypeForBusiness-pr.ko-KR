# <a name="blocked-user-licenses-for-hosting-skype-meeting-broadcast"></a>Skype 모임 브로드캐스트 호스팅에 대 한 차단 된 사용자 라이선스

불행 하 게도 사용자가 https://broadcast.skype.com Skype 모임 브로드캐스트를 주관 하려고 할 때 차단 되는 특정 사용자 라이선스가 있습니다. 사용자에 게이 메시지가 표시 되 면 차단 된 라이선스 중 하나가 할당 된 것입니다. 해당 앱을 계속 사용 하려면 다른 Office 365 라이선스를 할당 합니다.
  
 *"회사 정책에 따라 Skype 모임 브로드캐스트를 사용할 수 없습니다. 시스템 관리자에 게 문의 하 여 서비스를 사용 하도록 설정 하 고 올바른 라이선스를 보유 하 고 있는지 확인 합니다. "* 
  
## <a name="blocked-user-licenses"></a>차단 된 사용자 라이선스 

사용자에 게 할당 된 사용자 라이선스가 다음 중 하나라도 있으면 Skype 모임 브로드캐스트 이벤트 호스팅 차단 됩니다. 스트리밍 브로드캐스트 이벤트를 사용 하는 참석자는 사용자 라이선스가 필요 하지 않습니다.
  
- Office 365 교육 Fac/스태프
    
- Office 365 교육 학생
    
- Office 365 교육 및 Fac/스태프
    
- Office 365 교육 플러스 학생
    
- Office 365 교육 E1 Fac/스태프
    
- Office 365 교육 E1 학생
    
- Office 365 교육 E3 Fac/스태프
    
- Office 365 교육 E3 Fac/스태프 대형
    
- Office 365 교육 E3 학생
    
- Office 365 A1 학생 사용 혜택
    
- Microsoft 365 A1 학생 사용 혜택
