# <a name="skype-meeting-broadcast-preview-settings"></a>Skype 모임 브로드캐스트 미리 보기 설정

비즈니스용 Skype 고객 미리 보기 프로그램에서는 새 제품 및 기능에 대 한 초기 액세스를 제공 합니다. 이를 통해 조직에서 제공 되는 내용을 한눈에 살펴보고, 자신의 환경에서 새로운 기능을 테스트 하 고, 제품 빌드를 일반 공개로 릴리스하기 전에 의견을 제공할 수 있습니다. 자세한 정보는 [비즈니스용 Skype preview](https://www.skypepreview.com/)를 참조 하세요.
