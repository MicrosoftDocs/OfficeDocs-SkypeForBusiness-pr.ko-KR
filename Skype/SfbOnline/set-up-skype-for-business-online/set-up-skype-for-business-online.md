# <a name="set-up-skype-for-business-online"></a>비즈니스용 Skype 온라인 설정

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

전역 관리자 권한이 있어야 비즈니스용 Skype. 웹의 일부에 대한 액세스를 제한하는 방화벽 또는 프록시 서버가 있는 경우 [Microsoft](https://go.microsoft.com/fwlink/?linkid=391089) 파트너를 고용하여 사용자에 비즈니스용 Skype 있습니다.

## <a name="setting-up-skype"></a>설정 Skype

구독 또는 구독을 사용하여 Skype 설정하는 데 Microsoft 365 Office 365 모양입니다. 이 문서의 단계를 따라 설정을 완료할 수 있습니다.

## <a name="1-plan-for-skype-for-business"></a>1. 비즈니스용 Skype

표준 또는 **[Microsoft 365 Business Premium](https://products.office.com/business/office-365-business-premium)** 비즈니스 필수 요소가 있는 경우 구독을 비즈니스용 Skype 비즈니스의 다른 사용자와 온라인 통화를 할 수 있습니다. 예를 들어 비즈니스에 10명이 있는 경우 아래 2-6단계를 수행한 후 비즈니스용 SKYPE 및 [](https://support.office.com/article/2eed8424-581a-4497-b505-c08c152e5851) 온라인 모임에 대해 비즈니스용 Skype 비즈니스용 Skype 시작할 수 있습니다. [](https://support.office.com/article/cc05afa6-1894-4a82-9dd9-6222061f50fd) 또한 온라인 [모임에 비즈니스용 Skype](https://support.office.com/article/b8305620-d16e-4667-989d-4a977aad6556#bkmk_OWA) Outlook 설정할 수 있습니다.

비즈니스 외부 사용자로부터 전화를 걸고 비즈니스용 Skype  위해  다음을 사용하려는 경우:

- **옵션 1. 무료 앱 [Skype 를 사용하세요.](https://www.skype.com/)** 소규모 비즈니스(예: 1-2명)가 있는 경우 Skype 앱을 사용하는 것이 더 나은 방법입니다. 국내 및 국제 통화에 사용하는 비용은 저렴합니다. 프레젠테이션을 위해 전화 회의를 개최하고, 화상 통화를 할 수 있으며, 데스크톱을 공유할 수 있습니다. [요금 및 결제 옵션을 체크 아웃합니다.](https://secure.skype.com/en/calling-rates?wt.mc_id=legacy&amp;expo365=bundled)

- **옵션 2. 요금제 업그레이드 및** 전화 시스템 및 전화 요금제 Office 365. 이 비용의 정도를 확인한 다음 스위치를 전환하는 가장 쉬운 방법은 비즈니스 제품에 대한 지원에 [문의하고](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b) 관리자 도움말을 통해 모든 작업을 수행하는 것입니다.

자세한 내용은 비즈니스용 Office 365 [계획을 참조합니다.](https://support.office.com/article/eb926624-018b-4486-bf11-5fba6ee4d645#bkmk_skype)

## <a name="2-sign-in-to-office-365"></a>2. 로그인하여 Office 365

비즈니스용 Skype 온라인은 서비스 Office 365 제품군의 일부입니다. 온라인에서 비즈니스용 Skype 설정하려면 로그인하여 로그인해야 Office 365. 이 작업을 하는 방식은 다음과 같습니다.

1. 사용자 MICROSOFT 365 또는 Office 365 ID(예: <em>rob@fourthcoffee.com).를 찾습니다.</em> 온라인을 구매할 때 만든 Microsoft Online Services Microsoft 365 Office 365 사용자 ID가 포함된 전자 메일을 비즈니스용 Skype. 메일은 다음과 같습니다.

    ![온라인에 등록한 후 받은 환영 전자 메일의 비즈니스용 Skype 예입니다. 이 문서에는 사용자 Microsoft 365 Office 365 포함되어 있습니다.](../images/977c5c96-29c5-40c0-a4c4-1ba66ba3a1fb.png)

2. 관리 센터에 [로그인하고](https://admin.microsoft.com) 사용자 ID 및 암호를 Microsoft 365 Office 365 입력합니다. 

## <a name="3-set-up-your-domain-and-users"></a>3. 도메인 및 사용자 설정
<a name="bkmk_users"> </a>

이제 로그인하여 Office 365 조직 내 사용자와 도메인을 비즈니스용 Skype 있습니다.

1. [도메인 및](https://support.office.com/article/6383f56d-3d09-4dcb-9b41-b5f5a5efd611)Office 365 추가 : Office 365 설정 마법사를 사용하여 사용자 지정 도메인(예: *fourthcoffee.com)을* Office 365. **기본적으로 Office 365 설정 마법사에는 온라인 비즈니스용 Skype 설정하고 사용자 ID를 비즈니스용 Skype 포함됩니다.** 마법사를 사용하여 도메인을 Office 365 경우 이 단계를 완료합니다.

2. [도메인 및 DNS](https://support.office.com/article/2b54e1b0-47a7-4018-a1e4-c2b924e7c5a0)연결 확인 : 도메인 문제 해결사 - 도메인 및 DNS 설정이 올바른지 확인합니다. 이 작업을 수행하면 나중에 DNS 설정을 향후 문제의 원인으로 제거할 수 있게 됐기 때문에 나중에 설정 문제를 알아내기 위해 많은 도움이 됩니다.

3. [Office 365 URL](https://support.office.com/article/8548a211-3fe7-47cb-abb1-355ea5aa88a2#BKMK_LYO)및 IP 주소 범위: 대부분의 소규모 기업에서는 이 단계를 할 필요가 없습니다. **그러나 웹의** 일부에 대한 액세스를 제한하는 방화벽 또는 프록시 서버가 있는 경우 온라인 엔드포인트에 대한 액세스를 허용하는 비즈니스용 Skype 만들어야 합니다. 방화벽 및 프록시 서버를 구성한 경험이 있는 사용자가 가장 잘 수행한 고급 단계입니다. 전에 이 일을 수행하지 않은 경우 [Microsoft](https://go.microsoft.com/fwlink/?linkid=391089) 파트너를 고용하여 사용자에 대한 비즈니스용 Skype 고려하세요.

## <a name="4-set-up-im-and-presence-in-your-organization"></a>4. 조직에서 IM 및 현재 상태 설정
<a name="bkmk_IM"> </a>

메신저(메신저) 및 현재[](https://support.office.com/article/fea86e34-60cf-4dd0-bfb2-169a42afd92c)상태(비즈니스용 Skype 현재 상태 정보에 대한 제어 액세스)는 기본 비즈니스용 Skype. 기본적으로 비즈니스의 사람들은 서로 Skype IM을 사용할 수 있습니다.

1. **사용자가 통신할 수 있는 비즈니스용 Skype 다른 사용자를 선택합니다.**

   - [사용자가 외부 사용자와](allow-users-to-contact-external-skype-for-business-users.md) 비즈니스용 Skype 허용 사용자와 *다른* 비즈니스 모두 시스템을 구성해야 합니다.

     **중요:** 비즈니스에 두 개의 도메인(예: rob@contosowest.com ina@contosoeast.com)이 있는 경우 이 단계를 통해 모든 사용자가 서로 통신할 수 있습니다.

   - [사용자가 비즈니스용 Skype 외부에](let-skype-for-business-users-add-skype-contacts.md) Skype 추가

2. **공동 작업자가 온라인인지** 여부를 보는 사용자 선택: 현재 상태 기능은 온라인 상태인 사람 및 사용 가능한 가용성(예: 사용 가능, 사용 중, 비우기 또는 발표)을 보여줍니다.

    ![개인 메시지가 있는 사람의 온라인 상태의 예입니다.](../images/ab6c10b4-6ad5-453c-bf0e-459b977b6e23.png)

    비즈니스의 모든 사용자에 대한 기본 설정을 선택할 수 있습니다.

   - 조직의 모든 사용자에 대한 사람의 온라인 현재 상태 자동으로 표시

   - 연락처에만 사람의 온라인 현재 상태 표시

지침은 온라인에서 존재 [비즈니스용 Skype 참조하세요.](configure-presence-in-skype-for-business-online.md)

### <a name="set-up-audio-conferencing"></a>오디오 회의 설정

조직의 사람들이 전화를 사용하여 모임에 전화해야 하는 경우도 있습니다. 비즈니스용 Skype 상황에 대한 오디오 회의 기능이 포함되어 있습니다. 모바일 장치 또는 PC에서 비즈니스용 Skype 대신 휴대폰을 사용하여 비즈니스용 Skype 모임에 호출할 수 있습니다.

### <a name="set-up-phone-system-and-the-calling-plans-in-office-365"></a>전화 시스템 및 통화 계획을 Office 365

전화 시스템 기능으로 Office 365 전화 시스템을 제공합니다. 조직의 비즈니스용 Skype 다른 사용자에 대한 통화는 무료이며, 직원들은 서로 및 외부 발신자로부터 음성 메시지를 받을 수 있습니다. 다음은 사용자와 함께 얻을 전화 시스템.

전화 요금제 서비스를 추가하면 직원들은 전화 번호에 기본 전화 번호를 비즈니스용 Skype. 비즈니스 외부에서 전화를 걸고 받을 수 있습니다. VoIP 휴대폰, PC 및 모바일 장치에서 음성 통화를 할 수 있습니다. 또한 긴급한 경우 911을 호출하여 도움을 줄 수 있습니다.

단계별 설정 지침은 통화 계획 설정을 참조하세요.

### <a name="set-up-skype-meeting-broadcast"></a>브로드캐스트 Skype 모임 설정

Skype 모임 브로드캐스트는 최대 10,000명이 참석하는 모임을 생성, 호스트 및 브로드캐스트할 수 있는 기능입니다. **작동 방식에 대한 자세한 내용은 브로드캐스트의 Skype 모임 [참조하세요.](https://support.office.com/article/c472c76b-21f1-4e4b-ab58-329a6c33757d)**

다음은 브로드캐스트를 설정하는 Skype 모임 개요입니다.

1. [비즈니스용](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc)라이선스 할당 또는 Office 365 : 브로드캐스트 모임을 비즈니스용 Skype 모든 비즈니스용 Skype 온라인 또는 Enterprise  계획 라이선스를 할당합니다.  

2. [브로드캐스트 Skype 모임](../set-up-your-network-for-skype-meeting-broadcast/enable-skype-meeting-broadcast.md)사용 : 기본적으로 이 기능은 사용하도록 설정되지 않습니다. 설정한 후 사용자는 조직의 다른 사용자와 브로드캐스트 모임을 호스트할 수 있습니다.

3. [브로드캐스트에 Skype 모임](../set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast.md)네트워크 설정 : 조직 외부의 참석자들과 웨비나 및 기타 브로드캐스트를 호스트하려면 네트워크를 구성해야 합니다.

4. [브로드 Skype 모임캐스트를 Skype 모임](https://support.office.com/article/c3995bc9-4d32-4f75-a004-3bc5c477e553) 브로드캐스트 참가 : [브로드캐스트에서 브로드캐스트를](https://support.office.com/article/14689da0-821d-48d4-9035-ea762de80ebe)예약한 다음 Skype 모임 모임에 참가하려고 하여 브로드캐스트 모임이 작동해야 *https://portal.broadcast.skype.com* 합니다.

## <a name="learn-about-network-connectivity-requirements"></a>네트워크 연결 요구 사항에 대해 자세히 알아보기
<a name="bkmk_more"> </a>

오디오, 비디오 및 애플리케이션 공유의 품질은 비즈니스용 Skype 네트워크 연결의 품질에 크게 영향을 미치게 됩니다. 최적의 환경을 위해 회사 네트워크와 온라인 간에 고품질의 비즈니스용 Skype 것이 중요합니다. 네트워크 및 튜닝 정보는 온라인 성능 [비즈니스용 Skype 을 참조하세요.](https://support.office.com/article/beec23c2-c5d6-4e84-a8af-e82aefca7802)
