# <a name="customize-meeting-invitations"></a>모임 초대장의 사용자 지정

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

관리자는 모든 모임 초대에 지원 웹 사이트비즈니스용 Skype 법적 고지 조항에 대한 링크, 회사 로고 및 포함하려는 기타 텍스트와 같은 사용자가 필요할 수 있는 몇 가지 유용한 정보를 포함할 수 있습니다. 
  
## <a name="tips-for-creating-a-logo-for-meeting-invitations"></a>모임 초대장에 대한 로고 만들기 팁
<a name="__top"> </a>

1. 높이가 30픽셀(매우 작음)으로 188픽셀보다 작은 이미지를 만듭니다.
    
2. 이미지를 JPG 형식으로 저장합니다.
    
3. 네트워크 공유, 공유 폴더, 공유 폴더, 비즈니스용 OneDrive 등 비즈니스의 모든 사람이 액세스할 수 있는 Dropbox 저장합니다.
    
    이제 이것을 모임 초대장에 추가할 수 있습니다. 다음 단계를 참조하세요.
    
## <a name="customize-your-meeting-invitations"></a>모임 초대장의 사용자 지정
<a name="__top"> </a>

![비즈니스용 skype 로고를 나타내는 아이콘](../images/sfb-logo-30x30.png) **비즈니스용 Skype 관리 센터 사용**

1. 직장 또는 학교 계정으로 로그인합니다.
    
2. 관리 센터에서 **관리** 센터를  >  비즈니스용 Skype.
    
3. 관리 비즈니스용 Skype 에서 온라인 **모임을 선택하세요.** 
    
4. 다음 상자를 완료합니다.
    
    ![모임 초대를 사용자 지정하기 위해 양식을 완료하면 다음과 같은 모양이 될 수 있습니다.](../images/b0a7c3c6-0d86-41c6-b116-331143bbe398.png) 

   **로고 URL:** 로고가 저장되는 URL을 입력합니다. JPG 형식이 있어야 합니다. 
 
   **도움말 URL:** 조직에 지원 웹 사이트가 있는 경우 사용자가 문제가 있는 경우 여기에 URL을 입력합니다. 

   **법적 URL:** 조직에 법적 문제가 있는 경우 사용자가 이동하려는 법적 웹 사이트가 있는 경우 여기에 URL을 입력합니다.
    
   **발자국 텍스트:** 사용자에게 적합한 텍스트 전용 각주를 입력합니다.  
  
   
5. 저장을 **선택 합니다.**
    
6. 변경 내용이 모든 서비스 또는 서비스에서 채워질 때까지 Microsoft 365 Office 365 기다렸다. 그런 다음 Outlook 변경 내용을 Skype 모임을 예약합니다. 
    
    ![그래픽이 없는 사용자 지정 초대의 모양은 다음과 같습니다.](../images/ebb5c03c-c23d-4da7-97f1-9b13e26a6cf8.png)
