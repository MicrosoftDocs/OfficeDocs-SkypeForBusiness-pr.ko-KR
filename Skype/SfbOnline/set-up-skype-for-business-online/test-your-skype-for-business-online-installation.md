# <a name="test-your-skype-for-business-online-installation"></a>비즈니스용 Skype Online 설치 테스트

조직의 모든 사용자에 대해 설정하기 전에 비즈니스용 Skype Online 설치를 테스트하여 시간을 절약하고, 지원 통화를 줄이고, 사용자 만족도를 높입니다.

필요한 사항은 다음과 같습니다.

- 3개 이상의 Microsoft 365 또는 Office 365 계정(사용자 및 두 개 이상의 다른 계정)

- 각 테스트 계정에 대한 컴퓨터입니다. 조직에 있는 일반적인 컴퓨터처럼 설정할 수 있습니다.

- 비즈니스용 Skype Online용 오디오 회의 공급자가 있는 계정입니다.

## <a name="set-up-test-accounts"></a>테스트 계정 설정
<a name="__toc328126910"> </a>

1. 관리   >  **Microsoft 365** 또는 **Office 365** 사용자 및 그룹으로 이동한 다음 추가를 선택하고 필요한  >   ![ 정보를 ](../images/328ffb57-5f31-430a-b653-4a6b8e76d338.png) 입력합니다.

2. 4단계(전자 메일)에 오면 자신의 전자 메일 주소를 입력합니다. 그런 다음 새 사용자의 이름 및 암호에 대한 레코드가 생성됩니다.

3. 원하는 테스트 계정 수가 될 때까지 1단계와 2단계를 반복합니다. 비즈니스용 Skype Online의 온라인 모임 기능을 테스트하려면 사용자 계정 외에 두 개 이상의 계정이 필요합니다.

## <a name="set-up-test-computers"></a>테스트 컴퓨터 설정
<a name="__toc328126911"> </a>

각 테스트 컴퓨터에서:

1. Microsoft 365 또는 Office 365 홈페이지로 이동하여 테스트 계정 중 하나의 자격 증명으로 로그인합니다.

2. 설정 **설정으로** 이동: 프로필을 업데이트하고, 소프트웨어를 설치하고, 클라우드에 연결한 다음, 소프트웨어 설치를 클릭하고 클라우드에 ![ ](../images/4b83e9cb-c7e4-46c8-b3d1-cfee017123ae.png) **연결합니다.**

## <a name="set-up-audio-conferencing"></a>오디오 회의 설정
<a name="__toc328126912"> </a>

비즈니스용 Skype Online 모임에 대한 전화 액세스를 제공하려면 오디오 회의 공급자를 통해 계정을 설정하세요. 다음과 같은 것을 얻을 수 있습니다.

- 무료 전화 접속 번호 및 사용 가능한 경우 무료 번호

- 모임을 예약하거나 진행하는 조직의 각 사용자에 대해 회의 코드 및 PIN(개인 식별 번호)입니다.

오디오 회의를 위해 사용자를 설정한 후 전화 접속 번호와 전화 회의 코드가 있는 자동화된 전자 메일 메시지를 받게 됩니다. 이 정보는 새 비즈니스용 Skype 모임 요청에도 자동으로 추가됩니다.

 **테스트 사용자 중 하나의 계정에 오디오 회의 정보를 추가하려면**

1. 오디오 **회의 사용자를**  >  **클릭합니다.**

2. 전화 접속 회의를 설정하려는 사용자의 이름을 클릭한 다음 **편집을** ![ ](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png) 클릭합니다.

3. 오디오 회의 공급자를 선택하고 요청된 정보를 입력한 다음 저장을 **클릭합니다.**

|**오디오 회의 설정**|**설명**|
|:-----|:-----|
|**공급자** <br/> |목록에서 오디오 회의 공급자를 선택합니다.  <br/> |
|**전화 번호(필수)** <br/> |오디오 회의 공급자로부터 받은 전화 번호입니다. 비즈니스용 Skype 모임 초대에 표시하려는 번호 서식을 지정합니다.  <br/> |
|**무료 번호** <br/> |오디오 회의 공급자로부터 받은 전화 번호입니다. 비즈니스용 Skype 모임 초대에 표시하려는 번호 서식을 지정합니다.  <br/> |
|**암호** <br/> |이 사용자의 암호 또는 회의 코드입니다.  <br/> |
