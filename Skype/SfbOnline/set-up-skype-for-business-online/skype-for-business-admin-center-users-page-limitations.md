# <a name="skype-for-business-online-admin-center-users-page-limitations"></a>비즈니스용 Skype 온라인 관리 센터 사용자 페이지 제한 사항

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

성능상의 이유로 비즈니스용 Skype 관리 **센터에** 사용자 목록에 처음 1,000명이 표시됩니다. 사용자가 1,000명을 넘은 경우 **필터 필터를** ![ 사용 합니다.](../images/d360528a-a05a-4f0d-a7fd-25a6c696206a.png) 검색을 통해 원하는 특정 인재를 찾을  ![ 수 ](../images/5cc29d50-4eec-48a4-bdad-bbeae230f9ae.png) 있습니다.
  
## <a name="bulk-editing"></a>대량 편집
<a name="__top"> </a>

1. 목록 맨 위에 있는 표시 이름 옆에 있는 상자를 선택하여 처음 1,000명 또는 필터 또는 검색에서 반환된 모든 사용자를 선택합니다. 
    
2. 편집을  ![ 클릭합니다. ](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png) .
    
목록 맨 아래에 있는 컨트롤을 사용하여 각 페이지에 표시된 사용자 수를 변경할 수 있습니다.
  
![페이지의 사용자 양을 변경하기 위한 사용자 페이지 탐색입니다.](../images/7d8ba24e-e3e8-4ff8-92f7-98ac143aab6d.png)
