# <a name="skype-for-business-online-support-in-outlook-on-the-web"></a>웹용 Outlook의 비즈니스용 Skype Online 지원

Office 365의 웹용 outlook (Outlook Web App)은 탐색 모음에서 기본적인 비즈니스용 Skype 웹 클라이언트를 제공 합니다. 관리자가 Office 365 조직에 대 한 vanity URL을 구성 하지 않은 온라인 사용자는이 기본 클라이언트를 사용할 수 있습니다. 사용자 계정이 온라인 상태이 고 vanity URL이 없는 한, 조직에는 온-프레미스 사용자 계정이 있는 경우에도 계속 해 서 환경이 표시 됩니다. 온-프레미스 (vanity URL이 있는지 여부) 또는 Microsoft에서 관리 하는 사용자 계정이 있는 사용자는 Outlook web app에서 Lync 환경을 볼 수 있습니다.
  
다음 표에는 사용할 수 있는 다양 한 설치 및 사용 되는 웹 클라이언트가 요약 되어 있습니다.
