# <a name="turn-on-or-off-mobile-phone-notifications"></a>휴대폰 알림 설정 또는 해제

> [!IMPORTANT]
> Microsoft Teams 관리 센터가 비즈니스용 Skype 관리 센터(레거시 포털)를 대체했습니다. 이제 비즈니스용 Skype를 관리하기 위한 모든 설정이 Teams 관리 센터에 있습니다. 자세한 내용은 [Microsoft Teams 관리 센터에서 비즈니스용 Skype 설정 관리](https://docs.microsoft.com/MicrosoftTeams/skype-for-business-settings?toc=/skypeforbusiness/sfbotoc/toc.json&bc=/skypeforbusiness/breadcrumb/toc.json)를 참조하세요.

**[조직에 대한 Microsoft 365](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** 비즈니스용 앱의 관리자 역할 할당으로, 비즈니스용 Skype 사용자가 휴대폰이나 태블릿에 있는 경우 수신 및 부재 중 인스턴트 메시지에 대한 알림을 받을지 여부를 선택할 수 있습니다.
  
Android 및 Windows Phone에서 비즈니스용 Skype 알림이 실시간으로 표시됩니다. 그러나 Windows Phone, iPhone 및 iPad 장치의 경우 푸시 알림은 휴대폰이나 태블릿에서 비즈니스용 Skype를 적극적으로 사용하지 않을 때마다 알림을 표시하는 데 사용됩니다.
  
## <a name="turn-push-notifications-off-for-all-the-windows-phone-or-apple-devices-in-your-organization"></a>조직의 모든 Windows Phone 또는 Apple 디바이스에 대한 푸시 알림 끄기
<a name="__top"> </a>

![비즈니스용 skype 로고를 나타내는 아이콘](../images/sfb-logo-30x30.png) **비즈니스용 Skype 관리 센터 사용**

1. Microsoft 365에 로그인합니다.
    
2. 비즈니스용 Skype의 > **관리 센터로 이동하세요.**
    
3. 조직 **일반으로**  >  **이동** 
    
4. 휴대폰 **알림에서** 사용하지 않도록 설정할 알림 서비스 옆에 있는 상자의 선택을 취소한 다음 저장을 **클릭합니다.**
    
유의 사항: 
  
- 푸시 알림을 끄면 사용자가 모바일 장치에서 비즈니스용 Skype를 다시 시작할 때 모든 알림이 계속 수신됩니다.
    
- 푸시 알림은 기본적으로 켜져 있습니다. 개별 사용자는 모바일 장치에서 적절한 비즈니스용 Skype 옵션을 선택하여 해제할 수 있습니다.
    
- 푸시 알림을 해제하면 사용자가 푸시 알림을 다시 끄지 못합니다.
    
> [!IMPORTANT]
>  Microsoft는 다른 회사를 사용하여 Windows Phone, iPhone 및 iPad 사용자에게 비즈니스용 Skype 모바일 알림을 실시간으로 제공합니다. Microsoft 비즈니스용 Skype 제품에 대한 [개인정보처리방침을 참조하세요.](https://go.microsoft.com/fwlink/p/?linkid=247732)
