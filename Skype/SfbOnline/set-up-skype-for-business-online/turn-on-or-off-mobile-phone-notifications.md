# <a name="turn-on-or-off-mobile-phone-notifications"></a>휴대폰 알림 설정 또는 해제

> [!IMPORTANT]
> Microsoft Teams 관리 센터가 비즈니스용 Skype 관리 센터(레거시 포털)를 대체했습니다. 이제 비즈니스용 Skype를 관리하기 위한 모든 설정이 Teams 관리 센터에 있습니다. 자세한 내용은 [Microsoft Teams 관리 센터에서 비즈니스용 Skype 설정 관리](/MicrosoftTeams/skype-for-business-settings?bc=%2fskypeforbusiness%2fbreadcrumb%2ftoc.json&toc=%2fskypeforbusiness%2fsfbotoc%2ftoc.json)를 참조하세요.

**[조직에 대한 Microsoft 365 비즈니스용](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** 앱의 관리자 역할 할당으로 비즈니스용 Skype 사용자가 휴대폰 또는 태블릿에 있는 경우 수신 및 누락된 인스턴트 메시지에 대한 경고를 받을지 여부를 선택할 수 있습니다.
  
Android 및 Windows Phone에서 비즈니스용 Skype 알림이 실시간으로 표시됩니다. 그러나 Windows Phone, iPhone 및 iPad 디바이스의 경우 푸시 알림은 휴대폰 또는 태블릿에서 비즈니스용 Skype를 적극적으로 사용하지 않을 때마다 경고를 표시하는 데 사용됩니다.
  
## <a name="turn-push-notifications-off-for-all-the-windows-phone-or-apple-devices-in-your-organization"></a>조직의 모든 Windows Phone 또는 Apple 디바이스에 대한 푸시 알림을 해제합니다.
<a name="__top"> </a>

![비즈니스용 skype 로고를 나타내는 아이콘](../images/sfb-logo-30x30.png) **비즈니스용 Skype 관리 센터 사용**

1. Microsoft 365에 로그인합니다.
    
2. 비즈니스용 Skype > **관리 센터로 이동하세요.**
    
3. 조직 일반  >  **으로 이동** 
    
4. 휴대폰 **알림에서** 사용하지 않도록 설정하려는 알림 서비스 옆에 있는 상자를 선택 취소한 다음 저장을 **클릭합니다.**
    
유의 사항: 
  
- 푸시 알림을 해제하면 사용자가 모바일 장치에서 비즈니스용 Skype를 다시 시작할 때 모든 경고가 계속 수신됩니다.
    
- 푸시 알림은 기본적으로 켜져 있습니다. 개별 사용자는 모바일 장치에서 적절한 비즈니스용 Skype 옵션을 선택하여 해제할 수 있습니다.
    
- 푸시 알림을 해제하면 사용자가 푸시 알림을 다시 켜지 못합니다.
    
> [!IMPORTANT]
>  Microsoft는 다른 회사를 사용하여 Windows Phone, iPhone 및 iPad 사용자에 대한 실시간 비즈니스용 Skype 모바일 알림을 제공합니다. 비즈니스용 [Microsoft Skype 제품에 대한 개인 정보 취급 방침 을 참조하세요.](https://go.microsoft.com/fwlink/p/?linkid=247732)
