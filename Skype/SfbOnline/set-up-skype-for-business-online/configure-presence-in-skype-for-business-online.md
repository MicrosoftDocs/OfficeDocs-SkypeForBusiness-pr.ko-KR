# <a name="configure-presence-in-skype-for-business-online"></a>비즈니스용 Skype Online에서 현재 상태 구성

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

> [!IMPORTANT]
> Microsoft Teams 관리 센터가 비즈니스용 Skype 관리 센터(레거시 포털)를 대체했습니다. 관리에 대한 모든 비즈니스용 Skype 관리 센터에 Teams 있습니다. 전역 관리자 또는 비즈니스용 Skype [관리자의 Azure AD](/azure/active-directory/roles/permissions-reference) 관리자 역할이 비즈니스용 Skype 관리 센터에서 Teams 합니다. 자세한 내용은 [Microsoft Teams 관리 센터에서 비즈니스용 Skype 설정 관리](/MicrosoftTeams/skype-for-business-settings?bc=%2fskypeforbusiness%2fbreadcrumb%2ftoc.json&toc=%2fskypeforbusiness%2fsfbotoc%2ftoc.json)를 참조하세요.

기본적으로 조직의 사용자 중 한 명이 해당 사용자와 통신할 수 비즈니스용 Skype 온라인인지도 볼 수 있습니다. 비즈니스용 Skype 온라인, 모임, 오프라인 또는 다른 표시기에서 사용할 수 있는지 여부를 보여줍니다.

![에 있는 사람의 온라인 상태의 비즈니스용 Skype.](../images/f0849132-1ddb-480f-bca6-cfe9eaa0486d.png)

비즈니스의 **[](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** 모든 사용자에 대한 관리자로, 온라인에서 자신의 온라인 존재를 보는 사람을 선택할 수 비즈니스용 Skype.

![비즈니스용 skype 로고를 나타내는 아이콘](../images/sfb-logo-30x30.png) **비즈니스용 Skype 관리 센터 사용**

1. 관리 센터 > 관리 **센터로**  >  **비즈니스용 Skype.**

2. 관리 **비즈니스용 Skype** 에서 조직 을 **선택하세요.**

3. 현재 **상태 개인 정보 모드에서** 다음 설정 중 하나를 선택한 다음 저장을 **선택합니다.**

|**설정**|**Who 사용자의 현재 상태 보기**|
|:-----|:-----|
|**현재 상태 정보를 자동으로 표시합니다.** <br/> |비즈니스용 Skype 외부 목록에 추가되지 않은 모든 사용자 또는 사용자의  온라인  현재 상태는 볼 수 있습니다. <br/> |
|**사용자의 연락처에만 현재 상태 정보 표시** <br/> |외부 또는 차단된 목록에 추가하지 않은 사람의  연락처 목록에 있는 모든 **사람이 나열됩니다.** <br/> 개인은 도구 옵션 에서 기본 설정을 비즈니스용 Skype **수** 있습니다. 설정  >    >  **있습니다.** <br/> |

사용자가 변경될 수 있는 내용에 대한 비즈니스용 Skype 다음 문서를 참조하세요.

- [현재 상태 정보에 대한 액세스 제어 비즈니스용 Skype](https://support.office.com/article/fea86e34-60cf-4dd0-bfb2-169a42afd92c)

- [에서 상태 옵션 비즈니스용 Skype](https://support.office.com/article/efd25395-c8ef-4510-b9cb-6f70e2fff8a0)
