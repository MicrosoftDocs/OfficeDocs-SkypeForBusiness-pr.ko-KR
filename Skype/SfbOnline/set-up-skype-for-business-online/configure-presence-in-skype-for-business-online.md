# <a name="configure-presence-in-skype-for-business-online"></a>비즈니스용 Skype Online에서 현재 상태 구성

> [!IMPORTANT]
> Microsoft Teams 관리 센터가 비즈니스용 Skype 관리 센터(레거시 포털)를 대체했습니다. 이제 비즈니스용 Skype를 관리하기 위한 모든 설정이 Teams 관리 센터에 있습니다. Teams 관리 센터에서 비즈니스용 Skype 기능을 관리하려면 전역 관리자 또는 비즈니스용 Skype 관리자의 [Azure AD](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference) 관리자 역할이 할당되어야 합니다. 자세한 내용은 [Microsoft Teams 관리 센터에서 비즈니스용 Skype 설정 관리](https://docs.microsoft.com/MicrosoftTeams/skype-for-business-settings?toc=/skypeforbusiness/sfbotoc/toc.json&bc=/skypeforbusiness/breadcrumb/toc.json)를 참조하세요.

기본적으로 비즈니스용 Skype를 사용하여 조직의 사용자 중 하나와 통신할 수 있는 모든 사람은 해당 사람이 온라인인지 여부를 볼 수 있습니다. 비즈니스용 Skype는 온라인, 모임, 오프라인 또는 다른 표시기에서 해당 사람을 사용할 수 있는지 여부를 보여줍니다.

![비즈니스용 Skype의 온라인 상태 예입니다.](../images/f0849132-1ddb-480f-bca6-cfe9eaa0486d.png)

비즈니스의 **[모든](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)** 사용자에 대한 관리자로 비즈니스용 Skype에서 온라인 상태의 누가 볼 수 있는지 선택할 수 있습니다.

![비즈니스용 skype 로고를 나타내는 아이콘](../images/sfb-logo-30x30.png) **비즈니스용 Skype 관리 센터 사용**

1. 관리 센터 >   >  **비즈니스용 Skype로 이동하세요.**

2. 비즈니스용 **Skype 관리 센터에서** 조직을 **선택하세요.**

3. 현재 **상태 개인 정보 모드에서** 다음 설정 중 하나를 선택한 다음 저장을 **선택합니다.**

|**설정**|**사용자의 현재 상태 볼 수 있는 사용자**|
|:-----|:-----|
|**현재 상태 정보 자동으로 표시** <br/> |사용자의 외부 또는 차단 목록에 추가되지 않은 비즈니스용  Skype  사용자는 해당 사용자의 온라인 현재 상태 정보를 볼 수 있습니다. <br/> |
|**사용자의 연락처에만 현재 상태 정보 표시** <br/> |외부 또는 차단 목록에 추가하지 않은 사람의 연락처 목록에 **있는** **모든 사람입니다.** <br/> 개인은 비즈니스용 Skype 앱인 설정 도구 옵션에서 기본 설정을   >  **오버라이드할 수**  >  **있습니다.** <br/> |

비즈니스용 Skype에서 사용자가 변경할 수 있는 내용에 대한 자세한 내용은 다음 문서를 참조하세요.

- [비즈니스용 Skype에서 현재 상태 정보에 대한 액세스 제어](https://support.office.com/article/fea86e34-60cf-4dd0-bfb2-169a42afd92c)

- [비즈니스용 Skype에서 상태 옵션 설정](https://support.office.com/article/efd25395-c8ef-4510-b9cb-6f70e2fff8a0)
