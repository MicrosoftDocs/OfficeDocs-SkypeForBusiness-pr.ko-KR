# <a name="the-users-skype-for-business-online-service-plan-isnt-valid"></a>사용자의 비즈니스용 Skype Online 서비스 요금제가 유효 하지 않음

죄송 합니다만,이 사용자에 게 할당 된 비즈니스용 Skype Online 서비스 요금제가 인식 되지 않거나 잘못 되었습니다.
  
*Microsoft 365 관리 센터** > **활성 사용자** 로 이동 하 여이 사용자에 게 유효한 서비스 계획을 할당 합니다.
