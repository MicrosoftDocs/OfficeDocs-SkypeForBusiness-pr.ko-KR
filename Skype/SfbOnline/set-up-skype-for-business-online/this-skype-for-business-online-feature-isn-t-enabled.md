# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>이 비즈니스용 Skype Online 기능은 사용할 수 없습니다.

죄송합니다. 이 기능에 대한 라이선스가 없는 것 같아서.
  
|**잠정 원인**|**제안된 작업**|
|:-----|:-----|
|사용 가능한 라이선스가 있지만 이 사용자에 대한 기능을 설정하지 않은 것입니다.  <br/> |이 사용자에게 라이선스를 부여하려면 **Microsoft 365** 관리 센터  >  **활성** 사용자로 이동하세요. <br/> |
|사용자가 이 기능을 사용할 수 없는 위치에 있습니다.  <br/> |가능한 경우 **Microsoft 365** 관리 센터 활성 사용자로 이동하여 이 사용자를 기능을 사용할 수 있는 위치에  >   할당합니다. <br/> |
|이 기능은 조직에 대해 설정되지 않은 것입니다.  <br/> |Microsoft **365** 관리 센터 비즈니스용 Skype로 이동하여 전체 조직에  >   대한 기능을 켜면 됩니다. <br/> |
