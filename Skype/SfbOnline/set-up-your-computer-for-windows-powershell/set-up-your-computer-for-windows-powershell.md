# <a name="set-up-your-computer-for-windows-powershell"></a>Windows PowerShell용 컴퓨터 설정

Microsoft PowerShell을 사용하여 비즈니스용 Skype Online의 원격 관리는 다음 운영 체제 중 하나를 실행하는 64비트 컴퓨터에서만 지원됩니다.

- Windows Server 2016

- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
지원되는 운영 체제 외에도 컴퓨터가 다음을 실행해야 합니다.
  
- PowerShell
    
- Teams PowerShell 모듈 
    
다음 항목에서 이러한 항목의 다운로드 및 설치에 대한 정보를 찾을 수 있습니다.
  
- [Teams Powershell 모듈 다운로드 및 설치](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-install#install-the-teams-powershell-module)
    
- [Windows PowerShell 5.1 다운로드 및 설치](download-and-install-windows-powershell-5-1.md)
