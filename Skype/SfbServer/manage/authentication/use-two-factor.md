# <a name="use-two-factor-authentication-with-skype-for-business-client-and-skype-for-business-server"></a>비즈니스용 Skype 클라이언트 및 비즈니스용 Skype 서버에서 2 단계 인증 사용
 
**요약:** 비즈니스용 skype 서버 및 비즈니스용 Skype에서 2 단계 인증을 사용 합니다.
  
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>처음으로 비즈니스용 Skype에 로그인

로그인 정보는 일반적으로 비즈니스용 Skype가 설치 될 때 자동으로 구성 됩니다. 그러나 비즈니스용 Skype를 처음 사용 하는 경우에는 수동으로 클라이언트를 시작 해야 할 수 있습니다.
  
### <a name="to-sign-in-for-the-first-time"></a>처음으로 로그인 하려면

1. 조직의 네트워크에 로그온 합니다.
    
2. 모든 **** > **** 프로그램 > 시작**비즈니스용 Skype**를 선택 합니다.
    
    로그인 화면이 표시 됩니다.
    
    - 로그인 주소 상자가 이미 입력 되어 있는 경우 표시 된 주소가 올바른지 확인 합니다.
    
    - 올바르지 않은 경우 또는 상자가 비어 있는 경우에는 로그인 주소를 입력 합니다 (일반적으로 전자 메일 주소와 동일).
    
    - 빈 암호 상자가 표시 되 면 암호를 추가 합니다.
    
3. **로그인**을 선택 합니다.

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>스마트 카드를 사용 하 여 비즈니스용 Skype에 로그인

일부 조직에서는 2 단계 인증 이라는 다단계 로그인 프로세스를 사용 하 여 사용자의 보안을 강화 합니다. 이 옵션을 사용 해야 하는 경우 비즈니스용 Skype에 로그인 하는 데 "스마트 카드"가 필요 합니다. 스마트 카드는 실제 또는 가상 모두 일 수 있습니다.
  
- **실제** 신용 카드 크기에 대해 알아보세요. 로그인 하면 스마트 카드 판독기에 삽입 됩니다.
    
- **가상** 실제 개체는 아니지만 컴퓨터의 특별 칩에 기록 된 전자 식별자로, 본질적으로 컴퓨터에 스마트 카드를 작성 합니다. TPM (신뢰할 수 있는 플랫폼 모듈) 칩이 포함 된 Windows 8 컴퓨터 에서만 사용할 수 있습니다.
    
### <a name="enroll-your-smart-card"></a>스마트 카드 등록

스마트 카드를 사용 하 여 로그인 하기 전에 카드를 "등록" 해야 하며, 즉 사용자 자격 증명을 해당 카드를 사용 하 여 식별할 수 있어야 합니다. 이는 카드가 물리적 이거나 가상 인지에 대 한 것입니다. 이 프로세스는 비즈니스용 Skype 서버 관리자가 이미 수행 했을 수 있습니다. 해당 작업이 완료 되었는지 확인 합니다.
  
> [!NOTE]
> 각 가상 스마트 카드는 설치 된 장치에만 연결 되기 때문에 사용 하는 각 Windows 8 컴퓨터에 대해 별도의 카드를 등록 해야 합니다. 
  
### <a name="to-manually-enroll-your-smart-card"></a>스마트 카드를 수동으로 등록 하려면

1. 비즈니스용 Skype를 실행할 컴퓨터에 로그온 합니다.
    
2. Internet Explorer를 사용 하 여 조직의 인증 기관 웹 등록 페이지로 이동 합니다. 
    
    아직 없는 경우 비즈니스용 Skype 서버 관리자에 게이 리소스의 웹 주소를 문의 하세요. URL은 다음과 https://MyCA같이 표시 됩니다. [모든 companyname] .com/certsrv.
    
    > [!NOTE]
    > Internet Explorer 10을 사용 중인 경우 호환 모드에서이 웹 사이트를 확인 해야 할 수 있습니다. 
  
3. 인증 페이지에 로그온 하 라는 메시지가 표시 되 면 컴퓨터의 관리자가 아닌 도메인 계정을 사용 하 여 로그온 합니다.
    
4. 웹 사이트 시작 페이지에서 **인증서 요청**을 선택 합니다.
    
5. **고급 요청**을 선택 합니다.
    
6. **이 CA에 대 한 요청 만들기 및 제출을**선택 하 고 **다음**을 클릭 합니다.
    
7. 이제 스마트 카드 등록 스테이션 이라는 페이지가 표시 됩니다. ActiveX 컨트롤 설치 요청을 승인 하 고 고급 인증서 요청 양식을 다음과 같이 수행 합니다.
    
    에서. **인증서 템플릿** 드롭다운 목록에서 **스마트 카드 사용자** 를 선택 합니다.
    
    b. **새 키 집합 만들기**를 선택 합니다.
    
    c. 스마트 카드의 레이블에서 제조업체 정보를 찾고 **CSP** 드롭다운 목록에서 해당 제조업체를 선택 합니다.
    
    a. **CSP** 를 요청 형식 (아직 선택 하지 않은 경우)으로 선택 합니다.
    
    z.e.n.works. 해시 알고리즘 드롭다운 목록에서 **sha1** 을 선택 합니다 (선택 되어 있지 않은 경우).
    
    f. 인증서에 인식할 수 있는 이름을 지정 하 고 **제출을**클릭 합니다.
    
8. 이제 빈 스마트 카드를 등록 스테이션에 연결 된 카드 리더에 삽입 하 고 **등록**을 클릭 합니다.
    
9. 메시지가 표시 되 면 PIN (개인 식별 번호)을 입력 한 다음 **확인**을 클릭 합니다.
    
    > [!NOTE]
    > 기술 지원 담당자가 스마트 카드를 등록 하는 데 사용할 특수 PIN을 제공 하지 않은 경우 기본 스마트 카드 핀 값인 12345678을 사용 합니다. 
  
10. 스마트 카드를 처음 사용할 때 사용자를 강제로 PIN을 변경 하도록 하는 옵션을 선택 합니다.
    
11. 이제 빈 스마트 카드를 등록 스테이션에 연결 된 카드 리더에 삽입 하 고 **등록**을 클릭 합니다.
    
12. 메시지가 표시 되 면 PIN (개인 식별 번호)을 입력 한 다음 **확인**을 클릭 합니다.
    
    > [!NOTE]
    > 기술 지원 담당자가 스마트 카드를 등록 하는 데 사용할 특수 PIN을 제공 하지 않은 경우 기본 스마트 카드 핀 값인 12345678을 사용 합니다. 
  
13. 스마트 카드를 처음 사용할 때 사용자를 강제로 PIN을 변경 하도록 하는 옵션을 선택 합니다.
    
14. **확인** 을 클릭 하 여 표시 된 인증서에 정보가 있는지 확인 합니다.
    
15. 인증서가 발급 되었음을 알리는 메시지가 표시 되 면 **이 인증서 설치** 를 클릭 하 여 등록 프로세스를 완료 합니다.
    
### <a name="sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>스마트 카드 자격 증명을 사용 하 여 비즈니스용 Skype에 로그인

스마트 카드를 처음 사용 하기 전에 비즈니스용 Skype 로그인 페이지에서 **내 로그인 정보 삭제** 를 클릭 하는 것이 좋습니다. 이렇게 하면 컴퓨터에 저장 된 로그인 자격 증명이 지워지고 오류의 원인이 제거 됩니다.
  
### <a name="to-sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>스마트 카드 자격 증명을 사용 하 여 비즈니스용 Skype에 로그인 하려면

1. 비즈니스용 Skype 클라이언트를 시작 합니다.
    
2. 로그인 화면에서 **로그인 주소** 상자에 로그인 사용자 계정 이름을 입력 한 다음 **로그인**을 클릭 합니다.
    
3. 가상 스마트 카드를 사용 하는 경우에는이 단계를 건너뛰십시오.
    
    실제 스마트 카드를 사용 하는 경우 스마트 카드 판독기에 스마트 카드를 삽입 하 고 메시지가 감지 되 면 **확인** 을 클릭 합니다.
    
4. 스마트 카드의 PIN 번호를 입력 하 고 **확인**을 클릭 합니다.
    
    > [!NOTE]
    > 지원 담당자에 게 스마트 카드 PIN 번호를 할당 하지 않은 경우 기본값 12345678를 사용 합니다.
