# <a name="use-two-factor-authentication-with-skype-for-business-client-and-skype-for-business-server"></a>클라이언트 및 클라이언트에서 비즈니스용 Skype 2단계 인증 비즈니스용 Skype 서버
 
**요약:** 2단계 인증을 비즈니스용 Skype 서버 비즈니스용 Skype.
  
## <a name="sign-in-to-skype-for-business-for-the-first-time"></a>처음으로 비즈니스용 Skype 로그인

로그인 정보는 일반적으로 설치 시 자동으로 비즈니스용 Skype 구성됩니다. 그러나 처음 비즈니스용 Skype 클라이언트를 수동으로 시작해야 할 수 있습니다.
  
### <a name="to-sign-in-for-the-first-time"></a>처음 로그인하는 경우

1. 조직의 네트워크에 로그온합니다.
    
2. 에서 **모든**  >  **프로그램**  >  **시작을 비즈니스용 Skype.**
    
    로그인 화면이 표시됩니다.
    
    - 로그인 주소 상자가 이미 채워진 경우 표시된 주소가 올바른지 확인합니다.
    
    - 맞지 않는 경우 또는 상자가 비어 있는 경우 로그인 주소(일반적으로 전자 메일 주소와 동일)를 입력합니다.
    
    - 빈 암호 상자가 표시되면 암호를 추가합니다.
    
3. 로그인 **을 선택합니다.**

## <a name="sign-in-to-skype-for-business-with-a-smart-card"></a>스마트 비즈니스용 Skype 로그인합니다.

일부 조직에서는 이제 2단계 인증이라는 다단계 로그인 프로세스를 사용하여 사용자의 보안을 강화합니다. 이 옵션을 사용할 것으로 예상되는 경우 "스마트 카드"를 사용하여 로그인해야 비즈니스용 Skype. 스마트 카드는 물리적 또는 가상일 수 있습니다.
  
- **물리적** 신용 카드의 크기 로그인할 때 스마트 카드 판독기에 삽입합니다.
    
- **가상** 물리적 개체가 아니라 컴퓨터의 특수 칩에 기록되는 전자 식별자입니다. 이 식별자는 기본적으로 스마트 카드를 컴퓨터에 빌드합니다. TPM(신뢰할 수 있는 플랫폼 모듈) Windows 8 컴퓨터에만 사용할 수 있습니다.
    
### <a name="enroll-your-smart-card"></a>스마트 카드 등록

스마트 카드로 로그인하려면 먼저 카드가 "등록"되어야 합니다. 즉, 사용자 자격 증명을 카드로 식별해야 합니다. 카드가 실제인지 가상 카드인지에 따라 이 경우에 해당합니다. 이 프로세스는 사용자 관리자가 이미 비즈니스용 Skype 서버 있습니다. 완료 여부가 확실하지 않은 경우 확인을 합니다.
  
> [!NOTE]
> 각 가상 스마트 카드는 설치된 장치와만 연결하기 때문에 사용하는 각 가상 컴퓨터에 대해 별도의 Windows 8 등록해야 합니다. 
  
### <a name="to-manually-enroll-your-smart-card"></a>스마트 카드를 수동으로 등록하려면

1. 로그온할 컴퓨터에 로그온합니다비즈니스용 Skype.
    
2. 이 Internet Explorer 사용하여 조직의 인증 기관 웹 등록 페이지로 이동합니다. 
    
    아직 비즈니스용 Skype 서버 없는 경우 관리자에게 이 리소스의 웹 주소를 요청하세요. URL은 https://MyCA 다음과 같습니다. .[ yourcompanyname].com/certsrv.
    
    > [!NOTE]
    > 웹 사이트를 사용하는 Internet Explorer 10 호환 모드에서 이 웹 사이트를 보아야 할 수 있습니다. 
  
3. 인증 페이지에 로그온하라는 메시지가 표시될 경우 컴퓨터 관리자 대신 도메인 계정을 사용하여 로그온합니다.
    
4. 웹 사이트 시작 페이지에서 인증서 **요청을 선택합니다.**
    
5. 고급 **요청을 선택합니다.**
    
6. 이 CA에 요청 만들기 및 **제출을 선택하고** 다음 을 **클릭합니다.**
    
7. 이제 스마트 카드 등록 스테이션이라는 페이지가 표시됩니다. ActiveX 설치 요청을 승인하고 다음과 같이 고급 인증서 요청 양식을 작성합니다.
    
    a. 인증서 **템플릿 드롭다운** **목록에서 스마트** 카드 사용자를 선택합니다.
    
    b. 새 **키 집합 만들기 를 선택합니다.**
    
    c. 스마트 카드의 레이블에 대한 제조업체 정보를 찾아 **CSP** 드롭다운 목록에서 해당 제조업체를 선택합니다.
    
    d. 아직 선택하지 않은 경우 요청 형식으로 **CSP를** 선택합니다.
    
    e. 아직 선택되지 않은 경우 해시 알고리즘 드롭다운 목록에서 **sha1을** 선택합니다.
    
    f. 인증서에 인식할 수 있는 이름을 지정하고 제출을 **클릭합니다.**
    
8. 이제 등록 스테이션에 연결된 카드 판독기에 빈 스마트 카드를 삽입하고 등록을 **클릭합니다.**
    
9. 메시지가 표시될 때 PIN(개인 식별 번호)을 입력한 다음 확인 을 **클릭합니다.**
    
    > [!NOTE]
    > 기술 지원 팀에서 스마트 카드를 등록하는 데 사용할 특수 PIN을 지정하지 않은 경우 기본 스마트 카드 PIN 값을 12345678. 
  
10. 스마트 카드를 처음 사용할 때 사용자(사용자)가 PIN을 강제로 변경하도록 하는 옵션을 선택합니다.
    
11. 이제 등록 스테이션에 연결된 카드 판독기에 빈 스마트 카드를 삽입하고 등록을 **클릭합니다.**
    
12. 메시지가 표시될 때 PIN(개인 식별 번호)을 입력한 다음 확인 을 **클릭합니다.**
    
    > [!NOTE]
    > 기술 지원 팀에서 스마트 카드를 등록하는 데 사용할 특수 PIN을 지정하지 않은 경우 기본 스마트 카드 PIN 값을 12345678. 
  
13. 스마트 카드를 처음 사용할 때 사용자(사용자)가 PIN을 강제로 변경하도록 하는 옵션을 선택합니다.
    
14. **확인을** 클릭하여 표시된 인증서에 대한 정보가 표시되는지 확인할 수 있습니다.
    
15. 인증서가 발급된 알림이 표시되고 나면  이 인증서 설치를 클릭하여 등록 프로세스를 완료합니다.
    
### <a name="sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>스마트 카드 자격 비즈니스용 Skype 로그인합니다.

스마트 카드를 처음으로 사용하기 전에 비즈니스용 Skype 로그인 페이지에서 내  로그인 정보 삭제를 클릭하는 것이 좋습니다. 이렇게 하면 컴퓨터에 저장된 로그인 자격 증명이 지워지며 가능한 오류 원본이 제거됩니다.
  
### <a name="to-sign-in-to-skype-for-business-with-your-smart-card-credentials"></a>스마트 카드 자격 비즈니스용 Skype 로그인하는 경우

1. 클라이언트 비즈니스용 Skype 시작 합니다.
    
2. 로그인 화면에서 로그인 주소 상자에 로그인 사용자  계정 이름을 입력한 다음 로그인 **을 클릭합니다.**
    
3. 가상 스마트 카드를 사용하는 경우 이 단계를 건너뛰어도 무방합니다.
    
    실제 스마트 카드를 사용하는 경우 스마트 카드 판독기에 스마트 카드를 삽입하고 이를 확인하라는 메시지가 표시되면 카드가 감지되면 확인을 클릭합니다. 
    
4. 스마트 카드에 사용할 PIN 번호를 입력한 다음 확인 을 **클릭합니다.**
    
    > [!NOTE]
    > 지원 담당자가 스마트 카드 PIN 번호를 할당받지 않은 경우 기본 설정인 12345678.
