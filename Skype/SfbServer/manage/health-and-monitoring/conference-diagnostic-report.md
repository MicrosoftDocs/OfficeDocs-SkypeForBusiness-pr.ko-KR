# <a name="conference-diagnostic-report-in-skype-for-business-server"></a>회의 진단 보고서 비즈니스용 Skype 서버
 
**요약:** 회의에서 사용되는 회의 진단 보고서에 비즈니스용 Skype 서버.
  
회의 진단 보고서는 모든 회의 세션의 성공 및 실패에 대한 정보를 제공합니다. 각 비즈니스용 Skype 서버 오류 종류를 구분합니다.
  
- **예상 오류입니다.** 예상 오류는 일반적으로 가장 기술적 의미에서만 실패입니다. 예를 들어 누군가 회의를 시작하지만 누구나 참가하기 전에 전화를 끊었다고 가정해 보겠습니다. 기술적으로 실패: 회의가 시작되지만 완료되지는 않습니다. 그러나 이 오류는 예상할 수 있는 오류입니다. 이끌이가 참가하기 전에 회의를 취소하면 회의가 완료되지 않을 것으로 예상됩니다.
    
- **예기치 않은 오류입니다.** 예기치 않은 오류는 이름에서 의미하는 정확히, 상황에 따라 발생할 것으로 예상되지 않는 오류입니다. 예를 들어 이끌이의 모임 정책을 검색할 수 없는 경우 회의를 열 수 없다고 가정해 보겠습니다. 이는 예기치 않은 오류입니다. 결국 항상 사용자의 모임 정책을 검색할 수 있습니다.
    
성공, 예상 오류 및 예기치 않은 오류 메트릭은 총 세션 메트릭에 추가되지 않을 수 있습니다. 예를 들어 보고서에 다음 값이 표시될 수 있습니다.
  
|**성공**|**예상 오류**|**예기치 않은 오류**|**총 세션**|
|:-----|:-----|:-----|:-----|
|2024  <br/> |469  <br/> |16   <br/> |2521  <br/> |
   
2024 + 469 + 16을 추가하면 총 세션 수가 2,509개가 됐지만 총 세션 열에는 총 2,521개 세션이 표시됩니다. "누락된" 세션 12개는 시스템에서 성공 또는 실패로 분류할 수 없는 세션입니다. 타사 제품이 모니터링 서버에 익숙하지 않은 새로운 진단 코드를 도입하는 경우도 있습니다. 이와 같은 경우 해당 제품을 사용하여 전화를 걸고 해당 진단 코드를 보고하는 경우를 항상 성공, 예상 오류 또는 예기치 않은 오류로 분류할 수 있는 것은 아닙니다.
  
## <a name="accessing-the-conference-diagnostic-report"></a>회의 진단 보고서 액세스

회의 진단 보고서는 모니터링 보고서 홈 페이지에서 액세스합니다. 다음 메트릭 [](failure-distribution-report.md) 중 하나를 클릭하여 오류 비즈니스용 Skype 서버 보고서에 액세스할 수 있습니다.
  
- 예기치 않은 오류량
    
- 예상 오류량
    
## <a name="making-the-best-use-of-the-conference-diagnostic-report"></a>회의 진단 보고서를 가장 잘 활용

회의 진단 보고서에는 일련의 그래프가 포함되어 있습니다. 그래프에 표시된 각 열은 실제로 하이퍼링크입니다. 열을 클릭하면 해당 기간 및 해당 회의 [유형에](failure-distribution-report.md) 대한 비즈니스용 Skype 서버 보고서로 드릴다운됩니다.
  
## <a name="filters"></a>필터

필터를 사용하면 여러 방식으로 반환된 데이터를 보거나 보다 세부적으로 대상화된 데이터 집합을 반환할 수 있습니다. 예를 들어 회의 진단 보고서를 사용하면 수행되는 회의 유형(예: 포커스 기반 회의) 또는 회의에 사용된 에지 서버에서 수행되는 회의 유형에 대해 필터링할 수 있습니다. 또한 데이터의 그룹 지정 방식도 선택할 수 있습니다. 이 경우 전화 회의는 시간, 일, 주 또는 월별로 그룹화됩니다.
  
다음 표에는 회의 진단 보고서에 사용할 수 있는 필터가 나열됩니다.
  
**회의 진단 보고서 필터**

|**이름**|**설명**|
|:-----|:-----|
|**시작** <br/> |시간 범위의 시작 날짜/시간입니다. 시간별 데이터를 보려면 다음과 같이 시작 날짜 및 시간을 입력합니다.  <br/> 2015/7/7 오후 1:00  <br/> 시작 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시부터 시작됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/7/2015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/3/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**To** <br/> |시간 범위의 종료 날짜/시간입니다. 시간별 데이터를 보려면 다음과 같이 종료 날짜 및 시간을 입력합니다.  <br/> 2015/7/7 오후 1:00  <br/> 종료 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시에 종료됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/7/2015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/3/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**Interval** <br/> | 시간 간격입니다. 다음 중 하나를 선택합니다. <br/>  시간별(최대 25시간 표시 가능) <br/>  일별(최대 31일 표시 가능) <br/>  주별(최대 12주 표시 가능) <br/>  월별(최대 12개월 표시 가능) <br/>  시작 및 종료 날짜가 선택한 간격에 허용되는 최대 값 수를 초과하면 시작 날짜로부터 최대 값 수만 표시됩니다. 예를 들어 시작 날짜가 2015년 7월 7일과 종료 날짜가 2015년 2월 28일인 일별 간격을 선택하면 데이터가 8/7/2015 오전 12:00에서 2015/9/7 오전 12:00(총 31일 데이터) 동안 표시됩니다. <br/> |
|**풀** <br/> |등록자 풀 또는 에지 서버의 FQDN(정규화된 도메인 이름)입니다. 개별 풀을 선택하거나 **[모두]** 를 클릭하여 모든 풀에 대한 데이터를 봅니다. 이 드롭다운 목록은 데이터베이스의 레코드에 따라 자동으로 채워집니다.<br/> |
|**회의 세션** <br/> | 회의 세션의 유형을 나타냅니다. 다음 중 하나를 선택합니다. <br/>  [모두] <br/>  회의 센터 세션 <br/>  모든 MCU 세션 <br/>  IM 회의 <br/>  응용 프로그램 공유 <br/>  A/V 회의 <br/> |
   
## <a name="metrics"></a>메트릭

다음 표에는 각 회의 세션 유형에 대한 회의 진단 보고서에 제공된 정보가 나열됩니다.
  
**회의 진단 보고서 메트릭**

|**이름**|**이 항목에 대한 정렬 가능 여부**|**설명**|
|:-----|:-----|:-----|
|**성공량** <br/> |아니오  <br/> |성공한 총 전화 회의 수입니다.  <br/> |
|**성공 비율** <br/> |아니오  <br/> |중요한 문제로 완료된 회의의 비율입니다. 성공량을 총 세션으로 나눠서 계산됩니다.  <br/> |
|**예상 오류량** <br/> |아니오  <br/> |"예상 오류"가 발생한 총 전화 회의 수입니다.  <br/> 예상 오류는 발생할 것으로 예상된 오류입니다. 예를 들어 사용자가 자신의 상태를 방해 금지로 설정한 경우 해당 사용자에 대한 통화가 실패할 것으로 예상할 수 있습니다.  <br/> |
|**예상 오류 비율** <br/> |아니요  <br/> |예상 오류가 발생하는 회의의 비율입니다. 예상 오류량을 총 세션으로 나눠서 계산됩니다.  <br/> |
|**예기치 않은 오류량** <br/> |아니요  <br/> |"예기치 않은 오류"가 발생한 총 전화 회의 수입니다.  <br/> 예기치 않은 오류는 일반적으로 정상 상태의 시스템으로 보이지만 예기치 않게 발생한 오류입니다. 예를 들어 발신자가 보류 상태일 때는 통화가 종료되지 않아야 합니다. 하지만 통화가 종료되면 바로 예기치 않은 오류로 플래그 지정됩니다.  <br/> |
|**예기치 않은 오류 비율** <br/> |아니오  <br/> |예기치 않은 오류가 발생하는 회의의 비율입니다. 예기치 않은 오류량을 총 세션으로 나눠서 계산됩니다.  <br/> |
|**총 세션** <br/> |아니오  <br/> |성공한 회의, 실패한 회의(예상 오류 및 예기치 않은 오류) 및 미분할 수 있는 회의를 비롯한 총 전화 회의 수입니다.  <br/> |
