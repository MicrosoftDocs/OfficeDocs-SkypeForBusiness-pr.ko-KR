# <a name="conference-summary-report-in-skype-for-business-server"></a>회의 요약 보고서의 비즈니스용 Skype 서버
 
**요약:** 2013의 회의 요약 보고서에 비즈니스용 Skype 서버.
  
회의 요약 보고서에서는 온라인 회의 세션에 대한 전반적인 보기를 제공합니다. 회의에는 일반적으로 2명 이상의 사용자가 포함하며 회의 서비스를 사용해야 합니다. 반면, 피어 투 피어 세션은 일반적으로 사용자 2명만 포함하며 회의 서비스를 사용할 필요가 비즈니스용 Skype 서버 않습니다. 피어 투 피어 활동은 의 피어 투 피어 활동 [요약 보고서에 비즈니스용 Skype 서버.](peer-to-peer-activity-summary-report.md)
  
전화 회의 요약 보고서는 특정 기간(시간별, 일별, 주별, 월별)에 진행된 전화 회의의 수뿐만 아니라 해당 회의에 참여한 총 사용자 수와 고유한 전화 회의 이끌이의 총 수를 알 수 있습니다.
  
"고유한" 이끌이는 하나 이상의 회의를 예약하는 모든 사람입니다. 예를 들어 Pilar Ackerman이 하나의 회의를 예약했다면 한 명의 고유한 이끌이가 됩니다. Ken Myer가 148개의 회의를 예약했어도 한 명의 고유한 이끌이로 계산됩니다. 예를 들어 다음 표에는 예약된 회의가 8개 있지만 고유한 이끌이는 3명(Ken Myer, Pilar Ackerman 및 David Ahs)만 표시됩니다.
  
|**회의 이끌이**|**회의 날짜**|
|:-----|:-----|
|Ken Myer  <br/> |2015/7/7 오전 10:00  <br/> |
|David Ahs  <br/> |2015/7/7 오전 10:00  <br/> |
|Ken Myer  <br/> |2015/7/7 오전 11:00  <br/> |
|Pilar Ackerman  <br/> |2015/7/7 오전 11:00  <br/> |
|Ken Myer  <br/> |2015/7/7 오후 1:00  <br/> |
|Pilar Ackerman  <br/> |2015/7/7 오후 2:00  <br/> |
|Ken Myer  <br/> |2015/7/2 오전 10:00  <br/> |
|Pilar Ackerman  <br/> |2015/7/2 오전 10:00  <br/> |
   
회의 요약 보고서에는 또한 오디오 및/또는 비디오가 포함된 회의 개수가 표시됩니다.
  
## <a name="accessing-the-conference-summary-report"></a>회의 요약 보고서 액세스

회의 요약 보고서는 모니터링 보고서 홈 페이지에서 액세스됩니다. 다음 메트릭 중 하나를 클릭하여 회의 활동 보고서로 드릴다운할 수 있습니다.
  
- 총 회의
    
- 총 참가자
    
## <a name="making-the-best-use-of-the-conference-summary-report"></a>회의 요약 보고서의 모범 사례 만들기

회의 요약 보고서에 사용되는 대부분의 메트릭에 대한 총 값은 보고서 아래쪽에서 찾을 수 있습니다. 아래로 스크롤하여 지정된 기간 동안 진행된 총 전화 회의 수, 해당 회의에 참가한 총 사람 수 등의 값을 볼 수 있습니다. 보고서 맨 아래에 합계가 계산되지 않은 메트릭 중 하나는 총 고유 회의 이끌이입니다. 그 이유는 무엇인가요? 한 가지 이유가 있습니다. 한 달의 데이터만 보고 있는 경우를 가정해 볼 수 있습니다. 1일차에는 34명 고유한 회의 이끌이가 있습니다. 2일차에는 27명에 달하는 고유한 회의 이끌이가 있습니다. 이 2일 동안 61개의 고유한 회의 이끌이가 있는 것입니까? 반드시 그다지 필요는 없습니다. 결국 2일차에 회의를 주최한 27명이 모두 1일차에 회의를 주최한 34명 중일 수 있습니다. 예를 들어 이 간단한 보고서에서는 Ken Myer와 Pilar Ackerman이 2015년 7월 7일과 2015년 7월 2일에 예정된 회의를 예약했습니다.
  
|**회의 이끌이**|**회의 날짜**|
|:-----|:-----|
|Ken Myer  <br/> |2015/7/7 오전 10:00  <br/> |
|David Ahs  <br/> |2015/7/7 오전 10:00  <br/> |
|Ken Myer  <br/> |2015/7/7 오전 11:00  <br/> |
|Pilar Ackerman  <br/> |2015/7/7 오전 11:00  <br/> |
|Ken Myer  <br/> |2015/7/7 오후 1:00  <br/> |
|Pilar Ackerman  <br/> |2015/7/7 오후 2:00  <br/> |
|Ken Myer  <br/> |2015/7/2 오전 10:00  <br/> |
|Pilar Ackerman  <br/> |2015/7/2 오전 10:00  <br/> |
   
회의를 주최한 고유한 사용자의 총 수를 보다 쉽게 이해하기 위해 시간 간격을 바꿔보겠습니다. 예를 들어 하루가 아니라 한 달을 기준으로 데이터를 살펴 보겠습니다.
  
## <a name="filters"></a>필터

필터를 사용하면 여러 방식으로 반환된 데이터를 보거나 보다 세부적으로 대상화된 데이터 집합을 반환할 수 있습니다. 예를 들어 전화 회의 요약 보고서에서는 데이터의 그룹화 방법을 선택할 수 있습니다. 이 경우 전화 회의는 시간, 일, 주 또는 월별로 그룹화됩니다.
  
다음 표에서는 전화 회의 요약 보고서에서 사용할 수 있는 필터를 보여 줍니다.
  
**전화 회의 요약 보고서 필터**

|**이름**|**설명**|
|:-----|:-----|
|**시작** <br/> |시간 범위의 시작 날짜/시간입니다. 시간별 데이터를 보려면 다음과 같이 시작 날짜 및 시간을 입력합니다.  <br/> 2015/7/7 오후 1:00  <br/> 시작 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시부터 시작됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/7/2015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/3/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**To** <br/> |시간 범위의 종료 날짜/시간입니다. 시간별 데이터를 보려면 다음과 같이 종료 날짜 및 시간을 입력합니다.  <br/> 2015/7/7 오후 1:00  <br/> 종료 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시에 종료됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/7/2015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/3/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**Interval** <br/> | 시간 간격입니다. 다음 중 하나를 선택합니다. <br/>  시간별(최대 25시간 표시 가능) <br/>  일별(최대 31일 표시 가능) <br/>  주별(최대 12주 표시 가능) <br/>  월별(최대 12개월 표시 가능) <br/>  시작 및 종료 날짜가 선택한 간격에 허용되는 최대 값 수를 초과하면 시작 날짜로부터 최대 값 수만 표시됩니다. 예를 들어 시작 날짜가 2015년 7월 7일과 종료 날짜가 2015년 2월 28일인 일별 간격을 선택하면 데이터가 8/7/2015 오전 12:00에서 2015/9/7 오전 12:00(총 31일 데이터) 동안 표시됩니다. <br/> |
