# <a name="location-trend-report-in-skype-for-business-server"></a>비즈니스용 Skype 서버의 위치 추세 보고서
 
**요약:** 비즈니스용 Skype 서버의 위치 추세 보고서에 대해 알아보세요.
  
위치 추세 보고서는 네트워크 위치에 대 한 통화 품질 추세 정보를 제공 합니다.
  
## <a name="filters"></a>필터가

필터를 통해 더욱 세밀 하 게 대상 지정 된 데이터 집합을 반환 하거나 반환 된 데이터를 다양 한 방식으로 볼 수 있습니다. 예를 들어 위치 추세 보고서를 사용 하면 액세스 형식 (즉, 간격 액세스와 외부 액세스) 또는 유선/무선 네트워크 연결을 통해 반환 된 데이터를 필터링 할 수 있습니다. 데이터 그룹화 방법을 선택할 수도 있습니다. 이 경우 통화는 시간, 일 또는 주별로 그룹화 됩니다.
  
다음 표에는 위치 추세 보고서와 함께 사용할 수 있는 필터가 나와 있습니다. 
  
**위치 추세 보고서 필터**

|**이름**|**설명**|
|:-----|:-----|
|**보낸 사람** <br/> |시간 범위의 시작 날짜/시간입니다. 시간별로 데이터를 보려면 시작 날짜와 시간을 모두 다음과 같이 입력 합니다.  <br/> 오후 7/7/2015 1:00  <br/> 시작 시간을 입력 하지 않으면 지정한 날짜에 오전 12:00부터 보고서가 자동으로 시작 됩니다. 날짜별로 데이터를 보려면 날짜만 입력 합니다.  <br/> 7/7/2015  <br/> 주별 또는 월별로 보려면 보려는 주 또는 월의 어디에 나 날짜를 입력 합니다 (주 또는 월의 첫 번째 요일을 입력할 필요는 없음).  <br/> 7/3/2015  <br/> 주는 항상 일요일 ~ 토요일을 실행 합니다.  <br/> |

|**간격만** <br/> | 시간 간격. 다음 중 하나를 선택 합니다. <br/>  매시간 (최대 25 시간 표시할 수 있음) <br/>  매일 (최대 31 일이 표시 될 수 있음) <br/>  주간 (최대 12 주를 표시할 수 있음) <br/>  시작 및 끝 날짜가 선택한 기간에 허용 되는 최대 값을 초과 하는 경우에는 시작 날짜부터 시작 하 여 최대 값만 표시할 수 있습니다. 예를 들어 시작 날짜가 1/1/2011이 고 종료 날짜가 2/28/2011 인 일일 간격을 선택 하는 경우에는 8/1/2011 12:00 AM ~ 9/1/2011 12:00 AM (즉, 총 31 일 "의 데이터)에 대 한 데이터가 표시 됩니다. <br/> |
|**Access 형식** <br/> | 전화를 걸 때 클라이언트가 내부 네트워크 또는 외부 네트워크에 로그온 되었는지 여부를 나타냅니다. 다음 중 하나를 선택 합니다. <br/>  모든 <br/>  내부용 <br/>  내부 <br/> |
|**네트워크 유형** <br/> | 전화를 걸 때 클라이언트가 연결 된 네트워크의 유형을 나타냅니다. 다음 중 하나를 선택 합니다. <br/>  모든 <br/>  유 <br/>  Wireless-n <br/> |
