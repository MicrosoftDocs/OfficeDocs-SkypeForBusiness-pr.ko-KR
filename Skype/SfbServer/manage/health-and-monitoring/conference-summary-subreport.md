# <a name="conference-summary-subreport-in-skype-for-business-server"></a>비즈니스용 Skype 서버의 컨퍼런스 요약 하위 보고서
 
**요약:** 비즈니스용 Skype 서버의 회의 요약 하위 보고서에 대해 알아보세요.
  
회의 요약 하위 보고서는 실패 한 회의 세션의 전체적인 보기를 제공 합니다. 이러한 실패 한 세션은 세션 유형: 포커스 세션 및 MCU 세션으로 더욱 세분화 됩니다.
  
## <a name="filters"></a>필터가

필터를 통해 더욱 세밀 하 게 대상 지정 된 데이터 집합을 반환 하거나 반환 된 데이터를 다양 한 방식으로 볼 수 있습니다. 다음 표에는 회의 요약 하위 보고서와 함께 사용할 수 있는 필터가 나열 되어 있습니다.
  
**컨퍼런스 요약 하위 보고서 필터**

|**이름**|**설명**|
|:-----|:-----|
|**보낸 사람** <br/> |시간 범위의 시작 날짜/시간입니다. 시간별로 데이터를 보려면 시작 날짜와 시간을 모두 다음과 같이 입력 합니다.  <br/> 오후 7/7/2015 1:00  <br/> 시작 시간을 입력 하지 않으면 지정한 날짜에 오전 12:00부터 보고서가 자동으로 시작 됩니다. 날짜별로 데이터를 보려면 날짜만 입력 합니다.  <br/> 7/7/2015  <br/> 주별 또는 월별로 보려면 보려는 주 또는 월의 어디에 나 날짜를 입력 합니다 (주 또는 월의 첫 번째 요일을 입력할 필요는 없음).  <br/> 7/3/2015  <br/> 주는 항상 일요일 ~ 토요일을 실행 합니다.  <br/> |
|**받는 사람** <br/> |시간 범위의 종료 날짜/시간입니다. 시간별로 데이터를 보려면 다음과 같이 종료 날짜와 시간을 모두 입력 합니다.  <br/> 오후 7/7/2015 1:00  <br/> 종료 시간을 입력 하지 않으면 지정한 날짜에 12:00 오전에 보고서가 자동으로 종료 됩니다. 날짜별로 데이터를 보려면 날짜만 입력 합니다.  <br/> 7/7/2015  <br/> 주별 또는 월별로 보려면 보려는 주 또는 월의 어디에 나 날짜를 입력 합니다 (주 또는 월의 첫 번째 요일을 입력할 필요는 없음).  <br/> 7/3/2015  <br/> 주는 항상 일요일 ~ 토요일을 실행 합니다.  <br/> |
|**풀** <br/> |등록자 풀 또는 Edge 서버의 FQDN (정규화 된 도메인 이름)입니다. 개별 풀을 선택 하거나 **[모두]** 를 클릭 하 여 모든 풀에 대 한 데이터를 볼 수 있습니다. 이 드롭다운 목록은 데이터베이스의 레코드에 따라 자동으로 채워집니다. <br/> |
   
## <a name="metrics"></a>매트릭스

다음 표에는 회의 요약 하위 보고서에서 제공 하는 정보가 나열 되어 있습니다.
  
**컨퍼런스 요약 하위 보고서 메트릭**

|**이름**|**이 항목을 정렬할 수 있나요?**|**설명**|
|:-----|:-----|:-----|
|**총 컨퍼런스** <br/> |아니요  <br/> |보유 한 총 컨퍼런스 수입니다.  <br/> |
|**총 회의 세션** <br/> |아니요  <br/> |회의 세션의 총 수입니다. 단일 회의는 여러 세션을 가질 수 있습니다. 예를 들어 컨퍼런스에는 포커스 세션과 MCU 세션이 모두 포함 될 것입니다.  <br/> |
|**전체 세션 실패 율** <br/> |아니요  <br/> |실패 한 모든 컨퍼런스의 백분율입니다.  <br/> |
|**포커스 세션** <br/> |아니요  <br/> |총 포커스 세션 수입니다.  <br/> |
|**초점 고장 속도** <br/> |아니요  <br/> |실패 한 포커스 세션의 백분율입니다.  <br/> |
|MCU 세션  <br/> |아니요  <br/> |총 MCU 세션 수입니다.  <br/> |
|**MCU 실패 율** <br/> |아니요  <br/> |실패 한 MCU 세션의 백분율입니다.  <br/> |
|**모달에의 한 MCU 세션** <br/> |아니요  <br/> |형식 (예: IM 회의)으로 그룹화 된 총 MCU 세션 수입니다.  <br/> |
|**모달의 실패 비율** <br/> |아니요  <br/> |실패 한 MCU 세션의 백분율 (예: IM 회의)  <br/> |
