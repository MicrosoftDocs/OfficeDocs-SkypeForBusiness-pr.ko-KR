# <a name="conference-activity-report-in-skype-for-business-server"></a>비즈니스용 Skype 서버
 
**요약:** 이 보고서에서 사용되는 회의 활동 보고서에 비즈니스용 Skype 서버.
  
전화 회의 활동 보고서를 사용하면 매일 열리는 회의 수와 회의가 언제 열리는지와 같은 질문에 쉽게 답변할 수 있습니다. 이와 같은 정보는 자체적으로 사용할 수 있는 것이 아니라 문제 해결 도구로도 유용합니다. 예를 들어 사용자가 하루 중에는 네트워크 속도가 특히 느려졌다고 불평하는 경우를 가정해 보겠습니다. 회의 활동 보고서를 빠르게 보면 가능한 한 가지 이유가 제안될 수 있습니다. 오전 10시에서 오후 2시 사이에 더 많은 회의가 예약되고 있습니다.
  
네트워크 속도가 느려서 문제가 발생하면 사용자가 하루 중 트래픽이 적은 시간 동안 일부 회의를 다시 시작하도록 장려할 수 있습니다.
  
## <a name="accessing-the-conference-activity-report"></a>회의 활동 보고서 액세스

회의 활동 보고서는 다음 [](conference-summary-report.md) 메트릭 중 하나를 클릭하여 비즈니스용 Skype 서버 회의 요약 보고서에서 액세스합니다.
  
- 총 회의
    
- 총 참가자
    
## <a name="making-the-best-use-of-the-conference-activity-report"></a>회의 활동 보고서를 가장 잘 활용

기본적으로 전화 회의 활동 보고서에는 지정된 기간의 총 전화 회의 수(예: 일당 총 회의 수 또는 하루 시간당 총 전화 회의 수)가 표시됩니다. 그러나 해당 기간의 총 참가자 수 또는 총 참가자 수(분)를 표시하기로 선택할 수도 있습니다. 이렇게하려면 매개 변수 표시/숨기기 단추를 클릭하여 필터링 옵션을 표시한 다음 보고서 드롭다운 목록에서 다음 중 하나를 선택합니다.
  
- 참가자 수
    
- 참가자 시간(분)
    
- 회의 수
    
## <a name="filters"></a>필터

필터를 사용하면 여러 방식으로 반환된 데이터를 보거나 보다 세부적으로 대상화된 데이터 집합을 반환할 수 있습니다. 다음 표에는 회의 활동 보고서에 사용할 수 있는 필터가 나열됩니다.
  
**회의 활동 보고서 필터**

|**이름**|**설명**|
|:-----|:-----|
|**시작** <br/> |시간 범위의 시작 날짜/시간입니다. 시간별 데이터를 보려면 다음과 같이 시작 날짜 및 시간을 입력합니다.  <br/> 2015/7/7 오후 1:00  <br/> 시작 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시부터 시작됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/7/2015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/3/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**To** <br/> |시간 범위의 종료 날짜/시간입니다. 시간별 데이터를 보려면 다음과 같이 종료 날짜 및 시간을 입력합니다.  <br/> 2015/7/7 오후 1:00  <br/> 종료 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시에 종료됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/7/2015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/3/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**Interval** <br/> | 시간 간격입니다. 다음 중 한 개를 선택합니다. <br/>  시간별(최대 25시간 표시 가능) <br/>  일별(최대 31일 표시 가능) <br/>  주별(최대 12주 표시 가능) <br/>  월별(최대 12개월 표시 가능) <br/>  시작 및 종료 날짜가 선택한 간격에 허용되는 최대 값 수를 초과하면 시작 날짜로부터 최대 값 수만 표시됩니다. 예를 들어 시작 날짜가 2015년 7월 7일과 종료 날짜가 2015년 2월 28일인 일별 간격을 선택하면 데이터가 8/7/2015 오전 12:00에서 2015/9/7 오전 12:00(총 31일 데이터) 동안 표시됩니다. <br/> |
|**보고서 작성자** <br/> | 보고서에 사용할 값을 나타냅니다. 다음 중 하나를 선택할 수 있습니다. <br/>  참가자 수 <br/>  참가자 시간(분) <br/>  회의 수 <br/> |
