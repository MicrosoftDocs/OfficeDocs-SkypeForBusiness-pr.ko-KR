# <a name="server-media-quality-trend-report-in-skype-for-business-server"></a>서버 미디어 추세 보고서 비즈니스용 Skype 서버
 
**요약:** 추세 서버 미디어 품질 보고서에 대해 비즈니스용 Skype 서버.
  
서버 미디어 품질 추세 보고서를 사용하면 통화량, 불량 통화율, 패킷 손실 및 지터와 같은 체감 품질 메트릭에서 최대 5개의 서버를 그래픽으로 비교할 수 있습니다. 이를 통해 성능 품질이 낮은 서버, 활용률이 낮은 서버, 초과 사용되는 서버 등을 쉽게 식별할 수 있습니다.
  
## <a name="accessing-the-server-media-quality-trend-report"></a>서버 미디어 품질 추세 보고서 액세스

서버 미디어 품질 추세 보고서는 다음 보고서 중 하나에서 액세스할 수 있습니다.
  
- [보고서의](server-performance.md) 서버 비즈니스용 Skype 서버 보고서(추세 클릭)
    
- [A/V 에지 비즈니스용 Skype 서버](call-detail-report.md) 메트릭을 클릭하여 호출 세부 정보 보고서를 클릭합니다. 발신자 또는 수신자가 서버인 경우 끝점 이름을 클릭하여 서버 품질 미디어 추세 보고서에 연결할 수도 있습니다.)
    
## <a name="making-the-best-use-of-server-media-quality-trend-report"></a>서버 미디어 품질 추세 보고서 활용

특정 서버에 추세 서버 성능 보고서의 비즈니스용 Skype 서버 추세 보고서가 열립니다. [](server-performance.md) 그러나 해당 보고서의 빈 인스턴스만 표시됩니다. 서버 성능 보고서에서 선택한 서버가 화면에 표시되지 않습니다. 대신 서버 드롭다운에서 해당 서버를 선택해야 합니다. 서버 드롭다운에는 모두 선택 옵션이 포함되어 있습니다. 서버가 5대를 넘을 경우 이 옵션이 작동하지 않습니다. 서버 미디어 품질 추세 보고서에는 한에 최대 5대의 서버만 표시할 수 있습니다.
  
서버 미디어 품질 보고서에서 표시하는 그래프에서 통화 추세 불량 통화율 포인트는 핫링크입니다. 그래프에서 지점을 클릭하면 지정된 기간의 총 통화 수(또는 불량 [통화)비즈니스용 Skype 서버](call-list-report-0.md) 표시될 수 있는 통화 목록 보고서 인스턴스가 열립니다.
