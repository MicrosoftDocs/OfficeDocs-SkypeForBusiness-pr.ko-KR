# <a name="peer-to-peer-activity-summary-report-in-skype-for-business-server"></a>비즈니스용 Skype 서버의 피어 투 피어 활동 요약 보고서
 
**요약:** 비즈니스용 Skype 서버의 피어 투 피어 활동 요약 보고서에 대해 자세히 알아보습니다.
  
피어 투 피어 활동 요약 보고서는 피어 투 피어 통신 세션의 전체적인 보기를 제공합니다. 피어 투 피어 세션은 일반적으로 두 사용자만 포함하며 비즈니스용 Skype 서버 회의 서비스를 사용할 필요가 없습니다. 반면, 전화 회의에는 일반적으로 두 개 이상의 사용자가 포함하며 비즈니스용 Skype 서버 회의 서비스를 사용해야 합니다. 회의 활동은 회의 요약 보고서에 보고됩니다.
  
피어 투 피어 활동 요약 보고서는 다음과 같은 질문에 대한 답변을 제공합니다.
  
- 사용자가 평소에 보내는 피어 투 피어 인스턴트 메시지의 수
    
- 내 사용자가 실제로 비즈니스용 Skype 서버 응용 프로그램 공유 및 파일 전송 기능을 활용하나요?
    
- 사용자는 하루 중 특정 시간 동안 네트워크가 느려졌다고 불만을 나타냈습니다. 해당 기간 동안 피어 투 피어 오디오 및 비디오 세션에 얼마나 많은 시간이 사용하나요?
    
## <a name="accessing-the-peer-to-peer-activity-summary-report"></a>피어 투 피어 활동 요약 보고서 액세스

모니터링 보고서 홈 페이지에서 피어 투 피어 활동 요약 보고서에 액세스합니다. 다음 메트릭 중 하나를 클릭하여 비즈니스용 [Skype 서버에서](im-report.md) 피어 투 피어 IM 보고서를 열 수 있습니다.
  
- 총 피어 투 피어 IM 세션
    
- 총 피어 투 피어 IM 메시지
    
마찬가지로, 다음 메트릭 중 원하는 경우 피어 투 피어 음성 및 비디오 보고서를 열 수 있습니다.
  
- 총 피어 투 피어 오디오 세션
    
- 총 피어 투 피어 오디오 세션 시간(분)
    
- 총 피어 투 피어 오디오 세션
    
- 총 피어 투 피어 오디오 세션 시간(분)
    
## <a name="making-the-best-use-of-the-peer-to-peer-activity-summary-report"></a>피어 투 피어 활동 요약 보고서를 가장 잘 활용

피어 투 피어 활동 요약 보고서의 맨 아래에는 총 피어 투 피어 IM 세션 및 총 피어 투 피어 IM 메시지와 같은 메트릭에 대한 합계가 표시됩니다. 이렇게 하면 보고서 본문에 있는 자세한 정보를 빠르게 요약할 수 있습니다.
  
## <a name="filters"></a>필터

필터를 사용하면 여러 방식으로 반환된 데이터를 보거나 보다 세부적으로 대상화된 데이터 집합을 반환할 수 있습니다. 예를 들어 피어 투 피어 활동 요약 보고서를 사용하면 데이터를 그룹화하는 방법을 선택할 수 있습니다. 이 경우 활동은 시간, 일, 주 또는 월으로 그룹화됩니다.
  
다음 표에는 피어 투 피어 활동 요약 보고서에 사용할 수 있는 필터가 나열됩니다.
  
**피어 투 피어 활동 요약 보고서 필터**

|**이름**|**설명**|
|:-----|:-----|
|**시작** <br/> |시간 범위의 시작 날짜 및 시간입니다. 시간별 데이터를 보려면 다음과 같이 시작 날짜와 시간을 모두 입력합니다.  <br/> 2015/7/17 오후 1:00  <br/> 시작 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시부터 시작됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/17/12015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/13/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**To** <br/> |시간 범위의 종료 날짜 및 시간입니다. 시간별 데이터를 보려면 다음과 같이 종료 날짜 및 시간을 입력합니다.  <br/> 2015/7/17 오후 1:00  <br/> 종료 시간을 입력하지 않으면 보고서가 자동으로 지정된 날짜의 오전 12시에 종료됩니다. 일별 데이터를 보려면 날짜만 입력합니다.  <br/> 7/17/12015  <br/> 주 또는 월별로 보려면 데이터를 보려는 해당 주 또는 월에 속하는 날짜를 입력합니다. 주 또는 월의 첫 번째 날짜를 입력할 필요가 없습니다.  <br/> 7/13/2015  <br/> 주는 항상 일요일부터 토요일까지로 실행됩니다.  <br/> |
|**간격** <br/> | 시간 간격입니다. 다음 중 하나를 선택합니다. <br/>  시간별(최대 25시간 표시 가능) <br/>  일별(최대 31일 표시 가능) <br/>  주별(최대 12주 표시 가능) <br/>  월별(최대 12개월 표시 가능) <br/>  시작 및 종료 날짜가 선택한 간격에 허용되는 최대 값 수를 초과하면 시작 날짜로부터 최대 값 수만 표시됩니다. 예를 들어 시작 날짜가 12015년 7월 17일과 종료 날짜가 2015년 2월 28일인 일별 간격을 선택하면 데이터가 8/7/12015 오전 12:00에서 12015/9/7 오전 12:00(총 31일 데이터 가치)에 표시됩니다. <br/> |
