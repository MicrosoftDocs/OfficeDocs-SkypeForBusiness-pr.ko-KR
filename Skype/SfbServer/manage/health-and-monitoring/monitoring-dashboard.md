# <a name="using-the-monitoring-dashboard-in-skype-for-business-server"></a>비즈니스용 Skype 서버에서 모니터링 대시보드 사용
 
**요약:** 비즈니스용 Skype 서버의 모니터링 대시보드에 대해 자세히 알아보세요.
  
모니터링 대시보드는 관리자에 게 비즈니스용 Skype Server 시스템 상태 및 시스템 사용량에 대 한 간략 한 개요를 제공 합니다. 대시보드는 주요 시스템 메트릭의 집계 보기를 표시 하 고 다음 중 하나를 표시 하 여 수행 하도록 디자인 되었습니다.
  
- 현재 날짜의 합계입니다. 현재 날짜에 표시 되는 값은 현재 시간까지 자정에서 기록 된 데이터를 나타냅니다 (보고 서버의 현지 시간 기준). 즉, 사용자는 일반적으로 하루 종일 데이터를 볼 수 있으며, 24 시간 동안은 그렇지 않습니다. 예를 들어 서버의 현지 시간이 오전 8:00이 고 오전 8:00 AM의 현재 시간인 8 시간이 있으므로 8 시간 분량의 데이터가 표시 됩니다.
    
- 주 및 지난 6 주에 대 한 총 추세 합계
    
- 월의 합계 및 지난 6 개월 동안의 추세 합계 (시스템 사용에만 해당)
    
[CsReportingConfiguration](https://docs.microsoft.com/powershell/module/skype/get-csreportingconfiguration?view=skype-ps) cmdlet을 사용 하 여 비즈니스용 Skype 서버 모니터링 보고서에 액세스 하는 데 사용 되는 URL을 반환할 수 있습니다.
  
```
Get-CsReportingConfiguration
```

기본적으로 모니터링 대시보드에는 현재 주와 지난 6 주에 대 한 추세 합계에 대 한 다음 메트릭에 대 한 데이터가 표시 됩니다.

## <a name="working-with-the-monitoring-dashboard"></a>모니터링 대시보드 작업

참고로, 현재 주에는 기본적으로 합계가 표시 되 고 지난 6 주 동안 추세 값이 표시 됩니다. 현재 월의 합계 및 지난 6 개월의 추세 값을 확인 하려면 대시보드의 오른쪽 위 모서리에 있는 **월 보기** 링크를 클릭 합니다. 월 합계를 표시 하기로 결정 한 경우 링크 텍스트는 **주별 보기로**변경 됩니다. 해당 링크를 클릭 하 여 주 보기로 다시 전환할 수 있습니다.
  
> [!TIP]
> 모니터링 대시보드는 지난 6 주 (또는 월)에 대 한 현재 주 (또는 월) 및 추세 값에 대 한 합계를 볼 수 있도록 제한 합니다. 이러한 날짜와 시간은 변경할 수 없습니다. 예를 들어 대시보드를 사용 하 여 9 개월 전에 시작 되는 기간의 보고서 합계를 볼 수 없습니다. 
  
이번 **주**, **이번 달**또는 **오늘** 열에 표시 되는 값은 항목에 대 한 자세한 정보를 연결 합니다. 열 이름과 해당 열에 표시 되는 값은 선택한 메트릭에 따라, 주 보기 또는 월 단위 보기를 선택 했는지에 따라 다를 수 있다는 것에 유의 하세요. 예를 들어 **고유한 사용자 로그온** 메트릭에 대해 표시 된 요약을 클릭 하면 지정 된 기간에 대 한 **사용자 등록 보고서** 가 표시 됩니다. **대시보드**를 클릭 하 여 언제 든 지 모니터링 대시보드로 돌아갈 수 있습니다.
  
> [!TIP]
> 대시보드의 오른쪽 위 모서리에 있는 **보고서** 링크를 클릭 하 여 모니터링 서버 보고서 홈 페이지에 액세스할 수도 있습니다.
  
**추세** 열에는 지난 6 주 (또는 메트릭과 시간 간격, 지난 6 일간 또는 지난 6 개월)에 대 한 합계를 보여 주는 간단한 꺾은선형 그래프가 표시 됩니다. 이러한 간단한 선 그래프는 각 기간에 대해 레이블이 지정 되지 않은 데이터 요소 (예: 지난 6 주 각각에 대해 레이블이 지정 되지 않은 데이터 요소 한 개)를 표시 합니다. 그러나 그래프 위에 마우스 포인터를 놓으면 이러한 그래프의 실제 값을 검색할 수 있습니다. 이 경우 도구 팁에 그래프의 최대값 및 최소값을 보여 주는 값이 표시 됩니다.
  
## <a name="exporting-data-from-the-monitoring-dashboard"></a>모니터링 대시보드에서 데이터 내보내기

모니터링 대시보드는 현재 대시보드 보기를 내보낼 수 있는 여러 가지 방법을 제공 합니다. 대시보드 도구 모음에 녹색 화살표가 연결 된 플로피 디스크 모양의 아이콘이 표시 됩니다. 이 아이콘을 클릭 하면 다음과 같은 데이터 내보내기 형식을 제공 하는 드롭다운 목록이 표시 됩니다.
  
- 보고서 데이터를 포함 하는 XML 파일
    
- CSV (쉼표로 분리)
    
- PDF
    
- MHTML (웹 보관)
    
- 0:excel}
    
- TIFF 파일
    
- 이면
    
현재 대시보드 보기 (및 해당 값)를 내보내려면 원하는 내보내기 옵션을 클릭 합니다. 비즈니스용 Skype Server는 지정 된 형식으로 보고서를 생성 한 다음 해당 보고서를 열거나 저장 하는 옵션을 제공 합니다. 기본적으로 비즈니스용 Skype 서버는 보고서 **모니터링 대시보드의** 제목을, 다운로드 폴더에 저장 합니다. 보고서에 다른 이름을 지정 하거나 다른 폴더에 저장 하려면 **저장** 단추 옆에 있는 화살표를 클릭 한 다음 **저장**을 클릭 합니다. 이름 **모니터링 대시보드에** 대해 잘 알고 있고 보고서가 다운로드 폴더에 저장 되어 있는 경우 **저장** 단추를 클릭 하기만 하면 됩니다.
  
대시보드 데이터를 내보내려고 하면 "현재 설정이이 파일을 다운로드 하도록 허용 하지 않습니다." 라는 메시지와 함께 **보안 알림** 대화 상자가 표시 될 수 있습니다. 이런 경우 다음을 수행 합니다.
  
- Internet Explorer에서 **인터넷 옵션**을 선택 합니다.
    
- **인터넷 옵션** 대화 상자의 **보안** 탭에서 **신뢰할 수 있는 사이트** 를 클릭 한 다음 **사이트**를 클릭 합니다.
    
- 신뢰할 수 **있는 사이트** 대화 상자에서 **추가** 를 클릭 하 여 비즈니스용 skype 서버 보고서를 실행 하는 비즈니스용 skype 서버를 신뢰할 수 있는 웹 사이트 모음에 추가 합니다.
    
- **닫기를** 클릭 한 다음 **확인**을 클릭 합니다.
    
변경 내용을 적용 하려면 모니터링 대시보드를 새로 고쳐야 합니다. 이렇게 하려면 F5 키를 누르거나 대시보드 도구 모음에서 **새로 고침** 아이콘을 클릭 합니다. **새로 고침** 아이콘은 녹색 화살표가 있는 원입니다.
  
최신 모니터링 대시보드 데이터에 대 한 링크를 포함 하는 라이브 데이터 피드를 포함 하는 Excel 스프레드시트를 만들 수도 있습니다. 라이브 데이터 피드 파일을 만들려면 도구 모음에서 주황색 **으로 내보내기 데이터 피드** 아이콘을 클릭 합니다.
  
현재 대시보드를 인쇄 하려면 도구 모음에서 프린터 아이콘을 클릭 합니다.
