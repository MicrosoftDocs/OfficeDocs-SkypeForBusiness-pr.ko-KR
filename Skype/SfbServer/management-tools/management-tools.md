# <a name="skype-for-business-server-2015-management-tools"></a>비즈니스용 Skype 서버 2015 관리 도구
 
**요약:** 비즈니스용 Skype Server 2015의 서비스 관리 도구에 대해 알아보세요.
  
비즈니스용 Skype 서버 2015 통신 소프트웨어 (이전의 Lync Server)는 엔터프라이즈 수준의 공동 작업 요구 사항을 지 원하는 인스턴트 메시지 (IM), 현재 상태, 회의 및 전화 통신 솔루션을 제공 합니다. 이러한 서비스를 관리 하는 도구는 유연 하 고 강력 합니다.
