# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>비즈니스용 Skype 서버용 통화 품질 대시보드 개발
 
**요약:** 통화 품질 대시보드용 솔루션을 개발하는 방법을 알아보십시오. 통화 품질 대시보드는 비즈니스용 Skype 서버용 도구입니다.
  
이 항목에서는 비즈니스용 Skype 서버용 CQD(통화 품질 대시보드)에 대한 개발 설명서를 제공합니다.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>비즈니스용 Skype 서버용 통화 품질 대시보드 개발

사용자 지정 개발을 사용하여 CQD를 보고 환경에 확장하고 통합할 수 있습니다. 
  
두 가지 웹 API 집합을 사용할 수 있습니다. 하나는 큐브의 데이터에 액세스하기 위한 집합(데이터 API를 통해)과 보고서 메타데이터(리포지토리 API를 통해)에 액세스하기 위한 것입니다. 
  
- [비즈니스용 Skype 서버의 CQD(통화 품질 대시보드)에 대한 데이터 API](data-api.md)
    
- [비즈니스용 Skype 서버의 CQD(통화 품질 대시보드)에 대한 리포지토리 API](repository-api.md)
    
- [CQD 개발 샘플](cqd-development-samples.md)
