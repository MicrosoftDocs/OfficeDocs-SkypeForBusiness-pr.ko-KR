# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>사용자용 통화 품질 대시보드 비즈니스용 Skype 서버
 
**요약:** 통화 품질 대시보드용 솔루션을 개발하는 방법에 대해 자세히 알아보십시오. 통화 품질 대시보드는 통화 품질 대시보드를 위한 비즈니스용 Skype 서버.
  
이 항목에서는 CQD(통화 품질 대시보드)에 대한 개발 설명서를 비즈니스용 Skype 서버.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>사용자용 통화 품질 대시보드 비즈니스용 Skype 서버

사용자 지정 개발을 사용하여 CQD를 보고 환경에 확장하고 통합할 수 있습니다. 
  
두 가지 웹 API 집합을 사용할 수 있습니다. 하나는 큐브의 데이터에 액세스하기 위한 집합(데이터 API를 통해)과 보고서 메타데이터(리포지토리 API를 통해)에 액세스하기 위한 것입니다. 
  
- [CQD(통화 품질 대시보드)에 대한 데이터 API를 비즈니스용 Skype 서버](data-api.md)
    
- [CQD(통화 품질 대시보드)에 대한 리포지토리 API(비즈니스용 Skype 서버](repository-api.md)
    
- [CQD 개발 샘플](cqd-development-samples.md)
