# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>비즈니스용 Skype 서버에 대 한 통화 품질 대시보드 개발
 
**요약:** 통화 품질 대시보드의 솔루션을 개발 하는 방법에 대해 알아보세요. 통화 품질 대시보드는 비즈니스용 Skype 서버용 도구입니다.
  
이 항목에서는 비즈니스용 Skype 서버용 CQD (통화 품질 대시보드)에 대 한 개발 설명서를 제공 합니다.

사용자 지정 개발을 사용 하 여 CQD를 확장 하 고 보고 환경에 통합할 수 있습니다. 
  
두 개의 웹 Api 집합, 즉 데이터 Api를 통해 큐브 데이터에 액세스 하는 방법과 리포지토리 API를 통해 보고서 메타 데이터에 액세스 하는 데 사용할 수 있습니다.
