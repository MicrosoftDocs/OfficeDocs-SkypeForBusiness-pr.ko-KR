# <a name="call-quality-dashboard-for-skype-for-business-server"></a>비즈니스용 Skype 서버에서 통화 품질 대시보드 사용
 
**요약:** 비즈니스용 Skype 서버의 보고 도구인 통화 품질 대시보드에 대해 자세히 알아보습니다.
 
비즈니스용 Skype 서버 CQD(통화 품질 대시보드)는 비즈니스용 Skype 서버 및 Lync Server 2013의 모니터링 서버의 환경 품질 데이터베이스 위에 있는 보고 계층입니다. CQD는 Microsoft SQL Server Analysis Services를 사용하여 집계 사용 현황 및 통화 품질 정보와 데이터 집합에 대한 필터링 및 피벗을 제공합니다.
  
다음 항목은 이 섹션에 포함되어 있으며 통화 품질 대시보드를 배포하기 전에 검토해야 합니다.
