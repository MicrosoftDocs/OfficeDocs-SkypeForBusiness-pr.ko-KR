# <a name="federation-planning-tool"></a>페더링(계획 도구)
 
내부 및 외부 사용자가 파트너 조직 또는 서비스의 연락처에 액세스할 수 있도록 에지 서버를 구성할 수 있습니다.
  
 페더전은 Microsoft Lync 2013과 마찬가지로 작동합니다. 페더링을 위해 환경을 준비하는 데 대한 자세한 내용은 다음 항목을 참조하십시오.
  
- [Lync Server 2013 및 Office Communications Server 페더ation 계획](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [Lync Server 2013의 공용 인스턴트 메시징 연결 계획](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [Lync Server 2013의 XMPP(Extensible Messaging and Presence Protocol) 페더전 계획](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
