비즈니스용 Skype Server 2015는 SIP trunks, PSTN 게이트웨이로의 SIP 연결, PBX로 직접 SIP 연결을 사용 하 여 안정적이 고 확장 가능한 PSTN 연결을 제공 합니다. 자세한 내용은 [비즈니스용 Skype 서버 2015의 PSTN 연결 계획](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)을 참조 하세요.
