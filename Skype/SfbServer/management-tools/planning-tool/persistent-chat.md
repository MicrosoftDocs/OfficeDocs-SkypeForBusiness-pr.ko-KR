영구 채팅 서버의 환경을 준비하는 데 대한 자세한 내용은 [Plan for Persistent Chat Server in 비즈니스용 Skype 서버 2015을 참조하세요.](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md)
