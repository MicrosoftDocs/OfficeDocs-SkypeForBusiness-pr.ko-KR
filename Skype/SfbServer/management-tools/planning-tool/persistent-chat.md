# <a name="persistent-chat-planning-tool"></a>영구 채팅 (계획 도구)
 
영구 채팅 서버는 사용자가 시간에 따라 유지 되는 단체의 항목 기반 대화에 참여할 수 있도록 합니다.
  
영구 채팅 서버용 환경을 준비 하는 방법에 대 한 자세한 내용은 비즈니스용 [Skype 서버 2015의 영구 채팅 서버 계획](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md)을 참조 하세요.
