# <a name="persistent-chat-planning-tool"></a>영구 채팅(계획 도구)
 
영구 채팅 서버를 사용하면 시간이 지날 때 지속되는 주제 기반의 여러 대화에 참가할 수 있습니다.
  
영구 채팅 서버를 위한 환경을 준비하는 데 대한 자세한 내용은 비즈니스용 Skype 서버 2015의 영구 채팅 서버 [계획(Plan for Persistent Chat Server)을 참조하세요.](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md)
