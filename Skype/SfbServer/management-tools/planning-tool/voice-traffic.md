# <a name="voice-traffic-planning-tool"></a>음성 트래픽 (계획 도구)
 
계획 도구는 사용자 트래픽에 대 한 정보를 사용 하 여 배포에 사용할 포트 수를 결정 합니다. 이 번호는 필요한 중재 서버와 게이트웨이의 수를 결정 합니다. 자세한 내용은 [비즈니스용 Skype 서버 2015의 음성 사용량 및 트래픽 예측](../../plan-your-deployment/capacity/estimating-voice-traffic.md)을 참조 하세요.
