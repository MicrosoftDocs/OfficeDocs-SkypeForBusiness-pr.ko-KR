# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>설치 SQL Server Reporting Services 설치 비즈니스용 Skype 서버 
 
**요약:** 검색 창에서 사용하는 앱에 대한 SQL Server Reporting Services 위치를 비즈니스용 Skype 서버.
  
비즈니스용 Skype 서버 SSRS(SQL Server Reporting Services)를 사용하여 보고서를 보고 모니터링할 수 있습니다. 이 기능을 사용하려면 Reporting Services를 설치하고 구성해야 합니다.
  
## <a name="install-sql-server-reporting-services"></a>SQL Server Reporting Services 설치

비즈니스용 Skype 서버 모니터링 보고서를 사용하려는 경우(자세한 내용은 이 설명서의 다음 문서 참조)를 먼저 설치해야 SQL Server Reporting Services. Reporting Services는 설치 시와 동시에 설치할 수 Microsoft SQL Server 설치한 후 SQL Server 있습니다. 설치하지 않은 SQL Server 이 설명서의 앞부분에 제공된 지침을 따릅니다. 설치 SQL Server 기능 선택 페이지에서 Reporting Services를 선택해야 합니다. 이 경우 설치 SQL Server Reporting Services.
  
SSRS를 설치하는 SQL Server Reporting Services 자세한 내용은 [SQL Server Reporting Services(SSRS)를 참조하세요.](/sql/reporting-services/create-deploy-and-manage-mobile-and-paginated-reports)
  
이미 설치한 SQL Server 설치하지 않은 경우 SQL Server Reporting Services 지침에 따라 해당 기능을 추가할 수 SQL Server Reporting Services.
