# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>비즈니스용 Skype SQL Server Reporting Services 설치 
 
**요약:** 비즈니스용 Skype 서버에서 사용하는 SQL Server 대한 정보를 찾아보는 방법을 알아보는 방법을 배워야 합니다.
  
비즈니스용 Skype 서버는 SQL Server 보고 서비스(SSRS)를 사용하여 보고서를 보고 모니터링할 수 있습니다. 이 기능을 사용하려면 Reporting Services를 설치하고 구성해야 합니다.
  
## <a name="install-sql-server-reporting-services"></a>SQL Server Reporting Services 설치

비즈니스용 Skype 서버 모니터링 보고서를 사용하려는 경우(자세한 내용은 이 설명서의 다음 문서 참조) 먼저 Reporting Services SQL Server 설치해야 합니다. Reporting Services는 설치하는 동시에 설치하거나 Microsoft SQL Server 설치한 후 SQL Server 수 있습니다. 설치하지 않은 SQL Server 이 설명서의 앞부분에 제공된 지침을 따릅니다. 이 SQL Server 기능 선택 페이지에서 Reporting Services를 선택해야 합니다. 이 경우 Reporting Services SQL Server 설치됩니다.
  
Reporting Services를 설치하는 SQL Server 자세한 내용은 SQL Server [Reporting Services(SSRS)를 참조하세요.](https://technet.microsoft.com/library/ms159106.aspx)
  
Reporting Services에 SQL Server 설치했지만 SQL Server 설치하지 않은 경우 SQL Server Reporting Services에 대한 적절한 지침에 따라 해당 기능을 추가할 수 있습니다.
