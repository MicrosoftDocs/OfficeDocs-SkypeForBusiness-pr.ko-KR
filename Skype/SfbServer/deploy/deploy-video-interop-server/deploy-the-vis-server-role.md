# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>비즈니스용 Skype 서버에서 VIS 서버 역할 배포
 
**요약:** 비즈니스용 Skype 서버에서 VIS (비디오 Interop Server) 역할을 배포 합니다.
  
토폴로지 작성기에서 방금 만든 서버에서 VIS 서비스를 설정 하려면 비즈니스용 Skype 서버 배포 마법사를 시작 하 고 비즈니스용 **Skype Server System 설치 또는 업데이트** 를 누른 후 마법사에서 다음 단계를 수행 합니다.
  
1.  **로컬 구성 저장소 설치**를 선택 합니다.
    
2. **비즈니스용 Skype 서버 구성 요소 설치 또는 제거를**선택 합니다.
    
3. **요청, 설치 또는 인증서 할당을**선택 합니다.
    
4. **서비스 시작**을 선택 합니다.
    
이 서비스에 대 한 소프트웨어가 설치 되어 실행 중입니다. 서비스 mmc 도구를 열어 비즈니스용 **Skype 서버 영상 Interop 서버** 서비스가 다른 비즈니스용 skype 서버 서비스와 함께 실행 되 고 있는지 확인 하는 것이 좋습니다. 다음으로 VIS 서버 또는 풀을 구성 해야 합니다.
