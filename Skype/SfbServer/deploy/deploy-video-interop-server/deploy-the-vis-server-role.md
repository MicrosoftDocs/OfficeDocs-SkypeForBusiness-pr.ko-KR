# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>비즈니스용 Skype 서버에서 VIS 서버 역할 배포
 
**요약:** 비즈니스용 Skype 서버에서 VIS(Video Interop 서버) 역할을 배포합니다.
  
토폴로지 작성기에서 방금 만든 서버에서 VIS 서비스를 설정하려면 비즈니스용 Skype 서버 배포 마법사를 시작하고 비즈니스용 **Skype** 서버 시스템 설치 또는 업데이트를 누르고 마법사에서 다음 단계를 수행합니다.
  
1.  로컬 **구성 저장소 설치를 선택합니다.**
    
2. 설치 **또는 비즈니스용 Skype 서버 구성 요소를 제거합니다.**
    
3. 인증서 요청, 설치 또는 **할당을 선택합니다.**
    
4. 서비스 **시작을 선택합니다.**
    
이제 이 서비스의 소프트웨어가 설치되고 실행됩니다. 서비스 mmc 도구를 열어 비즈니스용 **Skype 서버 Video Interop 서버** 서비스가 다른 비즈니스용 Skype 서버 서비스와 함께 실행 중인지 볼 수 있습니다. 그런 다음 VIS 서버 또는 풀을 구성해야 합니다.
