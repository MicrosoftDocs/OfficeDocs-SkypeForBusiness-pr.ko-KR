# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>2016에서 비디오 interop 서버 비즈니스용 Skype 서버
 
**요약:** VIS 서버 역할을 배포할 비즈니스용 Skype 서버.
  
비즈니스용 Skype 서버 Cisco C60 또는 Cisco MX300과 같은 Cisco 원격 구성 시스템(VTC)과 직접 통합할 수 있습니다. 이렇게 하려면 VIS(Video Interop Server)라는 새 서버 역할을 도입하고 VIS와 상호 연동되는 장비와 VIS를 모두 올바르게 구성해야 합니다. VTC는 CUCM(Cisco Unified Communication Manager)과 같은 기존 Cisco 인프라에 등록되고 CUCM과 VIS 풀 간에 비디오 SIP 트렁크가 사용됩니다.

VIS 서버 또는 풀과 VTC 시스템 간의 상호 운영성을 구성하려면 다음 다섯 가지 절차를 수행해야 합니다.
