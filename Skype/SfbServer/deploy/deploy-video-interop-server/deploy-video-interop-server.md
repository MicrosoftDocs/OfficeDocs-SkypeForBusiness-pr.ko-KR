# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>비즈니스용 Skype 서버에서 비디오 Interop 서버 배포
 
**요약:** 비즈니스용 Skype 서버에서 VIS 서버 역할을 배포 합니다.
  
비즈니스용 Skype 서버는 이제 cisco C60 또는 Cisco MX300와 같은 Cisco teleconferencing 시스템 (VTCs)과 직접 통합할 수 있습니다. 이를 위해서는 비디오 Interop 서버 (VIS) 라는 새 서버 역할을 도입 하 고 VIS와 상호 운용 하는 장비의 올바른 구성을 수행 해야 합니다. CUCM (Cisco 통합 커뮤니케이션 관리자)와 같은 기존 Cisco 인프라를 사용 하 여 VTC를 등록 한 경우 비디오 SIP 트렁크가 CUCM와 VIS 풀 사이에 사용 됩니다.
