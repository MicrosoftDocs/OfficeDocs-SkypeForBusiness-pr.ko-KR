# <a name="optional-verify-call-park-deployment-in-skype-for-business"></a>) 비즈니스용 Skype에서 통화 공원 배포 확인
 
비즈니스용 Skype Server Enterprise Voice에서 통화 공원 배포를 확인 하는 중입니다. 
  
통화 대기를 설치 및 구성한 후에는 파킹 및 검색 통화가 예상 대로 작동 하는지 확인 하는 구성을 검증 해야 합니다. 최소한 다음 사항을 확인 하세요.
  
- 통화 공원을 사용 하도록 설정한 사용자에 게 전화를 걸고 사용자에 게 전화를 걸 수 있습니다.
    
    > [!NOTE]
    > 이 테스트를 수행 하기 전에 음성 정책에서 통화 대기를 사용 하도록 설정한 경우 통화를 진행 하는 사용자는 비즈니스용 Skype에서 로그 아웃 한 다음 다시 로그인 하 여 통화 전송 목록의 통화 대기 옵션을 볼 수 있습니다. 
  
- 통화를 검색 하려면 궤도 번호로 전화를 겁니다.
    
- 다른 전화를 걸고, 파킹 된 통화 시간을 초과 하 고, ringback를 선택 하지 않습니다. 시간 제한 통화가 **Ontimeouturi**에 대해 지정 된 대체 대상으로 올바르게 라우팅 되는지 확인 합니다.
