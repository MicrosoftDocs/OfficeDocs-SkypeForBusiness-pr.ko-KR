# <a name="optional-verify-response-group-deployment-in-skype-for-business"></a>반드시 비즈니스용 Skype에서 응답 그룹 배포 확인
 
비즈니스용 Skype 서버 Enterprise Voice에서 응답 그룹 배포가 성공적으로 작동 하는지 확인 합니다.
  
응답 그룹을 구성한 후에는 해당 구성에서 응답 그룹이 예상 대로 작동 하는지 확인 해야 합니다. 최소한 다음과 같은 유형의 사용자를 사용 하 여 다음 시나리오를 확인 합니다.
  
 **사용자**
  
- 비즈니스용 Skype에 있는 사용자
    
- 공중 전화망 (PSTN)을 사용 하는 외부 사용자
    
- 비즈니스용 Skype에 속한 에이전트
    
  **시나리오**
  
- 비즈니스용 Skype 사용자가 응답 그룹에 전화를 걸고 있습니다.
    
- 외부 사용자가 응답 그룹에 전화를 겁니다.
    
- 에이전트가 다른 전화를 받는 중에 사용자가 응답 그룹에 전화를 걸어 사용자가 큐로 이동됩니다.

    > [!NOTE]
    > 응답 그룹이 예상 대로 작동 하지 않으면 다음 링크를 확인 하세요.https://support.office.com/article/troubleshooting-for-response-groups-ca72d8f8-4054-4974-b832-4f173611bd89
