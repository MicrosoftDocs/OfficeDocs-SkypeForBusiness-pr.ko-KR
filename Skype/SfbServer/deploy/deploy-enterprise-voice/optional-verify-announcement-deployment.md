# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>(선택 사항) 비즈니스용 Skype에서 공지사항 배포 확인
 
비즈니스용 Skype 서버에서 공지 응용 프로그램 배포 Enterprise Voice. 이는 미지정 번호에 대한 호출이 처리된 방식에 영향을 미치게 됩니다.
  
알림을 설치 및 구성한 후에는 지정되지 않은 번호에 대한 통화가 예상대로 작동하는지 구성을 확인해야 합니다. 최소한 다음 사항을 확인하십시오.
  
- 조직에 대해 유효하지만 미지정 번호인 번호를 호출합니다.
    
- 지정되지 않은 번호에 전화를 걸어 올바른 알림이 재생되는지 확인합니다.
