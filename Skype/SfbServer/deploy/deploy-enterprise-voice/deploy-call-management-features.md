# <a name="deploy-call-management-features-in-skype-for-business"></a>비즈니스용 Skype에서 통화 관리 기능 배포
 
비즈니스용 Skype Server Enterprise Voice에서 통화 관리 기능을 배포 합니다.
  
기업 음성 통화 관리 기능은 수신 전화를 라우팅하거나 응답 하는 방법을 제어 합니다. 비즈니스용 Skype Server는 다음 통화 관리 기능을 제공 합니다. 
  
- **통화**대기: 음성 사용자가 통화를 일시적으로 충전 한 다음 같은 전화나 다른 전화기에서 선택할 수 있습니다.
    
- **그룹 통화 픽업**: 사용자가 통화 픽업 그룹 번호로 전화를 걸어 픽업 그룹에 할당 된 다른 사용자의 통화에 대 한 응답을 받을 수 있도록 합니다.
    
- **응답 그룹**: 헌트 그룹 또는 IVR (대화형 음성 응답) 질문 및 답변을 사용 하 여 수신 전화를 에이전트 그룹으로 라우팅합니다.
    
- **공지**사항: 지정 하지 않은 번호에 대 한 통화에 대 한 메시지를 재생 하거나 통화를 다른 곳으로 또는 둘 다로 라우팅합니다.
    
이 섹션에서는 엔터프라이즈 음성 배포 중에 이러한 통화 관리 기능을 구성 하는 방법에 대해 설명 합니다.
