# <a name="provisioning-of-skype-room-system-exchange-and-skype-accounts"></a>Skype 대화방 시스템 Exchange 및 Skype 계정 프로 비전
 
Skype 대화방 시스템용 Exchange 및 Skype 계정을 프로 비전 하는 방법에 대 한 자세한 내용은 다음 항목을 참조 하십시오. 

> [!NOTE]
> Microsoft 팀 대화방은 서로 다른 종속성 및 배포 절차를 사용 하는 다른 제품입니다. Microsoft 팀 대화방에 대 한 자세한 내용은 Microsoft 팀원 대화방 [배포 개요](room-systems-v2.md)를 참조 하세요.
  
> [!NOTE]
> Skype 대화방 시스템 계정 프로 비전은 조직에 있는 토폴로지의 유형에 따라 달라 집니다. Active Directory 토폴로지에 대 한 자세한 내용은 [비즈니스용 Skype 서버 2015에 대 한 환경 요구 사항을](../../plan-your-deployment/requirements-for-your-environment/environmental-requirements.md)참조 하세요. 
  
## <a name="provisioning-of-skype-room-system-exchange-amp-skype-for-business-accounts"></a>Skype 대화방 시스템 프로 비전 비즈니스용 &amp; Skype 계정

다음 항목에서는 다음에 대 한 Skype 대화방 시스템 교환 및 비즈니스용 Skype 계정을 구축 하 고 관리 하는 방법에 대해 설명 합니다.
  
- 단일 포리스트 온-프레미스 배포
    
- 다중 포리스트 온-프레미스 배포
    
- Microsoft 365 또는 Office 365
    
- 하이브리드 배포
    
- 비즈니스용 skype 대화방 시스템 및 비즈니스용 Skype 페더레이션 파트너
    
- Skype 대화방 시스템 계정 관리
