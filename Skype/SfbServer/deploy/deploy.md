# <a name="deploy-skype-for-business-server-2015"></a>2015 비즈니스용 Skype 서버 배포
 
**요약:** 다음 링크를 통해 비즈니스용 Skype 서버 2015를 설치 및 구성하고 이전 버전의 Lync Server에서 업그레이드하는 방법을 알아보는 방법을 참조하세요.
  
비즈니스용 Skype 서버 엔터프라이즈 수준의 공동 작업 요구 사항을 지원할 수 있는 IM(인스턴트 메시징), 현재 상태, 회의, 비디오 및 전화 통신 솔루션을 제공합니다. 이 항목에서는 배포 및 구성 방법에 대한 정보의 링크를 비즈니스용 Skype 서버. 
  
이러한 문서의 작성자는 새로운 정보가 제공되고 사용자가 의견을 제공함에 따라 문서를 지속적으로 업데이트합니다. 이 페이지는 자주 업데이트될 예정이기 때문에 자주 다시 확인하십시오.
