# <a name="latest-updates-for-versions-of-skype-for-business-that-use-windows-installer-msi"></a>MSI(설치 관리자)를 비즈니스용 Skype 버전의 최신 Windows 업데이트

이 페이지의 링크를 사용하여 비즈니스용 Skype 2016, 비즈니스용 Skype 2015 및 Lync 2013 클라이언트의 최신 업데이트에 대한 최신 업데이트를 다운로드할 수 있습니다.

> [!NOTE]
> - 이 문서의 정보는 MSI(비즈니스용 Skype Installer) 설치 기술을 사용하는 Windows 버전에만 적용됩니다. 예를 들어 볼륨 라이선스 버전의 비즈니스용 Skype 설치한 경우(예: 비즈니스용 Skype 포함) Office Professional Plus 2016.
> - 이 문서의 정보는 Microsoft 365 또는 Office 365 버전(예: 비즈니스용 Skype 포함)에는 적용되지 비즈니스용 Skype 엔터프라이즈용 Microsoft 365 앱.
> - 최신 모바일 클라이언트 업데이트의 경우 모바일 클라이언트(iTunes, Google Play 또는 Microsoft Store)의 앱 스토어로 이동하여 세부 정보를 보고 업데이트를 설치합니다.
> - 사용 중이 비즈니스용 Skype 버전에 대한 자세한 내용은 어떤 버전의 비즈니스용 Skype [있나요?](https://support.microsoft.com/help/3195481/how-to-check-the-version-of-skype-for-business-2016)
> - Office 업데이트 설치에 대한 자세한 내용은 [Office 업데이트 설치](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5)를 참조하세요.

비즈니스용 Skype 및 Microsoft 365 Office 365 환경과의 호환성은 배포에 대한 시스템 요구 사항에 따라 다릅니다.

- Microsoft 365 및 Office 365 2016의 최신 버전을 비즈니스용 Skype 지원되는 모든 버전의 비즈니스용 Skype 사용할 수 있습니다. 이전 버전의 비즈니스용 Skype 지원이 확장된 버전만 기능이 축소되어도 Microsoft 365 Office 365 계속 사용할 수 있습니다. 자세한 내용은 [Microsoft 수명 주기 정책을 참조하세요.](https://support.microsoft.com/lifecycle)

- 비즈니스용 Skype 서버 배포의 경우 배포 버전에 대한 시스템 요구 사항을 비즈니스용 Skype.

  - [사용자 환경의 비즈니스용 Skype 요구 사항](./plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md)

  - [Lync Server 2013 시스템 요구 사항](/previous-versions/office/lync-server-2013/lync-server-2013-system-requirements)
