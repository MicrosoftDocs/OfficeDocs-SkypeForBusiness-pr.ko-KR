# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>비즈니스용 Skype 서버 구성 요소 설치 또는 제거(소개)
 
2015 구성 요소의 설치 및 활성화 또는 비활성화 및 비즈니스용 Skype 서버 준비 중입니다. 실행할 프로세스는 중앙 관리 저장소에 게시된 정의의 현재 구성에 의해 정의됩니다. 작업을 성공적으로 완료하려면 로컬 Administrators 그룹의 구성원으로 로그인해야 합니다. 최소한 Active Directory 도메인 서비스를 읽을 수 있어야 합니다.
