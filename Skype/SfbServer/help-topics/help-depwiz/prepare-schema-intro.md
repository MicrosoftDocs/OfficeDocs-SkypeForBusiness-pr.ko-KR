# <a name="prepare-schema-intro"></a>스키마 준비(소개)
 
비즈니스용 Skype Server 2015의 스키마를 확장 하기 위해 작업을 시작 하려면 **스키마 준비** 페이지에서 시작 합니다. 스키마 준비를 계속하려면 **다음**을 클릭합니다.
