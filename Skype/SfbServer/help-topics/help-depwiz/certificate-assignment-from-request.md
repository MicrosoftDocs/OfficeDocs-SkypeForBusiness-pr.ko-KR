# <a name="certificate-assignment-from-request"></a>인증서 할당(요청에서)
 
**인증서 할당** 페이지는 역할 서버에 대해 요청된 사용에 인증서를 할당하는 프로세스를 시작합니다. **인증서 세부 정보 보기**를 클릭하여 인증서를 봅니다. **일반** 탭, **자세히** 탭 또는 **인증 경로** 탭을 선택할 수 있습니다. **일반** 탭에는 발급 인증 기관 및 개인 키 상태와 같은 인증서에 대한 기본 정보가 표시됩니다. 자세히 탭에는 주체 이름, 주체 대체 이름 및 기타 세부 정보와 같은 인증서 정보가 표시됩니다. **인증 경로** 탭에는 인증서에서 발급 또는 중간 인증 기관(있는 경우) 및 루트 인증 기관에 이르는 인증서 "신뢰 체인"이 표시됩니다.
  
계속하려면 **다음**을 클릭합니다.
