# <a name="prepare-schema-invoke"></a>스키마 준비(호출)
 
명령 **실행** 페이지에는 2015년 6월 20일까지의 비즈니스용 Skype 서버 실행에 대해 실행된 명령의 요약이 표시됩니다. 작업이 완료되면 요약 창에 작업이 성공했거나 실패한 것으로 표시될 수 있습니다. 요약 화면을 검토하고 요약 창 아래에 작업 상태 **: 완료** 가 표시되는지 검토합니다. 요약 창의 드롭다운 목록을 사용하여 로그 파일인 Prepare Schema를 선택하고 볼 수 있습니다. 로그 **보기를 클릭하여** 로그 파일을 검토합니다. 스마마 준비 중에 오류가 발생하면 해당 작업이 여기에 기록됩니다. 그런 다음 오류를 수정하기 위한 조치를 수행한 다음, 다시 실행하여 오류를 수정할 수 있습니다.
  
**마침** 을 클릭하여 작업을 완료합니다.
