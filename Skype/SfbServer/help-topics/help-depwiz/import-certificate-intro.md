# <a name="import-certificate-intro"></a>인증서 가져오기(소개)
 
인증서를 가져오려면 인증서 파일에 대 한 경로를 제공 해야 합니다. **인증서 파일 선택** 텍스트 상자에서 전체 경로와 파일 이름을 입력 하거나, **찾아보기** 단추를 클릭 하 고 경로 위치와 파일 이름 (일반적으로. p7b, .pfx 또는 .cer 파일)으로 이동할 수 있습니다.
  
인증서에 개인 키가 포함 되어 있으면 인증서 **파일에 인증서의 개인 키가 포함**된 확인란을 선택 합니다. 이 확인란을 선택 하면 **암호** 텍스트 입력을 사용할 수 있습니다. 개인 키와 연결 된 인증서가 있는 경우 일반적으로 인증서가 만들어질 때 개인 키에 암호가 저장 됩니다. 개인 키에 대 한 암호를 입력 하 여 인증서와 개인 키를 인증서 저장소로 가져올 수 있도록 합니다. 인증서 파일 경로에 대 한 정보를 제공 하 고 필요에 따라 개인 키 암호를 입력 하는 경우 **다음**을 클릭 합니다.
  
> [!IMPORTANT]
> 개인 키에 대 한 암호를 모르는 경우에는 가져오지 못합니다.
