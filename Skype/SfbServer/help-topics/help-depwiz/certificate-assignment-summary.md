# <a name="certificate-assignment-summary"></a>인증서 할당(요약)
 
인증서 할당을 마치기 전에 할당될 인증서에 대한 정보가 나열된 **인증서 할당 요약** 이 표시됩니다. 정보를 검토하여 주체 이름과 주체 대체 이름이 올바른지 확인합니다. 인증서 요약 검토를 마쳤으면 **다음** 을 클릭합니다.
