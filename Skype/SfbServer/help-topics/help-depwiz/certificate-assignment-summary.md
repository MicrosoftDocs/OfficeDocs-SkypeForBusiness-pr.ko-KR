# <a name="certificate-assignment-summary"></a>인증서 할당(요약)
 
인증서 할당이 완료 되기 전에 할당 될 인증서의 세부 정보를 나열 하는 **인증서 할당 요약이** 표시 됩니다. 정보를 검토 하 여 주체 이름 및 주체 대체 이름이 올바른지 확인 합니다. 인증서 요약 검토가 완료 되 면 **다음**을 클릭 합니다.
