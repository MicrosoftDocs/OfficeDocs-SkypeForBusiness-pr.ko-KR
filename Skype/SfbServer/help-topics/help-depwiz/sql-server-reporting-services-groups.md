# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services(그룹)

읽기 전용 **지정 그룹**에서 모니터링 보고서에 읽기 전용 액세스 권한을 부여할 그룹을 지정 합니다. 도메인 &lt;&gt;\\액세스 형식 그룹&gt;을 사용&lt;합니다. 예: contoso\RTCUniversalReadOnlyAdmins.
  
그룹 지정 읽기 전용 액세스는 선택 사항입니다. 지금 읽기 전용 액세스 권한을 부여 하지 않으려는 경우 나중에 SQL Server Reporting Services에서 직접 구성할 수 있습니다.
