# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services(그룹)

**읽기 전용 그룹 지정** 에서 모니터링 보고서에 대한 읽기 전용 권한을 부여할 그룹을 지정합니다. &lt;domainaccess 그룹&lt;&gt;\\ 형식을 사용 합니다.&gt; 예를 들어 contoso\RTCUniversalReadOnlyAdmins와 같이 지정합니다.
  
그룹에 읽기 전용 권한을 할당하는 것은 선택 사항입니다. 지금 읽기 전용 권한을 부여하지 않으려는 경우 나중에 SQL Server Reporting Services에서 직접 읽기 전용 권한을 구성할 수 있습니다.
