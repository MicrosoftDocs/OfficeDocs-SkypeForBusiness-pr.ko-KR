# <a name="install-or-update-skype-for-business-server-system"></a>비즈니스용 Skype 서버 시스템 설치 또는 업데이트

설치를 시작 하거나 비즈니스용 Skype Server 2015의 기존 배포를 업데이트 하려면 비즈니스용 **Skype 서버 설치 또는 업데이트**를 클릭 합니다. 이 섹션의 단계에서는 토폴로지 작성기 문서를 기반으로 지정 된 서버 역할에 해당 하는 구성 요소를 설치 합니다. 이 일련의 단계를 시작 하기 전에 토폴로지 작성기에서 배포를 정의 해야 합니다. 로컬 관리자 및 서버가 위치해 있는 도메인의 Domain Admins 그룹 구성원으로 배포할 컴퓨터에 로그인해야 합니다. 이 단계를 완료하기 전에 컴퓨터를 도메인에 가입시켜야 합니다.

현재 배포 된 서버를 업데이트 하는 경우 토폴로지 작성기에서 변경 하려는 내용을 설정 해야 합니다.
