# <a name="certificate-request-invoke"></a>인증서 요청(호출)
 
**명령 실행** 페이지에는 인증서 요청 생성을 위해 명령이 실행됨에 따라 작업 상태 창이 표시됩니다. 작업이 완료된 후 요약 창 아래의 드롭다운 목록에 로그 파일 목록이 포함됩니다. 드롭다운 목록을 사용하여 볼 로그 파일을 선택한 다음 **로그 보기** 를 클릭합니다. Internet Explorer가 시작되어 로그 파일을 표시합니다. 실패한 요청의 경우 로그 파일을 검토하여 실패의 원인을 확인해야 합니다. 문제를 해결하기 위해 인증서 요청을 다시 실행하여 로그 파일에 표시된 문제를 해결할 수 있습니다. 인증서 요청 페이지를 통해 계속하고 인증서를 다시 요청합니다.
