After the tasks are completed, the drop-down list below the summary pane contains a list of log files. Use the drop-down list to select a log file to view, and then click **View Log**. Internet Explorer will start and display the log file. In case of a failed request, you should review the log file(s) to determine the cause of the failure. You can rerun the certificate request to resolve the issue, correcting the problem indicated in the log file. Continue though the certificate request pages and request the certificate again.</span></span>
