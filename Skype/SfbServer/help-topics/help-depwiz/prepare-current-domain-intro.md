# <a name="prepare-current-domain-intro"></a>현재 도메인 준비(소개)
 
도메인 **준비 페이지에서** 는 2015년 2015에 대해 도메인을 준비하는 비즈니스용 Skype 서버 시작됩니다. 도메인 준비를 시작하려면 **다음** 을 클릭합니다.
