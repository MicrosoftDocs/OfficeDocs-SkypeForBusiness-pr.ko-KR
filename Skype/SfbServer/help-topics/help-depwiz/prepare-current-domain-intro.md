# <a name="prepare-current-domain-intro"></a>현재 도메인 준비(소개)
 
**도메인 준비** 페이지는 비즈니스용 Skype Server 2015에 대 한 도메인을 준비 하는 프로세스를 시작 합니다. 도메인 준비를 시작하려면 **다음**을 클릭합니다.
