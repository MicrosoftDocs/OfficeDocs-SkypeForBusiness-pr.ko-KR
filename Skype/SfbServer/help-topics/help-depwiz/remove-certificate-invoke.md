# <a name="remove-certificate-invoke"></a>인증서 제거(호출)
 
**명령 실행** 페이지에서는 인증서 제거에 대한 세부 정보가 제공됩니다. 인증서 제거 요약 정보를 검토합니다. 인증서 제거 로그 파일이 드롭다운 목록에 표시되는지 확인하고 **로그 보기** 를 클릭하여 인증서 제거 세부 정보를 검토합니다. 요약 및 로그 파일 검토를 마쳤으면 **마침** 을 클릭합니다.
