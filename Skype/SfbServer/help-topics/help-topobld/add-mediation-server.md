# <a name="add-mediation-server"></a>중재 서버 추가
 
중재 서버를 정의 하려면 중재 서버 풀에 대 한 FQDN (정규화 된 도메인 이름)을 지정 합니다. 중재 서버 풀을 만드는 경우 배포 하는 중재 서버에 대 한 풀의 이름을 지정 합니다. 다른 단계에서 개별 서버의 FQDN을 정의합니다. 단일 중재 서버를 배포 하는 경우 사용자가 지정 하는 FQDN은 단일 중재 서버의 FQDN입니다. 자세한 내용은 [비즈니스용 Skype 서버 2015의 토폴로지 작성기에서 중재 서버 배포](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)를 참조 하세요.
