# <a name="add-mediation-server"></a>중재 서버 추가
 
중재 서버를 정의하려면 중재 서버 풀의 FQDN(정규화된 도메인 이름)을 지정합니다. 중재 서버 풀을 만드는 경우 배포하는 중재 서버 풀의 이름을 지정합니다. 다른 단계에서 개별 서버의 FQDN을 정의합니다. 단일 중재 서버를 배포하는 경우 지정하는 FQDN은 단일 중재 서버의 FQDN이 됩니다. 자세한 내용은 [비즈니스용 Skype 서버 2015의](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)토폴로지 작성기에서 중재 서버 배포를 참조하세요.
