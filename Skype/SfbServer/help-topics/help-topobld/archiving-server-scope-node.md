# <a name="archiving-server-scope-node"></a>보관 서버 범위 노드
 
토폴로지 작성기에서는 콘솔 트리에서 풀을 마우스 오른쪽 단추로 클릭 하 고 **속성 편집**을 선택 하 여 보관을 실행 하는 서버와 관련 된 작업을 수행할 수 있습니다. **속성 편집** 페이지에서는 보관을 실행 중인 서버의 FQDN(정규화된 도메인 이름), 보관 SQL Server 데이터베이스 및 보관 파일 저장소를 변경할 수 있습니다.
