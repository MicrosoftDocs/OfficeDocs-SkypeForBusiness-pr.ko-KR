# <a name="add-director-association-for-lync-server-2010"></a>Lync Server 2010에 대 한 디렉터 연결 추가
 
디렉터 또는 디렉터 풀에 연결 되는 모니터링 서버가 있는지 여부를 정의 합니다. 모니터링 서버를 디렉터 또는 디렉터 풀에 연결 하려면 확인란을 선택 합니다.
  
> [!TIP]
> 모니터링 서버 데이터베이스 (및 선택 인스턴스)를 아직 만들지 않은 경우이 확인란을 선택 하 여 모니터링 서버를 디렉터 또는 디렉터 풀에 연결 하면 나중에이를 만들 수 있는 기회가 주어 집니다.
