# <a name="add-director-features"></a>Director 기능 추가

디렉터 풀에 모니터링 서버를 포함 하려면 **모니터링 (CDR 및 체감 품질 메트릭스)** 을 선택 합니다.

디렉터 풀을 추가 하는 방법에 대 한 자세한 내용은 [디렉터에 대 한 시나리오](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx) 및 [디렉터 설정을](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)참조 하세요.
