# <a name="add-default-sip-domain"></a>기본 SIP 도메인 추가
 
배포에서 지원하도록 할 기본 SIP(Session Initiation Protocol) 도메인을 지정합니다. 기본 SIP 도메인은 Active Directory 도메인 서비스 도메인과 같거나 다를 수 있습니다. 예를 들어 Active Directory 도메인이 fabrikam.net일 수 있으며 기본 SIP 도메인을 contoso.net으로 지정할 수 있습니다.
