# <a name="edit-office-web-apps-server-settings"></a>Office Web Apps 서버 설정 편집

구성 된 Office Web Apps 서버의 속성을 편집 합니다. 편집할 수 있는 속성은 다음과 같습니다.

 **Office Web Apps 서버 FQDN**:이 속성은 Office Web apps 서버의 정규화 된 도메인 이름을 정의 하 고 DNS (domain name system) 호스트 a 또는 AAAA (IPv6이 사용 되는 경우) 레코드와 일치 해야 합니다.

 **Office Web Apps 서버 검색 url**: Office Web apps 서버에 대 한 클라이언트 액세스를 위한 url (uniform resource locator)을 사용 하는 경우, 서버가 다른 네트워크 영역에 있는 경우에는이 주소를 기본 네트워크로 편집 해야 할 수 있습니다. 배포가.

경계 네트워크 또는 더 적은 신뢰 네트워크를 분리 하는 내부 방화벽 외부의 경계 네트워크 또는 기타 네트워크 영역에이 서버를 배포 하는 경우 **Office Web Apps 서버가 외부 네트워크에 배포 되** 는 확인란을 선택 합니다. 내부 배포의 인터넷.
