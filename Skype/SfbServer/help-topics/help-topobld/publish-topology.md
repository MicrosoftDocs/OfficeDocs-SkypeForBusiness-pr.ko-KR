# <a name="publish-topology"></a>토폴로지 게시
 
게시 프로세스를 시작 하면 **토폴로지 게시** 페이지가 표시 됩니다. 진행 하기 전에 나열 된 작업이 완료 되었는지 확인 합니다. 계속할 준비가 되 면 **다음**을 클릭 합니다. 게시 프로세스를 종료 해야 하는 경우 **취소**를 클릭 합니다.
