# <a name="legacy-merge-results"></a>레거시 병합 결과

레거시 배포와 레거시 배포의 비즈니스용 Skype 서버 다음을 **클릭합니다**. 토폴로지 작성기 병합 마법사를 실행하는 데 대한 자세한 내용은 마이그레이션 설명서에서 [Merge Using Topology Builder Merge Wizard](/previous-versions/office/lync-server-2013/merge-using-topology-builder-merge-wizard) 를 참조하십시오.
