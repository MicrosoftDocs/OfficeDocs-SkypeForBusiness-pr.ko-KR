# <a name="legacy-merge-results"></a>레거시 병합 결과

비즈니스용 Skype 서버와 레거시 배포의 병합을 시작 하려면 **다음**을 클릭 합니다. 토폴로지 작성기 병합 마법사를 실행하는 방법에 대한 자세한 내용은 마이그레이션 설명서에서 [Merge Using Topology Builder Merge Wizard](https://technet.microsoft.com/library/c3f3c425-dab6-4dcd-bf0e-d7fde05f2ebf.aspx)을 참조하세요.
