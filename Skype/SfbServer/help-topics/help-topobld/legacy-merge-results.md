# <a name="legacy-merge-results"></a>레거시 병합 결과

레거시 배포와 비즈니스용 Skype 서버의 Merging을 시작하려면 다음을 **클릭합니다.** 토폴로지 작성기 병합 마법사를 실행하는 [](https://technet.microsoft.com/library/c3f3c425-dab6-4dcd-bf0e-d7fde05f2ebf.aspx) 자세한 내용은 마이그레이션 설명서에서 토폴로지 작성기 병합 마법사를 사용하여 병합을 참조하십시오.
