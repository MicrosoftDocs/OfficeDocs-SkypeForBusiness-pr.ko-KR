# <a name="skype-for-business-server-root-scope-node"></a>비즈니스용 Skype 서버 루트 범위 노드
 
**Lync Server** 노드에 토폴로지의 중앙 사이트가 나열 됩니다. 다음 작업을 수행할 수 있습니다.
  
- **새 중앙 사이트** 토폴로지에 새 중앙 사이트를 추가 하는 데 사용 하는 새 중앙 사이트 정의 마법사를 시작 합니다.
    
- **속성 편집**. 지원 되는 SIP (Session 착수 프로토콜) 도메인 및 단순 Url과 같은 전체 토폴로지에 영향을 주는 설정을 편집할 수 있습니다.
    
- **새 토폴로지** 배포를 위한 새 토폴로지 만들기를 시작할 수 있습니다. 이 옵션은 서버를 설치 하기 전에 배포를 계획 하는 경우에 유용 합니다. 현재 토폴로지를 이미 사용 하 여 배포를 설치한 경우에는이 옵션을 클릭 하지 마십시오.
    
- **토폴로지를 엽니다**. 저장 한 토폴로지 파일을 엽니다.
    
- **토폴로지를 다운로드**합니다. 중앙 관리 저장소에서 현재 배포 된 토폴로지를 로드 합니다. 이미 토폴로지를 정의 하 고 중앙 관리 저장소를 설정한 경우에는 항상이 옵션을 선택 하 여 토폴로지를 로드 해야 합니다.
    
- **토폴로지 복사본을로 저장**합니다. 토폴로지 작성기에서 나중에 다시 로드할 수 있는 파일에 현재 토폴로지를 저장 합니다.
    
- **토폴로지를 게시**합니다. 서버 배포를 시작할 수 있도록이 토폴로지를 중앙 관리 저장소에 게시 하려면이 옵션을 사용 합니다.
    
- **데이터베이스를 설치**합니다. 배포에 필요한 데이터베이스를 만들 수 있습니다.
    
- **Office Communications Server 2007 R2 토폴로지를 병합**합니다. 토폴로지 작성기 병합 마법사를 시작 합니다. 이 도구는 Edge Server 정보를 포함 하 여 이전 Office Communications Server 환경에 대 한 정보를 수집 하 고 해당 정보를 공유 데이터베이스에 게시 합니다. 
    
- **배포를 제거**합니다. 전체 배포 제거를 시작할 수 있는 배포 제거 마법사를 시작 합니다.
    
이러한 옵션 외에 다음과 같은 일반적인 작업을 수행할 수 있습니다.
  
- **보기**입니다. 이 옵션을 사용 하 여 콘솔 트리 보기에서 표시 하거나 숨길 항목을 선택 하 여 토폴로지 작성기 보기를 사용자 지정할 수 있습니다.
    
- **도움말**. 이 옵션을 사용하여 활성 노드에 대한 도움말에 액세스할 수 있습니다.
