# <a name="skype-for-business-server-root-scope-node"></a>비즈니스용 Skype 서버 루트 범위 노드
 
**Lync Server** 노드에 토폴로지의 중앙 사이트가 나열됩니다. 다음 작업을 수행할 수 있습니다.
  
- **새 중앙 사이트**. 토폴로지에 새 중앙 사이트를 추가하는 데 사용하는 새 중앙 사이트 정의 마법사를 시작합니다.
    
- **속성 편집**. 지원되는 SIP(Session Initiation Protocol) 도메인 및 단순 URL과 같이 전체 토폴로지에 영향을 주는 설정을 편집할 수 있습니다.
    
- **새 토폴로지**. 배포에 대한 새 토폴로지 만들기를 시작할 수 있습니다. 이 옵션은 서버를 설치하기 전에 배포를 계획할 때 유용합니다. 배포를 설치하기 위해 현재 토폴로지가 이미 사용된 경우 이 옵션을 클릭하지 않습니다.
    
- **토폴로지 열기.** 저장한 토폴로지 파일을 니다.
    
- **토폴로지 다운로드.** 중앙 관리 저장소에서 현재 배포된 토폴로지 로드 토폴로지가 이미 정의되어 있으며 중앙 관리 저장소를 설정한 경우 항상 이 옵션을 선택해야 토폴로지가 로드됩니다.
    
- **토폴로지 복사본을 로 저장합니다.** 현재 토폴로지가 나중에 토폴로지 작성기에서 다시 로드할 수 있는 파일에 저장합니다.
    
- **토폴로지** 게시. 이 옵션을 사용하여 서버 배포를 시작할 수 있도록 이 토폴로지가 중앙 관리 저장소에 게시됩니다.
    
- **데이터베이스를 설치합니다.** 배포에 필요한 데이터베이스를 만들 수 있습니다.
    
- **Communications Server Office R2** 토폴로지 병합 토폴로지 작성기 병합 마법사를 시작합니다. 이 도구는 에지 서버 정보를 포함하여 이전 Office Communications Server 환경에 대한 정보를 수집하고 해당 정보를 공유 데이터베이스에 게시합니다. 
    
- **배포를 제거합니다.** 전체 배포 제거를 시작할 수 있는 배포 제거 마법사를 시작합니다.
    
이러한 옵션 외에 다음과 같은 일반적인 작업을 수행할 수 있습니다.
  
- **보기.** 이 옵션을 사용하여 콘솔 트리 보기에 표시하거나 숨길 항목을 선택하여 토폴로지 작성기 보기를 사용자 지정할 수 있습니다.
    
- **도움말**. 이 옵션을 사용하여 활성 노드에 대한 도움말에 액세스할 수 있습니다.
