# <a name="define-computers-in-persistent-chat-pool"></a>영구 채팅 풀에서 컴퓨터 정의
 
컴퓨터 **FQDN** 에 FQDN(FQDN)을 입력하여 영구 채팅 서버 풀에 참여할 컴퓨터를 정의합니다. 이 FQDN은 DNS(Domain Name System) 호스트(A 또는 AAAA) 레코드에 정의된 컴퓨터의 이름이어야 합니다.
  
풀의 컴퓨터 목록에 FQDN을 추가하려면 **추가** 를 클릭합니다.
  
풀에 대해 정의된 컴퓨터 목록을 구성하는 컴퓨터에서 컴퓨터 목록의 항목을 새로 고치려면 **업데이트** 를 클릭합니다.
  
풀에서 정의된 컴퓨터를 삭제하려면 **제거** 를 클릭합니다.
  
이전 풀 정의 대화 상자로 돌아가려면 **뒤로** 를 클릭합니다.
  
이 **풀** 의 컴퓨터 구성원 입력을 마친 후 다음을 클릭하여 영구 채팅 서버 풀 정의를 계속 진행합니다.
  
모든 변경 내용을 취소하고 **새 영구 채팅 풀 정의** 마법사를 종료하려면 **취소** 를 클릭합니다.
  
이 페이지와 같은 상황에 맞는 도움말에 액세스하려면 **도움말** 을 클릭합니다.
