# <a name="monitoring-server-general-settings-expander"></a>모니터링 서버 일반 설정 확장기
 
토폴로지 작성기에서는 콘솔 트리에서 모니터링 서버를 마우스 오른쪽 단추로 클릭 하 고 도구 **모음에서 작업을 클릭 하거나** , 작업 창에서 작업을 클릭 하 고 **속성 편집**을 클릭 하 고 다음 옵션을 변경 하 여 개별 모니터링 서버에 대 한 속성을 편집할 수 있습니다.
  
- **FQDN**: 모니터링 서버로 배포할 서버의 FQDN(정규화된 도메인 이름)을 변경합니다.
    
- Sql **저장소**-모니터링 sql server 데이터베이스에 사용할 sql server 인스턴스를 변경 합니다. 모니터링 서버의 SQL Server 데이터베이스를 변경 하는 경우에는 모니터링 서버를 다시 시작 해야 변경 내용이 적용 됩니다.
