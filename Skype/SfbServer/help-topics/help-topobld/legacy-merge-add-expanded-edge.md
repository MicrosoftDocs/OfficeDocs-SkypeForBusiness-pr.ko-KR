# <a name="legacy-merge-add-expanded-edge"></a>레거시 병합 확장된 에지 추가
 
비즈니스용 Skype 서버는 레거시 Office 통신 서버 2007 R2 확장 Edge 서버 구성의 병합을 지원 하지 않습니다. 에지 서버 정보를 병합하기 전에 먼저 확장 에지 서버를 통합 에지 서버 구성으로 변환해야 합니다.
