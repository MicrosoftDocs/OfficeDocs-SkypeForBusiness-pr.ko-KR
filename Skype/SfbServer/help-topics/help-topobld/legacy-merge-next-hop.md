# <a name="legacy-merge-next-hop"></a>레거시 병합 다음 홉
 
**다음 홉 FQDN** 은 레거시 에지 서버가 내부 트래픽을 라우팅하는 다음 홉 서버를 식별합니다. 이 에지 서버가 내부 트래픽을 라우팅하는 다음 홉 서버의 FQDN(정규화된 도메인 이름)을 입력한 후 다음 홉 서버가 **프런트 엔드 풀** 인지 아니면 **디렉터** 인지를 선택합니다.
