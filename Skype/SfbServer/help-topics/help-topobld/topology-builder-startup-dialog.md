# <a name="topology-builder-startup-dialog"></a>토폴로지 작성기 시작 대화 상자

토폴로지 작성기를 시작 하 여 Microsoft Lync Server 2013 배포의 속성 및 구성을 구성 합니다. 시작할 때 토폴로지 작성기에는 토폴로지 문서의 출처를 묻는 대화 상자가 표시 됩니다. 다음 세 가지 옵션 중에서 선택합니다.

- **기존 배포에서 토폴로지 다운로드** 토폴로지 작성기를 실행 하 고 토폴로지가 중앙 관리 서버에 성공적으로 게시 된 경우이 옵션을 선택 합니다. 이 옵션을 선택하는 경우 현재 구성의 로컬 복사본을 요청 및 다운로드하므로 기존 배포를 편집하고 구성할 수 있습니다.

- **로컬 파일에서 토폴로지 열기** 배포를 구성 하는 중이지만 아직 구성을 게시 하지 않은 경우이 옵션을 선택 합니다. 이 옵션을 사용 하면 토폴로지 작성기에서 구성 작업의 로컬 복사본을 저장 한 후 나중에 다시 돌아올 수 있습니다.

- **새 토폴로지**. 새 토폴로지 문서 구성을 시작하려면 이 옵션을 선택합니다. 이 옵션을 선택하는 경우 구성하려는 배포의 요구 사항을 충족하기 위해 구성할 새 토폴로지 문서가 열립니다.
