# <a name="legacy-merge-add-machine"></a>레거시 병합 컴퓨터 추가
 
에지 서버 풀을 구성하는 컴퓨터를 정의하려면 레거시 부하가 조정된 통합 에지 풀에 있는 하나 이상의 컴퓨터의 FQDN(정식 도메인 이름)을 입력합니다.
