# <a name="legacy-merge-summary-page"></a>레거시 병합 요약 페이지

Microsoft Office Communications Server 2007 R2에서 Lync Server 2013으로 마이그레이션하는 경우 토폴로지 작성기 기능을 사용하여 기존 토폴로지가 병합됩니다. 요약 화면은 Office Communications Server 2007 R2를 Lync Server 2013 토폴로지 작성기 구성으로 병합하는 동안 수행한 모든 작업 및 선택을 지정합니다.

요약을 검토하고 선택 사항이 올바르게 정의되었는지 확인합니다. 요약에 반영된 선택 사항이 올바르면 병합을 진행합니다.
