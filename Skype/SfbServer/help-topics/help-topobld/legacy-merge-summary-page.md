# <a name="legacy-merge-summary-page"></a>레거시 병합 요약 페이지

Microsoft Office Communications Server 2007 R2에서 Lync Server 2013로 마이그레이션하면 토폴로지 작성기의 기능을 사용 하 여 기존 토폴로지가 병합 됩니다. 요약 화면에서는 Office Communications Server 2007 R2의 합병을 Lync Server 2013 토폴로지 작성기 구성으로 구성 하는 동안 수행한 모든 작업과 선택 사항을 지정 합니다.

요약을 검토 하 고 선택 항목이 올바르게 정의 되었는지 확인 합니다. 요약에 반영 된 선택 항목이 만족 스 러 우면 합병을 진행 합니다.
