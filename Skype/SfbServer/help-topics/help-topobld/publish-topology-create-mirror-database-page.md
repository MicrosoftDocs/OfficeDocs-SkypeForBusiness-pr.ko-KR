# <a name="publish-topology-create-mirror-database-page"></a>토폴로지 게시 미러 데이터베이스 페이지 만들기
 
이 문서는 예비 설명서로, 변경될 수 있습니다. 빈 항목은 자리 표시자로 포함되어 있습니다.
  
토폴로지 작성기에서는 정의된 토폴로지 문서의 모든 구성 설정을 구문 분석합니다. 미러된 데이터베이스를 SQL Server 경우 미러 저장소, 기본 저장소, 파일 공유 경로 및 데이터베이스 경로를 표시하는 화면이 표시됩니다. 데이터베이스 미러 저장소와 연결된 확인란을 선택하여 미러된 데이터베이스 및 파일 공유 경로를 만듭니다. 데이터베이스를 만들고 정의된 데이터베이스 데이터베이스에 배포하지 않을 경우 데이터베이스 미러 확인란의 선택을 SQL Server.
  
 현재 이러한 데이터베이스 중 하나 또는 전체를 설치하지 않을 경우 설치하지 않을 데이터베이스를 지워야 합니다. 나중에 토폴로지 작성기에서 대부분의  노드 항목을 마우스 오른쪽 단추로 클릭하여 데이터베이스 설치를 사용할 수 있습니다.
