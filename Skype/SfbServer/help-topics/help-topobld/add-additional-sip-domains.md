# <a name="add-additional-sip-domains"></a>추가 SIP 도메인 추가
 
배포에서 지원할 각 추가 도메인의 도메인 이름을 지정합니다. 예를 들어 contoso.net을 기본 SIP (Session 착수 프로토콜) 도메인으로 지정 하 고 배포에서 SIP 트래픽용 fabrikam.net도 지원 하려면 추가 지원 도메인으로 fabrikam.net를 지정 합니다.
