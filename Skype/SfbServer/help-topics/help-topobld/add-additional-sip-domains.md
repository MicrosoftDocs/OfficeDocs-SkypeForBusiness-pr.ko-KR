# <a name="add-additional-sip-domains"></a>추가 SIP 도메인 추가
 
배포에서 지원할 각 추가 도메인의 도메인 이름을 지정합니다. 예를 들어 contoso.net SIP(Session Initiation Protocol) 도메인으로 지정한 경우 배포에서 SIP 트래픽에 대한 fabrikam.net 지원하려는 경우 fabrikam.net 도메인을 추가 지원 도메인으로 지정합니다.
