# <a name="add-mediation-server-nexthop"></a>중재 서버 NextHop 추가
 
정의할 중재 서버의 다음 홉을 정의하려면 다음 홉 풀 드롭다운 목록에서 중재 서버  의 다음 홉으로 사용할 Enterprise Edition 서버 또는 Standard Edition 서버를 선택합니다. 계속하려면 **다음** 을 클릭합니다.
