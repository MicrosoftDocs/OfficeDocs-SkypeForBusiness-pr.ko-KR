# <a name="add-mediation-server-nexthop"></a>중재 서버 NextHop 추가
 
정의 중인 중재 서버의 다음 홉을 정의 하려면 중재 서버의 다음 홉으로 작동할 Enterprise Edition 서버 또는 Standard Edition server의 **다음 홉 풀** 드롭다운 목록에서 선택 합니다. 계속하려면 **다음**을 클릭합니다.
