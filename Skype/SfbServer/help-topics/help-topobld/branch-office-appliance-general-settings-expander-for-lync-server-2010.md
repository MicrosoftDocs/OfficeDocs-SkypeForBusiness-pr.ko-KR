# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010에 대한 Branch Office Appliance 일반 설정 확장기
 
Survivable Branch 기기 또는 Survivable Branch 서버의 속성을 편집 하려면 **일반**에서 다음을 구성 합니다.
  
- **FQDN**: Survivable branch 기기 또는 Survivable branch 서버의 정규화 된 도메인 이름을 입력 합니다.
    
    **구성 된 모든 IP 주소 사용** Survivable branch 기기 또는 Survivable branch 서버에서 구성 된 ip 주소를 모두 용도에 적합 합니다.
    
    **선택한 IP 주소로 서비스 사용 제한** PSTN에 사용할 IP 주소 및 서버를 정의하는 고유 주소를 구성합니다.
    
    **기본 IP 주소**: PSTN 연결 기능을 제외하고는 모든 용도로 정의 및 구성되는 IP 주소입니다.
    
    **PSTN IP 주소**: 공중 전화망(PSTN) 기능과 연결된 IP 주소입니다.
    
- **연결** 을 구성 하 여 다른 서버 역할이 Survivable branch 기기 또는 Survivable branch 서버와 연결 되어 있는지 확인 합니다.
    
    **보관 서버 연결** Survivable Branch 기기 또는 Survivable Branch 서버와 연결 하려는 보관 서버 목록에서 선택 합니다.
    
    이 Survivable Branch 기기 또는 Survivable Branch 서버와 연결 하려는 보관 서버를 만들지 않은 경우 **새로 만들기** 를 클릭 합니다.
    
    **모니터링 서버 연결** Survivable Branch 기기 또는 Survivable Branch 서버와 연결 하려는 모니터링 서버 목록에서 선택 합니다.
    
    이 Survivable Branch 기기 또는 Survivable Branch 서버와 연결 하려는 모니터링 서버를 만들지 않은 경우 **새로 만들기** 를 클릭 합니다.
    
    **Edge 풀 연결 (미디어 구성 요소의 경우)** 목록에서 Survivable Branch 기기 또는 Survivable Branch 서버와 연결 하려는 Edge 서버 또는 Edge 풀을 선택 합니다.
    
  **확인**: 변경 내용을 적용하고 대화 상자로 커밋합니다.
  
  **취소**: 변경 내용을 취소하고 대화 상자를 닫습니다.
  
  **도움말**: 이 도움말 화면을 표시합니다.
