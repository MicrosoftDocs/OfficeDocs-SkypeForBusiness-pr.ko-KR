# <a name="publish-topology-next-step"></a>토폴로지 게시 다음 단계

토폴로지 게시를 완료하면 **게시 마법사 완료** 페이지가 표시됩니다.

토폴로지 게시 프로세스의 단계를 마침에 따라 수행된 작업에 대한 정보가 포함된 로그가 만들어집니다. 해당 단계에 관련된 로그 파일을 보려면 로그 파일을 검토해야 할 단계를 선택한 다음 **로그 보기** 를 클릭합니다.

> [!NOTE]
> 토폴로지 게시 프로세스의 모든 단계에 관련된 로그 파일이 있는 것은 아닙니다. 단계를 선택하고 **로그 보기** 를 사용할 수 있는 경우 이 단계에서는 로그 파일이 만들어진 것입니다. **로그 보기** 단추를 사용할 수 없는 경우(회색으로 표시됨) 이 단계에 대해 만들어진 로그 파일이 없는 것입니다.

토폴로지 완료에 대해 제안된 다음 단계를 검토하려면 **할 일 목록을 열려면 여기를 클릭하십시오** 링크를 클릭합니다. 이 링크를 클릭하여 연 텍스트 파일에는 로컬 설치 프로그램을 실행해야 할 서버와 배포를 완료하기 위해 수행해야 할 추가 단계가 나와 있습니다. 나열된 각 작업에 대해서는 배포 설명서에서 적절한 섹션을 참조하십시오. 자세한 내용은 [Deploying Lync Server 2013 Preview](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-lync-server) 및 할 일 목록에 나와 있는 각 서버 역할에 대한 관련 섹션을 참조하십시오.

**마침** 을 클릭합니다.

자세한 내용은 [Publish the Topology](/previous-versions/office/lync-server-2013/lync-server-2013-publish-the-topology)를 참조하십시오.
