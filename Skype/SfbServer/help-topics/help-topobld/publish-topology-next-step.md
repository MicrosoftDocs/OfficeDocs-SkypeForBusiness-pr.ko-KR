# <a name="publish-topology-next-step"></a>토폴로지 게시 다음 단계

토폴로지 게시가 완료 되 면 **게시 마법사 완료** 페이지가 표시 됩니다.

토폴로지 게시 프로세스 중에 단계를 완료 하면 수행 된 작업에 대 한 정보가 포함 된 로그가 만들어집니다. 지정 된 단계와 관련 된 로그 파일을 보려면 로그 파일을 검토 하는 데 필요한 단계를 선택한 다음 **로그 보기**를 클릭 합니다.

> [!NOTE]
> 토폴로지 게시 프로세스 중 일부 단계에는 연결 된 로그 파일이 포함 되지 않습니다. 단계를 선택 하 고 **보기 로그** 를 사용할 수 있는 경우이 단계에서는 로그 파일을 만들었습니다. **로그 보기** 단추를 사용할 수 없는 경우 (회색으로 표시)이 단계에 대해 로그 파일이 만들어지지 않은 것입니다.

토폴로지를 완료 하기 위해 제안 하는 다음 단계를 검토 하려면 **여기를 클릭 하 여 할 일 목록 열기**링크를 클릭 합니다. 이 링크를 클릭 하 여 연 텍스트 파일은 로컬 설정을 실행 하는 데 필요한 서버와 배포를 완료 하기 위해 수행 해야 하는 추가 단계를 나타냅니다. 나열 된 각 작업에 대해 배포 설명서의 해당 섹션을 참조 하세요. 자세한 내용은 [Lync Server 2013 Preview 배포](https://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)를 참조 하 고 할 일 목록에 명시 된 각 서버 역할에 대 한 관련 섹션을 참조 하세요.

**완료** 클릭

자세한 내용은 [토폴로지 게시](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx)를 참조 하세요.
