# <a name="front-end-start-page"></a>프런트 엔드 시작 페이지

이 마법사를 사용 하 여 사이트에 대 한 새 프런트 엔드 풀을 정의 합니다. 각 사이트에는 Enterprise Edition pool 또는 Standard Edition 서버가 하나 이상 있어야 합니다. 마법사를 사용 하 여 새 프런트 엔드 풀을 만드는 데 필요한 요구 사항에 대 한 자세한 내용은 배포 설명서에서 [프런트 엔드 풀 정의 및 구성을](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) 참조 하세요.
