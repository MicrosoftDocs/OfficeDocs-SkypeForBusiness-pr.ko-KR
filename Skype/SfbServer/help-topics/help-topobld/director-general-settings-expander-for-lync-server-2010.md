# <a name="director-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010에 대한 디렉터 일반 설정 확장기
 
Director **또는** Director 풀의 일반 속성을 구성합니다.
  
 **FQDN:** Director 또는 Director 풀의 정식 도메인 이름 입력
  
 **연결**
  
 **파일 저장소:** 목록에서 Director 또는 Director 풀의 파일 저장소 선택
  
**새로 만들기** 를 클릭하여 새 파일 저장소를 정의합니다.
  
모니터링 **서버** 연결 확인란을 선택하고 목록에서 모니터링 서버 선택
  
**새로하기를** 클릭하여 새 모니터링 서버 정의
  
 **확인**: 변경 내용을 적용하고 대화 상자로 커밋합니다.
  
 **취소**: 변경 내용을 취소하고 대화 상자를 닫습니다.
  
 **도움말**: 이 도움말 화면을 표시합니다.
