# <a name="add-edge-server-nat-ip-2010"></a>에지 서버 NAT IP 2010 추가
 
이 에지 풀의 공용 IP 주소가 NAT(Network Address Translation)에 의해 변환되는 경우 변환할 공용 IP 주소를 지정해야 합니다. 이 주소는 A/V 에지 서비스의 외부 IP 주소여야 합니다.
  
에지 풀의 NAT 변환 사용 여부를 변경하려면 에지 풀 이름을 마우스 오른쪽 단추로 클릭하고 **속성 편집**을 클릭합니다.
