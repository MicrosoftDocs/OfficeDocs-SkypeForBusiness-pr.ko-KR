# <a name="add-front-end-features-2010"></a>프런트 엔드 기능 2010 추가

프런트 엔드 풀에 대해 사용하도록 설정할 각 기능을 선택합니다. 최소한 IM(인스턴트 메시징)을 배포해야 하며, IM은 항상 사용하도록 설정되고 선택 취소할 수 없습니다. 회의, Enterprise Voice 및 CAC (통화 허용 제어)와 같은 다른 기능을 배포 하는 옵션도 제공 됩니다. 일부 기능은 토폴로지에 따라 달라집니다.

Standard Edition 서버 또는 Enterprise Edition 풀에 대 한 기능을 지정 하는 방법에 대 한 자세한 내용은 배포 설명서에서 [Define And Configure a Front End pool Or Standard Edition server](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) 을 참조 하십시오.
