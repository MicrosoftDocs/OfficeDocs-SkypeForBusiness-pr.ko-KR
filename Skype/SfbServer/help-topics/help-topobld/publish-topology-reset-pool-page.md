# <a name="publish-topology-reset-pool-page"></a>토폴로지 게시 풀 다시 설정 페이지
 
이 문서는 예비 설명서로, 변경될 수 있습니다. 빈 항목은 자리 표시자로 포함되어 있습니다.
  
새 컴퓨터를 기존 풀에 추가하려면 풀을 다시 설정하여 Windows Fabric에서 풀의 모든 구성원 간에 연결을 다시 정의하도록 허용해야 합니다. 이렇게 하려면 서버를 추가한 풀을 선택하고 **다음** 을 클릭합니다.
