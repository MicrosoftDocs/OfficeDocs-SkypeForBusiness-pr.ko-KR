# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Lync Server 2010에 대한 프런트 엔드 SQL 저장소 추가 페이지
 
구성할 새 프런트 엔드 SQL Server 저장소의 속성을 정의합니다.
  
> [!IMPORTANT]
> 서버 Standard Edition 구성하는 경우 이 대화 상자가 표시되지만 설정을 구성할 수 없습니다. 계속하려면 **다음** 을 클릭합니다.
  
구성할 수 있는 설정은 다음과 같습니다.
  
- **이전에 정의한** SQL Server 저장소 사용: 프런트 엔드 SQL Server 저장소의 목록에서 기존 저장소를 선택합니다.
    
- 새 **저장소를 만들 SQL Server 저장소** 정의를 SQL Server 선택합니다.
    
    **FQDN에** SQL Server 도메인 SQL Server 입력합니다.
    
    처음에는 **기본 인스턴스** 가 선택되어 있습니다. 인스턴스를 정의하려면 **명명된 인스턴스** 를 선택하고 만들어야 하는 인스턴스의 이름을 입력합니다.
    
  **뒤로**: 이전 대화 상자로 돌아갑니다.
  
  **다음**: 변경 내용을 커밋하고 다음 단계 대화 상자로 이동합니다.
  
  **취소**: 변경 내용을 무시하고 구성을 취소합니다.
  
  **도움말**: 이 항목과 같은 도움말 정보를 표시합니다.
