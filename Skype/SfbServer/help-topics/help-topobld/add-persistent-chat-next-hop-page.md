이 페이지를 사용 하 여 배포의 영구 채팅 서버 서버에서 다음 홉을 정의 합니다. 영구 채팅 서버를 배포 하는 방법에 대 한 자세한 내용은 비즈니스용 [Skype 서버 2015에서 영구 채팅 서버 배포](../../deploy/deploy-persistent-chat-server/deploy-persistent-chat-server.md)를 참조 하세요.
