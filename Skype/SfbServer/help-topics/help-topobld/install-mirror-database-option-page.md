# <a name="install-mirror-database-option-page"></a>미러 데이터베이스 설치 옵션 페이지
 
다음 항목을 정의하여 **미러 데이터베이스 설정**을 구성합니다.
  
- 미러된 데이터베이스에 대 한 백업 SQL Server 파일의 위치를 정의 하는 **파일 공유 경로** 를 입력 합니다.
    
    > [!NOTE]
    > 기본 SQL Server 인스턴스 (명명 된 인스턴스 또는 기본 인스턴스)에는 여기에서 정의한 파일 공유에 대 한 쓰기 권한이 있어야 합니다. 미러 SQL Server 인스턴스 (명명 된 인스턴스 또는 기본 인스턴스)에는 동일한 파일 공유에 대 한 읽기 권한이 있어야 합니다. 
  
  **확인**: 변경 내용을 적용하고 대화 상자로 커밋합니다.
  
  **취소**: 변경 내용을 취소하고 대화 상자를 닫습니다.
  
  **도움말**: 이 도움말 화면을 표시합니다.
