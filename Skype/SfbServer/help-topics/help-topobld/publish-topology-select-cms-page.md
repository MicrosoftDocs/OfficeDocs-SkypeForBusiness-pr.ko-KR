# <a name="publish-topology-select-cms-page"></a>토폴로지 게시 CMS 선택 페이지

토폴로지 작성기를 사용 하 여 구성한 토폴로지를 게시 합니다. 중앙 관리 저장소를 보유 하는 역할을 갖는 프런트 엔드 서버 또는 프런트 엔드 풀 목록에서 선택 하 라는 메시지가 표시 됩니다. 주어진 시간에이 역할을 하나의 프런트 엔드 서버 또는 프런트 엔드 풀에만 포함할 수 있습니다.

중앙 관리 저장소에 대 한 자세한 내용은 [lync server 2010 중앙 관리 서버를 Lync server 2013로 이동을](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx)참조 하십시오.
