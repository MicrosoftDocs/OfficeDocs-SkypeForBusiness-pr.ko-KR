# <a name="add-edge-machine-external-ip-2010"></a>Edge 컴퓨터 외부 IP 2010 추가

이 페이지의 속성을 구성 하 여 Edge 서버를 Access Edge 서비스, 웹 회의에 지 서비스 및 A/V Edge 서비스의 외부 위치에 가리키도록 구성할 수 있습니다.

- **외부 ipv4 주소**아래에 edge 서버의 외부 ipv4 주소를 **Access edge 서비스**, **웹 회의**에 지 서비스 및 **A/V Edge 서비스**에 입력 합니다.

- **확인** 을 클릭 하 여 변경 내용을 저장 하 고 토폴로지 문서를 커밋합니다.

- 변경 내용을 취소 하 고 **Edge 풀에 서버 추가** 마법사를 닫으려면 **취소** 를 클릭 합니다.

- 도움말 **** 을 클릭 하 여이 도움말 항목을 읽습니다.

기존 토폴로지에 Edge 서버를 추가 하는 방법에 대 한 자세한 내용은 [Edge 토폴로지 정의](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)를 참조 하세요. Edge 서버 마이그레이션에 대 한 자세한 내용은 [5 단계: Lync server 2013 Preview Edge 서버를 파일럿 풀에 추가](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)를 참조 하세요.
