# <a name="add-edge-machine-external-ip-2010"></a>에지 컴퓨터 외부 IP 2010 추가

액세스 에지 서비스, 웹 회의 에지 서비스 및 A/V 에지 서비스의 외부 위치를 에지 서버를 설정하도록 이 페이지의 속성을 구성할 수 있습니다.

- 외부 **IPv4** 주소 아래에 액세스 에지 **서비스,** 웹 회의 에지 서비스 및 A/V 에지 서비스에 에지 서버의 외부 IPv4 주소를 **입력합니다.** 

- 토폴로지 문서에 대한 변경 내용을 저장 및 커밋하려면 **확인** 을 클릭합니다.

- 변경 내용을 취소하고 **에지 풀에 서버 추가** 마법사를 닫으려면 **취소** 를 클릭합니다.

- 이 도움말 항목을 읽으려면 **도움말** 을 클릭합니다.

기존 토폴로지에 에지 서버를 추가하는 데 대한 자세한 내용은 [에지 토폴로지 정의를 참조합니다.](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) 에지 서버 마이그레이션에 대한 자세한 내용은 [5단계: 파일럿 풀에 Lync Server 2013 Preview 에지 서버 추가를 참조하세요.](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)
