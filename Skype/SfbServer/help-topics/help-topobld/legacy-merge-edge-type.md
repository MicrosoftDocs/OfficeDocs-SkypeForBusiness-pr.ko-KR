# <a name="legacy-merge-edge-type"></a>레거시 병합 에지 유형
 
**단일 에지 서버** 또는 **부하 분산 통합 에지** 서버의 경우 **다음** 을 클릭하여 구성 정보를 지정합니다. 확장 **에지의** 경우 각 서버의 구성을 통합 에지 구성으로 변경해야 Lync Server 2013과 병합할 수 있습니다.
