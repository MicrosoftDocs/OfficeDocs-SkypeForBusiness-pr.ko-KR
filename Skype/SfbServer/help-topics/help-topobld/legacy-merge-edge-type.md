# <a name="legacy-merge-edge-type"></a>레거시 병합 에지 유형
 
**단일 에지 서버** 또는 **부하 분산 통합 에지** 서버의 경우 **다음**을 클릭하여 구성 정보를 지정합니다. 확장 된 **edge**의 경우 각 서버의 구성을 Lync server 2013와 병합 하기 전에 통합에 지 구성으로 변경 해야 합니다.
