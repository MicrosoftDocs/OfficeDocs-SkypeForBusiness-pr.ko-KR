# <a name="remove-deployment"></a>배포 제거
 
완료 **를** 클릭하여 전체 배포를 해제하는 첫 번째 단계로 빈 토폴로지 만들기를 클릭합니다.
  
> [!CAUTION]
> 이 단계는 실행 취소할 수 없습니다.
