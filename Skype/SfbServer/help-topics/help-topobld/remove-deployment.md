# <a name="remove-deployment"></a>배포 제거
 
전체 배포를 해제 하는 첫 단계로 빈 토폴로지를 만들려면 **마침을** 클릭 합니다.
  
> [!CAUTION]
> 이 단계는 실행 취소할 수 없습니다.
