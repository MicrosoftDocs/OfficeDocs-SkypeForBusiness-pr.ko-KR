# <a name="define-the-pstn-gateway-fqdn"></a>PSTN 게이트웨이 FQDN 정의

Enterprise Voice 구성 요소를 PSTN 또는 PBX (private branch exchange)에 연결 하는 PSTN (공중 전화망) 게이트웨이의 FQDN (정규화 된 도메인 이름)을 지정 합니다.

자세한 내용은 [비즈니스용 Skype 서버 2015의 PSTN 연결 구성 요소](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md) 를 참조 하세요.
