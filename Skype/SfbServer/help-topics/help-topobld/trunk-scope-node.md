# <a name="trunk-scope-node"></a>트렁크 범위 노드
 
토폴로지 작성기에서 기존 SIP 트렁크를 마우스 오른쪽 단추로 클릭 하 여 다음을 수행 합니다.
  
- **속성 편집**: 기존 SIP 트렁크 구성을 편집하거나 다시 정의하는 데 사용되는 **속성 편집** 마법사를 시작합니다.
    
속성 편집 옵션 외에 다음과 같은 일반적인 작업을 수행할 수 있습니다.
  
- **토폴로지**. 이 옵션을 사용 하 여 새 토폴로지를 정의 하 고, 기존 토폴로지를 열고, 토폴로지 복사본을 저장 하 고, 토폴로지에 대해 토폴로지를 게시 하 고, 데이터베이스를 설치 하거나, 이전 버전의 Lync Server에서 구성 요소를 병합 하거나, 배포를 제거할 수 있습니다.
    
- **삭제**. 이 선택 항목을 사용하여 현재 선택되어 있는 항목을 제거합니다. 기타 종속성으로 인해 삭제가 차단될 수 있습니다. 항목을 제거하려면 확인해야 하는 서비스 및 종속성을 보여 주는 대화 상자가 표시됩니다.
    
- **도움말**. 이 옵션을 사용하여 활성 노드에 대한 도움말에 액세스할 수 있습니다.
