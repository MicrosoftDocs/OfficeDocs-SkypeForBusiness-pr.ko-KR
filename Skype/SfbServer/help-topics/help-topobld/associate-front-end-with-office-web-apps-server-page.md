# <a name="associate-front-end-with-office-web-apps-server-page"></a>Office Web Apps Server와 프런트 엔드 연결 페이지

Office Web Apps 서버는 발표자가 모임에 사용할 문서를 업로드 하지만 이전 버전에서 사용 하지 않았던 추가 기능을 사용 하도록 설정 하는 것과 비슷한 서비스를 제공 합니다.

비즈니스용 Skype 서버 2015 구성에 대 한 기능 옵션 중 하나로 회의를 선택한 경우 배포용 Office Web Apps 서버를 선택 합니다.

이 대화 상자에서는 다음과 같은 옵션을 선택할 수 있습니다.

1. **Office Web Apps 서버와 풀 연결** -회의 사용자에 게 Office Web apps 서버를 제공 하 고 배포 하려면이 확인란을 선택 합니다.

2. 확인란을 선택한 경우 목록을 사용하여 이미 정의된 Office Web Apps Server를 선택합니다.

3. **새로 만들기**를 클릭하여 배포용으로 새 Office Web Apps Server를 정의합니다.
