# <a name="create-persistent-chat-servers-scope-node"></a>영구 채팅 서버 범위 노드 만들기
 
토폴로지 작성기에서 **영구 채팅 풀** 을 마우스 오른쪽 단추로 클릭 하 여 다음을 수행 합니다.
  
- **새 영구 채팅 풀**: 새 영구 채팅 서버 또는 풀을 정의 하는 데 사용 되는 **새 영구 채팅 풀 정의** 마법사를 시작 합니다.
    
새 영구 채팅 풀 옵션 외에 다음과 같은 일반적인 작업을 수행할 수 있습니다.
  
- **토폴로지**. 이 옵션을 사용 하 여 새 토폴로지를 정의 하 고, 기존 토폴로지를 열고, 토폴로지 복사본을 저장 하 고, 토폴로지에 대해 토폴로지를 게시 하 고, 데이터베이스를 설치 하거나, 이전 버전의 Lync Server에서 구성 요소를 병합 하거나, 배포를 제거할 수 있습니다.
    
- **도움말을 참조**하세요. 이 옵션을 사용하여 활성 노드에 대한 도움말에 액세스할 수 있습니다.
