# <a name="add-edge-server-external-ip-2010"></a>에지 서버 외부 IP 2010 추가

이 페이지의 속성을 구성 하 여에 지 서버에서 액세스에 지 서비스, 웹 회의에 지 서비스 및 A/V에 지 서비스의 외부 위치를 가리키도록 구성할 수 있습니다.

- **외부 ipv4 주소**에 **액세스에 지 서비스**, **웹 회의**에 지 서비스 및 **A/V에 지 서비스**에에 지 서버의 외부 IPv4 주소를 입력 합니다.

- 토폴로지 문서에 대한 변경 내용을 저장 및 커밋하려면 **확인**을 클릭합니다.

- 변경 내용을 취소하고 **에지 풀에 서버 추가** 마법사를 닫으려면 **취소**를 클릭합니다.

- 이 도움말 항목을 읽으려면 **도움말**을 클릭합니다.

기존 토폴로지에에 지 서버를 추가 하는 방법에 대 한 자세한 내용은 [Define 사용자에 지 토폴로지](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)를 참조 하십시오. 에 지 서버를 마이그레이션하는 방법에 대 한 자세한 내용은 [Phase 5: Lync server 2013 Preview Edge server를 파일럿 풀에 추가](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)를 참조 하십시오.
