# <a name="add-site-name"></a>사이트 이름 추가
 
배포의 첫 번째 사이트를 정의 하려면 첫 번째 사이트의 이름을 지정 합니다. 선택적으로 첫 번째 사이트에 대 한 설명을 지정할 수 있습니다.
