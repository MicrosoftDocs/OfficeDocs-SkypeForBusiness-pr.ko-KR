# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Lync Server 2010 모니터링 페이지와 Director 연결
 
모니터링 서버를 구성 하 고 디렉터 또는 디렉터 풀에 연결 하려면 다음을 수행 합니다.
  
모니터링 **서버** 목록에서 모니터링 서버를 선택 합니다.
  
모니터링 서버가 없고 지금 모니터링 서버를 만들려면 **새로**만들기를 클릭 합니다.
  
 **뒤로**: 이전 대화 상자로 돌아갑니다.
  
 **완료**: 변경 내용을 커밋하고 마법사를 완료 합니다.
  
 **취소**: 변경 내용을 취소 하 고 구성을 취소 합니다.
  
 **도움말**:이 항목과 같은 도움말 정보를 표시 합니다.
