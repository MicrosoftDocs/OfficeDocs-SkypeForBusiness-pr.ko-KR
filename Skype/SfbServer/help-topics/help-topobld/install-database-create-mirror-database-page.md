# <a name="install-database-create-mirror-database-page"></a>데이터베이스 설치 미러 데이터베이스 페이지 만들기
 
이 문서는 예비 문서로, 변경 될 수 있습니다. 빈 항목은 개체 틀로 포함 됩니다.
  
게시 프로세스 중에 SQL Server 미러 데이터베이스를 설치 하지 않도록 선택한 경우에는 토폴로지 작성기의 노드를 마우스 오른쪽 단추로 클릭 하면 설치 **데이터베이스** 를 사용 하 여 설치 합니다. 설치할 미러 저장소를 선택 합니다. **설정을** 클릭 하 여 고급 설정을 정의 합니다.
