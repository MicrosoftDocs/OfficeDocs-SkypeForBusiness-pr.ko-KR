# <a name="add-monitoring-server"></a>모니터링 서버 추가

모니터링 서버를 토폴로지 디자인에 통합하려면 모니터링 서버를 배포할 서버의 FQDN(FQDN)을 지정해야 합니다. 모니터링 서버를 포함하는 토폴로지 게시 및 서버에 비즈니스용 Skype 서버 전에 모니터링 배포를 위한 모든 선행 조건이 완료되어야 합니다. 이러한 필수 구성 요소에 대한 자세한 내용은 배포 설명서에서 [Deploying Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring)를 참조하십시오.
