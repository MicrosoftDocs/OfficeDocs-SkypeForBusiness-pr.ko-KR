# <a name="add-monitoring-server"></a>모니터링 서버 추가

모니터링 서버를 토폴로지 디자인에 통합 하려면 모니터링 서버를 배포할 서버의 FQDN (정규화 된 도메인 이름)을 지정 해야 합니다. 모니터링 서버를 포함 하 고 서버에 비즈니스용 Skype 서버를 설치 하는 토폴로지를 게시 하기 전에 모니터링 배포를 위한 모든 필수 구성 요소를 완료 해야 합니다. 이러한 필수 구성 요소에 대한 자세한 내용은 배포 설명서에서 [Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)를 참조하십시오.
