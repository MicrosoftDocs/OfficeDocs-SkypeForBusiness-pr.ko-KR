# <a name="add-front-end-monitoring-store-page"></a>프런트 엔드 모니터링 저장소 추가 페이지
 
다음 속성을 구성하여 **모니터링 SQL Server 저장소 정의**를 수행합니다.
  
- **모니터링 Sql server 저장소**: 목록에서 sql server 정규화 된 도메인 이름 (및 인스턴스)을 선택 합니다.
    
    **새로** 만들기를 클릭 하 여 새 SQL Server FQDN 정의를 만들고, 필요에 따라 모니터링 서버 저장소의 인스턴스 이름을 만듭니다.
    
- 모니터링 서버에 대 한 데이터베이스 미러링을 추가 하려면 **SQL Server 저장소 미러링 사용** 확인란을 선택 합니다.
    
    목록에서 기존 **모니터링 SQL Server 저장소 미러**를 선택합니다.
    
    **새로** 만들기를 클릭 하 여 새 SQL Server FQDN 정의를 만들고 원하는 경우 미러 저장소의 인스턴스 이름을 만듭니다.
    
- **Sql server 저장소 미러링 사용**을 선택한 경우 필요에 따라 **sql Server 미러링 모니터 서버를 사용 하 여 자동 장애 조치 (failover)** 가 목록에서 sql server 미러링 모니터 저장소를 선택 하도록 설정 합니다.
    
    **새로** 만들기를 클릭 하 여 새 SQL Server FQDN 정의를 만들고 원하는 경우 미러링 모니터 서버 저장소의 인스턴스 이름을 만듭니다.
    
이전 풀 정의 대화 상자로 돌아가려면 **뒤로**를 클릭합니다.
  
이 대화 상자의 옵션을 모두 입력한 후 구성을 계속하려면 **다음**을 클릭합니다.
  
모든 변경 내용을 취소하고 마법사를 종료하려면 **취소**를 클릭합니다.
  
이 페이지와 같은 상황에 맞는 도움말에 액세스하려면 **도움말**을 클릭합니다.
