# <a name="add-branch-site-details"></a>분기 사이트 세부 정보 추가

사이트에 대한 사이트 세부 정보를 지정하려면 다음을 수행합니다.

새 분기 사이트의 구/군/시 이름 지정. 구/군/시 값은 선택적 항목입니다.

새 분기 사이트의 시/도 이름 지정. 시/도 값은 선택적 항목입니다.

새 분기 사이트의 국가/지역 코드 이름 지정. 국가/지역 코드 값은 선택적 항목입니다.

자세한 내용은 [Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx)를 참조하세요.
