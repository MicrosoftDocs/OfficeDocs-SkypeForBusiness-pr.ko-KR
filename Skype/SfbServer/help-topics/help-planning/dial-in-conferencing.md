# <a name="skype-for-business-server-dial-in-conferencing-planning-tool"></a>비즈니스용 Skype 서버 전화 접속 회의 계획 도구
 
비즈니스용 Skype 서버 2015 전화 접속 회의를 사용하면 타사 오디오 회의 공급자 없이도 PSTN(Public Switched Telephone Network) 전화를 사용하여 비즈니스용 Skype 서버 2015 전화 회의의 오디오 부분에 참가할 수 있습니다.
  
조직에 전화 접속 회의를 사용하도록 설정하는 방법을 계획하는 데 대한 자세한 내용은 다음 항목 및 연결되는 항목을 참조하십시오. 
  
- [2015년 비즈니스용 Skype 서버 회의 계획](../../plan-your-deployment/conferencing/conferencing.md)
