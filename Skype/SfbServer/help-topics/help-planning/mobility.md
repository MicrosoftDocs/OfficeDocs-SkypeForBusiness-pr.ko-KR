# <a name="mobility-planning-tool"></a>이동성 (계획 도구)
 
비즈니스용 Skype 서버 2015의 이동성 기능은 모바일 장치에서 비즈니스용 Skype 기능을 지원 합니다.
  
이동성 기능은 Lync Server 2013와 동일 합니다. 이동성을 위한 환경을 준비 하는 방법에 대 한 자세한 내용은 [Lync Server 2013의 이동성 계획](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)을 참조 하세요.
