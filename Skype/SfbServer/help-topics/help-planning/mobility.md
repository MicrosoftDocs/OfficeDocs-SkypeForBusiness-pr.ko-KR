# <a name="skype-for-business-server-mobility-planning-tool"></a>비즈니스용 Skype 서버: 모바일 계획 도구
 
비즈니스용 Skype 서버 2015의 모바일 기능은 모바일 비즈니스용 Skype 기능을 지원합니다.
  
모바일 기능은 Lync Server 2013과 동일합니다. 모바일을 위해 환경을 준비하는 데 대한 자세한 내용은 [Planning for mobility in Lync Server 2013을 참조합니다.](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-mobility)
