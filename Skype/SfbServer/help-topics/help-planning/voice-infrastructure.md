# <a name="voice-infrastructure-planning-tool"></a>음성 인프라(계획 도구)
 
비즈니스용 Skype 서버 2015는 SIP 트렁크, PSTN 게이트웨이에 대한 직접 SIP 연결 또는 PBX에 대한 직접 SIP 연결을 사용하여 안정적이고 확장 가능한 PSTN 연결을 제공합니다. 자세한 내용은 비즈니스용 Skype 서버 [2015의 PSTN](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)연결 계획을 참조하세요.
