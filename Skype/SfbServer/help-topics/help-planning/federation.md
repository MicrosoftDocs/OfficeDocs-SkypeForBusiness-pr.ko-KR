# <a name="federation-planning-tool"></a>페더레이션 (계획 도구)
 
내부 및 외부 사용자가 파트너 조직이 나 서비스에서 연락처에 액세스할 수 있도록에 지 서버를 구성할 수 있습니다.
  
 페더레이션은 Microsoft Lync 2013에서와 같은 방식으로 작동 합니다. 페더레이션에 대 한 환경을 준비 하는 방법에 대 한 자세한 내용은 다음 항목을 참조 하십시오.
  
- [Lync Server 2013 및 Office Communications Server federation 계획](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [Lync Server 2013의 공용 인스턴트 메시징 연결 계획](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [Lync Server 2013의 XMPP (extensible messaging and 거점 protocol) 페더레이션 계획](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
