# <a name="voice-infrastructure-planning-tool"></a>음성 인프라 (계획 도구)
 
비즈니스용 Skype Server는 SIP trunks를 사용 하 여 안정적이 고 확장 가능한 PSTN 연결을 제공 하며 PSTN 게이트웨이에 대 한 SIP 연결 또는 직접 SIP 연결을 PBX에 연결할 수 있습니다. 자세한 내용은 [비즈니스용 Skype 서버의 PSTN 연결 계획](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)을 참조 하세요.
