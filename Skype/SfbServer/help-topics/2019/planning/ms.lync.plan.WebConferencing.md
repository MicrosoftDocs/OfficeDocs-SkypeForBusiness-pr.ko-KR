# <a name="web-conferencing-planning-tool"></a>웹 회의(계획 도구)
 
비즈니스용 Skype 서버 웹 회의에는 문서 공동 작업, 응용 프로그램 공유 및 데스크톱 공유가 포함됩니다.
  
> [!NOTE]
> 비즈니스용 Skype 서버는 Office Web Apps 및 Office Web Apps 서버를 사용하여 PowerPoint 프레젠테이션의 공유 및 렌더링을 처리합니다. 웹 회의 기능은 Microsoft Lync 2013에서와 정확히 동일하게 기능합니다. Office Web Apps 서버 설치 및 구성에 대한 자세한 내용은 Office Web Apps 서버 및 [Lync Server 2013과의](https://technet.microsoft.com/library/jj204792%28v=ocs.15%29.aspx)통합 구성을 참조하세요.
