# <a name="ip-version-planning-tool"></a>IP 버전 (계획 도구)
 
비즈니스용 Skype 서버에서 IP 주소를 구성할 때는 세 가지 옵션이 있습니다. IPv4 (IP 버전 4)만 지원 하도록 비즈니스용 Skype 서버를 구성 하거나 (IP 버전 6), IPv6 또는 둘 다 (이중 스택 이라고 함)를 조합 하 여 사용할 수 있습니다.
  
IP 인프라 선택에 대 한 자세한 내용은 비즈니스용 [Skype의 IPv6 계획](../../../plan-your-deployment/network-requirements/ipv6.md)을 참조 하세요.
