# <a name="mediation-collocation-planning-tool"></a>중재 위치 (계획 도구)
 
중재 서버는 기본적으로 프런트 엔드 서버와 collocated 됩니다. 조정 서버는 성능상의 이유 때문에 독립 실행형 풀에 배포 되거나 SIP 트렁크를 배포 하는 경우에도 독립 실행형 풀을 강력히 권장 합니다. 비즈니스용 Skype 서버의 collocation은 Lync Server 2013에서와 마찬가지로 작동 합니다. 자세한 내용은 다음 항목을 참조 하세요.
  
- [Lync Server 2013의 지원되는 서버 배치](https://technet.microsoft.com/en-us/library/gg425885%28v=ocs.15%29.aspx)
    
- [비즈니스용 Skype 서버의 중재 서버 구성 요소](../../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
