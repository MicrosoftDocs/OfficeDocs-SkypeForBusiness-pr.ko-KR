# <a name="legacy-merge-next-hop"></a>레거시 병합 Next Hop
 
**다음 홉 FQDN** 은 레거시 Edge 서버가 내부 트래픽을 라우팅하는 다음 홉 서버를 식별 합니다. 이 Edge 서버가 내부 트래픽을 라우팅하는 다음 홉 서버의 FQDN (정규화 된 도메인 이름)을 입력 한 다음 **프런트 엔드 풀** 또는 **디렉터**중에서 선택 합니다.
