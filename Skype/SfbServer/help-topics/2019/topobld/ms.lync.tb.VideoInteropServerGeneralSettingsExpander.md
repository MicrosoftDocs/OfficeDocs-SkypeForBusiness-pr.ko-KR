# <a name="video-interop-server-general-settings"></a>비디오 Interop 서버 일반 설정
 
토폴로지 작성기에서 새 VIS 또는 비디오 게이트웨이를 정의하는 데 필요한 단계는 Create [a VIS pool in 비즈니스용 Skype 서버](../../../deploy/deploy-video-interop-server/create-a-vis-pool.md).
  
서버를 사용하여 서버에 Video Interop 서버 역할을 할당하는 Windows PowerShell 단계는 2016년 8월의 [VIS](../../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md) 서버 역할 배포에서 비즈니스용 Skype 서버
  
이 기능을 사용하는 새 비디오 상호 Windows PowerShell 구성 단계는 2016년 8월에 비디오 [Interop 서버 구성](../../../deploy/deploy-video-interop-server/configure-the-vis.md)에서 비즈니스용 Skype 서버.
  
 VIS와의 상호 연동을 위해 Cisco GUI에서 설정해야 하는 비디오 게이트웨이에 대한 설정은 온라인에서 [Configure CUCM for Interoperation with 비즈니스용 Skype 서버](../../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md).
  
 비디오 게이트웨이 및 VIS와의 상호 연동을 위해 Cisco GUI에서 설정해야 하는 VTC에 대한 설정은 Configure [a VTC for Interoperation with 비즈니스용 Skype 서버](../../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md).
