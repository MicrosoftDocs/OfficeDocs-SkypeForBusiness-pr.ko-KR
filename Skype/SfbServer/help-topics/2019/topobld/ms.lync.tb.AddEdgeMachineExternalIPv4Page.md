# <a name="add-edge-machine-external-ipv4"></a>에지 컴퓨터 외부 IPv4 추가
 
외부 **IPv4에서** 액세스 에지 서비스, 웹 회의 에지 서비스 및 A/V 에지 서비스를 지점하는 외부 IP 주소를 입력합니다.
