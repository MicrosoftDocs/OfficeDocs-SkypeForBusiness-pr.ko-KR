# <a name="legacy-merge-landing"></a>레거시 병합 Landing
 
**에지 설정 지정 페이지에서** 레거시 에지 서버 정보를 구성할 수 있습니다. Office Communications Server 배포의 레거시 에지 서버 버전을 지정하려면 **추가** 를 클릭합니다. 레거시 환경에 에지 서버가 없는 경우 계속하려면 **다음** 을 클릭합니다.
