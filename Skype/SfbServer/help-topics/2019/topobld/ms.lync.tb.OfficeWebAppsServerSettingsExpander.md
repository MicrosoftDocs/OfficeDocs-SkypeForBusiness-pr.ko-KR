 <span data-ttu-id="19a33-107">**Office Web Apps 서버 FQDN**:이 속성은 Office Web apps 서버의 정규화 된 도메인 이름을 정의 하 고 DNS (domain name system) 호스트 a 또는 AAAA (IPv6이 사용 되는 경우) 레코드와 일치 해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="19a33-107">**Office Web Apps Server FQDN**: This property defines the fully qualified domain name of the Office Web Apps Server and should match a domain name system (DNS) host A or AAAA (if IPv6 is being used) record.</span></span>

<span data-ttu-id="19a33-109">확인란 선택 Office Web Apps 서버는 경계 네트워크에 배포 되는 경우 **외부 네트워크에 배포 되며** 경계 네트워크, 더 적은 신뢰 네트워크 및 내부 배포의 인터넷을 분리 하는 내부 방화벽 외부의 다른 네트워크 영역</span><span class="sxs-lookup"><span data-stu-id="19a33-109">Select the check box **Office Web Apps Server is deployed in an external network** if this server is deployed in your perimeter network or other network zone that is outside of your internal firewall separating the perimeter network, less trusted networks, and the Internet from your internal deployment.</span></span>
