# <a name="monitoring-servers-scope-node"></a>모니터링 서버 범위 노드
 
**모니터링 서버** 노드는 토폴로지의 모니터링 서버를 나열 합니다. 다음 작업을 수행할 수 있습니다.
  
- **새 모니터링 서버**. 새 모니터링 **서버 정의** 마법사를 시작 하 여 토폴로지에 새 모니터링 서버를 추가 하는 데 사용 합니다.
    
모니터링 서버 옵션 외에도 다음과 같은 일반적인 작업을 수행할 수 있습니다.
  
- **토폴로지**. 이 옵션을 사용 하 여 새 토폴로지를 정의 하 고, 기존 토폴로지를 열고, 토폴로지 복사본을 저장 하 고, 토폴로지를 게시 하 고, 데이터베이스를 설치 하 고, 이전 버전의 Office Communications Server의 구성 요소를 병합 하거나, 제거할 수 있습니다. 배포가.
    
- **보기**. 이 옵션을 사용하여 콘솔 트리 보기에 표시할 항목과 숨길 항목을 선택하여 토폴로지 작성기 보기를 사용자 지정할 수 있습니다.
    
- **도움말**. 이 옵션을 사용하여 활성 노드에 대한 도움말에 액세스할 수 있습니다.
