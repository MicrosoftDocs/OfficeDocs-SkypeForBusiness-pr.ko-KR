# <a name="add-nexthop-trusted-application-server"></a>다음 홉 신뢰할 수 있는 응용 프로그램 서버 추가
 
신뢰할 수 있는 응용 프로그램 서버 또는 풀에 대한 다음 홉을 설정하려면 다음을 수행합니다.
  
**다음 홉 풀** 드롭다운 목록에서 신뢰할 수 있는 응용 프로그램 풀을 다음 홉으로 사용할 Enterprise Edition 풀 또는 Standard Edition 프런트 엔드 서버를 선택합니다.
  
**마침** 을 클릭하여 신뢰할 수 있는 응용 프로그램 풀 만들기를 완료합니다.
