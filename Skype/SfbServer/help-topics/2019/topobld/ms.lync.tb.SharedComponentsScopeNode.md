# <a name="shared-components-scope-node"></a>공유 구성 요소 범위 노드
 
토폴로지 **작성기** 계층 구조 트리에서 공유 구성 요소를 마우스 오른쪽 단추로 클릭하여 새 공유 구성 요소를 만들거나 토폴로지 수준 옵션에 액세스합니다.
  
![공유 구성 요소 범위 노드](../../../media/Shared_Components_Scope_Node.jpg)
  
새로 **만들기를** 선택하여 새 파일 저장소, 새 IP/PSTN 게이트웨이, 새 Office Web Apps 서버, 새 SQL Server 저장소 또는 새 트렁크를 만들 수 있습니다.
  
> [!NOTE]
> 이러한 각 옵션에 대한 도움말 항목을 참조하세요. 
  
![공유 구성 요소 범위 노드](../../../media/Shared_Components_Scope_NodeB.jpg)
  
**토폴로지 선택 시** 새 토폴로지 문서를 여는 옵션을 클릭합니다. 기존 토폴로지 문서를 열고, 중앙 관리 저장소에서 현재 토폴로지 다운로드, 현재 토폴로지 문서 복사본 저장, 현재 토폴로지 문서 게시, 현재 구성된 데이터베이스 구성을 만들기 위한 데이터베이스 설치, Office Communications Server 2007 R2를 현재 배포에 병합 또는 배포 제거
  
![공유 구성 요소 범위 노드 C](../../../media/Shared_Components_Scope_NodeC.jpg)
