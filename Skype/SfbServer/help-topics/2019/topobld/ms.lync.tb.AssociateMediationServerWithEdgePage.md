# <a name="associate-mediation-server-with-edge"></a>에지와 중재 서버 연결
 
중재 서버를 정의하려면 중재 서버를 프런트 엔드 서버 및 에지 서버 또는 풀과 연결하는 설정을 지정합니다. 중재 서버 또는 풀을 에지 서버 또는 에지 서버 풀과 연결하려면 다음을 수행합니다.
  
1. 드롭다운 목록에서 중재 서버와 연결할 에지 풀 또는 에지 서버를 선택합니다.
    
2. 계속하려면 **다음** 을 클릭합니다.
    
> [!NOTE]
> 배포에서 에지 서버 또는 에지 서버 풀을 아직 정의하지 않은 경우 드롭다운 목록에 포함된 항목이 없습니다. **다음** 을 클릭하여 계속하고 배포에 에지 서버 또는 풀을 추가한 후 중재 서버의 설정을 편집할 수 있습니다.
  
자세한 내용은 Deploy [a Mediation Server in Topology Builder in 비즈니스용 Skype 서버.](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)
