# <a name="add-survivable-branch-appliance-pool"></a>Survivable Branch 기기 풀 추가
 
**Survivable Branch 기기 FQDN을 정의**하려면 **fqdn** 텍스트 상자에 Survivable Branch 기기 또는 Survivable branch 서버의 fqdn (정규화 된 도메인 이름 시스템) (DNS (domain name System)에 정의 됨)을 입력 합니다. 계속하려면 **다음**을 클릭합니다.
  
> [!NOTE]
> 다른 방법으로 명시 되지 않는 한, Survivable Branch 기기 또는 Survivable Branch 서버 정의는 동일한 절차입니다.
