# <a name="add-file-store"></a>파일 저장소 추가
 
새 파일 저장소를 정의하려면 다음을 지정합니다.
  
- 새 파일 저장소를 호스팅하는 서버의 FQDN(정규화된 도메인 이름). 토폴로지 게시 전에 서버의 FQDN이 DNS(Domain Name System)에 있어야 합니다.
    
- 호스트 서버에 있는 파일 공유 이름. 파일 공유는 토폴로지를 게시하기 전에 있어야 합니다.
