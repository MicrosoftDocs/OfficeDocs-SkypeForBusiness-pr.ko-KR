# <a name="add-file-store"></a>파일 저장소 추가
 
새 파일 저장소를 정의 하려면 다음을 지정 합니다.
  
- 새 파일 저장소를 호스팅할 서버의 정규화 된 도메인 이름 (FQDN)입니다. 토폴로지를 게시 하기 전에 DNS (Domain Name System)에 서버의 FQDN이 있어야 합니다.
    
- 호스트 서버의 파일 공유 이름입니다. 토폴로지를 게시 하기 전에 파일 공유가 존재 해야 합니다.
