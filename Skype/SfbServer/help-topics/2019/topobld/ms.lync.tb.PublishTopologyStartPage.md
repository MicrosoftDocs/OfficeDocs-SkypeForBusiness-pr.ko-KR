# <a name="publish-topology"></a>토폴로지 게시
 
게시 프로세스 시작 시 **토폴로지 게시** 페이지가 표시됩니다. 계속하기 전에 나열된 작업을 살펴보고 완료되었는지 확인합니다. 계속할 준비가 되면 **다음** 을 클릭합니다. 게시 프로세스를 끝내야 할 경우 **취소** 를 클릭합니다.
