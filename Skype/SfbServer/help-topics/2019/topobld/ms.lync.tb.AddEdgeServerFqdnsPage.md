# <a name="add-edge-server-fqdn"></a>Edge 서버 FQDN 추가
 
액세스에 지 서비스에 대해 정규화 된 도메인 이름 (FQDN)을 지정 해야 합니다. **기능 선택** 페이지에서 **단일 FQDN &amp; IP 주소 사용** 옵션을 선택 하지 않은 경우에는 웹 회의 Edge 서비스에 대 한 fqdn과 a/V Edge 서비스도 지정 해야 합니다.
  
또한 **단일 FQDN &amp; IP 주소 사용** 옵션을 선택한 경우 각 Edge services에 대해 다른 포트 번호를 지정 해야 합니다 (권장 포트 설정: 액세스에 지 서비스용 444, 웹 회의 edge 서비스의 경우 8057, a/V edge 서비스의 경우 443). 옵션을 선택 하지 않은 경우 세 가지 서비스 모두에 동일한 포트 번호 (예: 443)를 사용할 수 있습니다.
