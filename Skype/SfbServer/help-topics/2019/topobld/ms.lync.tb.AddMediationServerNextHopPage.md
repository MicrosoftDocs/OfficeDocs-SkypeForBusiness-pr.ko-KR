# <a name="add-mediation-server-nexthop"></a>중재 서버 다음 홉 추가
 
정의 하는 중재 서버의 다음 홉을 정의 하려면 중재 서버의 다음 홉 역할을 하는 Enterprise Edition server 또는 Standard Edition 서버를 **다음 홉 풀** 드롭다운 목록에서 선택 합니다. 계속하려면 **다음**을 클릭합니다.
