# <a name="define-the-ip-address-of-the-ip-or-pstn-gateway"></a>IP 또는 PSTN 게이트웨이의 IP 주소 정의

**Ipv4 사용** 또는 **IPv6 사용**을 선택 하 여 IP 또는 PSTN (휴대폰 전환 전화 네트워크) 게이트웨이의 ipv4 또는 ipv6 주소를 정의 합니다. **구성 된 모든 IP 주소 사용** 을 선택 하거나 **선택한 ip 주소로 서비스 사용량을 제한할**수 있습니다.

**구성 된 모든 IP 주소 사용**을 선택 하면 게이트웨이에 구성 된 모든 주소가 사용 됩니다. 선택적으로 **서비스 사용량 제한을 선택 하**여 **PSTN ip 주소** 항목을 사용할 수 있도록 하는 선택 된 IP 주소로 제한 합니다. 이 게이트웨이에 대해 할당 된 IPv4 또는 IPv6 주소를 입력 합니다.
