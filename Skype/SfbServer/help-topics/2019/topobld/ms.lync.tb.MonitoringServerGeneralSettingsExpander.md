# <a name="monitoring-server-general-settings-expander"></a>모니터링 서버 일반 설정 확장기
 
토폴로지 작성기의 콘솔 트리에서 모니터링 서버를 마우스 오른쪽 단추로 클릭하고 도구 모음에서 **동작**을 클릭하거나 동작 창에서 작업을 클릭한 다음 **속성 편집**을 클릭하고 다음 옵션 중에서 원하는 옵션을 변경하여 개별 모니터링 서버의 속성을 편집할 수 있습니다.
  
- **FQDN**: 모니터링 서버로 배포할 서버의 FQDN(정규화된 도메인 이름)을 변경합니다.
    
- **SQL 저장소**: 모니터링 SQL Server 데이터베이스에 사용할 SQL Server 인스턴스를 변경합니다. 모니터링 서버의 SQL Server 데이터베이스를 변경하는 경우 변경 내용을 적용하려면 모니터링 서버를 다시 시작해야 합니다.
