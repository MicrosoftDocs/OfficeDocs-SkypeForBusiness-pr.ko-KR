# <a name="file-store-settings-expander"></a>파일 저장소 설정 확장기
 
정의 된 파일 저장소의 속성을 편집 하려면 다음을 수행 합니다.
  
파일 공유 이름에 대해 새 공유의 이름을 정의할 수 있습니다. 그런 다음 파일 서버의 FQDN (정규화 된 도메인 이름) 속성에 지정 된 서버에서 공유를 만들어야 합니다.
  
> [!NOTE]
> 파일 서버 FQDN 속성에 지정 된 서버의 FQDN을 변경할 수 없습니다. 파일 저장소 파일 공유를 다른 서버에 배치 해야 하는 경우 새 파일 저장소 항목을 만듭니다.
