# <a name="edge-server-add-machine"></a>Edge 서버 추가 컴퓨터

이 풀에 Edge 서버로 추가 하려는 각 컴퓨터의 FQDN (정규화 된 도메인 이름)을 지정 합니다. 컴퓨터를 목록에 추가한 후에는 토폴로지를 게시 하기 전에 언제 든 지 컴퓨터의 FQDN을 업데이트 하거나 풀에서 제거할 수 있습니다. 토폴로지를 게시 한 후 FQDN을 변경 하려면 토폴로지 작성기에서 서버를 삭제 한 다음 새 FQDN을 사용 하 여 풀에 새 서버를 추가 해야 합니다. Edge 풀에 대 한 토폴로지를 디자인 하는 방법에 대 한 자세한 내용은 Edge 배포 설명서에서 [Edge 토폴로지 정의](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) 를 참조 하세요.
