# <a name="archiving-servers-scope-node"></a>보관 서버 범위 노드
 
토폴로지 작성기에서 콘솔 트리에서 보관 서버를 마우스 오른쪽 단추로 클릭하고 도구 모음에서 동작을 클릭하거나  작업 창에서 작업을 클릭한 다음 다음 보관 서버 옵션을 클릭하여 개별 보관 서버와 관련된 작업을 수행할 수 있습니다. 
  
- **새 보관 서버**. FQDN(정규화된 도메인 이름), 보관 SQL Server 데이터베이스, 보관 파일 저장소 및 풀 연결을 비롯한 새 보관 서버의 설정을 지정하는 **새 보관 서버 정의** 마법사를 시작합니다.
    
> [!NOTE]
> 토폴로지 작성기에서 이전에 보관 서버를 토폴로지에 추가한 경우 보관 서버 노드에 보관 서버가 포함되어 있습니다. 그러나 보관 서버도 설정해야만 인스턴트 메시징 또는 웹 회의(메시징)를 보관할 수 있습니다. 여기에는 보관 서버로 사용할 서버에 비즈니스용 Skype 서버 설치 및 보관 지원 구성이 포함됩니다. 
  
이 항목의 앞부분에서 설명한 보관 서버 옵션 외에 콘솔 트리의 다른 노드에서 사용 가능한 보관 서버 노드의 일반 옵션도 있습니다. 여기에는 다음 옵션이 포함됩니다.
  
- **토폴로지.** 이 옵션을 사용하여 새 토폴로지 정의, 기존 토폴로지 열기, 현재 토폴로지 다운로드, 토폴로지 복사본 저장, 토폴로지 게시, 데이터베이스 설치, 구성 요소 병합 또는 배포 제거
    
- **보기**. 이 옵션을 사용하여 콘솔 트리 보기에 표시할 항목과 숨길 항목을 선택하여 토폴로지 작성기 보기를 사용자 지정합니다.
    
- **도움말**. 이 옵션을 사용하여 활성 노드에 대한 도움말에 액세스합니다.
