# <a name="add-archiving-server"></a>보관 서버 추가
 
보관 서버를 토폴로지 디자인으로 배포하려면 보관을 배포할 서버의 FQDN(정규화된 도메인 이름)을 지정해야 합니다. 비즈니스용 Skype Server 보관 및 설치를 포함 하는 토폴로지를 게시 하기 전에 보관 배포에 대 한 모든 필수 조건을 완료 해야 합니다.
