# <a name="add-edge-server-support"></a>Edge 서버 지원 추가

초기 토폴로지를 배포 하거나 나중에 외부 사용자 액세스에 대 한 지원을 추가할 수 있습니다. 기존 토폴로지에 에지 서버를 추가하는 방법에 대한 자세한 내용은 에지 서버 배포 설명서에서 [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)를 참조하세요.
