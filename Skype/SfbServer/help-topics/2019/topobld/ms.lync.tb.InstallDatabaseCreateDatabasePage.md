# <a name="install-and-create-databases"></a>데이터베이스 설치 및 만들기

배포에 대해 만들려는 데이터베이스를 선택 합니다. 기본적으로 데이터베이스는 정의 된 사이트의 정의 된 SQL Server에서 만들어지고 데이터베이스를 배치 하는 SQL Server에 따라 데이터베이스 파일을 자동으로 배포 하 고 구성 합니다.

 **만들려는 데이터베이스를 선택**합니다. 배포 하 고 구성 하려는 데이터베이스의 확인란을 선택 합니다. 배포할 데이터베이스 중 하나 또는 모두의 확인란을 선택 합니다.

> [!CAUTION]
> 해당 데이터베이스를 배포 하는 인스턴스를 수용 하기 위해 SQL Server가 이미 인스턴스에 대해 구성 되어 있어야 하며 방화벽 포트를 열어야 합니다. 자세한 내용은 [SQL Server 구성](https://technet.microsoft.com/library/375e5cc4-e436-46dc-9b02-5063f35cdcc1.aspx) 을 참조 하세요.

 **고급**: sql server를 클릭 하 고 **고급** 단추를 클릭 하 여 sql server의 데이터베이스 파일 위치에 대 한 옵션을 선택 합니다. 고급 데이터베이스 파일 배치에 대 한 자세한 내용은 [Lync Server Management Shell을 사용 하 여 데이터베이스 설치](https://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx) 를 참조 하세요.

 **뒤로**:이 단추를 클릭 하면 이전 화면으로 돌아갑니다 (이 대화 상자에 표시 된 방법에 따라 항상 사용 가능 하지 않을 수도 있음).

 **다음**:이 단추를 클릭 하면 현재 대화 상자에서 선택한 내용이 커밋되고 추가 정보를 구성 하는 다음 대화 상자로 이동 합니다.

 **취소**:이 단추를 클릭 하면 구성이 종료 되 고 변경 내용이 취소 됩니다. 일부 구성 화면에서는 종료 하 고 변경 내용을 취소 하 라는 메시지가 표시 됩니다. **예** 를 선택 하면 현재 구성이 닫히고 현재 구성이 닫히고 토폴로지 작성기로 돌아갑니다. **아니요** 를 선택 하면 현재 구성 대화 상자로 돌아가고 구성을 계속 진행할 수 있습니다.

 **도움말**: **도움말** 단추를 클릭 하면 현재 구성 대화 상자와 연결 된 도움말 정보가 표시 됩니다.
