# <a name="legacy-merge-edge-type"></a>레거시 병합 가장자리 유형
 
**단일 Edge 서버** 또는 **부하 분산 된 통합에 지** 서버에서 **다음** 을 클릭 하 여 구성 정보를 지정 합니다. 확장 된 **edge**의 경우 각 서버의 구성을 통합 된 edge 구성으로 변경 해야 병합이 가능 합니다.
