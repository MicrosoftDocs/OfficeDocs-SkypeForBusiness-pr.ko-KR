# <a name="select-pstn-usage-record"></a>PSTN 사용 레코드 선택
 
**PSTN 사용 레코드 선택** 대화 상자를 사용하여 사용할 수 있는 기존 공중 전화망(PSTN) 사용 레코드를 찾습니다.
  
> [!NOTE]
> 사용 가능한 모든 PSTN 사용 레코드를 검색하려면 검색 상자에 아무런 내용을 입력하지 않고 검색 아이콘을 클릭합니다.
