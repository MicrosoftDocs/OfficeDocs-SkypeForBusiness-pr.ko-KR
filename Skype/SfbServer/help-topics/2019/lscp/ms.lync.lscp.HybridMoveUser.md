# <a name="hybrid-wizard-move-user"></a>하이브리드 마법사: 사용자 이동

사용자 이동 마법사를 사용하여 비즈니스용 Skype Online에 있는온-프레미스 풀에 있는 사용자 계정을 이동할 수 있습니다. 자세한 내용은 하이브리드 관리 가능성 [마법사 사용을 참조하세요.](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)
