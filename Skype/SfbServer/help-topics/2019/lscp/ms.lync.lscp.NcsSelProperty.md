# <a name="assign-properties"></a>속성 지정
 
**속성 지정** 대화 상자를 사용 하 여 한 번에 여러 서브넷에 대 한 설명 및 네트워크 사이트 ID를 지정 합니다. **설명**에 선택한 서브넷에 대해 사용할 설명을 입력 합니다. **네트워크 사이트 id**에서 선택한 서브넷에 할당 하려는 네트워크 사이트 id를 선택 합니다. 이러한 속성을 선택한 서브넷에 할당 하려면 **확인**을 클릭 합니다.
