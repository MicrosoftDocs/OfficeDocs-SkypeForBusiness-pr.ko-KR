# <a name="select-from-active-directory"></a>Active Directory에서 선택
 
Active **Directory에서 선택** 대화 상자를 사용하여 Active Directory에 대해 사용하도록 설정할 사용자를 비즈니스용 Skype 서버. 검색 상자에 찾을 사용자의 SIP 주소 또는 표시 이름을 입력합니다. 사용 가능한 모든 사용자를 검색하기 위해 검색 상자에 아무 것도 입력하지 않습니다.
