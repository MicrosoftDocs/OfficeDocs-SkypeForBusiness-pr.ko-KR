# <a name="uncommitted-voice-configuration-settings-review"></a>커밋되지 않은 음성 구성 설정: 검토
 
**커밋되지 않은 음성 구성 설정** 대화 상자를 사용하여 적용한 모든 변경 내용을 검토합니다. 대화 상자를 스크롤하여 커밋되지 않은 모든 변경 내용을 검토하거나 화살표를 클릭하여 다양한 섹션을 확장 또는 축소합니다. 검토를 마친 후 **닫기** 를 클릭합니다. 이 대화 상자를 닫은 후 변경 내용을 수정하거나 커밋할 수 있습니다.
