# <a name="pin-set-successfully"></a>PIN 설정 완료
 
전자 메일을 사용하여 사용자에게 PIN을 보내려면 전자 메일 응용 프로그램 열기를 클릭하여 사용자에게 새 **PIN을 전송합니다.** PIN을 전자 메일 메시지에 복사하려면 **PIN 표시를 선택합니다.**
