# <a name="select-an-announcement-service"></a>공지 사항 서비스 선택
 
**알림 서비스 선택** 대화 상자를 사용 하 여 지정 되지 않은 숫자의 범위에 대 한 수신 전화를 처리할 알림 서비스를 선택 합니다.
  
검색 상자에 사용 하려는 알림 서비스 이름의 전부 또는 일부를 입력 합니다. 사용 가능한 모든 알림 서비스를 반환 하려면 검색 상자를 비워 두고 검색 아이콘을 클릭 합니다.
  
검색에 의해 반환 되는 알림 서비스 목록에서 지정 되지 않은 숫자 범위에 대 한 수신 전화를 처리 하는 데 사용할 서비스를 선택 합니다.
