# <a name="clients-mobility-policy"></a>클라이언트: 이동성 정책

**클라이언트** **이동성 정책** 페이지에는 배포에서 사용할 수 있는 이동성 정책이 표시 됩니다.

## <a name="tasks-you-can-perform"></a>수행할 수 있는 작업

**모바일 정책** 페이지에서 다음 작업을 수행할 수 있습니다.

- 기존 모바일 정책 보기

- 새 이동성 정책 만들기

- 이동성 정책 편집

## <a name="ui-reference"></a>UI 참조

다음 목록에서는 페이지의 메뉴, 명령, 필드 및 속성에 대해 설명합니다.



- **검색 필드** 표시 되는 정책을 필터링 하려면 텍스트 상자에 문자열을 입력 합니다.

- **새로운** **새로**만들기를 클릭 한 다음 **사이트** 또는 **사용자** 중 하나를 선택 하 여 새 이동성 정책을 만듭니다.

- **편집** 정책을 선택한 다음 **편집** 을 클릭 하 여 선택한 정책의 속성을 편집 합니다.

- **새로 고침** 표시 된 정책 목록을 새로 고칩니다. **새로 고침**을 클릭 하면 검색 필터 문자열이 지워지고 기존의 모든 정책이 표시 됩니다.

모바일 정책 구성에 대 한 자세한 내용은 [모바일 정책 구성을](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx)참조 하세요.
