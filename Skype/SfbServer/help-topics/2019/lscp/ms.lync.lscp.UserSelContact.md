# <a name="select-contacts-from-active-directory"></a>Active Directory에서 연락처 선택
 
Active Directory에서 선택 대화 상자를 사용하여 Active **Directory** 에 대해 사용하도록 설정할 연락처를 비즈니스용 Skype. 검색 상자에 찾을 연락처의 SIP 주소나 표시 이름을 입력합니다. 사용 가능한 모든 연락처를 검색하려면 검색 상자에 아무런 내용을 입력하지 마십시오.
