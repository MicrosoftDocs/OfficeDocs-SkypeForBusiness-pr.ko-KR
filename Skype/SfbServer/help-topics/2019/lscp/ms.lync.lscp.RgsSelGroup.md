# <a name="select-groups"></a>그룹 선택

응답 그룹 큐는 상담원이 전화에 대답할 때까지 응답 그룹에 대 한 통화를 보류 합니다. 큐에 에이전트 그룹이 할당 되었습니다. 큐를 만들거나 편집할 때 **그룹 선택** 대화 상자를 사용 하 여 큐에 할당할 하나 이상의 에이전트 그룹을 선택 합니다.

## <a name="ui-reference"></a>UI 참조

다음 목록에서는 **그룹 선택** 대화 상자의 컨트롤 및 필드에 대해 설명 합니다.

- 검색 상자 에이전트 그룹의 이름을 검색 합니다. 이름의 전부 또는 일부를 입력 합니다. 검색 상자를 비워 두어 조직의 모든 에이전트 그룹을 표시 합니다.

- **이름** 반환 되는 에이전트 그룹의 이름을 표시 합니다.

- **참가 정책** 에이전트 그룹에 대해 정의 된 참가 정책 (공식적인 또는 비공식적인)을 표시 합니다.

- **라우팅 방법** 에이전트 그룹에 대해 정의 된 라우팅 유형을 표시 합니다.

큐 사용에 대한 자세한 내용은 작업 설명서의 [Managing Response Group Queues](https://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx)를 참조하세요.
