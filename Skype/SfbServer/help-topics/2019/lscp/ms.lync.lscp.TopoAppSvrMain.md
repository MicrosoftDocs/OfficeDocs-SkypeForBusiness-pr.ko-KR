# <a name="topology-server-application"></a>토폴로지: 서버 응용 프로그램

**토폴로지**: **서버 응용 프로그램** 페이지에는 배포의 서버 응용 프로그램이 표시됩니다.

## <a name="tasks-you-can-perform"></a>수행할 수 있는 작업

**토폴로지**: **서버 응용 프로그램** 페이지에서는 다음 작업을 수행할 수 있습니다.

- 배포의 서버 응용 프로그램 보기

- 서버 응용 프로그램을 사용하거나 사용하지 않도록 설정

- 응용 프로그램을 중요로 선택 또는 선택 취소
