# <a name="users-view-mobility-policy"></a>사용자: 모바일 정책 보기

**보기 - 모바일 정책** 대화 상자에는 선택한 모바일 정책의 속성이 표시됩니다.

모바일 정책을 할당하는 방법에 대한 자세한 내용은 [Assign a Per-User Mobility Policy](/previous-versions/office/lync-server-2013/lync-server-2013-assign-a-per-user-mobility-policy) 및 [Configuring Mobility Policy](/previous-versions/office/lync-server-2013/lync-server-2013-configuring-mobility-policy)을 참조하십시오.
