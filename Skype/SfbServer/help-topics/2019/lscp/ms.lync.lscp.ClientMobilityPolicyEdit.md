# <a name="edit-client-mobility-policy"></a>클라이언트 모바일 정책 편집

**모바일 정책 편집** 대화 상자에서 모바일 정책 세부 정보를 편집할 수 있습니다.

## <a name="ui-reference"></a>UI 참조

다음 목록에서는 페이지의 메뉴, 명령, 필드 및 속성에 대해 설명합니다.


- **커밋** 정책 설정에 대한 변경 내용을 저장하려면 클릭합니다.

- **취소** 클릭하여 정책 편집을 중지하고 정책 설정에 대한 변경 내용을 삭제합니다.

- **이름** 모바일 정책의 이름입니다.

- **설명** 모바일 정책에 대한 설명입니다.

모바일 정책에 대한 자세한 내용은 [Mobility Policies](/previous-versions/office/lync-server-2013/lync-server-2013-mobility-policies)을 참조하십시오.
