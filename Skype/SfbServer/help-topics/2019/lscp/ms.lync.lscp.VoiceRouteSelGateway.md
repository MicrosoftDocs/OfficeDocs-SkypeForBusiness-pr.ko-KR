# <a name="select-trunk"></a>트렁크 선택
 
**트렁크 선택** 대화 상자를 사용하여 사용할 수 있는 기존 트렁크를 찾습니다.
  
> [!NOTE]
> 사용 가능한 모든 트렁크를 검색하려면 검색 상자에 아무런 내용을 입력하지 않고 검색 아이콘을 클릭합니다.
