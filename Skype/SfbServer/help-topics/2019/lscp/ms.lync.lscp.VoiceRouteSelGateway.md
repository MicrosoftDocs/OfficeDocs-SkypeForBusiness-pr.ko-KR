# <a name="select-trunk"></a>트렁크 선택
 
**트렁크 선택** 대화 상자를 사용 하 여 사용할 수 있는 기존 트렁크를 찾습니다.
  
> [!NOTE]
> 사용 가능한 모든 trunks를 검색 하려면 검색 상자에 아무것도 입력 하지 않은 다음 검색 아이콘을 클릭 합니다.
