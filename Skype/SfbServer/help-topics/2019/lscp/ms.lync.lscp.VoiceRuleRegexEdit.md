# <a name="translation-rule-type-a-regular-expression"></a>번역 규칙: 정규식을 입력 합니다.
 
**이 패턴에 일치** 필드에서 번역할 숫자와 일치 시키는 데 사용할 패턴을 지정 합니다. **번역 규칙** 필드에서 번역 된 숫자의 형식에 대 한 패턴을 지정 합니다. 예를 들어 **이 패턴 일치** 필드\+에 ^{9}(\d \d +) $를 입력 하 고 **번역 규칙** 필드에 011 $1을 입력할 경우 규칙은 + 441235551010을 011441235551010로 변환 합니다.
