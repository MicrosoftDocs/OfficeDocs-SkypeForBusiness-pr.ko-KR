# <a name="administrative-roles-and-skype-for-business-server-control-panel"></a>관리 역할 및 비즈니스용 Skype 서버 제어판

높은 보안 표준을 유지하면서 관리 작업을 위임할 수 있도록 비즈니스용 Skype 서버는 RBAC(역할 기반 액세스 제어)를 지원합니다. RBAC에서는 미리 정의된 관리 역할에 사용자를 할당하여 관리 권한을 부여합니다. 비즈니스용 Skype 서버에는 다양한 기본 제공 관리 역할 집합이 포함되어 있으며, 새 역할을 만들고 각 새 역할에 대한 사용자 지정 cmdlet 목록을 지정할 수도 있습니다. 또한 미리 정의된 RBAC 역할 및 사용자 지정 RBAC 역할 둘 다에 대해 허용되는 작업에 cmdlet 스크립트를 추가할 수도 있습니다. 비즈니스용 Skype 서버에서 사용할 수 있는 미리 정의한 관리 역할을 비롯한 자세한 내용은 역할 기반 [액세스 제어를 참조하세요.](https://technet.microsoft.com/library/41204ba3-ce5b-41a8-a6c3-b444468fa328.aspx)
