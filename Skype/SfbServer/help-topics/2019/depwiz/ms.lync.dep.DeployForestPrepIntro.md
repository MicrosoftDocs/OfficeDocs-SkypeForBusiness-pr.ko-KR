# <a name="prepare-current-forrest-intro"></a>현재 포리스트 준비(소개)
 
**포리스트 준비** 작업을 시작하려면 **다음** 을 클릭합니다. 포리스트 **준비 작업을 실행** 하면 전역 설정이 만들어지며 포리스트에서 사용할 유니버설 그룹이 비즈니스용 Skype 서버.
