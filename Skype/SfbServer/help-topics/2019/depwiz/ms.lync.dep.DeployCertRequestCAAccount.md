# <a name="certificate-request-certificate-authority-account"></a>인증서 요청(인증 기관 계정)
 
요청을 제출하려면 CA(인증 기관)에 현재 로그인한 사용자의 자격 증명이 아니라 다른 자격 증명이 필요할 수 있습니다. 인증서 요청을 다른 사용자로 허용하려면 인증 기관에 대한 대체 자격 증명 지정 확인란을 선택한 다음 인증서를 요청할 수 있는 사용자의 사용자 이름 또는 _domainusername_ \  을 입력합니다. 암호 **텍스트** 상자에 지정한 사용자의 암호를 입력합니다. 그러면 사용자 이름 및 암호가 인증서 요청 프로세스(그러나 실제 인증서 요청에서는 아님)의 일부로 온라인 CA로 전송됩니다.
