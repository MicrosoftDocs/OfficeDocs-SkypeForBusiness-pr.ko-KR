# <a name="prepare-current-domain-intro"></a>현재 도메인 준비(소개)
 
도메인 **준비 페이지에서** 는 도메인을 준비하는 프로세스를 비즈니스용 Skype 서버. 도메인 준비를 시작하려면 **다음** 을 클릭합니다.
