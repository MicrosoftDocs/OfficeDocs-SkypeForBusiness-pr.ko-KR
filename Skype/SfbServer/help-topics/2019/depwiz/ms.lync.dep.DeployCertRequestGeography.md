# <a name="certificate-request-geography"></a>인증서 요청 (지리)
 
**지리 정보** 페이지는 인증서 요청에 대 한 추가 정보를 요청 합니다. 이 페이지에서 **국가**를 선택 하 고 텍스트 상자에 시 **/도** 및 **구/군/시** 를 입력 합니다. 약어를 사용해 서는 안 되며 시/도 및 구/군/시의 전체 이름을 기록해 야 합니다.
