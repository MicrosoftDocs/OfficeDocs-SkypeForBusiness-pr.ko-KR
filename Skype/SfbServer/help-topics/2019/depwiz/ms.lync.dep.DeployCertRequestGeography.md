# <a name="certificate-request-geography"></a>인증서 요청(배치)
 
**지역 정보** 페이지에서는 인증서 요청에 대한 추가 정보를 요청합니다. 이 페이지에서 **국가** 를 선택한 다음 텍스트 상자에 **시/도** 및 **구/군/시** 를 입력합니다. 약어를 사용하지 말고 시/도 및 구/시의 전체 이름을 입력해야 합니다.
