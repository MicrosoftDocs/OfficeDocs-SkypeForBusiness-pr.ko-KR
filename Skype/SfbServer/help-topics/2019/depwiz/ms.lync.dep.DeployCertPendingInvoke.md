# <a name="pending-certificate-invoke"></a>보류 중인 인증서(인보크)
 
**명령 실행** 페이지에는 보류 중인 인증서를 검색 하는 명령의 상태가 표시 됩니다. 작업이 완료 되 면 **보류 중인 인증서 검색** 로그 파일을 사용할 수 있습니다. **로그 보기** 를 클릭 하 여 처리 및 작업 완료 상태를 검토 합니다. 계속하려면 **다음**을 클릭합니다.
  
> [!IMPORTANT]
> 인증 기관 관리자가 인증서를 승인 받지 않은 경우 보류 중인 요청 로그 파일 검색은 대기 중인 요청이 있음을 나타냅니다. 인증 기관 관리자가 요청을 거부 한 경우 로그 파일에 오류가 표시 되 고, 요청이 거부 되었음을 알리는 오류 및 요청을 거부 한 사람의 사용자 이름이 보고 됩니다.
