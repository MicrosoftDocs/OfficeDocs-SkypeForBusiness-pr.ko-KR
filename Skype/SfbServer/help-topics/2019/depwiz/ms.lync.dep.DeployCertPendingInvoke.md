# <a name="pending-certificate-invoke"></a>대기 중인 인증서(호출)
 
**명령 실행** 페이지에 대기 중인 인증서를 검색하는 명령 상태가 표시됩니다. 작업이 완료되면 **대기 중인 인증서 검색** 로그 파일을 사용할 수 있습니다. **로그 보기** 를 클릭하여 처리 및 작업 완료 상태를 검토합니다. 계속하려면 **다음** 을 클릭합니다.
  
> [!IMPORTANT]
> 인증 기관 관리자가 아직 인증서를 승인하지 않은 경우 대기 중인 요청 검색 로그 파일에 대기 중인 요청이 있음이 표시됩니다. 인증 기관 관리자가 요청을 거부한 경우 로그 파일에 실패가 표시되고, 요청이 거부되었음과 요청을 거부한 사람의 사용자 이름이 표시된 오류가 보고됩니다.
