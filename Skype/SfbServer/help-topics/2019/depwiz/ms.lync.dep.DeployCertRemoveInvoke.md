# <a name="remove-certificate-invoke"></a>인증서 제거 (호출)
 
**명령 실행** 페이지에서는 인증서를 제거 하는 방법에 대 한 세부 정보를 제공 합니다. 인증서 제거에서 요약 정보를 검토 합니다. 드롭다운 목록에 인증서 로그 파일 제거가 표시 되는지 확인 한 다음 **로그 보기** 를 클릭 하 여 인증서 제거에 대 한 세부 정보를 검토 합니다. 요약 및 로그 파일 검토가 완료 되 면 **마침을**클릭 합니다.
