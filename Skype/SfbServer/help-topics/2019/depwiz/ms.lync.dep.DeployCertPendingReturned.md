# <a name="pending-certificate-returned"></a>보류 중인 인증서(반환됨)
 
**온라인 인증서 요청 상태** 페이지에서 인증서의 현재 상태를 반환 합니다. 인증서가 인증 기관 관리자와 여전히 승인 상태에 있는 경우, 즉 아직 승인 되지 않은 경우 상태는 인증서가 여전히 처리 되 고 있음을 나타냅니다.
  
보류 중인 요청이 승인 되 고 인증서를 검색 하는 데 성공 하면 인증서가 로컬 인증서 저장소에 할당 되었음을 나타내고 상태 창에도 인증서 지문이 표시 됩니다.
  
인증 기관 관리자가 인증서 요청을 거부 하는 경우 상태 창에는 지정 된 ID (인증 기관에서 유지 관리)로 보류 중인 인증서 요청이 거부 됨이 보고 됩니다.
  
**완료** 를 클릭 하 여 보류 중인 인증서 처리를 완료 합니다.
