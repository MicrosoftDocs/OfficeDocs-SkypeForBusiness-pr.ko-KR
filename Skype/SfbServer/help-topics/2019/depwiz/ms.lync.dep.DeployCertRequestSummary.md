# <a name="certificate-request-summary"></a>인증서 요청(요약)
 
**인증서 요청 요약** 페이지에 인증서의 모든 옵션 및 구성에 대한 자세한 요약이 표시됩니다. 요약은 인증서 요청에 포함되는 내용과 이후에 인증서가 발급되는 방식을 나타냅니다. 요약을 신중하게 검토하여 선택한 모든 항목 및 옵션이 올바른지 확인합니다. 선택한 것이 올바른 경우  다음을 클릭하여 요청 생성을 계속하고 CA(인증 기관)로 보낼 준비를 합니다.
