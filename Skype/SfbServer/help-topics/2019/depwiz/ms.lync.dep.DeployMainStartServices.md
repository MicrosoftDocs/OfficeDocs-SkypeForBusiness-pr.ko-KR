# <a name="start-services"></a>서비스 시작
 
배포 된 서버 역할에서 서비스를 시작 하려면 **4 단계: 서비스 시작**에서 **실행** 을 클릭 합니다. 로컬 관리자 그룹 또는 동등한 사용자의 구성원으로 로그온 해야 합니다. 
  
> [!NOTE]
> **서비스 시작** 을 실행 한 경우 **다시 실행**을 클릭 하 여 명령을 다시 실행할 수 있습니다.
