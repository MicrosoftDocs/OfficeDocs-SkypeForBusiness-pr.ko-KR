# <a name="certificate-request-online-offline"></a>인증서 요청(온라인/오프라인)
 
지연 된 요청 또는 오프 라인 요청을 만들려면 **지금 요청 준비를 선택 하지만 나중에 (오프 라인 인증서 요청) 보냅니다**. 온라인 인증 기관 (일반적으로 조직의 공개 키 인프라의 내부 시스템)에 대 한 즉각적인 요청을 만들려면 **온라인 인증 기관에 요청을 즉시 보냅니다**.를 선택 합니다.
  
선택한 후 계속하려면 **다음**을 클릭합니다. 변경하지 않고 취소하려면 **취소**를 클릭합니다. 이전 단계로 돌아가려면 **뒤로**를 클릭합니다.
