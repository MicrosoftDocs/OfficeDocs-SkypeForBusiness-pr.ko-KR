# <a name="certificate-request-invoke"></a>인증서 요청 (호출)
 
**명령 실행** 페이지에는 인증서 요청 생성을 위해 실행 되는 명령으로 작업 상태 창이 표시 됩니다. 작업이 완료 되 면 요약 창 아래의 드롭다운 목록에 로그 파일 목록이 포함 됩니다. 드롭다운 목록을 사용 하 여 보려는 로그 파일을 선택한 다음 **로그 보기**를 클릭 합니다. Internet Explorer가 시작 되 고 로그 파일이 표시 됩니다. 요청이 실패 한 경우 로그 파일을 검토 하 여 오류의 원인을 확인 해야 합니다. 인증서 요청을 다시 실행 하 여 문제를 해결 하 고 로그 파일에 표시 된 문제를 수정할 수 있습니다. 인증서 요청 페이지를 계속 하 고 인증서를 다시 요청 합니다.
