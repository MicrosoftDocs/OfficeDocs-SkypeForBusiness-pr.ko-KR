# <a name="prepare-schema-invoke"></a>스키마 준비(호출)
 
**실행 명령** 페이지에는 비즈니스용 Skype 서버의 스키마를 확장 하기 위해 발급 된 명령의 요약이 표시 됩니다. 작업이 완료되면 요약 창에 작업의 성공 여부가 표시됩니다. 요약 화면을 검토하고 요약 창에 **작업 상태: 완료됨**이 표시되는지 확인합니다. 요약 창의 드롭다운 목록에서 스키마 준비 로그 파일을 선택하여 볼 수 있습니다. 로그 파일을 검토하려면 **로그 보기**를 클릭합니다. 스키마를 준비하는 동안 오류가 발생한 경우 작업에 발생한 오류가 여기에 기록됩니다. 그러면 오류를 수정하기 위한 작업을 수행한 후 스키마 준비를 다시 실행할 수 있습니다.
  
**마침**을 클릭하여 작업을 완료합니다.
