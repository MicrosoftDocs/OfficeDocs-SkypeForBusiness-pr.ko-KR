# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>비즈니스용 Skype 서버 구성 요소 설치 또는 제거(소개)
 
비즈니스용 Skype 서버 구성 요소를 설치 및 활성화 또는 비활성화 하 고 제거할 준비를 하 고 있습니다. 실행될 프로세스는 중앙 관리 저장소에 게시된 정의의 현재 구성에 의해 정의됩니다. 작업을 성공적으로 완료 하려면 로컬 관리자 그룹의 구성원으로 로그인 해야 하며, 최소한 Active Directory 도메인 서비스를 읽을 수 있어야 합니다.
  
**명령 실행** 화면으로 계속하려면 **다음**을 클릭합니다.
