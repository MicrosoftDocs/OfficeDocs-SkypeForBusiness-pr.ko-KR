# <a name="certificate-request-intro"></a>인증서 요청(소개)
 
인증서 요청 프로세스를 계속하려면 **다음** 을 클릭합니다. 시스템에 대한 변경 내용 없이 프로세스를 끝내려면 **취소** 를 클릭합니다.
