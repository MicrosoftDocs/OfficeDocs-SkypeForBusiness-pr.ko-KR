# <a name="install-local-configuration-store-invoke"></a>로컬 구성 저장소 (호출) 설치
 
**명령 실행** 창에 중앙 관리 저장소의 로컬 읽기 전용 복제본을 설치 하기 위한 작업의 상태가 표시 됩니다. 작업이 실행 되는 동안 명령이 실행 되 면 작업창이 업데이트 됩니다. 작업 상태를 검토 하 고 표시 되는 오류 또는 경고를 기록해 야 합니다. 작업이 완료 되 면 메시지 **작업 상태: 완료** 됨이 표시 되 고 로그 파일의 드롭다운 목록에 로그 파일의 이름이 표시 됩니다. 생성 된 로그 파일을 보려면 보려는 로그 파일을 선택 하 고 **로그 보기**를 클릭 합니다. 모든 로그 파일에서 성공, 오류 및 경고 메시지를 검토 합니다.
  
로그 파일을 검토 한 후 **마침을** 클릭 하 여 로컬 구성 저장소 설치를 완료 합니다.
