# <a name="pending-certificate-intro"></a>보류 중인 인증서(소개)
 
온라인 인증 기관에 발급 했지만 관리적으로 수동 승인 프로세스가 필요 하도록 설정 되어 있는 보류 중인 온라인 인증서 요청을 처리 하려면 **다음**을 클릭 합니다.
