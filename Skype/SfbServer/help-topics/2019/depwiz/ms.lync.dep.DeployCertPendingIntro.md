# <a name="pending-certificate-intro"></a>대기 중인 인증서(소개)
 
대기 중인 온라인 인증서 요청(즉, 온라인 인증 기관에 발급되었지만 관리자가 수동 인증 프로세스가 필요하다고 설정한 요청)을 처리하려면 **다음** 을 클릭합니다.
