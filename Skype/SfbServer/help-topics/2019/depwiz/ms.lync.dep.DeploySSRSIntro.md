# <a name="sql-server-reporting-services-intro"></a>SQL Server Reporting Services (소개)
 
각 프런트 엔드 풀 및 Survivable Branch 기기는 연결 된 모니터링 서버를 하나만 가질 수 있습니다. 사이트에 대해 모니터링을 사용할 수 있는 경우 모니터링 서버는 CDR (통화 정보 기록), 체감 품질 (환경 품질) 데이터 수집 및 보고 기능을 제공 합니다.
  
사용 현황이 모니터링 서버의 용량을 초과 하지 않는 경우 사이트의 모든 풀과 여러 중앙 사이트의 풀은 동일한 모니터링 서버를 사용할 수 있습니다. 모니터링을 지원 하기 위해 토폴로지를 디자인 하는 방법에 대 한 자세한 내용은 배포 설명서의 [비즈니스용 Skype 서버에서 프런트 엔드 풀을 사용 하 여 모니터링 저장소 연결](../../../deploy/deploy-monitoring/associate-a-monitoring-store.md) 을 참조 하세요.
