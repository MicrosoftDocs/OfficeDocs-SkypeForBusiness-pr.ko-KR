# <a name="import-certificate-summary"></a>인증서 가져오기(요약)
 
**인증서 가져오기 요약** 페이지에는 개인 키의 파일 이름, 경로, 현재 상태에 대 한 요약이 표시 됩니다. 요약을 검토 한 후 **다음** 을 클릭 하 여 계속 합니다.
