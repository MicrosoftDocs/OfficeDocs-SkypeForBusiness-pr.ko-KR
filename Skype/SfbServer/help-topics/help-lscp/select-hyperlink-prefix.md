# <a name="select-hyperlink-prefix"></a>하이퍼링크 접두사 선택
 
**하이퍼링크 접두사 선택** 대화 상자를 사용 하 여 차단 하려는 하이퍼링크 유형 선택, 경고 메시지 보내기 또는 허용 (경고 메시지 차단 또는 보내기) 또는 선택 (허용)에서 확장의 확인란을 **하이퍼링크 접두사**의 범주 화살표를 클릭 하 여 범주를 확장 하 고 해당 범주에 대 한 모든 하이퍼링크 접두사를 봅니다. 범주에 대 한 모든 하이퍼링크 접두사를 선택 하려면 범주 이름을 선택 합니다. 
  
**하이퍼링크**접두사에 선택 하려는 하이퍼링크 접두사가 표시 되지 않는 경우 **목록에 새 하이퍼링크 접두사 추가**아래에 있는 텍스트 상자에 확장명을 입력 한 다음 **추가**를 클릭 합니다. 
  
**선택한 하이퍼링크 접두사** 목록에 원하는 모든 하이퍼링크 접두사를 추가한 후 **확인**을 클릭 합니다.
  
비즈니스용 Skype 서버 제어판을 사용 하 여 수행할 수 있는 다양 한 절차에 대 한 자세한 내용은 비즈니스용 [Skype 서버 2015 관리](../../manage/manage.md)를 참조 하세요.
