# <a name="users-view-mobility-policy"></a>사용자: 이동성 정책 보기

**보기-이동성 정책** 대화 상자에 선택한 이동성 정책의 속성이 표시 됩니다.

이동성 정책 지정에 대 한 자세한 내용은 [사용자 단위 이동성 정책 할당](https://technet.microsoft.com/library/d8bf997f-4bc7-48d3-973b-323505f55e9d.aspx) 및 [이동성 정책 구성을](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx)참조 하세요.
