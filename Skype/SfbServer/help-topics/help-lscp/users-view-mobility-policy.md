# <a name="users-view-mobility-policy"></a>사용자: 모바일 정책 보기

**보기 - 모바일 정책** 대화 상자에는 선택한 모바일 정책의 속성이 표시됩니다.

모바일 정책을 할당하는 방법에 대한 자세한 내용은 [Assign a Per-User Mobility Policy](https://technet.microsoft.com/library/d8bf997f-4bc7-48d3-973b-323505f55e9d.aspx) 및 [Configuring Mobility Policy](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx)을 참조하십시오.
