# <a name="topology-server-application"></a>토폴로지: 서버 응용 프로그램

**토폴로지**: **서버 응용 프로그램** 페이지는 배포에 서버 응용 프로그램을 표시 합니다.

## <a name="tasks-you-can-perform"></a>수행할 수 있는 작업

**토폴로지**: **서버 응용 프로그램** 페이지에서 다음 작업을 수행할 수 있습니다.

- 배포의 서버 응용 프로그램 보기

- 서버 응용 프로그램 사용 또는 사용 안 함

- 중요 한 응용 프로그램을 선택 하거나 선택을 취소 합니다.
