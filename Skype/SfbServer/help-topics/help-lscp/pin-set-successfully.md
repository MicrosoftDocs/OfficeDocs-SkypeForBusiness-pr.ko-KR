# <a name="pin-set-successfully"></a>PIN 설정 완료
 
전자 메일을 사용하여 사용자에게 PIN을 보내려면 내 전자 메일 응용 프로그램 열기 를 클릭하여 사용자에게 새 **PIN을 전송합니다.** 전자 메일 메시지에 PIN을 복사하려면 **PIN 표시를 선택합니다.** 
  
비즈니스용 Skype 서버 제어판을 사용하여 수행할 수 있는 다양한 절차에 대한 자세한 내용은 [Manage 비즈니스용 Skype 서버 2015를 참조합니다.](../../manage/manage.md)
