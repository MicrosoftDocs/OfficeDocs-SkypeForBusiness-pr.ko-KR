# <a name="pin-set-successfully"></a>PIN 설정 완료
 
전자 메일을 사용 하 여 사용자에 게 PIN을 보내려면 **내 전자 메일 응용 프로그램 열기를 클릭 하 여 새 PIN을 사용자에 게 보냅니다**. 전자 메일 메시지에 PIN을 복사 하려면 **Pin 표시**를 선택 합니다. 
  
비즈니스용 Skype 서버 제어판을 사용 하 여 수행할 수 있는 다양 한 절차에 대 한 자세한 내용은 비즈니스용 [Skype 서버 2015 관리](../../manage/manage.md)를 참조 하세요.
