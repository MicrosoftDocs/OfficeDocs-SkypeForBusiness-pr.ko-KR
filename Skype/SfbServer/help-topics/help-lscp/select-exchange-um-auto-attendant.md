# <a name="select-exchange-um-auto-attendant"></a>Exchange UM 자동 전화 교환 선택
 
**EXCHANGE Um 자동 전화 교환 선택** 대화 상자를 사용 하 여 지정 되지 않은 범위의 범위에 대 한 수신 전화를 처리할 Exchange Um 자동 전화 교환을 선택 합니다.
  
검색 상자에 사용 하려는 Exchange UM 자동 전화 교환의 이름 전체 또는 일부를 입력 합니다. 사용할 수 있는 모든 Exchange UM 자동 전화 교환을 반환 하려면 검색 상자를 비워 두고 검색 아이콘을 클릭 합니다.
  
검색에 의해 반환 되는 Exchange UM 자동 전화 교환 목록에서 지정 되지 않은 숫자 범위에 대 한 수신 호출을 처리 하는 데 사용할 자동 전화 교환을 선택 합니다.
