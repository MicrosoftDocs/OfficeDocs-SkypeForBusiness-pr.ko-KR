# <a name="select-exchange-um-auto-attendant"></a>Exchange UM 자동 전화 교환 선택
 
UM Exchange 선택 **자동 전화 교환** 대화 상자를 사용하여 지정되지 않은 번호 범위에 대한 수신 Exchange 자동 전화 교환 UM 번호를 선택합니다.
  
검색 상자에 사용할 Exchange UM 자동 전화 교환 이름을 모두 또는 일부분 입력합니다. UM 자동 Exchange 사용할 수 있는 모든 기능을 반환하려면 검색 상자를 비워두고 검색 아이콘을 클릭합니다.
  
검색에서 Exchange 반환하는 UM 자동 전화 자동 전화 교환 지정되지 않은 번호 범위에 대한 수신 전화를 처리하는 데 사용할 UM 자동 전화 응답 목록을 선택합니다.
