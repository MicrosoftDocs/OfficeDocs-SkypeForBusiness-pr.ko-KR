# <a name="select-groups"></a>그룹 선택

응답 그룹 큐는 에이전트가 통화에 응답할 때까지 응답 그룹에 대한 통화를 보류합니다. 큐에는 에이전트 그룹이 할당되어 있습니다. 큐를 만들거나 편집할 때는 **그룹 선택** 대화 상자를 사용하여 큐에 할당할 에이전트 그룹을 하나 이상 선택합니다.

## <a name="ui-reference"></a>UI 참조

다음 목록에서는 **그룹 선택** 대화 상자의 컨트롤 및 필드에 대해 설명합니다.

- 검색 상자 에이전트 그룹의 이름을 검색합니다. 이름을 전체 또는 일부분 입력합니다. 조직의 모든 에이전트 그룹을 표시하려면 검색 상자를 비워 둡니다.

- **이름** 반환된 에이전트 그룹의 이름을 표시됩니다.

- **참가 정책** 에이전트 그룹에 대해 정의된 참가 정책(공식 또는 비공식)을 표시됩니다.

- **라우팅 방법** 에이전트 그룹에 대해 정의된 라우팅 유형을 표시됩니다.

큐를 사용하는 방법에 대한 자세한 내용은 작업 설명서에서 [Managing Response Group Queues](/previous-versions/office/lync-server-2013/lync-server-2013-managing-response-group-queues)를 참조하십시오.
