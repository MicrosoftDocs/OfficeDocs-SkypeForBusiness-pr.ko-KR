# <a name="set-pin"></a>PIN 설정

사용자의 PIN을 생성하려면 유효한 PIN을 자동으로 **생성합니다.**

자체 PIN을 만들려면 **특정 PIN을 수동으로 입력** 을 선택한 다음 텍스트 상자에 PIN 정책 설정에 지정되어 있는 PIN 요구 사항을 충족하는 PIN을 입력합니다.

비즈니스용 Skype 서버 제어판을 사용하여 수행할 수 있는 다양한 절차에 대한 자세한 내용은 비즈니스용 [Skype 서버 2015를 참조하세요.](../../manage/manage.md)
