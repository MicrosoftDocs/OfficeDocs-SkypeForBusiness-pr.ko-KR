# <a name="select-agents"></a>에이전트 선택

상담원은 응답 그룹 통화에 응답 하도록 지정 된 사용자입니다. 응답 그룹에는 해당 응답 그룹에 대한 전화를 받을 수 있는 에이전트를 식별하는 에이전트 그룹이 할당되어 있어야 합니다. 에이전트 그룹을 만드는 방법 중 하나는 적합한 사용자를 선택하여 사용자 지정 그룹을 정의하는 것입니다. 비즈니스용 Skype Server 및 Enterprise Voice에서 적격 사용자를 사용할 수 있습니다.

**에이전트 선택** 대화 상자를 사용하여 에이전트 그룹에 추가할 사용자를 선택합니다.

## <a name="ui-reference"></a>UI 참조

다음 목록에서는 **에이전트 선택**의 컨트롤에 대해 설명합니다.

- **찾기** 사용자의 SIP 주소 또는 표시 이름을 검색 합니다. 주소 또는 이름 전체 또는 일부를 입력 합니다. 검색 상자를 비워 두어 비즈니스용 Skype Server 및 Enterprise Voice에 대해 설정 된 모든 사용자를 표시 합니다.

- **표시할 최대 사용자 수** 표시 되는 반환 되는 결과의 수를 변경 합니다. 많은 결과가 예상 되는 경우이 카운터를 사용 하 여 검색을 제한 합니다.

다음 목록에서는 **에이전트 선택** 대화 상자의 필드에 대해 설명합니다.

- **상담원** 검색에서 반환 된 사용자 이름을 표시 합니다.

- **SIP 주소** 검색 결과로 반환 되는 사용자 SIP 주소를 표시 합니다.

- **전화 통신** 사용자에 대해 정의 된 **전화 통신** 필드의 값을 표시 합니다.

- **사용 하도록 설정** 사용자에 대해 정의 된 **Lync Server 사용** 필드의 값을 표시 합니다.

에이전트 그룹 사용에 대한 자세한 내용은 작업 설명서의 [Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx)를 참조하세요.
