# <a name="assign-properties"></a>속성 할당
 
**속성 할당** 대화 상자를 사용하여 한 번에 여러 서브넷에 설명 및 네트워크 사이트 ID를 지정합니다. **설명** 에 선택한 서브넷에 사용할 설명을 입력합니다. **네트워크 사이트 ID** 에서 선택한 서브넷에 할당할 네트워크 사이트 ID를 선택합니다. 선택한 서브넷에 이러한 속성을 할당하려면 **확인** 을 클릭합니다.
  
비즈니스용 Skype 서버 제어판을 사용하여 수행할 수 있는 다양한 절차에 대한 자세한 내용은 [Manage 비즈니스용 Skype 서버 2015를 참조합니다.](../../manage/manage.md)
