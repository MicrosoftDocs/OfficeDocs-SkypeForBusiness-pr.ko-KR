# <a name="hybrid-manageability-wizard"></a>하이브리드 관리성 마법사

하이브리드 관리 마법사를 사용 하 여 비즈니스용 Skype Online으로 하이브리드 배포를 설정 하는 데 필요한 몇 가지 구성 작업을 수행할 수 있습니다. 하이브리드 배포를 구성 하는 방법에 대 한 자세한 내용은 [하이브리드 관리성 마법사를 사용 하 여 환경 구성을](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)참조 하세요.
