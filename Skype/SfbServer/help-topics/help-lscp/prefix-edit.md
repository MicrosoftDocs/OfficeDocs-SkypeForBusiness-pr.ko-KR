# <a name="prefix-edit"></a>접두사 편집
 
접두사나 시작 숫자를 정의하려면 이 경로에서 처리해야 하는 번호 접두사를 추가합니다(필요한 경우 + 포함). 예를 들어 +123으로 시작하는 모든 번호를 일치시킬 수 있습니다.
  
비즈니스용 Skype 서버 제어판을 사용하여 수행할 수 있는 다양한 절차에 대한 자세한 내용은 [Manage 비즈니스용 Skype 서버 2015를 참조합니다.](../../manage/manage.md)
