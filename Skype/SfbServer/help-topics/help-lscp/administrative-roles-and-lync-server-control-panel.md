# <a name="administrative-roles-and-lync-server-control-panel"></a>관리 역할 및 Lync Server 제어판

보안에 대한 높은 표준을 유지하면서 관리 작업을 위임할 수 있도록 비즈니스용 Skype 서버 RBAC(역할 기반 액세스 제어)를 지원합니다. RBAC에서는 미리 정의된 관리 역할에 사용자를 할당하여 관리 권한을 부여합니다. 비즈니스용 Skype 서버 다양한 기본 제공 관리 역할 집합을 포함하며, 새 역할을 만들고 각 새 역할에 대한 사용자 지정 cmdlet 목록을 지정할 수도 있습니다. 또한 미리 정의된 RBAC 역할 및 사용자 지정 RBAC 역할 둘 다에 대해 허용되는 작업에 cmdlet 스크립트를 추가할 수도 있습니다. 역할 기반 액세스 제어에서 사용할 수 있는 미리 비즈니스용 Skype 서버 관리 역할을 비롯한 자세한 [내용은 Role-Based Access Control을 참조합니다.](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-role-based-access-control)
