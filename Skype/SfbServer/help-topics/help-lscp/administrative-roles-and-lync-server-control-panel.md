# <a name="administrative-roles-and-lync-server-control-panel"></a>관리 역할 및 Lync Server 제어판

보안을 위해 높은 표준을 유지 하면서 관리 작업을 위임할 수 있도록 비즈니스용 Skype Server는 RBAC (역할 기반 액세스 제어)를 지원 합니다. RBAC에서는 미리 정의된 관리 역할에 사용자를 할당하여 관리 권한을 부여합니다. 비즈니스용 Skype Server에는 다양 한 기본 제공 관리 역할이 포함 되어 있으며, 새 역할을 만들고 새 역할에 대 한 사용자 지정 cmdlet 목록을 지정할 수도 있습니다. 또한 미리 정의된 RBAC 역할 및 사용자 지정 RBAC 역할 둘 다에 대해 허용되는 작업에 cmdlet 스크립트를 추가할 수도 있습니다. 비즈니스용 Skype 서버에서 사용할 수 있는 미리 정의 된 관리 역할을 비롯 한 자세한 내용은 [역할 기반 액세스 제어](https://technet.microsoft.com/library/41204ba3-ce5b-41a8-a6c3-b444468fa328.aspx)를 참조 하세요.
