# <a name="select-normalization-rules"></a>정규화 규칙 선택
 
정규화 규칙은 .NET Framework 정규식을 사용하여 전화를 거는 번호를 E.164 형식으로 변환합니다. 각 다이얼 플랜에는 하나 이상의 정규화 규칙을 할당해야 합니다.
  
기존 정규화 규칙을 사용하려면 목록에서 규칙을 클릭한 다음 **확인**을 클릭합니다.
  
비즈니스용 Skype 서버 제어판을 사용 하 여 수행할 수 있는 다양 한 절차에 대 한 자세한 내용은 비즈니스용 [Skype 서버 2015 관리](../../manage/manage.md)를 참조 하세요.
