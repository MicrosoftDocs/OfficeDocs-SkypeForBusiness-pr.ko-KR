# <a name="users-view-persistent-chat"></a>사용자: 영구 채팅 보기

**CsPersistentChatAdministrator** 역할(영구 채팅 관리자)의 구성원은 채팅방 관리자 권한 및 사용 권한을 다른 사용자와 자신에게 부여할 수 있습니다.

영구 채팅 관리자는 다음을 할 수 있습니다.

- 방 범주 및 채팅방 만들기

- 모든 범주 및 채팅방의 구성원 자격 설정

- 모든 범주와 채팅방의 모든 설정 관리

- 영구 채팅 서버에 대한 정책을 사용하도록 설정하거나 사용하지 않도록 설정

- 영구 채팅 서버 풀에서 구성 설정을 설정하고 관리합니다.

자세한 내용은 배포 설명서에서 [Adding a Persistent Chat Administrator](/previous-versions/office/lync-server-2013/lync-server-2013-adding-a-persistent-chat-administrator)를 참조하십시오.

## <a name="tasks-that-you-can-perform"></a>수행할 수 있는 작업

**사용자: 영구 채팅 보기** 페이지에서는 다음 작업을 수행할 수 있습니다.

- [사용자가 범주, 대화방, 사용자 권한 및 사용 권한을 관리할 수 있도록 설정](/previous-versions/office/lync-server-2013/lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions)

- [영구 채팅 구성원 이해](/previous-versions/office/lync-server-2013/understanding-persistent-chat-membership)

비즈니스용 Skype 서버 제어판을 사용하여 수행할 수 있는 다양한 절차에 대한 자세한 내용은 [Manage 비즈니스용 Skype 서버 2015를 참조합니다](../../manage/manage.md).
