# <a name="select-contacts-from-active-directory"></a>Active Directory에서 연락처 선택
 
Active **Directory에서 선택** 대화 상자를 사용하여 비즈니스용 Skype를 사용하도록 설정할 연락처를 찾을 수 있습니다. 검색 상자에 찾을 연락처의 SIP 주소나 표시 이름을 입력합니다. 사용 가능한 모든 연락처를 검색하려면 검색 상자에 아무런 내용을 입력하지 마십시오.
  
비즈니스용 Skype 서버 제어판을 사용하여 수행할 수 있는 다양한 절차에 대한 자세한 내용은 비즈니스용 [Skype 서버 2015를 참조하세요.](../../manage/manage.md)
