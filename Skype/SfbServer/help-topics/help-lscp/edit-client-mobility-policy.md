# <a name="edit-client-mobility-policy"></a>클라이언트 이동성 정책 편집

**모바일 정책 편집** 대화 상자에서 모바일 정책에 대 한 세부 정보를 편집할 수 있습니다.

## <a name="ui-reference"></a>UI 참조

다음 목록에서는 페이지의 메뉴, 명령, 필드 및 속성에 대해 설명합니다.


- **Commit** 정책 설정에 대 한 변경 내용을 저장 하려면 클릭 합니다.

- **취소** 정책 편집을 중지 하 고 정책 설정에 대 한 변경 내용을 취소 하려면 클릭 합니다.

- **이름** 이동성 정책의 이름입니다.

- **설명** 이동성 정책에 대 한 설명입니다.

모바일 정책에 대 한 자세한 내용은 [모바일 정책을](https://technet.microsoft.com/library/8caa5525-e16a-4e38-b3cd-acc0ae9ea375.aspx)참조 하세요.
