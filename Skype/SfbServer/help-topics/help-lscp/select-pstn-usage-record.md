# <a name="select-pstn-usage-record"></a>PSTN 사용 기록 선택
 
**Pstn 사용 레코드 선택** 대화 상자를 사용 하 여 사용할 수 있는 기존 PSTN (공개 전화 통신망) 사용 레코드를 찾습니다.
  
> [!NOTE]
> 사용 가능한 모든 PSTN 사용 레코드를 검색 하려면 검색 상자에 아무것도 입력 하지 않은 다음 검색 아이콘을 클릭 합니다. 
  
비즈니스용 Skype 서버 제어판을 사용 하 여 수행할 수 있는 다양 한 절차에 대 한 자세한 내용은 비즈니스용 [Skype 서버 2015 관리](../../manage/manage.md)를 참조 하세요.
