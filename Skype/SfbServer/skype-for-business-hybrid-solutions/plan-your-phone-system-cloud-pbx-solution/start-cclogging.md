# <a name="start-cclogging"></a>Start-CcLogging
 
이 Start-CcLogging cmdlet은 비즈니스용 Skype 클라우드 커넥터 버전 어플라이언스에 대한 수신 및 발신 통화 로그를 생성합니다. 
  
```powershell
Start-CcLogging
```

## <a name="parameters"></a>매개 변수

없음
  
## <a name="examples"></a>예제
<a name="Examples"> </a>

### <a name="example-1"></a>예 1

다음 예에서는 수신 및 발신 전화 로그를 생성합니다.
  
```powershell
Start-CcLogging
```

## <a name="detailed-description"></a>자세한 정보
<a name="DetailedDescription"> </a>

이 Start-CcLogging cmdlet을 사용하면 관리자가 Cloud Connector 어플라이언스에서 수신 및 발신 전화 로깅을 시작할 수 있습니다. 기본적으로 로깅은 4시간 후에 자동으로 중지됩니다.
  
## <a name="input-types"></a>입력 형식
<a name="InputTypes"> </a>

없음 이 Start-CcLogging cmdlet은 파이프라인된 입력을 허용하지 않습니다.
  
## <a name="return-types"></a>반환 형식
<a name="ReturnTypes"> </a>

없음
