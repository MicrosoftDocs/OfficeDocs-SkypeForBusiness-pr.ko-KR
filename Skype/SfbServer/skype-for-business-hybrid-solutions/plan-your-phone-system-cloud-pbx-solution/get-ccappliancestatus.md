# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
CcApplianceStatus cmdlet은 비즈니스용 Skype 클라우드 커넥터 에디션 기기에 대 한 진단 정보를 제공 합니다.
  
이 cmdlet은 비즈니스용 Skype 클라우드 커넥터 에디션 2.1에 적용 됩니다.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>자세한 정보

클라우드 커넥터 호스트 컴퓨터에서 실행 되는 경우 CcApplianceStatus cmdlet은 비즈니스용 Skype 클라우드 커넥터 기기에 대 한 진단 정보를 제공 합니다.
  
## <a name="parameters"></a>매개 변수

없음
  
## <a name="input-types"></a>입력 형식

없음. Get-CcApplianceStatus cmdlet은 파이프라인 입력을 허용 하지 않습니다.
  
## <a name="return-types"></a>반환 형식

명령이 기기의 상태를 반환 합니다.
