# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
이 Get-CcApplianceStatus cmdlet은 비즈니스용 Skype 클라우드 커넥터 버전 어플라이언스에 대한 진단 비즈니스용 Skype 클라우드 커넥터 버전 제공합니다.
  
이 cmdlet은 비즈니스용 Skype 클라우드 커넥터 버전 2.1에 적용됩니다.
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>자세한 정보

이 Get-CcApplianceStatus cmdlet은 클라우드 커넥터 호스트 컴퓨터의 클라우드 커넥터 어플라이언스에 대한 진단 비즈니스용 Skype 제공합니다.
  
## <a name="parameters"></a>매개 변수

없음
  
## <a name="input-types"></a>입력 형식

없음 이 Get-CcApplianceStatus cmdlet은 파이프라인된 입력을 허용하지 않습니다.
  
## <a name="return-types"></a>반환 형식

이 명령은 어플라이언스 상태를 반환합니다.
