# <a name="validate-your-cloud-connector-deployment"></a>클라우드 커넥터 배포 확인

> [!Important]
> Cloud Connector Edition은 온라인과 함께 2021년 7월 31일 비즈니스용 Skype 있습니다. 조직에서 사용자 Teams 직접 라우팅을 사용하여 Teams 방법을 [확인합니다.](/MicrosoftTeams/direct-routing-landing-page)
 
배포 및 구성 단계를 완료한 후 비즈니스용 Skype 클라우드 커넥터 버전 전화를 걸고 받을 수 있는지 확인해야 합니다.
  
## <a name="validate-the-deployment"></a>배포 유효성 검사

1. 방금 설정한 비즈니스용 Skype 계정으로 비즈니스용 Skype 클라이언트에 로그인합니다.
    
2. 클라이언트에서 PSTN 번호로 전화를 걸고 통화가 성공하는지 확인할 수 있습니다.
    
3. 전화 또는 PSTN 번호와 연결된 다른 장치를 사용하여 비즈니스용 Skype 클라이언트에 전화를 걸고 통화가 성공하는지 확인합니다.
    
    > [!NOTE]
    > 사이트를 추가하거나 사이트를 변환하여 HA 지원을 사용하도록 설정한 후에 호출을 성공적으로 할 수 있도록 해야 합니다.
