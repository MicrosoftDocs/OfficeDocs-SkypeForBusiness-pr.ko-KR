# <a name="validate-your-cloud-connector-deployment"></a>클라우드 커넥터 배포 확인

> [!Important]
> Cloud Connector Edition은 비즈니스용 Skype Online과 함께 2021년 7월 31일부터 사용이 중지됩니다. 조직이 Teams로 업그레이드한 후 직접 라우팅을 사용하여 프레미스 전화 통신 네트워크를 Teams에 연결하는 [방법을 배워야 합니다.](/MicrosoftTeams/direct-routing-landing-page)
 
비즈니스용 Skype 클라우드 커넥터 버전 배포 및 구성 단계를 완료한 후 전화를 걸고 받을 수 있는지 확인해야 합니다.
  
## <a name="validate-the-deployment"></a>배포 유효성 검사

1. 방금 설정한 사용자 계정으로 비즈니스용 Skype 클라이언트에 로그인합니다.
    
2. 클라이언트에서 PSTN 번호로 전화를 걸고 통화가 성공하는지 확인할 수 있습니다.
    
3. 전화 또는 PSTN 번호와 연결된 다른 장치를 사용하여 비즈니스용 Skype 클라이언트에 전화를 걸고 통화가 성공하는지 확인합니다.
    
    > [!NOTE]
    > 사이트를 추가하거나 사이트를 변환하여 HA 지원을 사용하도록 설정한 후에 호출을 성공적으로 할 수 있도록 해야 합니다.
