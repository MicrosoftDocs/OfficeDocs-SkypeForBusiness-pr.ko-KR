# <a name="validate-your-cloud-connector-deployment"></a>클라우드 커넥터 배포 확인
 
비즈니스용 Skype 클라우드 커넥터 에디션을 배포 하 고 구성 하는 단계를 완료 한 후 전화를 걸고 받을 수 있는지 확인 합니다.
  
## <a name="validate-the-deployment"></a>배포 유효성 검사

1. 방금 설정한 사용자 계정을 사용 하 여 비즈니스용 Skype 클라이언트에 로그인 합니다.
    
2. 클라이언트에서 PSTN 번호로 전화를 걸고 호출이 성공 하는지 확인 합니다.
    
3. PSTN 번호와 연결 된 휴대폰 또는 기타 장치를 사용 하 여 비즈니스용 Skype 클라이언트에 전화 하 고 통화 성공 여부를 확인 합니다.
    
    > [!NOTE]
    > 다른 사이트를 추가 하거나 사이트를 변환 하 여 HA 지원을 사용 하도록 설정 하면 성공적으로 전화를 걸 수 있는지 확인 해야 합니다.
