# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>비즈니스용 Skype에 대 한 모바일 클라이언트 기능 비교
 
**요약:** 비즈니스용 Skype 서버를 계획 하는 동안 모바일 클라이언트에 대 한 기능 지원을 검토 합니다.
  
이 문서에서는 비즈니스용 Skype mobile 클라이언트와 비즈니스용 Skype 데스크톱 클라이언트의 기능 및 기능을 다음 범주로 비교 합니다.
  
- 로그인, 푸시 알림 및 일반 기능
    
- 향상 된 현재 상태
    
- 연락처 및 메일 그룹
    
- 인스턴트 메시지 (IM)
    
- 비즈니스용 skype for Business 오디오 및 비디오
    
- 회의
    
- 통신
    
- 외부 사용자
    
- 보관 및 규정 준수
    
-  최신 인증
    
다음 표에는 비즈니스용 Skype 서버의 온-프레미스 배포에서 비즈니스용 Skype 사용자가 사용할 수 있는 기능이 나와 있습니다. 표 각주에 달리 표시 되지 않는 한 비즈니스용 Skype Online 및 Microsoft Office 365 사용자도 동일한 기능을 사용할 수 있습니다.
  
> [!NOTE]
> 최종 사용자를 위한 온라인 도움말 및 리소스는 [비즈니스용 Skype 검색](https://go.microsoft.com/fwlink/p/?LinkId=528686)을 참조 하세요. 
  
> [!NOTE]
> 다른 비즈니스용 Skype 클라이언트에서 사용할 수 있는 기능을 비교 하려면 비즈니스용 [skype에 대 한 데스크톱 클라이언트 기능 비교](desktop-feature-comparison.md)를 참조 하세요. 

> [!NOTE]
> 이전 모바일 클라이언트에 대 한 MCX (Mobility Service) 지원은 더 이상 비즈니스용 Skype 서버 2019에서 사용할 수 없습니다. 현재 모든 비즈니스용 Skype 모바일 클라이언트는 이미 통합 커뮤니케이션 웹 API (인스턴트 메시징 (IM), 현재 상태, 대화 상대 지원)를 사용 합니다. MCX를 사용 하는 레거시 클라이언트가 있는 사용자는 현재 클라이언트로 업그레이드 해야 합니다.
