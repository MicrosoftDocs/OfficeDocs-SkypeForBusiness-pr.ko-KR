# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>비즈니스용 Skype의 모바일 클라이언트 기능 비교
 
**요약:** 비즈니스용 Skype 서버를 계획하는 동안 모바일 클라이언트에 대한 기능 지원을 검토합니다.
  
이 문서에서는 다음 범주의 비즈니스용 Skype 모바일 클라이언트와 비즈니스용 Skype 데스크톱 클라이언트의 기능을 비교합니다.
  
- 로그인, 푸시 알림 및 일반 기능
    
- 현재 상태 사용자 지정
    
- 대화 상대 및 대화 상대 그룹
    
- IM(인스턴트 메시징)
    
- 비즈니스용 Skype에서 비즈니스용 Skype로의 오디오 및 비디오
    
- 전화 회의
    
- 전화 통신
    
- 외부 사용자
    
- 보관 및 준수
    
-  최신 인증
    
다음 표에는 비즈니스용 Skype 서버의 전체 배포에서 비즈니스용 Skype 사용자가 사용할 수 있는 기능이 나열되어 있습니다. 테이블 각주에 다른 표시가 없는 한 비즈니스용 Skype Online 및 Microsoft 365 또는 Office 365 사용자에게도 동일한 기능을 사용할 수 있습니다.
  
> [!NOTE]
> 최종 사용자를 위한 온라인 도움말 및 리소스는 [비즈니스용 Skype 검색을 참조하세요.](https://go.microsoft.com/fwlink/p/?LinkId=528686) 
  
> [!NOTE]
> 다른 비즈니스용 Skype 클라이언트에서 사용할 수 있는 기능을 비교하기 위해 비즈니스용 Skype에 대한 데스크톱 클라이언트 기능 [비교를 참조하세요.](desktop-feature-comparison.md) 

> [!NOTE]
> 레거시 모바일 클라이언트에 대한 MCX(Mobility Service) 지원은 비즈니스용 Skype 서버 2019에서 더 이상 사용할 수 없습니다. 모든 현재 비즈니스용 Skype 모바일 클라이언트는 이미 UCWA(Unified Communications Web API)를 사용하여 IM(인스턴트 메시징), 현재 상태 및 연락처를 지원하고 있습니다. MCX를 사용하는 레거시 클라이언트를 사용하는 사용자는 현재 클라이언트로 업그레이드해야 합니다.
