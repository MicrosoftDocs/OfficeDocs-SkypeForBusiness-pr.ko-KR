# <a name="desktop-client-feature-comparison-for-skype-for-business-server-2015"></a>비즈니스용 Skype 서버 2015 데스크톱 클라이언트 기능 비교

**요약:** 비즈니스용 skype Server 2015 또는 비즈니스용 Skype Online 관리자는 이러한 표를 사용 하 여 어떤 클라이언트가 어떤 기능을 지원 하는지 이해할 수 있습니다.
  
 비즈니스용 Skype를 배포 하거나 업그레이드 하기 전에 조직에서 이미 사용 중인 클라이언트를 확인 합니다. 해당 클라이언트에 대 한 기능 지원 영향을 이해 하려면 아래 표를 사용 하세요. 이를 통해 사용자에 게 변경 내용을 전달 하 고, 롤아웃 프로세스의 속도를 개선 하 고, 최신 클라이언트로 업그레이드 하는 이점을 완전히 이해할 수 있습니다.
  
비즈니스용 skype Online에서 사용할 2015 수 있는 일부 기능을 온라인에서 사용할 수 없습니다. 자세한 내용은 [온라인 또는 하이브리드 사용자 계정 제한 사항을](desktop-feature-comparison.md#Online-Hybrid) 참조 하세요. 비즈니스용 skype Online 관리자가 사용할 수 있는 다양 한 요금제에 대 한 정보에 대 한 [비즈니스용 Skype Online 서비스 설명을](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) 참조 하 고 싶을 수 있습니다.

비즈니스용 skype Server 2019의 클라이언트 지원에 대 한 [비즈니스용 skype 2019의 데스크톱 클라이언트 기능 비교](../../../SfBServer2019/plan/feature-comparison.md) 를 참조 하세요.
  
다음 표에서는 비즈니스용 Skype Server 2015 또는 비즈니스용 Skype Online에서 작동 하는 각 클라이언트에서 사용할 수 있는 기능을 보여 줍니다. 스마트 전화와 태블릿 클라이언트 기능 비교를 위해 [비즈니스용 Skype에 대 한 모바일 클라이언트 기능 비교](mobile-feature-comparison.md) 를 참조 하 고 싶을 수도 있습니다. 조직 구입에 대 한 클라이언트 액세스 라이선스 또는 사용자 구독 라이선스는 사용자가 사용할 수 있는 기능에도 영향을 줍니다. 전체 또는 기본 클라이언트를 사용자에 게 배포할지 여부는 조직이 구매할 때 선택한 라이선스 또는 계획에 따라 달라 집니다. 자세한 정보는 [라이선스 가이드](https://products.office.com/en-us/skype-for-business/it-pros) 를 참조 하세요.
  
> [!IMPORTANT]
> 비즈니스용 skype 서버 2015 및 비즈니스용 Skype Online은 이전에 출시 된 다음 클라이언트: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition, Lync 2010 전화 교환 기능을 지원 합니다. 다른 서버와 함께 사용 하는 경우 이러한 클라이언트에 대 한 자세한 내용은 lync server [2013의 클라이언트 비교 표](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.15%29.aspx) 를 참조 하 고 [lync server 2010에 대 한 클라이언트 비교 표에서](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.14%29.aspx)

> [!NOTE]
> **Lync 2010 수행자** 클라이언트는 비즈니스용 Skype Online에서 지원 되지 않습니다.

> [!NOTE]
> 비즈니스용 Skype Web App 브라우저 클라이언트 및 Skype 모임 앱 Windows 10 앱은 [모임 지원도](desktop-feature-comparison.md#BKMK_Conferencing)제공 합니다. 이러한 클라이언트에 대 한 자세한 내용은 [모임 클라이언트 (웹 앱 및 모임 앱) 계획](meetings-clients.md) 을 참조 하세요.

## <a name="client-limitations"></a>클라이언트 제한 사항

<a name="Types"> </a>

### <a name="basic-client-limitations"></a>기본 클라이언트 제한 사항

<a name="Full-Basic"> </a>

아래 기능은 전체 클라이언트를 사용 하 여 사용할 수 있으며 기본 클라이언트에서는 사용할 수 없습니다.

- 팀 통화 설정 관리
- 대리인 관리
- 대리인으로 구성 된 경우 다른 사람의 통화 처리
- 대량 통화 관리
- 응답 그룹에 대 한 통화 시작
- 통화 공원
- 인사말 변경
- 그룹 통화 픽업
- 사용자 상태가 UM을 사용 하지 않도록 설정 되어 있고 레거시 Outlook 클라이언트 (2013 또는 이전 버전)를 사용 하는 경우 부재 중 전화 알림 전자 메일이 생성 되지 않습니다.

### <a name="online-or-hybrid-user-account-limitations"></a>온라인 또는 하이브리드 사용자 계정 제한 사항

<a name="Online-Hybrid"> </a>

사용자 계정은 온라인 또는 온-프레미스로 존재할 수 있으며 해당 사용자가 사용할 수 있는 기능에 영향을 줍니다. 비즈니스용 Skype Online에 계정이 있는 사용자는 전체 클라이언트를 사용 하는 경우에도 다음 기능에 액세스할 수 없습니다.
  
- 향상 된 현재 상태: 모든 공용 사이트의 사진을 사진으로 사용
- 연락처: 응답 그룹 검색
- 메신저 대화 지원: 영구 채팅 (그룹 채팅) 통합
- IM 지원: 한 번의 클릭으로 지속적인 채팅방을 비즈니스용 Skype 모임으로 확대
- 외부 사용자: 외부 사용자와 함께 2-파티 또는 단체 통화를 실시 합니다.
