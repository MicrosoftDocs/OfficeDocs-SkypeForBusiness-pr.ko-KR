# <a name="plan-for-clients-and-devices"></a>클라이언트 및 장치 계획

**요약:** 지원되는 클라이언트 및 비즈니스용 Skype 앱을 검토합니다.

오늘날의 인력은 계속해서 이동하고 있습니다. 직원들은 회사 사무실, 지역 위치, 집 사무실 또는 출장 중 등에서 근무하는지 여부에 따라 의사소통하고 공동 작업을 해야 합니다. 비즈니스용 Skype 서버는 조직의 사용자에게 배포할 수 있는 클라이언트 인터페이스 모음을 통해 이러한 요구 사항을 지원합니다. 신중하게 계획을 세우면 직원들이 필요한 것을 얻을 수 있으며 비즈니스용 Skype를 어디서든 사용할 수 있습니다.

## <a name="available-clients"></a>사용 가능한 클라이언트

비즈니스용 Skype 서버는 컴퓨터 설치 클라이언트 소프트웨어, 웹 기반 클라이언트 및 모바일 장치용 클라이언트를 포함하여 여러 유형의 클라이언트를 지원합니다. 지원되는 모든 클라이언트에 대한 자세한 목록은 비즈니스용 Skype 서버 [2015에](desktop-feature-comparison.md) 대한 데스크톱 클라이언트 기능 비교 또는 비즈니스용 [Skype 서버 2019에](../../../SfBServer2019/plan/feature-comparison.md)대한 데스크톱 클라이언트 기능 비교를 참조하세요. 이전에 Lync 클라이언트 조합을 사용한 경우 비즈니스용 Skype 서버 2019와는 다른 지원되지 않는 레거시 클라이언트가 있습니다. [](clients-and-devices.md#Legacy) 업데이트가 정기적으로 진행됩니다. 따라서 이 항목에서 최신 클라이언트 정보를 주기적으로 확인합니다.

### <a name="skype-for-business-2019"></a>비즈니스용 Skype(2019)

비즈니스용 Skype(2019)는 비즈니스용 Skype 서버 2015 및 2019에 대해 권장되는 완전한 기능을 제공합니다. 새 기능에 대한 설명은 [비즈니스용 Skype의](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 최신 업데이트 팔로우를 참조하세요. 클라이언트 기능 지원은 비즈니스용 [Skype에](desktop-feature-comparison.md)대한 데스크톱 클라이언트 기능 비교에서 자세히 설명하고 사용자 설명서는 비즈니스용 Skype 도움말 [에 있습니다.](https://support.office.com/Skype-for-business) 이 클라이언트는 사용자가 Microsoft 365 또는 Office 365를 설치할 때 포함됩니다.

더 적은 기능을 지원하는 무료 기본 클라이언트도 사용할 수 있습니다. 두 버전은 모든 장치에서 비즈니스용 [Skype 다운로드에서 다운로드할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 전체 클라이언트와 기본 클라이언트의 차이점은 기본 클라이언트 제한 [섹션에 설명되어](desktop-feature-comparison.md#Full-Basic) 있습니다.

### <a name="skype-for-business-2016"></a>비즈니스용 Skype 2016

비즈니스용 Skype 2016은 비즈니스용 Skype 서버 2015 또는 2019에 대한 완전한 기능을 하는 클라이언트입니다. 새로운 기능에 대한 설명은 [비즈니스용 Skype 2016의](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 새로운 기능을 참조하세요. 클라이언트 기능 지원은 비즈니스용 [Skype에](desktop-feature-comparison.md)대한 데스크톱 클라이언트 기능 비교에서 자세히 설명하고 사용자 설명서는 비즈니스용 Skype 도움말 [에 있습니다.](https://support.office.com/Skype-for-business) 이 클라이언트는 사용자가 Office 365를 설치할 때 포함됩니다.

더 적은 기능을 지원하는 무료 기본 클라이언트도 사용할 수 있습니다. 두 버전은 모든 장치에서 비즈니스용 [Skype 다운로드에서 다운로드할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 전체 클라이언트와 기본 클라이언트의 차이점은 기본 클라이언트 제한 [섹션에 설명되어](desktop-feature-comparison.md#Full-Basic) 있습니다.

### <a name="skype-for-business-2015"></a>비즈니스용 Skype 2015

비즈니스용 Skype 2015는 비즈니스용 Skype 서버 2015 또는 2019에 대한 완전한 기능을 하는 클라이언트입니다. 비즈니스용 Skype 사용자 인터페이스는 완전히 다시 디자인되어 통화 모니터, Skype 디렉터리 통합, 이모티콘 등의 새로운 통합 기능을 포함합니다. 변경 내용에 대한 요약은 Lync가 이제 비즈니스용 [Skype임 - 새로운 을 참조하세요.](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612) 클라이언트 기능 지원은 비즈니스용 [Skype에](desktop-feature-comparison.md)대한 데스크톱 클라이언트 기능 비교에서 자세히 설명하고 사용자 설명서는 비즈니스용 Skype 도움말 [에 있습니다.](https://support.office.com/Skype-for-business) 이 클라이언트는 사용자가 Office 365를 설치할 때 포함됩니다.

### <a name="skype-for-business-on-mac"></a>Mac의 비즈니스용 Skype

Mac의 [비즈니스용 Skype 클라이언트를](https://www.microsoft.com/download/details.aspx?id=54108) 다운로드할 수 있습니다. Mac의 [비즈니스용 Skype 클라이언트 요구 사항을](mac-requirements.md) 참조하여 선행 조건 검토

### <a name="skype-for-business-for-mobile-devices"></a>모바일 장치용 비즈니스용 Skype

클라이언트는 Windows Phone, iPhone/iPad 및 Android에서 사용할 수 있습니다. 사용자는 모든 장치에서 비즈니스용 Skype 다운로드 [에서 다운로드할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 이러한 클라이언트에 대한 기능 지원은 비즈니스용 Skype의 모바일 [클라이언트 기능 비교에 자세히 설명됩니다.](mobile-feature-comparison.md)

> [!NOTE]
> 레거시 모바일 클라이언트에 대한 MCX(Mobility Service) 지원은 비즈니스용 Skype 서버 2019에서 더 이상 사용할 수 없습니다. 모든 현재 비즈니스용 Skype 모바일 클라이언트는 이미 UCWA(Unified Communications Web API)를 사용하여 IM(인스턴트 메시징), 현재 상태 및 연락처를 지원하고 있습니다. MCX를 사용하는 레거시 클라이언트를 사용하는 사용자는 현재 클라이언트로 업그레이드해야 합니다.

### <a name="online-meeting-add-in-for-skype-for-business"></a>비즈니스용 Skype용 온라인 모임 추가 기능

비즈니스용 Skype용 온라인 모임 추가 기능에서는 Windows의 Microsoft Outlook 메시징 및 공동 작업 클라이언트 내에서 모임 관리를 지원할 수 있습니다. 비즈니스용 Skype용 온라인 모임 추가 기능은 비즈니스용 Skype와 함께 자동으로 설치됩니다.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>비즈니스용 Skype Web App 및 Skype 모임 앱

비즈니스용 Skype가 사용자 컴퓨터에 설치되어 있지 않은 경우 사용자가 Windows 컴퓨터의 모임 요청에서 모임 링크를 클릭하면 Skype 모임 앱 또는 비즈니스용 Skype Web App이 설치 및 열립니다.  Skype 모임 앱은 조직 외부의 참가자가 선택할 수 있는 클라이언트입니다. (Mac에서는 Mac의 비즈니스용 Skype가 설치 및 열립니다.) 이러한 클라이언트를 사용하기 위한 요구 사항은 [Plan for Meetings clients (Web App and Meetings](meetings-clients.md) App)을 참조하세요.


### <a name="skype-for-business-web-scheduler"></a>비즈니스용 Skype 웹 스케줄러

[비즈니스용 Skype 웹](https://sched.lync.com) 스케줄러는 Microsoft Outlook에 액세스할 수 없는 비즈니스용 Skype Online 사용자 또는 Windows를 기반으로 하지 않는 운영 체제에 있는 사용자를 위한 웹 기반 모임 예약 및 관리 도구입니다. 비즈니스용 Skype 웹 스케줄러를 사용하여 사용자는 기본 설정 전자 메일 프로그램을 사용하여 새 모임을 만들고, 기존 모임을 수정하고, 초대를 보낼 수 있습니다. 비즈니스용 Skype 웹 스케줄러 [설명서에서는](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US)  자세한 정보를 제공합니다.

### <a name="vdi-plugins"></a>VDI 플러그 인

보안 및 규정 준수 문제가 특히 중요한 일부 조직에서는 VDI(가상 데스크톱 인프라) 환경이 사용됩니다. 가상 데스크톱에 있는 클라이언트에서 오디오 및 비디오 처리를 많이 요구하는 경우와 같은 연결에서 전체 오디오 및 비디오와 함께 비즈니스용 Skype를 사용하세요. 최종 사용자의 로컬 컴퓨터로 처리를 오프로드하고 가상 데스크톱의 부하를 줄이는 추가 VDI 플러그 인 소프트웨어를 사용할 수 있습니다. 이러한 플러그 인 사용에 대한 자세한 [내용은 VDI](vdi-environments.md) 환경에서 비즈니스용 Skype 계획을 참조하세요.

### <a name="microsoft-teams-rooms"></a>Microsoft Teams 룸

Microsoft Teams 룸은 친숙한 인터페이스를 사용하는 Microsoft의 최신 회의 솔루션으로, 쉽게 배포 및 관리하며, 쉽게 설치할 수 있도록, 더 쉽게 설치할 수 있도록 더하기 위해 이미 준비된 EQUIPMENT 패널과 같은 기존 장비를 활용합니다. Microsoft Teams 룸은 콘솔 모드에서 Surface Pro 4 또는 Surface Pro에서 실행되는 용도의 UWP 앱을 사용하며(UWP 앱을 배포한 후 디바이스에서 실행되는 유일한 앱) 구현 시 자체 디바이스 계정이 필요합니다. 소프트웨어는 Windows 스토어와 Windows 업데이트를 통해 업데이트됩니다. 배포에서 이러한 방 콘솔 사용에 대한 자세한 https://aka.ms/MTRDocs 내용은 을 참조합니다. 

### <a name="skype-for-business-on-surface-hub"></a>Surface Hub의 비즈니스용 Skype

Microsoft Surface Hub는 브레인스토Ming, 공동 작업 및 프레젠테이션을 위한 올인원 생산성 장치입니다. Microsoft Surface Hub 관리 가이드에 설명된 비즈니스용 Skype 클라이언트의 자체적으로 이 [클라이언트를 실행합니다.](/surface-hub/)

## <a name="choosing-your-organizations-preferred-client"></a>조직의 기본 설정 클라이언트 선택
<a name="BK_client_choose"> </a>

조직에서 적절한 라이선스를 구입한 경우 전체 클라이언트를 선택하고, 그렇지 않으면 기본 클라이언트를 선택하십시오.

사용자는 모든 장치에서 비즈니스용 Skype 다운로드에서 클라이언트를 [직접 설치할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 클라이언트는 사용자가 Windows에 Microsoft 365 또는 Office 365를 설치할 때도 설치됩니다. 일부 사용자에게 Mac이 있는 경우 이전 섹션에서 설명한 대로 해당 사용자에게는 다른 기능 집합이 있습니다.

비즈니스용 Skype 서버 2015에서 사용할 수 있는 일부 기능은 비즈니스용 Skype Online 또는 비즈니스용 Skype 서버 2019에서 사용할 수 없습니다. 자세한 내용은 [2015에](desktop-feature-comparison.md#Online-Hybrid) 대한 온라인 또는 하이브리드 사용자 계정 제한 또는 [2019에](desktop-feature-comparison.md#Online-Hybrid) 대한 하이브리드 사용자 계정 제한을 참조하세요. 비즈니스용 Skype Online 관리자는 사용할 수 있는 다양한 요금제에 대한 자세한 내용은 [비즈니스용 Skype Online](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description) 서비스 설명을 참조할 수 있습니다.

 비즈니스용 Skype를 배포하거나 업그레이드하기 전에 조직에서 이미 사용 중이던 클라이언트를 확인하세요. 비즈니스용 [Skype에](desktop-feature-comparison.md) 대한 데스크톱 클라이언트 기능 비교를 사용하여 해당 클라이언트에 대한 기능 지원 영향을 이해합니다. 이를 통해 사용자에게 변경 내용을 전달하고, 롤아웃 프로세스의 속도를 향상하고, 최신 클라이언트로 업그레이드할 때의 이점을 완전히 이해할 수 있습니다.

## <a name="ways-to-deploy-the-client-to-your-users"></a>사용자에게 클라이언트를 배포하는 방법
<a name="BK_User_Deploy"> </a>

클라이언트 설치 관리자는 MSI 및 Click-to-run 형식 설치 관리자 모두에 사용할 수 있습니다. 비즈니스용 Skype 클라이언트 유지 전략은 선택에 영향을 줄 수 있으므로 다음 지점을 이해해야 합니다.

- 일반적으로 비즈니스용 Skype는 이전에 릴리스된 클라이언트에 새 기능을 추가하지 않습니다.

- 일반적으로 비즈니스용 Skype는 초기 릴리스 이후 비즈니스용 Skype MSI의 새 기능을 배송할 계획이 없습니다. 릴리스 간의 MSI 개선은 기본적으로 품질/보안입니다.

- 비즈니스용 Skype 클라이언트 환경의 최신 및 가장 좋은 환경은 비즈니스용 Skype 2019 클릭과 실행 설치 관리자에서 찾을 수 있습니다.

비즈니스용 Skype 서버에서 Windows 클라이언트 설치 사용자 지정에 설명된 대로 클라이언트의 사용자 지정 [배포를 할 수 있습니다.](../../deploy/deploy-clients/customize-windows-client-installation.md) 설치 방법은 비즈니스용 Skype 서버용 클라이언트 배포에서 [자세히 설명되어 있습니다.](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Click-to-run

Click-to-run은 비즈니스용 Skype를 포함하여 Office 제품을 설치하고 업데이트하는 데 사용할 수 있는 Microsoft 스트리밍 및 가상화 기술입니다. 이러한 스트리밍 및 가상화 기능은 Microsoft App-V(Application Virtualization)의 기술을 기반으로 합니다. Click-to-run에는 다음과 같은 이점이 있습니다.

- 짧은 설치 시간으로 이어지며 Office 제품군의 스트리밍 설치

- Slipstreamed 업데이트 및 패치

- 디스크 공간을 더 적게 차지

- 사용자 기반 라이선싱: 사용자당 5회 설치

- 독립 프로그램 설치를 위한 XML 편집기를 통해 사용자 지정 가능

이 유형의 설치 관리자에 [Office 배포](https://www.microsoft.com/download/details.aspx?id=49117) 도구를 사용할 수 있습니다.

기본 및 전체 클라이언트 버전(32비트 및 64비트 버전 선택)은 모두 클릭 실행 설치 관리자를 사용하여 사용할 수 있으며 사용자는 모든 장치에서 비즈니스용 Skype 다운로드에서 다운로드할 [수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3)

### <a name="msi"></a>MSI

MSI는 2015 및 2016 클라이언트에서 비즈니스용 Skype 기존의 설치 방법입니다. 업데이트 및 패치를 수동으로 설치하고, 볼륨 라이선싱 및 정품 인증을 사용할 수 있으며, 사용자 지정 도구를 통해 사용자 Office [있습니다.](https://www.microsoft.com/download/details.aspx?id=49030) 그룹 정책을 적용하거나, 그룹 정책을 적용하거나, Microsoft Endpoint Configuration Manager 도구를 사용하여 클라이언트를 배포할 수 있습니다.



## <a name="legacy-clients"></a>레거시 클라이언트

비즈니스용 Skype 서버 2015는 이전에 릴리스된 Lync 2013, Lync 2010, Lync 2010 Mobile, Lync 전화 Edition 및 Lync 2010 Attendant 클라이언트를 지원합니다. 다른 서버와 함께 사용할 때 이러한 클라이언트에 대한 자세한 내용은 [Lync Server 2013의](/previous-versions/office/lync-server-2013/lync-server-2013-desktop-client-comparison-tables) 클라이언트 비교 표 및 [Lync Server 2010용](/previous-versions/office/skype-server-2010/gg425836(v=ocs.14))클라이언트 비교 표를 참조하십시오.
