# <a name="plan-for-clients-and-devices"></a>클라이언트 및 장치 계획

**요약:** 지원되는 클라이언트 및 앱에 대한 비즈니스용 Skype.

오늘날의 인력은 계속해서 이동하고 있습니다. 직원들은 회사 사무실, 지역 위치, 집 사무실 또는 출장 중 등에서 근무하는지 여부에 따라 의사소통하고 공동 작업을 해야 합니다. 비즈니스용 Skype 서버 조직의 사용자에게 배포할 수 있는 클라이언트 인터페이스 모음을 통해 이러한 요구 사항을 지원할 수 있습니다. 신중하게 계획을 세우면 직원들이 필요한 것을 얻을 수 있으며비즈니스용 Skype 어디서든 사용할 수 있습니다.

## <a name="available-clients"></a>사용 가능한 클라이언트

비즈니스용 Skype 서버 설치된 클라이언트 소프트웨어, 웹 기반 클라이언트 및 모바일 장치용 클라이언트를 포함하여 여러 유형의 클라이언트를 지원합니다. 지원되는 모든 클라이언트에 대한 자세한 목록은 비즈니스용 Skype 서버 [2015에](desktop-feature-comparison.md) 대한 데스크톱 클라이언트 기능 비교 또는 비즈니스용 Skype 서버 [2019에](../../../SfBServer2019/plan/feature-comparison.md)대한 데스크톱 클라이언트 기능 비교를 참조하세요. 이전에 Lync 클라이언트 조합을 사용한 경우 2019 또는 [](clients-and-devices.md#Legacy) 2019와는 다른 지원되지 않는 레거시 클라이언트가 비즈니스용 Skype 서버 있습니다. 업데이트가 정기적으로 진행됩니다. 따라서 이 항목에서 최신 클라이언트 정보를 주기적으로 확인합니다.

### <a name="skype-for-business-2019"></a>비즈니스용 Skype(2019)

비즈니스용 Skype (2019)는 2015 및 2019의 모든 기능을 비즈니스용 Skype 서버 클라이언트입니다. 새 기능에 대한 설명은 [비즈니스용 Skype](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 최신 업데이트 팔로우를 참조하세요. 클라이언트 기능 지원은 에 대한 데스크톱 클라이언트 기능 비교에서 자세히 [비즈니스용 Skype](desktop-feature-comparison.md)사용자 설명서는 비즈니스용 Skype [있습니다.](https://support.office.com/Skype-for-business) 이 클라이언트는 사용자가 설치 또는 설치를 Microsoft 365 Office 365.

더 적은 기능을 지원하는 무료 기본 클라이언트도 사용할 수 있습니다. 두 버전은 모든 디바이스에서 다운로드할 [비즈니스용 Skype 다운로드할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 전체 클라이언트와 기본 클라이언트의 차이점은 기본 클라이언트 제한 [섹션에 설명되어](desktop-feature-comparison.md#Full-Basic) 있습니다.

더 적은 기능을 지원하는 무료 기본 클라이언트도 사용할 수 있습니다. 두 버전은 모든 디바이스에서 다운로드할 [비즈니스용 Skype 다운로드할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 전체 클라이언트와 기본 클라이언트의 차이점은 기본 클라이언트 제한 [섹션에 설명되어](desktop-feature-comparison.md#Full-Basic) 있습니다.

### <a name="skype-for-business-2015"></a>비즈니스용 Skype 2015

비즈니스용 Skype 2015는 2015 또는 2019의 모든 기능을 비즈니스용 Skype 서버 클라이언트입니다. 비즈니스용 Skype 사용자 인터페이스는 완전히 다시 디자인되어 통화 모니터, Skype 디렉터리 통합, 이모티콘 등의 새로운 통합 기능을 포함합니다. 변경 내용에 대한 요약은 [Lync가](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612)현재 비즈니스용 Skype 새로운 을 참조하세요. 클라이언트 기능 지원은 에 대한 데스크톱 클라이언트 기능 비교에서 자세히 [비즈니스용 Skype](desktop-feature-comparison.md)사용자 설명서는 비즈니스용 Skype [있습니다.](https://support.office.com/Skype-for-business) 이 클라이언트는 사용자가 클라이언트를 설치할 때 Office 365.

## <a name="choosing-your-organizations-preferred-client"></a>조직의 기본 설정 클라이언트 선택
<a name="BK_client_choose"> </a>

조직에서 적절한 라이선스를 구입한 경우 전체 클라이언트를 선택하고, 그렇지 않으면 기본 클라이언트를 선택하십시오.

사용자는 모든 디바이스에서 다운로드 다운로드에서 비즈니스용 Skype [설치할 수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) 클라이언트는 사용자가 클라이언트에 설치하거나 Microsoft 365 Office 365 때 Windows. 일부 사용자에게 Mac이 있는 경우 이전 섹션에서 설명한 대로 해당 사용자에게는 다른 기능 집합이 있습니다.

비즈니스용 Skype 서버 2015에서 사용할 수 있는 일부 기능은 비즈니스용 Skype Online 또는 비즈니스용 Skype 서버 2019에서 사용할 수 없습니다. 자세한 내용은 [2015에](desktop-feature-comparison.md#Online-Hybrid) 대한 온라인 또는 하이브리드 사용자 계정 제한 또는 [2019에](desktop-feature-comparison.md#Online-Hybrid) 대한 온라인 또는 하이브리드 사용자 계정 제한을 참조하세요. 비즈니스용 Skype 온라인 관리자는 온라인 서비스 [](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description) 비즈니스용 Skype 사용 가능한 여러 계획에 대한 정보를 참조할 수 있습니다.

 조직에 배포하거나 업그레이드하기 비즈니스용 Skype 조직에서 이미 사용 중이던 클라이언트를 확인 합니다. 데스크톱 클라이언트 [기능 비교를 사용하여](desktop-feature-comparison.md) 비즈니스용 Skype 지원이 해당 클라이언트에 미치는 영향을 이해합니다. 이를 통해 사용자에게 변경 내용을 전달하고, 롤아웃 프로세스의 속도를 향상하고, 최신 클라이언트로 업그레이드할 때의 이점을 완전히 이해할 수 있습니다.

## <a name="ways-to-deploy-the-client-to-your-users"></a>사용자에게 클라이언트를 배포하는 방법
<a name="BK_User_Deploy"> </a>

클라이언트 설치 관리자는 MSI 및 Click-to-run 형식 설치 관리자 모두에 사용할 수 있습니다. 클라이언트 비즈니스용 Skype 전략은 선택에 영향을 줄 수 있으므로 다음 점에 대해 이해해야 합니다.

- 일반적으로 비즈니스용 Skype 릴리스된 클라이언트에 새 기능을 추가하지 않습니다.

- 일반적으로 비즈니스용 Skype 릴리스 후 비즈니스용 Skype MSI의 새 기능을 배송할 계획이 없습니다. 릴리스 간의 MSI 개선은 기본적으로 품질/보안입니다.

- 클라이언트 환경의 최신 비즈니스용 Skype 가장 좋은 환경은 비즈니스용 Skype 2019 Click-to-run installer에서 찾을 수 있습니다.

에서 클라이언트 설치 사용자 지정에 설명된 대로 클라이언트의 사용자 지정 배포를 Windows [수 비즈니스용 Skype 서버.](../../deploy/deploy-clients/customize-windows-client-installation.md) 설치 방법은 Deploy clients [for 비즈니스용 Skype 서버](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Click-to-run

Click-to-run은 Microsoft 스트리밍 및 가상화 기술로, 이 기술을 포함하여 Office 제품을 설치하고 업데이트하는 비즈니스용 Skype. 이러한 스트리밍 및 가상화 기능은 Microsoft App-V(Application Virtualization)의 기술을 기반으로 합니다. Click-to-run에는 다음과 같은 이점이 있습니다.

- 짧은 설치 시간을 Office 제품군의 스트림 설치

- Slipstreamed 업데이트 및 패치

- 디스크 공간을 더 적게 차지

- 사용자 기반 라이선싱: 사용자당 5회 설치

- 독립 프로그램 설치를 위한 XML 편집기를 통해 사용자 지정 가능

이 유형의 설치 [관리자에](https://www.microsoft.com/download/details.aspx?id=49117) Office 배포 도구를 사용할 수 있습니다.

기본 및 전체 클라이언트 버전(32비트 및 64비트 버전 선택)은 모두 Click-to-run installer를 사용하여 사용할 수 있으며 사용자는 모든 디바이스에서 다운로드 비즈니스용 Skype 다운로드할 [수 있습니다.](https://products.office.com/skype-for-business/download-app?tab=tabs-3)

### <a name="msi"></a>MSI

MSI는 2015 및 2016 클라이언트에서 비즈니스용 Skype 기존의 설치 방법입니다. 업데이트 및 패치를 수동으로 설치하고, 볼륨 라이선싱 및 정품 인증을 사용할 수 있으며, 사용자 지정 도구를 통해 사용자 Office [있습니다.](https://www.microsoft.com/download/details.aspx?id=49030) 그룹 정책을 적용하거나, 그룹 정책을 적용하거나, Microsoft Endpoint Configuration Manager 도구를 사용하여 클라이언트를 배포할 수 있습니다.
