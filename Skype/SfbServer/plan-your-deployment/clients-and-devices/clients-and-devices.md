# <a name="plan-for-clients-and-devices"></a>클라이언트 및 장치 계획

**요약:** 비즈니스용 Skype에 대해 지원 되는 클라이언트 및 앱을 검토 합니다.

오늘날의 직원 들이 끊임없이 이동 하 고 있습니다. 직원 들이 본사, 지역, 가정용 사무실 또는 여행 중 어디서 든 통신 하 고 공동 작업을 수행 해야 합니다. 비즈니스용 Skype 서버는 조직의 사용자에 게 배포할 수 있는 클라이언트 인터페이스 모음을 통해 이러한 요구 사항을 지원 합니다. 확인 계획을 통해 직원 들이 필요한 내용을 얻고 비즈니스용 Skype를 어디에서 든 사용할 수 있습니다.

## <a name="available-clients"></a>사용 가능한 클라이언트

비즈니스용 Skype 서버는 컴퓨터 설치 클라이언트 소프트웨어, 웹 기반 클라이언트 및 모바일 장치용 클라이언트를 비롯 한 여러 종류의 클라이언트를 지원 합니다. 이 섹션에서는 기본 클라이언트를 소개 하 고, 지원 되는 모든 클라이언트에 대 한 자세한 목록은 비즈니스용 Skype Server 2015 또는 [데스크톱 클라이언트 기능 비교 (비즈니스용 Skype server 2019](../../../SfBServer2019/plan/feature-comparison.md))에 대 한 [데스크톱 클라이언트 기능 비교표](desktop-feature-comparison.md) 를 참조 하세요. 이전에 Lync 클라이언트의 조합을 사용한 경우 비즈니스용 Skype 서버 2019와 호환 되지 않는 지원 되지 않는 [레거시 클라이언트가](clients-and-devices.md#Legacy) 있음을 참고 하세요. 업데이트가 정기적으로 수행 되므로 최신 클라이언트 정보에 대해이 항목을 정기적으로 확인 하세요.

### <a name="skype-for-business-2019"></a>비즈니스용 Skype (2019)

비즈니스용 skype (2019)는 비즈니스용 Skype for Business Server 2015 및 2019에서 권장 되는 모든 기능을 갖춘 클라이언트입니다. 새로운 기능에 대 한 설명은 비즈니스용 [Skype의 최신 업데이트 팔 로우](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 를 참조 하세요. 클라이언트 기능 지원은 비즈니스용 [skype에 대 한 데스크톱 클라이언트 기능 비교](desktop-feature-comparison.md)에 자세히 설명 되어 있으며, 사용자 문서는 [비즈니스용 skype 도움말](https://support.office.com/en-US/Skype-for-business)에 있습니다. 이 클라이언트는 사용자가 Office 365을 설치할 때 포함 됩니다.

더 적은 기능을 지 원하는 무료 기본 클라이언트만 사용할 수 있습니다. [모든 장치에서 비즈니스용 Skype를 다운로드 하 여](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)두 가지 버전을 다운로드할 수 있습니다. 전체 및 기본 클라이언트 간의 차이점은 [기본 클라이언트 제한](desktop-feature-comparison.md#Full-Basic) 섹션에 설명 되어 있습니다.

### <a name="skype-for-business-2016"></a>비즈니스용 Skype 2016

비즈니스용 skype 2016는 비즈니스용 Skype Server 2015 또는 2019 용 모든 기능을 갖춘 클라이언트입니다. 새로운 기능에 대 한 설명은 비즈니스용 [Skype 2016의 새로운](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 기능을 참조 하세요. 클라이언트 기능 지원은 비즈니스용 [skype에 대 한 데스크톱 클라이언트 기능 비교](desktop-feature-comparison.md)에 자세히 설명 되어 있으며, 사용자 문서는 [비즈니스용 skype 도움말](https://support.office.com/en-US/Skype-for-business)에 있습니다. 이 클라이언트는 사용자가 Office 365을 설치할 때 포함 됩니다.

더 적은 기능을 지 원하는 무료 기본 클라이언트만 사용할 수 있습니다. [모든 장치에서 비즈니스용 Skype를 다운로드 하 여](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)두 가지 버전을 다운로드할 수 있습니다. 전체 및 기본 클라이언트 간의 차이점은 [기본 클라이언트 제한](desktop-feature-comparison.md#Full-Basic) 섹션에 설명 되어 있습니다.

### <a name="skype-for-business-2015"></a>비즈니스용 Skype 2015

비즈니스용 skype 2015는 비즈니스용 Skype Server 2015 또는 2019 용 모든 기능을 갖춘 클라이언트입니다. 비즈니스용 Skype 사용자 인터페이스는 완전히 다시 디자인 되었으며, 통화 모니터, Skype 디렉터리 통합, 이모티콘 등의 새로 통합 된 기능을 포함 하 고 있습니다. 변경 내용에 대 한 요약은 Lync를 비즈니스용 [Skype로 업데이트-새로운 기능](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612)을 참조 하세요. 클라이언트 기능 지원은 비즈니스용 [skype에 대 한 데스크톱 클라이언트 기능 비교](desktop-feature-comparison.md)에 자세히 설명 되어 있으며, 사용자 문서는 [비즈니스용 skype 도움말](https://support.office.com/en-US/Skype-for-business)에 있습니다. 이 클라이언트는 사용자가 Office 365을 설치할 때 포함 됩니다.

### <a name="skype-for-business-on-mac"></a>Mac의 비즈니스용 Skype

[Mac 클라이언트의 비즈니스용 Skype](https://www.microsoft.com/en-us/download/details.aspx?id=54108) 를 다운로드할 수 있습니다. 필수 구성 요소를 검토 하려면 [Mac 용 비즈니스용 Skype 클라이언트 요구 사항을](mac-requirements.md) 참조 하세요.

### <a name="skype-for-business-for-mobile-devices"></a>모바일 장치용 비즈니스용 Skype

Windows Phone, iPhone/iPad 및 Android에서 클라이언트를 사용할 수 있습니다. 사용자는 [모든 장치에서 비즈니스용 Skype를 다운로드](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)하 여 받을 수 있습니다. 이러한 클라이언트에 대 한 기능 지원은 [비즈니스용 Skype에 대 한 모바일 클라이언트 기능 비교](mobile-feature-comparison.md)에 자세히 설명 되어 있습니다.

> [!NOTE]
> 이전 모바일 클라이언트에 대 한 MCX (Mobility Service) 지원은 더 이상 비즈니스용 Skype 서버 2019에서 사용할 수 없습니다. 현재 모든 비즈니스용 Skype 모바일 클라이언트는 이미 통합 커뮤니케이션 웹 API (인스턴트 메시징 (IM), 현재 상태, 대화 상대 지원)를 사용 합니다. MCX를 사용 하는 레거시 클라이언트가 있는 사용자는 현재 클라이언트로 업그레이드 해야 합니다.

### <a name="online-meeting-add-in-for-skype-for-business"></a>비즈니스용 Skype 용 온라인 모임 추가 기능

비즈니스용 Skype 용 온라인 모임 추가 기능을 통해 Windows의 Microsoft Outlook 메시징 및 공동 작업 클라이언트에서 모임 관리를 지원 합니다. 비즈니스용 skype 소프트웨어의 온라인 모임 추가 기능이 비즈니스용 Skype에 자동으로 설치 됩니다.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>비즈니스용 skype Web App 및 Skype 모임 앱

사용자의 컴퓨터에 비즈니스용 Skype가 설치 되어 있지 않은 경우 사용자가 Windows 컴퓨터의 모임 요청에서 모임 링크를 클릭 하면 Skype 모임 앱 또는 비즈니스용 Skype Web App이 설치 되 고 열립니다.  Skype 모임 앱은 조직 외부의 참가자가 선택할 수 있는 클라이언트입니다. (Mac에서 비즈니스용 Skype for Mac이 설치 되 고 열립니다.) 이 클라이언트를 사용 하기 위한 요구 사항에 대 한 [모임 클라이언트 (웹 앱 및 모임 앱)에 대 한 계획](meetings-clients.md) 을 참조 하세요.


### <a name="skype-for-business-web-scheduler"></a>비즈니스용 Skype Web Scheduler

비즈니스용 [Skype Web Scheduler](https://sched.lync.com) 는 Microsoft Outlook에 액세스할 수 없는 비즈니스용 skype Online의 사용자를 위한 웹 기반 모임 예약 및 관리 도구 이거나, Windows를 기반으로 하지 않는 운영 체제입니다. 비즈니스용 Skype Web Scheduler를 통해 사용자는 새 모임을 만들고, 기존 모임을 수정 하 고, 선호 하는 전자 메일 프로그램을 사용 하 여 초대장을 보낼 수 있습니다. 비즈니스용 Skype Web Scheduler [설명서](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) 에는 자세한 내용이 나와 있습니다.

### <a name="vdi-plugins"></a>VDI 플러그 인

VDI (가상 데스크톱 인프라) 환경은 보안 및 준수 문제가 특히 중요 한 일부 조직에 사용 됩니다. 예를 들어 전체 오디오 및 비디오를 사용 하 여 가상 데스크톱에 있는 클라이언트의 오디오 및 비디오 처리 로드가 필요 합니다. 추가 VDI 플러그 인 소프트웨어를 사용 하 여 최종 사용자의 로컬 컴퓨터로 처리 하는 작업을 오프 로드 하 고 가상 데스크톱의 부하를 줄일 수 있습니다. 이러한 플러그인을 사용 하는 방법에 대 한 자세한 내용은 [VDI 환경에서 비즈니스용 Skype 계획](vdi-environments.md) 을 참조 하세요.

### <a name="microsoft-teams-rooms"></a>Microsoft Teams 룸

Microsoft 팀 대화방은 친숙 한 인터페이스를 사용 하며 쉽게 배포 하 고 관리 하며, 설치 하기 쉽도록 LCD 패널과 같은 기존 장비를 활용 하는 Microsoft의 최신 회의 솔루션입니다. Microsoft 팀 대화방은 콘솔 모드에서 Surface Pro 4 또는 Surface Pro에서 실행 되는 목적으로 제작 된 UWP 앱을 사용 합니다 (배포 된 후에는 UWP 앱이 디바이스에서 실행 되는 유일한 앱) 이므로 구현에 고유한 디바이스 계정이 필요 합니다. 소프트웨어는 Windows 스토어와 Windows 업데이트를 통해 업데이트 됩니다. 배포 https://aka.ms/MTRDocs 에서 이러한 방 콘솔을 사용 하는 방법에 대 한 자세한 내용은을 참조 하세요. 

### <a name="skype-for-business-on-surface-hub"></a>Surface Hub의 비즈니스용 Skype

Microsoft Surface Hub는 브레인스토밍, 공동 작업, 프레젠테이션을 위한 일대일 생산성 장치입니다. [Microsoft Surface Hub 관리 가이드](https://docs.microsoft.com/surface-hub/)에 명시 된 비즈니스용 Skype 클라이언트의 이터레이션이 있습니다.

## <a name="choosing-your-organizations-preferred-client"></a>조직의 기본 클라이언트 선택
<a name="BK_client_choose"> </a>

조직이 적절 한 라이선스를 구매한 경우 전체 클라이언트를 선택 하 고, 그렇지 않으면 기본 클라이언트를 선택 합니다.

사용자는 [모든 장치에서 비즈니스용 Skype를 다운로드](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)하 여 클라이언트 자체를 설치할 수 있습니다. 사용자가 Windows에 Office 365을 설치할 때에도 클라이언트가 설치 됩니다. 일부 사용자가 Mac을 사용 하는 경우 이러한 사용자는 이전 섹션에 설명 된 것과 다른 기능 집합을 사용할 수 있습니다.

비즈니스용 skype Server 2015에서 제공 하는 일부 기능은 비즈니스용 Skype Online 또는 비즈니스용 Skype Server 2019에서 사용할 수 없습니다. 2015 또는 [온라인 또는](desktop-feature-comparison.md#Online-Hybrid) 하이브리드 사용자 계정 [제한 사항](desktop-feature-comparison.md#Online-Hybrid) 에 대 한 자세한 내용은 2019 비즈니스용 skype Online 관리자가 사용할 수 있는 다양 한 요금제에 대 한 정보에 대 한 [비즈니스용 Skype Online 서비스 설명을](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) 참조 하 고 싶을 수 있습니다.

 비즈니스용 Skype를 배포 하거나 업그레이드 하기 전에 조직에서 이미 사용 중인 클라이언트를 확인 합니다. 비즈니스용 [Skype에 대 한 데스크톱 클라이언트 기능 비교](desktop-feature-comparison.md) 를 사용 하 여 해당 클라이언트에 대 한 기능 지원 영향을 이해 하세요. 이를 통해 사용자에 게 변경 내용을 전달 하 고, 롤아웃 프로세스의 속도를 개선 하 고, 최신 클라이언트로 업그레이드 하는 이점을 완전히 이해할 수 있습니다.

## <a name="ways-to-deploy-the-client-to-your-users"></a>사용자에 게 클라이언트를 배포 하는 방법
<a name="BK_User_Deploy"> </a>

MSI 및 간편 실행 형식 설치 관리자에서 클라이언트 설치 관리자를 사용할 수 있습니다. 비즈니스용 Skype 클라이언트-sustainment 전략이 선택 사항에 영향을 줄 수 있으므로 다음 사항을 이해 해야 합니다.

- 일반적으로 비즈니스용 Skype는 이전에 출시 된 클라이언트에 새 기능을 추가 하지 않습니다.

- 일반적으로 비즈니스용 Skype는 최초 릴리스 이후 비즈니스용 Skype MSI에서 새로운 기능을 제공 하는 계획을 받지 않습니다. 릴리스 간의 MSI 향상은 기본적으로 품질/보안입니다.

- 비즈니스용 2019 Skype 클라이언트 환경에 대 한 최신 및 최고의 비즈니스용 설치 프로그램에서 찾을 수 있습니다.

[비즈니스용 Skype 서버에서 Windows 클라이언트 설치 사용자 지정](../../deploy/deploy-clients/customize-windows-client-installation.md)에서 설명한 대로 클라이언트의 사용자 지정 된 배포를 수행할 수 있습니다. 설치 방법에 대 한 자세한 내용은 [비즈니스용 Skype 서버용 클라이언트 배포](../../deploy/deploy-clients/deploy-clients.md) 를 참조 하십시오.

### <a name="click-to-run"></a>간편 실행

간편 실행은 비즈니스용 Skype를 포함 하 여 Office 제품을 설치 하 고 업데이트 하는 데 사용할 수 있는 Microsoft 스트리밍 및 가상화 기술입니다. 이러한 스트리밍 및 가상화 기능은 Microsoft Application Virtualization (App-v)의 기술에 기반을 둔 것입니다. 간편 실행을 통해 얻을 수 있는 이점은 다음과 같습니다.

- 설치 시간을 단축 하는 Office suite 스트리밍 설치

- Slipstreamed 업데이트 및 패치

- 디스크 공간을 덜 차지 합니다.

- 사용자 기반 라이선스: 사용자 당 5 개 설치

- 독립 프로그램 설치를 위해 XML 편집기를 통해 사용자 지정할 수 있습니다.

[Office 배포 도구](https://www.microsoft.com/en-us/download/details.aspx?id=49117) 를 사용 하 여 이런 유형의 설치 관리자를 할 수 있습니다.

기본 및 전체 클라이언트 버전 (32 및 64 비트 버전 선택)은 모두 간편 실행 설치 관리자를 사용 하 여 사용할 수 있으며, 사용자는 [모든 장치에서 비즈니스용 Skype 다운로드](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3)시 다운로드할 수 있습니다.

### <a name="msi"></a>MSI

MSI는 비즈니스용 Skype 2015 및 2016 클라이언트에 사용 되는 전통적인 설치 방법입니다. 이를 통해 수동으로 업데이트 및 패치를 설치 하 고, 볼륨 라이선스 및 활성화를 사용 하 고, [Office 사용자 지정 도구](https://www.microsoft.com/en-us/download/details.aspx?id=49030)를 통해 사용자 지정할 수 있습니다. Microsoft 끝점 구성 관리자를 사용 하거나 타사 도구를 사용 하 여 그룹 정책을 적용 하 여 클라이언트를 배포할 수 있습니다.



## <a name="legacy-clients"></a>레거시 클라이언트
<a name="Legacy"> </a>

비즈니스용 skype 서버 2019 및 비즈니스용 skype Online은 이전에 출시 된 다음 클라이언트: 비즈니스용 Skype 2016, 비즈니스용 Skype 2015, Lync 2013을 지원 합니다.

비즈니스용 Skype 서버 2015는 이전에 출시 된 다음 클라이언트 (Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition, Lync 2010 전화 교환)를 지원 합니다. 다른 서버와 함께 사용 하는 경우 이러한 클라이언트에 대 한 자세한 내용은 lync server [2013의 클라이언트 비교 표](https://technet.microsoft.com/library/gg425836%28v=ocs.15%29.aspx) 를 참조 하 고 [lync server 2010에 대 한 클라이언트 비교 표에서](https://technet.microsoft.com/library/gg425836%28v=ocs.14%29.aspx)
