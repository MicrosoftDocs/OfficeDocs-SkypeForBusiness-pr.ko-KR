# <a name="plan-for-security-in-skype-for-business-server"></a>비즈니스용 Skype 서버에서 보안 계획 
 
비즈니스용 Skype 서버 보안 콘텐츠. 
  
이 콘텐츠는 비즈니스용 Skype 서버 보안과 관련이 있습니다. 보안은 매우 광범위 한 주제입니다. 비즈니스에 대 한 비즈니스용 Skype 서버의 모든 기능, 그리고 에코 시스템을 구성 하는 데이터베이스, 서비스 및 하드웨어의 보안에도 도달 합니다. 이 콘텐츠는 비즈니스용 Skype 서버에 대 한 특정 보안 항목을 보완 합니다.
