# <a name="plan-for-security-in-skype-for-business-server"></a>비즈니스용 Skype 보안 계획 
 
비즈니스용 Skype 서버 콘텐츠 관리 
  
이 콘텐츠는 보안과 비즈니스용 Skype 서버 관련이 있습니다. 보안은 매우 광범위한 항목입니다. 보안은 에코시스템을 구성하는 비즈니스용 Skype 서버, 서비스 및 하드웨어뿐만 아니라 모든 기능에서 사용할 수 있습니다. 이 콘텐츠는 보안과 관련한 특정 보안 항목에 대한 비즈니스용 Skype 서버.
