# <a name="plan-for-security-in-skype-for-business-server"></a>비즈니스용 Skype 서버의 보안 계획 
 
비즈니스용 Skype 서버 보안 콘텐츠입니다. 
  
이 콘텐츠는 비즈니스용 Skype 서버 보안과 관련이 있습니다. 보안은 매우 광범위한 주제입니다. 보안은 에코시스템을 구성하는 데이터베이스, 서비스 및 하드웨어뿐만 아니라 비즈니스용 Skype 서버의 모든 기능에도 도달합니다. 이 콘텐츠는 비즈니스용 Skype 서버와 관련하여 특정 보안 항목에 대한 추가 정보입니다.
