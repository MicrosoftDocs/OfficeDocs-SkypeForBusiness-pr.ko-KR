# <a name="requirements-for-your-skype-for-business-2015-environment"></a>2015 비즈니스용 Skype 요구 사항
 
**요약:** 2015년의 시스템 요구 사항 및 네트워크 또는 인프라 요구 사항에 비즈니스용 Skype 서버 개요입니다. 2015를 설치 및 배포하기 위한 모든 것을 설치하거나 구성해야 하는 경우 비즈니스용 Skype 서버 여기에 있습니다.
  
이 페이지는 2015년 6월 배포와 관련한 요구 사항에 비즈니스용 Skype 서버 페이지입니다. 여기에는 서버 요구 사항과 도메인 및 네트워킹 구조, 인증서 등 준비해야 할 일반적인 사항이 모두 포함됩니다.
  
## <a name="whats-in-this-topic"></a>이 항목의 항목

환경 요구 사항 섹션의 두 문서에서는 조직의 서버 환경에서 비즈니스용 Skype 서버 2015로 배포하거나 업그레이드하기 전에 필요한 사항을 간략하게 설명합니다.
  
### <a name="server-requirements"></a>서버 요구 사항

[비즈니스용 Skype 서버 2015](server-requirements.md)의 서버 요구 사항은 배포에서 사용하려는 개별 서버에서 준비해야 하는 사항입니다.
  
### <a name="environmental-requirements"></a>환경 요구 사항

[비즈니스용 Skype 서버 2015](environmental-requirements.md)에 대한 환경 요구 사항은 2015년을 배포하기 전에 인프라에서 준비해야 하는 광범위한 비즈니스용 Skype 서버 요구 사항입니다.
