# <a name="plan-for-unified-contact-store-in-skype-for-business-server-2015"></a>비즈니스용 Skype 서버 2015의 통합 연락처 저장소 계획
 
**요약:** 비즈니스용 Skype 서버 2013 또는 2016과 통합할 Exchange 이 항목을 검토하세요.
  
통합 연락처 저장소는 Microsoft Office 제품 전반에 걸쳐 일관된 연락처 환경을 제공하며 사용자가 Exchange 2013에 모든 연락처 정보를 저장할 수 있지만 이러한 정보를 비즈니스용 Skype, Exchange, Outlook 및 Outlook  웹 액세스.
  
## <a name="requirements-for-unified-contact-store"></a>통합 연락처 저장소에 대한 요구 사항

다음을 통해 통합 연락처 저장소를 비즈니스용 Skype 서버.
  
- 2013 또는 비즈니스용 Skype 서버 Exchange 실행 중이야 합니다.
    
- 사용자는 연락처를 비즈니스용 Skype 2013 또는 비즈니스용 Skype 서버 Exchange 마이그레이션해야 합니다.
    
- 사용자 사서함은 2013에서 2013으로 Exchange 합니다.
    
- 2013 또는 2016과 비즈니스용 Skype 서버 Exchange 서버 간 인증을 구성해야 합니다.
    
    > [!NOTE]
    > 비즈니스용 Skype 서버 및 Exchange 2013 또는 2016 간에 인증을 설정하는 데 대한 자세한 요구 사항은 작업 설명서에서 [Manage server-to-server authentication (OAuth) and partner applications in 비즈니스용 Skype 서버](../../manage/authentication/server-to-server-and-partner-applications.md) 참조하십시오.
