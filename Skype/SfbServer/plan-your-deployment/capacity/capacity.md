# <a name="capacity-planning-for-skype-for-business-server"></a>용량 계획 비즈니스용 Skype 서버
 
이 섹션의 항목은 조직의 사용자 수를 적절하게 계획하고 활동이 생성하는 서버 부하를 계획할 수 있도록 비즈니스용 Skype 서버 배포하는 방법을 이해하는 데 도움이 됩니다. 
  
> [!NOTE]
> 이 섹션의 모든 권장 사항은 서버에 비즈니스용 Skype 2015년 11월 이상 누적 업데이트를 설치했다고 가정합니다.
