# <a name="tblsiopwhitelist"></a>tblSiopWhiteList
 
tblSiopWhiteList는 노드와 연결할 수 있는 등록 된 추가 기능 목록입니다.
  
**열**

|**열**|**유형**|**설명**|
|:-----|:-----|:-----|
|siopID  <br/> |GUID (null 아님)  <br/> |추가 기능의 GUID입니다.  <br/> |
|siopName  <br/> |nvarchar (50), null 아님  <br/> |Display-추가 기능의 이름입니다.  <br/> |
|siopUrl  <br/> |nvarchar (255), null 아님  <br/> |추가 기능의 URL입니다.  <br/> |
   
**키**

|**열**|**설명**|
|:-----|:-----|
|siopID  <br/> |기본 키입니다.  <br/> |
