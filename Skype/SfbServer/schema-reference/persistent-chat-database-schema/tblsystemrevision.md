# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision에는 여러 관리자 클라이언트 간에 일관성을 유지하기 위해서 tblAdminLock 테이블에서 사용되는 수정 버전 번호가 포함됩니다.
  
**열**

|**열**|**유형**|**설명**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint, null이 아님  <br/> |수정 버전 번호입니다.  <br/> |
