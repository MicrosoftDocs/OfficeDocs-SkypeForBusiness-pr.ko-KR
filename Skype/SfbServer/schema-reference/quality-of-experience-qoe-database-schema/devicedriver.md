# <a name="devicedriver-table"></a>DeviceDriver 테이블
 
DeviceDriver 테이블은 지원 테이블입니다. 각 레코드는 캡처 장치 또는 렌더 장치에 사용 되는 드라이버를 나타냅니다.
  
|**열**|**데이터 형식**|**키/인덱스**|**세부적인**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |주요한  <br/> |이 장치 드라이버 레코드를 식별 하는 고유 번호입니다.  <br/> |
|**DeviceDriver** <br/> |varchar (256)  <br/> |독특한  <br/> |장치 드라이버 이름입니다.  <br/> |
