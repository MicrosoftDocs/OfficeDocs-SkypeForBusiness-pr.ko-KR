# <a name="networkconfigurationsettings-view"></a>NetworkConfigurationSettings 보기
 
NetworkConfigurationSettings 보기는 네트워크 구성에 대 한 정보를 저장 합니다. 이 보기는 Microsoft Lync Server 2013에서 도입 되었습니다.
  
|**열**|**데이터 형식**|**세부적인**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |서브넷 IP 주소 (IPv4 주소만 지원 됨)  <br/> |
|UserSiteName  <br/> |name  <br/> |사용자의 사이트 이름입니다.  <br/> |
|국가 이름  <br/> |name  <br/> |사용자의 국가/지역 이름입니다.  <br/> |
