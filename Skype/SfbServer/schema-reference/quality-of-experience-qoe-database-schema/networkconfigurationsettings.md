# <a name="networkconfigurationsettings-view"></a>NetworkConfigurationSettings 보기
 
NetworkConfigurationSettings 보기에는 네트워크 구성에 대한 정보가 저장됩니다. 이 보기는 Microsoft Lync Server 2013에서 도입했습니다.
  
|**열**|**데이터 형식**|**세부 정보**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |서브넷 IP 주소(IPv4 주소만 지원)  <br/> |
|UserSiteName  <br/> |nvarchar(128)  <br/> |사용자의 사이트 이름입니다.  <br/> |
|RegionName  <br/> |nvarchar(128)  <br/> |사용자의 국가/지역의 이름입니다.  <br/> |
