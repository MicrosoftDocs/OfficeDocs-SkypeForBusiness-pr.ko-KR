# <a name="schema-attributes-and-descriptions-in-skype-for-business-server"></a>2016의 Schema 특성 및 비즈니스용 Skype 서버
 
이 섹션에서는 이 섹션에서 사용하는 모든 schema 특성에 대해 비즈니스용 Skype 서버. 특성과 연결된 클래스에 대한 자세한 내용은 [schema attributes by class in 비즈니스용 Skype 서버.](schema-attributes-by-class.md)
  
연결된 쌍인 특성은 전달 링크 또는 뒤로 링크로 지정됩니다. 다른 개체를 참조하는 특성은 앞으로 연결입니다. 첫 번째 개체를 다시 참조하는 다른 개체의 특성은 뒤로 링크입니다. 전달 링크는 업데이트할 수 있는 반면, 뒤로 링크는 읽기 전용입니다.
  
일부 특성에는 비트 마스크 값이 있습니다. 이러한 특성의 경우 각 설정은 비트로 표시되고 표시되는 10진수 값은 비트 위치를 표시합니다. 비트 위치는 비트 0으로 시작됩니다. 예를 들어 1(이진)은 비트 0 집합, 10000(이진)은 비트 4 집합입니다. 각 비트는 속성을 나타내는 것입니다. 다음은 몇 가지 예입니다. 
  
- 100000(이진수)의 10진수 값이 16(비트 4가 설정)입니다.
    
- 1000000000(이진수)의 10진수 값이 256(비트 8이 설정)입니다.
    
- 1100(이진)의 10진수 값이 12입니다. 즉, 비트 2와 3이 설정되어 있으며 두 비트로 나타내는 속성이 모두 활성화됩니다.
    
- 1111000001(이진)의 10진수 값은 961입니다( 즉, 비트 0, 6, 7, 8 및 9가 설정됩니다. 이러한 각 비트에 대한 속성이 사용하도록 설정됩니다.
