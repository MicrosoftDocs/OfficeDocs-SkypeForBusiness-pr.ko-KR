# <a name="pools-table"></a>Pools 테이블
 
Pools 테이블은 다양한 풀에 대한 정보를 저장하는 지원 테이블입니다. 테이블의 각 레코드는 하나의 풀을 나타냅니다.
  
|**열**|**데이터 형식**|**키/인덱스**|**세부 정보**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Primary  <br/> |이 풀을 식별하는 고유 번호입니다.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |풀 FQDN입니다.  <br/> |
