# <a name="mediationservers-table"></a>MediationServers 테이블
 
MediationServers 테이블은 지원 테이블입니다. 각 레코드는 데이터베이스의 레코드를 포함 하는 호출에 관련 된 중재 서버 하나에 대 한 정보를 저장 합니다.
  
|**열**|**데이터 형식**|**키/인덱스**|**세부적인**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |주요한  <br/> |이 중재 서버를 식별 하는 고유 번호입니다.  <br/> |
|**MediationServer** <br/> |nvarchar (256)  <br/> | <br/> |중재 서버 이름.  <br/> |
