# <a name="mediationservers-table"></a>MediationServers 테이블
 
MediationServers 테이블은 지원 테이블입니다. 각 레코드는 데이터베이스에 레코드가 있는 통화와 관련된 중재 서버 하나에 대한 정보를 저장합니다.
  
|**열**|**데이터 형식**|**키/인덱스**|**세부 정보**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Primary  <br/> |이 중재 서버를 식별하는 고유 번호입니다.  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |중재 서버 이름입니다.  <br/> |
