# <a name="mcus-view"></a>Mcus 보기
 
Mcus 보기는 회의 세션에 참가 한 MCUs에 대 한 정보를 저장 합니다. 이 보기는 Microsoft Lync Server 2013에서 도입 되었습니다.
  
|**열**|**데이터 형식**|**세부적인**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |MCU를 식별 하는 고유 번호입니다.  <br/> |
|**McuUri** <br/> |nvarchar (450)  <br/> |MCU의 URI입니다.  <br/> |
|**McuUriType** <br/> |nvarchar (256)  <br/> |MCU URI의 유형입니다. 자세한 내용은 [UriTypes 테이블](uritypes.md) 을 참조 하세요. <br/> |
