# <a name="mcus-view"></a>Mcus 보기
 
Mcus 보기에는 회의 세션에 참가한 MCU에 대한 정보가 저장됩니다. 이 보기는 Microsoft Lync Server 2013에서 도입했습니다.
  
|**열**|**데이터 형식**|**세부 정보**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |MCU를 식별하는 고유 번호입니다.  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |MCU의 URI입니다.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |MCU URI의 형식입니다. 자세한 내용은 [UriTypes 테이블](uritypes.md) 을 참조하십시오. <br/> |
