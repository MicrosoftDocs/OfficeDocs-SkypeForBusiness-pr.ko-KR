# <a name="servers-table"></a>Servers 테이블
 
Servers 테이블은 다양한 서버에 대한 정보를 저장하는 지원 테이블입니다. 테이블의 각 레코드는 하나의 서버를 나타내며,
  
|**열**|**데이터 형식**|**키/인덱스**|**세부 정보**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Primary  <br/> |이 서버를 식별하는 고유 번호입니다.  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |서버 FQDN.  <br/> |
