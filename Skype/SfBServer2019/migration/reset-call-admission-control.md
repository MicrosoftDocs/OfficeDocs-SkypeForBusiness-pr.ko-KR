# <a name="reset-call-admission-control"></a>통화 허용 제어 서비스 다시 설정

레거시 프론트 엔드 풀이 CAC (통화 허용 제어)를 호스트 하는 경우에는 레거시 프런트 엔드 풀을 제거 하려면 먼저 CAC 호스팅을 비즈니스용 Skype 서버 2019 풀로 이동 해야 합니다.
  
### <a name="to-reset-cac"></a>CAC를 다시 설정 하려면

1. 토폴로지 작성기를 엽니다.
    
2. 사이트 노드를 마우스 오른쪽 단추로 클릭 한 다음 **속성 편집**을 클릭 합니다.
    
3. **통화 허용 제어 설정**에서 **통화 허용 제어가** 선택 되어 있는지 확인 합니다. 
    
4. **프런트 엔드 풀에서 cac (호출 허용 컨트롤)를 실행 하려면**cac를 호스트할 비즈니스용 Skype 서버 2019 풀을 선택한 다음 **확인**을 클릭 합니다.
    
5. 토폴로지를 게시 합니다.
