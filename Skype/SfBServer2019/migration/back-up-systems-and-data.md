# <a name="back-up-systems-and-data"></a>시스템 및 데이터 백업

마이그레이션을 시작 하기 전에 전체 시스템 백업을 수행 하 고 각 풀에 있는 사용자 계정 인벤토리를 포함 하 여 기존 시스템을 문서화 하 여 필요에 따라 롤백할 수 있도록 합니다. 데이터, 설정, 시스템 백업 및 복원에는 여러 도구 및 프로그램을 사용할 수 있습니다.

자세한 내용과 절차는 해당 버전에 대 한 백업 및 복원 절차를 참조 하십시오.
