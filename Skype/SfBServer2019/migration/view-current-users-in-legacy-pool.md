# <a name="view-current-users-in-legacy-pool"></a>레거시 풀에서 현재 사용자 보기

풀 간에 사용자를 이동할 수 있는 다양한 방법을 알아보기 전에 먼저 레거시 풀에 있는 사용자를 결정해야 합니다. 등록자 **풀** 열은 레거시 풀에 대해 구성된 사용자를 식별합니다. 2019 풀로 이동할 테스트 비즈니스용 Skype 서버 사용자입니다.
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>레거시 풀의 사용자 목록을 표시

1. RTCUniversalServerAdmins 그룹의 구성원 또는 CsAdministrator 또는 CsUserAdministrator 관리 역할의 구성원인 계정으로 레거시 프런트 엔드 서버에 로그온합니다.
    
2. **제어판을 비즈니스용 Skype 서버 를 니다.**
    
3. **사용자,** **검색을 클릭한** 다음 찾기를 **클릭합니다.**
