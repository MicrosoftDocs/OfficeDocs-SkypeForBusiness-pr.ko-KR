# <a name="verify-federation-and-remote-access-for-external-users"></a>외부 사용자를 위한 페더레이션 및 원격 액세스 확인

페더링 경로를 비즈니스용 Skype 서버 에지 서버로 전환한 후 일부 기능 테스트를 수행하여 페더링이 예상대로 수행되는지 확인해야 합니다. 외부 사용자 액세스를 위한 테스트에는 다음 중 일부 또는 모두를 포함하여 조직에서 지원하는 각각의 외부 사용자 유형이 포함됩니다.
  
### <a name="test-connectivity-of-external-users-and-external-access"></a>외부 사용자 및 외부 액세스의 연결 테스트

- 하나 이상의 페더니트 도메인의 사용자, 비즈니스용 Skype 서버 2019의 내부 사용자 및 레거시 설치의 사용자 IM(인스턴트 메시징), 현재 상태, A/V(오디오/비디오) 및 데스크톱 공유를 테스트합니다.
    
- 조직에서 비즈니스용 Skype 서버 2019의 사용자 및 레거시 설치의 사용자와의 통신을 지원하고 프로비전이 완료된 각 공용 IM 서비스 공급자의 사용자입니다. 
    
- 익명 사용자가 회의에 참가할 수 있는지 확인합니다.
    
- 비즈니스용 Skype 서버 2019의 사용자 및 레거시 설치의 사용자와 원격 사용자 액세스를 사용하여 레거시 설치에 호스트된 사용자(vpn 없이 인트라넷 외부에서 i nto Lync Server/비즈니스용 Skype 로깅) IM, 현재 상태, A/V 및 데스크톱 공유를 테스트합니다.
    
- 비즈니스용 Skype 서버 2019의 사용자 및 레거시 설치의 사용자와 원격 사용자 액세스(VPN 없이 인트라넷 외부에서 비즈니스용 Skype 서버 2019에 로그인)를 사용하여 비즈니스용 Skype 서버 2019에 호스트된 사용자. IM, 현재 상태, A/V 및 데스크톱 공유를 테스트합니다.
