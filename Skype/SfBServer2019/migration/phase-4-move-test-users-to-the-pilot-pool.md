# <a name="phase-4-move-test-users-to-the-pilot-pool"></a>4 단계: 테스트 사용자를 파일럿 풀로 이동

비즈니스용 Skype 서버 제어판 및 비즈니스용 Skype 서버 관리 셸에서 다음과 같은 두 가지 방법을 사용 하 여 단일 사용자 또는 사용자 그룹을 새 Microsoft 비즈니스용 Skype 서버 2019 배포로 이동할 수 있습니다. 이 섹션의 항목에서는 파일럿 배포 시에 수행 해야 하는 작업에 대해 설명 하 고 파일럿 개발 환경에서 프로덕션 수준의 배포로 비즈니스용 Skype 서버 2019의 배포를 이동 하기 전 합니다.
