# <a name="skype-for-business-server-prepare-active-directory-for-skype"></a>비즈니스용 Skype 서버: 사용자에 대해 Active Directory Skype

동시 사용 상태로 비즈니스용 Skype 서버 2019를 배포하기 전에 2019년 2019에 대해 더 많은 Active Directory 작업을 수행하여 비즈니스용 Skype 서버 구성해야 합니다. 이 Schema extensions는 2019년 8월에 필요한 Active Directory 클래스 및 특성을 비즈니스용 Skype 서버 추가합니다. <!-- For more information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>2019년도에 대해 Active Directory를 비즈니스용 Skype 서버

1. 비즈니스용 Skype 서버 2019 프런트 엔드 서버에서 비즈니스용 Skype 서버 2019 설치를 실행합니다.
    
2. **Active Directory 준비** 를 선택합니다.
  
3. 마법사에서 1~5단계를 완료합니다.
