# <a name="administering-servers-after-migration"></a>마이그레이션 후 서버 관리

일반적으로 관리하려는 서버 버전에 해당하는 관리 도구를 사용해야 합니다. 다른 릴리스의 관리 도구는 동일한 컴퓨터에 설치할 수 없습니다.
  
> [!IMPORTANT]
> 비즈니스용 Skype 서버 파일럿 풀이 배포된 후 토폴로지 작성기 또는 제어판을 사용하여 2019 리소스를 관리할 수 없습니다. 2019 도구를 사용하여 모든 현재 및 이전 버전 리소스를 관리해야 합니다.
