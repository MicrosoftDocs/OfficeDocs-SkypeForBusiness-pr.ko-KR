# <a name="administering-servers-after-migration"></a>마이그레이션 후 서버 관리

일반적으로 관리 하려는 서버 버전에 해당 하는 관리 도구를 사용 해야 합니다. 동일한 컴퓨터에 여러 릴리스의 관리 도구를 설치할 수 없습니다.
  
> [!IMPORTANT]
> 비즈니스용 Skype 서버 파일럿 풀이 배포 된 후에는 토폴로지 작성기 또는 제어판을 사용 하 여 2019 리소스를 관리할 수 없습니다. 2019 도구를 사용 하 여 현재 및 이전 버전 리소스를 모두 관리 해야 합니다.
