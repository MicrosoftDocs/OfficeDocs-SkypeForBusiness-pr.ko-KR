# <a name="whats-in-skype-for-business-server-2019"></a>비즈니스용 Skype 서버 2019의 기능

**요약:** 비즈니스용 Skype 서버 2019의 새로운 기능에 대 한 자세한 내용을 보려면이 항목을 읽어 보십시오.  

비즈니스용 Skype 서버 2019의 새로운 기능은 다음과 같습니다.
  
- 클라우드 음성 메일  
- 통화 데이터 커넥터
- 병행 마이그레이션

## <a name="unified-messaging-services-cloud-voicemail"></a>통합 메시징 서비스: 클라우드 음성 메일

Exchange UM은 비즈니스용 Skype 2019를 Exchange 2013 또는 Exchange 2016와 통합할 때 비즈니스용 Skype 서버 2019에서 계속 사용할 수 있습니다. Exchange 2019의 지원 변경 사항으로 인해, 클라우드 음성 메일과 클라우드 자동 전화 교환 기능을 사용 하 여 Exchange UM 통합을 보다 강조 하 고 있습니다.  

클라우드 음성 메일을 사용 하면 모든 비즈니스용 Skype 2019 사용자가 Microsoft 클라우드의 동일한 음성 메일 서비스에 액세스할 수 있도록 온-프레미스에 있는지, 아니면 온라인&#x2014;든 지&#x2014;수 있습니다. 클라우드 음성 메일은 온-프레미스 및 온라인 사용자 모두에 게 다음과 같은 이점을 제공 합니다.

- 비즈니스용 Skype Online, 팀 또는 Outlook 클라이언트를 사용 하 여 Exchange 사서함의 음성 메일 액세스
- 웹 기반 포털을 사용 하 여 음성 메일 옵션을 관리 하는 기능

자세한 내용은 [Plan Cloud 음성 메일 서비스](../sfbhybrid/hybrid/plan-cloud-voicemail.md) 및 [비즈니스용 Skype 서버 및 Exchange server 마이그레이션을](../sfbhybrid/hybrid/plan-um-migration.md) 계획을 참조 하십시오.
  
## <a name="call-monitoring-call-data-connector"></a>통화 모니터링: Call Data Connector

통화 데이터 커넥터는 서로 다른 온-프레미스 및 온라인 도구 집합을 사용 하 여 모든 사용자 통화 품질을 모니터링할 필요가 없기 때문에 하이브리드 환경에서 호출 모니터링이 매우 간단해 집니다.  사용자가 온-프레미스에 있거나 온라인 상태 인지 여부에 관계 없이 전체 조직에 대 한 통화 품질을 온라인으로 볼 수 있습니다.

Call Data Connector에서는 단일 도구 집합을 사용 하 여 다음 작업을 수행할 수 있습니다.

- Microsoft 팀원, 비즈니스용 Skype Online 및 비즈니스용 Skype 서버에서 사용자 환경을 모니터링 합니다.
- 네트워크에서 문제 보기 및 문제 해결
- 헬프데스크 및 관리자 역할을 할당 하 여 지원 자가 해당 분야를 확인 하 고 문제를 해결할 수 있도록 합니다.

자세한 내용은 [요금제 Call Data Connector](../sfbhybrid/hybrid/plan-call-data-connector.md) 을 참조 하십시오.
